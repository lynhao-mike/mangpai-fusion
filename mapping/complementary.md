## 九、互补层使用规则

### 1. 触发条件

```
当多个派别对同一对象给出"同向但路径不同"的结论时
→ 触发互补层匹配
→ 启动联立加权置信度（详见 engine/confidence.yaml）
```

### 2. 输出标注

报告中必须显式列出：
```
[段+杨+任] 婚期四重机制叠加 → 2018 戊戌年 ★★★★ (86%)
（段派从应期签字 / 杨派从人来印来 / 任派从触发第 1 道，三派指向同一年）
```

### 3. 联立加权公式

```
joint_score = max(各派 score) + Σ (其余派 score × 0.15)
封顶 100。

例：
  - M2-Y-064 (杨派) static = 80
  - M3-R-031 (任派) static = 78  
  - M1-D-XXX (段派) static = 75

joint_score = 80 + 78×0.15 + 75×0.15 = 80 + 11.7 + 11.25 = 102.95 → 100 ★★★★★

加权后：单派 ★★★★ → 联立 ★★★★★
```

### 4. 升级到共识层

互补层规律满足以下条件可升入共识层：
- ✅ 4 派全部有对应规律
- ✅ 历史命中率 ≥ 80% 且样本 ≥ 3 例
- ✅ 无任何派别持反对意见
