# 独门层 · EXCLUSIVE · 单派独有规律

> 金字塔**上层**。本文件登记**单派独有、其他派别没有对应规律**的高价值实战武器。  
> 这是用户最希望被显式提示的层——独门规律 = 该派的核心竞争力。

---

## 编号规则

```
EXC-{派别}-{topic}-{NNN}

例：EXC-G-MG-001 = 高派独门·命宫·第1条
   EXC-Y-TH-002 = 杨派独门·调候·第2条
```

派别字母：G=高 / D=段 / Y=杨 / R=任

## 五、独门层使用规则

### 1. 触发独门规律的条件

实战分析时，独门规律的触发遵循：
```
A. 4 派中只有 1 派对该现象有规律 → 自动启用独门
B. 4 派中多派有同向但表述粗略 + 1 派精细 → 共识结论 + 独门补充
C. 历史命中率 ≥ 80% 的独门规律 → 视同共识层
D. 用户在策略选择器中明示要"领域专家"模式 → 独门规律全开
```

### 2. 独门规律的输出标注

报告中必须显式标注：
```
[杨派独门] 人来印来 → 2027 丁未年 6-8 月婚期 ★★★★ (82%)
[高派独门] 词馆 + 学堂 = 985 院校 ★★★★ (85%)
[任派独门] 18 道法门第 6 道触发 → 应期到 ★★★★ (80%)
[段派独门] L4 富贵层级签字律 → 正处级 ★★★★ (83%)
```

### 3. 独门规律的置信度

独门规律**默认起始置信度低于共识层**（因为缺少跨派交叉验证），但可通过实战命中率快速升级：
```
独门规律默认：★★★★ (75%)
连续 3 例命中且历史 hit_rate ≥ 80% → 升级 ★★★★★ (90%)
连续 3 例失验 → 降级 ★★★ (65%) 等待人工 review
```

### 4. 高派的"独门密度"最高

| 派别 | 独门规律量 | 独门主题 |
|---|---|---|
| **高派** | ~80 条 | 命宫长生诀 / 神煞宝典 / 词馆学堂 / 32 字诀 |
| 段派 | ~50 条 | L0-L5 / 4 层签字 / 风火集 / 滴天髓 |
| 杨派 | ~40 条 | 调候改运 / 五合明暗 / 学历 14 条 / 六穿医学 |
| 任派 | ~30 条 | 18 道法门 / 6 触发应期 / 操作清单 |
| **合计** | **~200 条** | 占总规律 22% |

**这意味着高派是 4 派中独门规律最多的，是用户实战最大的差异化武器。**
