# 冲突仲裁层 · CONFLICTS · 派别冲突登记与裁定

> 本文件登记**多派对同一对象给出相反结论**的冲突案例，及其仲裁规则。  
> 冲突 ≠ 错误，是命主信息的一部分——必须显式标注，不可隐瞒。

---

## 编号规则

```
CFL-{topic}-{NNN}

例：CFL-HUNYIN-001 = 冲突·婚姻·第 1 条
```

---

## 冲突登记格式

```yaml
- id: CFL-HUNYIN-001
  layer: conflicts
  topic: hunyin
  title: 桃花格局结论分歧
  positions:
    - school: gao
      rule: G-LF-022
      stance: 多变      # 高派认为阳包阴 = 婚姻多变
      strength: 0.78
    - school: yang
      rule: M2-Y-XXX  
      stance: 不破婚    # 杨派认为已婚态官杀混杂分流不破婚
      strength: 0.85
  resolution_rule: domain-weight-yang   # 婚姻领域杨派权重最高
  winner: yang
  loser: gao
  winner_score: 0.85 × 0.85 - 0.78 × 0.3 = 0.4885
  output_format: |
    "[杨派主导] 不破婚 ★★★ (75%)  
    （高派持反对意见：阳包阴 → 婚姻多变 ★★ 60%，权重低于本结论）"
  notes: 这是命主信息的两种可能性，必须如实呈现
```

## 一、仲裁原则总纲

### 原则 1：领域权重优先

```
婚姻领域      → 杨派 lead，结论冲突时杨派胜
应期领域      → 任派 lead，结论冲突时任派胜
格局定性      → 段派 lead，结论冲突时段派胜
神煞应用      → 高派 lead，结论冲突时高派胜
财运层级      → 段派 lead，高派副
学业判定      → 杨派 lead，高派副
健康灾厄      → 高派 lead，杨派副
调候改运      → 杨派 lead（独家）
六亲画像      → 杨派 lead，段派副
子女          → 高派 lead，杨派副
```

详见 [`engine/domain-weights.yaml`](../engine/domain-weights.yaml)。

### 原则 2：高强度规律优先

如果 lead 派的支持规律 strength < 0.7，而非 lead 派 strength ≥ 0.85：
- 仍由 lead 派胜出（领域权重优先于单条规律强度）
- 但报告中必须标注："本结论 lead 派支持较弱，请重视另一方意见"

### 原则 3：历史命中率优先

如果某条规律历史命中率 < 50% 且样本 ≥ 5 例：
- 该规律降权（× 0.5）
- 即使 lead 派持有，也可能输给非 lead 派

### 原则 4：显式呈现

冲突的最终输出格式必须显式呈现两方观点：
```
[X 派主导] 结论 A ★★★★ (82%)
[Y 派持反对] 结论 B ★★ (60%) - 权重低于本结论
```

绝不允许隐瞒一方观点。

### 原则 5：不可一票否决

任何派别都没有对其他 3 派的"一票否决权"。即使 4 派中只有 1 派反对，结论中也要列出反对意见。

## 二、已知冲突登记

### CFL-HUNYIN-001 · 阳包阴格局婚姻分歧

**冲突描述**：
- 高派 G-LF-022：女命八字两组干支阴阳对立 + 互相包夹 = **婚姻多变铁断信号**
- 杨派 M2-Y-XXX：已婚态官杀混杂分流（夫宫安静 + 金舆嫁贵）= **不破婚**

**潜在矛盾点**：当一个女命同时满足「阳包阴格局」+「夫宫安静 + 有金舆」时，两派结论相反。

**仲裁原则**：婚姻领域杨派 lead → 杨派结论胜出  
**输出格式**：
```
[杨派主导] 已婚态不破婚 ★★★★ (82%)  
（高派 G-LF-022 阳包阴格局提示："婚姻多变信号" ★★ (60%) 
- 杨派仲裁：已婚态 + 夫宫稳 → 桃花结构性存在但不一定离婚）
```

**实战使用**：当遇到此冲突时，**必须告知命主两种可能性**，让命主自己确认是否有桃花情况。

### CFL-LF-001 · 旺衰判定 vs 不论旺衰

**冲突描述**：
- 高派 G-LF-005..006：必须先看清浊判格局（隐含强弱判断）
- 任派 M3-R-002：**不论旺衰、不取用忌神**

**真实情况**：这不是真冲突，而是表述差异——
- 高派"清浊"是指**用神能量**而非身强弱
- 任派"不论旺衰"是反对**为旺衰而旺衰**

**仲裁原则**：4 派实际共识——旺衰是工具不是目的（已入 CON-LF-003）  
**输出格式**：不作为冲突呈现，作为同向不同表述。

## 四、冲突分级

| 级别 | 描述 | 处理 |
|---|---|---|
| **L1 显性冲突** | 多派对同一现象给出明确相反结论 | 必须仲裁 + 双方观点全列 |
| **L2 路径差异** | 多派结论同向但路径不同 | 不算冲突，作互补层处理 |
| **L3 适用边界差异** | 多派同向但触发条件不同 | 取交集 + 显式说明边界 |
| **L4 表述差异** | 同一现象多派叫法不同 | 建立"派别术语对照表" |

## 七、冲突的价值

> 派别冲突 ≠ 命理混乱。  
> 冲突 = 命主存在两种可能性的真实信号。  
> 命理师的任务是把两种可能性都呈现给命主，由命主在反馈中告诉我们哪种是真。  
> 这是 mangpai-fusion 系统**比单派更强**的根本原因。
