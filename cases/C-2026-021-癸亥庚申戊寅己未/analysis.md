## Step 7 · 四大生命主线画像

### 7.1 事业/职业

**4 派共识**：
- 格局：**食神格 + 食伤生财 + 财生官三连**，最终官库未开需特定大运激活
- 路径：以技术/专业起步 → 在体制或大型机构内晋升 → 官命最高取化杀生枭
- 天花板：**L11-L13**（副厅/正厅/省部级 或 上市公司高管，取决于领域）
- P(体制内) = **0.85**，高概率走公门路线

**三段式发展**：
1. **前期（-37 岁）**：以专业/技术奠基，历经辛酉/壬戌/癸亥三运打磨
2. **中期（38-47 岁，甲子运）**：名气/学识/晋升积累，是"被看见"的阶段
3. **顶峰（48-57 岁，乙丑运）**：正官透干，**官命天花板最终兑现**

**空亡软制约**：月支申（食神之根）在空亡 → 专业才能被空约 20%，需通过大运流年冲开空亡才能全力释放（壬寅/甲寅等寅类流年最有力）

### 7.2 财运

**层级定位**：
- 段派：大富级·上（L12-L13），wealth_ceiling 罕见高配
- 杨派：官杀库 = 亿级天花板（库未开 → 属于隐性财富，非现金型）
- **核心结论**：这是**权力型财富**（power_hierarchy 框架），不是现金型富；财富体现为职位对应的资源支配权，而非账户余额

**财运触发规律**：
- 食伤生财路径（庚申→癸亥）是主要财路
- 寅申冲激活财星，逢冲之年财运启动
- 官杀库（时支未）需亥未合+冲开才能激活 → **乙丑大运后期（2035+）最有可能开库**

**当前财运（2026 丙午）**：午冲子大运财根，财星受冲 = **名气年，财不稳，不宜冒险投资**

### 7.3 婚姻/感情

**4 派综合**：
- **婚宫寅**：被月支申冲（最强），被月支申刑，亥寅六合（柔性稳定）
- **配偶星**：日支甲（七杀，贴身度 1.0）= 配偶性情**仁厚直率有担当**，但七杀性格也有强势冲动
- **婚运触发**：癸亥大运（28-37岁）亥寅六合最强；壬戌大运（18-27岁）戌合寅也有触发
- **婚姻稳定度**：易动（寅申冲不断）
- **高派补注**：阴差阳错（日柱）= 婚姻或感情有阴差阳错之感，可能有一次重大感情偏差；孤辰（日柱）= 命主骨子里有孤高气质，婚姻中习惯独立。金舆（日+时双现）= 配偶带来物质财富/房产，从物质层面看婚姻有升华

**当前（43岁）**：婚宫持续受压，2027 丁未（亥未合强化官杀）可能有关系重整；若未婚，乙丑大运（48+）乙官透干亦有机会。

### 7.4 健康/灾厄
