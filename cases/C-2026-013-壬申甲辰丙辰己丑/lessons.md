## 二、失验规律（❌ 已推翻·已收录黑名单）

### F-01 / XF-001 · ⚠️ "正官藏而不透 = 晚婚"铁律 — 重大失验

| 项目 | 内容 |
|---|---|
| **黑名单编号** | `XF-001`（已收录铁断册黑名单） |
| **原规律** | 坤造正官三藏不透 + 食伤极旺克官 = 晚婚命，黄金婚期 35-41 岁 |
| **派别来源** | 杨派初判 + 共识层误判 |
| **实际结果** | 29 岁首婚（提前 6-12 年），31 岁再婚 |
| **失验原因** | ① 官藏不透≠晚婚，仅代表缘分迂回；② 忽略官杀混杂的"曲折"含义；③ 未启用财动婚动校验 |
| **替代规则** | 改用 `MR-005 财动婚动` + `MR-004 空亡冲实` + 多派联合判断"婚路曲折再婚命"；**严禁**单凭"官藏"直接断晚婚 |
| **校准强制条件** | 坤造判婚期必须同时检查：① 官杀格局；② 财动婚动；③ 空亡冲实。三项中至少 2 项满足才认定婚期年 |

## 四、命理师沟通经验（v2.0 新增）

1. **官杀混杂坤造**：不要急于判断晚婚，首要确认"是否为再婚命/感情曲折命"
2. **空亡宫位**：强调"需流年冲动填实才应期"，而非"无"或"晚"
3. **原生家庭不稳定命主**：双辰/伏吟月柱者，早年经历多坎坷，应多共情，不要过度渲染负面
4. **体制内配偶**（高速公路公司·国企）：庚子大运财官旺 → 稳定型配偶入场符合大运气场
