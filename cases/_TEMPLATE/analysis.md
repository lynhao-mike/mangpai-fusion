## 11. Step 9 · 风险、不确定性与禁用规则

### 11.1 缺失事实对置信度的影响

| 缺失字段 | 影响 |
|---|---|
| 出生地/真太阳时 | 若出生地时差较大，时柱与应期细节需复核 |
| 学历/证书 | 学业与职业资质层级只能给区间 |
| 职业/行业 | 事业方向只能给适配类型，不能确认具体岗位 |
| 婚姻状态 | 婚期只可给窗口，不可给必婚/必离 |
| 收入/资产 | 财富层级只可定中位，不可确认资产规模 |
| 历史重大事件 | 应期强弱无法回测升降星 |

### 11.2 禁用/谨慎规则

| 规则 | 本案处理 |
|---|---|
| 见财强即断大富 | 禁用；必须结合承载、印比、风险结构、known_facts |
| 官杀强即断高官 | 禁用；必须结合印、职业事实、平台层级 |
| 文昌/国印即断高学历或体制高位 | 禁用；仅作旁证 boost |
| 婚期窗口即断必婚/必离 | 禁用；known_facts 为空时只能列高概率触发窗口 |
| 五行偏枯即断重大疾病 | 禁用；仅提示风险方向，疾病需现实反馈与医学检查 |
| L+数字裸写 | 禁用；必须写成“Lx｜现实层级释义 + 起止界限 + 证伪条件” |
