# cases/ · 案例库

每个案例的归档结构：

```
cases/
├── cases-index.md         案例索引（含八字指纹防重 + 统计）
└── C-2026-NNN/
    ├── input.md           输入（去隐私后的命盘 + 提问）
    ├── analysis.md        4 派融合分析过程（含派别仲裁记录）
    ├── feedback.md        命主反馈（应验/失验逐条记录）
    └── lessons.md         本案学到的教训（用于规律升降级）
```

每个案例必须**同时**生成 `reports/C-YYYY-NNN-report.md` 正式报告。

详细规则在 M7 阶段定稿。
