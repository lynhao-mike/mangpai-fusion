## 一、4 派各自定性

### 段派（M1-D-*）

- **格局判定**：比劫极旺局；月令正官（丙）原透提纲，但巳被酉丑合化为金 → 官星化劫，正官格破
- **从格判定**：**不成立从革**（FIX-001/002）——时干丁火七杀坐酉为**长生**（阴干逆行：丁长生在酉），有根，破从革条件；判定为"比劫极旺·不从·七杀有根力弱"
- **五行比重**（FIX·ADD-001）：
  - 金：庚(年干)+申(年支·庚壬戊)+辛(月干)+巳(庚劫)+辛(日干)+丑(辛比)+酉(辛比)+丁酉纳音山下火含辛 ≈ **58%**
  - 土：丑(己癸辛·偏印主)+巳藏戊+申藏戊 ≈ **22%**
  - 火：丁(时干·七杀)+巳(丙官·月支主气) ≈ **12%**
  - 水：申藏壬(伤官·微弱) ≈ **5%**
  - 木：全局无木干，仅命宫己卯含乙余气 ≈ **3%**
- **富贵层级**：L1-L2（小康下到小康中）；做功层数 1-2 层（金太旺·火土微·无木水辅助）
- **关键签字**：群比劫夺财 + 七杀有根力弱 + 月令官化为金 + 调候缺壬甲
- **使用规律**：M1-D-027 / M1-D-088 / M1-D-119 / M1-D-173 / M1-D-188 / M1-D-201


### 杨派（M2-Y-*）

- **主位宾位**：日柱辛丑为主，时柱丁酉为宾；七杀宾位坐酉**长生**（v2修正），有根但被群金压制，力弱而非"无根虚透"
- **体用判定**：体（辛金）极强，用神缺位——喜壬甲，原局壬仅藏申余气、甲全无 → 调候严重不及

### 任派（M3-R-*）

- **七杀有根修正**（FIX-001）：丁酉七杀不再是"孤悬虚透"，而是"有根压制状态"；大运走火土可激活七杀威力，但当前被群金吸收
