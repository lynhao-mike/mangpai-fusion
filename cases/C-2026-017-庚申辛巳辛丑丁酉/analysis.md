## 一、4 派各自定性

### 段派（M1-D-*）

- **格局判定**：比劫极旺局；月令正官（丙）原透提纲，但巳被酉丑合化为金 → 官星化劫，正官格破
- **从格判定**：假从革破（七杀丁火虚透天干扰从）
- **富贵层级**：L1-L2（小康下到小康中）；做功层数 1-2 层
- **关键签字**：群比劫夺财 + 七杀虚透不能制 + 月令官化为金
- **使用规律**：M1-D-027（从格成立判）/ M1-D-088（比劫格性格）/ M1-D-119（比劫夺财论婚）/ M1-D-173（做功层数论财）/ M1-D-188（七杀坐死论子）/ M1-D-201（行业映射）

### 杨派（M2-Y-*）

- **主位宾位**：日柱辛丑为主，时柱丁酉为宾；杀星宾位坐酉死地，无力制主
- **体用判定**：体（辛金）极强，用神缺位 —— 喜壬甲，原局壬仅藏申、甲全无
- **调候情况**：巳月辛金贵在壬水洗淘 + 甲木通根，今二者俱无 → 调候不及
- **完整性评估**：日支偏印（丑）坐妻宫 → 妻劳/像姐/偏印型；时干七杀坐死 → 子缘薄
- **使用规律**：M2-Y-035（财富 7 等）/ M2-Y-061（性格底色）/ M2-Y-091（承载本质）/ M2-Y-127（偏印坐妻宫）/ M2-Y-149（偏印夺食论子）/ M2-Y-201（六亲位论父母）

### 任派（M3-R-*）

- **18 道法门相关**：三合化金的「封箱效应」 —— 巳酉丑三合金 + 旬空辰巳 → 原局做功路径已被合住，自力难解
- **6 触发**：偏弱；启动主力靠流年透壬水（食伤泄秀）或甲乙木（财星引动）
- **富贵两要素**：财库为丑（藏辛比 = 非标准财库）、官库为戌（大运丙戌引动） → 此命走丙戌运 = 一生事业高峰
- **应期清单**（见 §三）
- **使用规律**：M3-R-031（封箱启动诀）/ M3-R-031.3（透干引动）/ M3-R-067（6 触发之透干）/ M3-R-088（三合穿夫妻宫）/ M3-R-110（比劫格性格）/ M3-R-145（杀弱无库论财）/ M3-R-156（杀星行业线）

### 高派（G-*-*）

- **清浊判**：偏浊 —— 比劫满盘 + 七杀孤悬 + 调候失衡 + 神煞两极（贵人多但空亡夺力）
- **神煞要点**：
  - 年柱：月德/德秀贵人 + 羊刃 + 血刃 + 亡神 → 早年家中不顺或自身有伤
  - 月柱：国印/福星/天德/德秀贵人 + 空亡 + 劫煞 + 披麻 → 贵人有助但空亡削弱
  - 日柱：天乙/天德/德秀贵人 + 空亡 + 天喜 → 婚姻有喜兆但空亡有损
  - 时柱：红艳煞 + 禄神 + 桃花 + 将星 → 中晚年异性缘 + 名望同发
- **命宫长生诀**：命宫己卯，绝在亥；身宫与命宫同（己卯）→ 命身一体，主自力命
- **32 字诀自检**：满盘比劫无财官，调候失衡靠运扶；血刃亡神早年警，红艳将星晚景明
- **使用规律**：G-04-088（血刃应期）/ G-08-042（亡神 + 将星）/ G-12-018（官星化劫）/ G-15-031（披麻入月）/ G-18-022（披麻劫煞母位）/ G-19-013（红艳将星组合）/ G-22-007（将星禄神行业）

### 4 派一致性

- **【高】** 比劫旺、官杀虚、调候失衡、做功层数低 —— 4 派结论同向
- **【中】** 婚姻不顺论 —— 段+杨+任+高 4 派一致认为偏印坐妻宫 + 比劫夺财 → 婚晚或婚劳，但具体年份各派差异
- **冲突项数**：0（无需仲裁）

## 二、推断详情（按主题）

### 2.1 性格

**共识层断语 [4 派一致]**：
> 性格倔强、独立、有主见、不服管，做事讲原则但易孤介；同辈/合伙人关系易摩擦，宜独立做事不宜合伙。 **★4 (85%)**
> - 来源：M1-D-088 / M2-Y-061 / M3-R-110 / G-08-042
> - 证伪：若命主自评"性格随和、善协调、长期合伙稳定" → 失验

### 2.2 财富

**共识层断语 [4 派一致]**：
> 一生财富层级 L7-L8（小康中-上，年收入 15-50 万 / 资产 100-300 万级）；不主大富，温饱衣食无忧。 **★4 (80%)**
> - 来源：M1-D-173 / M2-Y-035 / M3-R-145
> - 证伪：年入 ≥ 200 万或资产 ≥ 千万 → 失验

### 2.3 婚姻

**共识层断语 [4 派一致]**：
> 婚姻晚 + 易波折；妻年纪偏大或像姐姐；婚后聚少离多或感情冷淡。 **★4 (78%)**
> - 来源：M1-D-119 / M2-Y-127 / M3-R-088 / G-15-031
> - 初婚最佳窗口：25-32 岁（2005-2012，甲申大运）
> - 备选窗口：47-48 岁（2026-2027，丙戌运中段）若 32 岁前未婚

### 2.4 事业

**互补层断语 [段+任+高]**：
> 适合金属 / 机械 / IT / 工程技术 / 医疗器械 / 钟表珠宝 / 牙科口腔等"金性 + 锐器 + 精细"行业；体制内偏军警司法纪检；不利贸易、文创、餐饮。 **★4 (83%)**

**互补层断语 [杨+高]**：

> - 来源：M2-Y-201 / G-18-022

**互补层断语 [杨+高]**：

**独门层断语 [高派]**：
