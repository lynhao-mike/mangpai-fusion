## 三、关键教训

### L1 · CFL-001 仲裁错误 → 仲裁引擎 Rule 1 需修正

**问题**：
- 原仲裁按"领域权重"给了杨派 1.0（婚姻 lead），高派 0.6（contributor）
- 结果杨派"婚凶"胜出 → **完全错误！**
- 高派"金舆嫁贵"铁应验

**教训**：
- **领域权重不应该是唯一仲裁依据**
- "金舆"是**时柱特定神煞**，其预测的不是"婚姻好坏"，而是**"配偶/岳家资源"**
- 杨派"五凶煞=婚凶"是**笼统模式判断**，高派"金舆=嫁贵"是**精确位置判断**
- → **精确位置判断 > 笼统模式判断**

**建议修正**：
- engine/arbitration.md 增加 Rule 4：**当一方引用"精确位置+精确神煞"，另一方引用"笼统模式"时，精确方 +15 bonus**
- 高派神煞应用在"配偶资源/岳家背景"场景中 → 升权到 0.9（与杨派平级）
- 杨派"日柱五凶煞=婚凶" → 需加条件限制：**仅在无时柱吉煞对冲时成立**

**影响范围**：所有涉及"日柱凶煞 vs 时柱吉煞"冲突的案例

### L2 · 婚期推算模型严重失误（差 8 年）

**问题**：
- 预测初婚在辛巳运（2008-2018），最有可能 2013 癸巳
- 实际初婚在 2005 年（庚辰运末尾），差 8 年

**根因分析**：
- 系统基于"大运辛巳=巳火带丙财禄→财动→婚"推算
- 但忽略了**更早的触发条件**：
  - 2005 乙酉年：乙木伤官透 + 酉金印根 → **伤官配印 + 财动**
  - 酉合丑（半三合金局）→ 时柱印库被激活 → 贵人资源启动
  - **25 岁在庚辰运**：庚印透干 + 辰为壬水库 → 印旺期可以借人成事
- 实际上命主是"借亲戚关系进事业编 + 借岳父权力结婚"= **全部是贵人/印/外力驱动**
- → 传统"财动=婚"模型在本案不适用！本案是"**印动=婚**"（因为婚姻本质是"借妻家之力"）

**教训**：
- 对于"体强用弱 + 印比一片"的命格，婚姻触发器不一定是"财动"
- 可能是"**印动/贵人动**"——因为这类人的婚姻本质上是"资源整合"而非"情感驱动"
- → 需要在婚期推算模型中增加一条路径："印动=借力成婚"

**建议修正**：
- mapping/complementary.md 增加新互补规律：  
  **COMP-NEW-001**：体强用弱 + 时柱金舆/天乙 + 印旺 → 婚姻触发器 = 印动年（而非财动年）
- 婚期模型增加"印动婚"路径，权重 0.6（低于财动婚 0.8 但必须考虑）

### L3 · "子丑合 = 印库合身"的正确解读

**原解读**：印库合身 → 印星变被动 → 印的力量受限

**正确解读（本案验证）**：
- 子丑合 = **妻宫（子）被时柱（丑=印库）合住** → 自己被妻家"合住/吸附"
- 不是"印变被动"，而是"**自身依附印/妻家资源**"
- "在女方的房里结婚" = 子（自己）被丑（妻家）合走的**字面画面**！
- 丑中己土（正官）→ 妻家有官权（岳父局长）→ "正官入妻宫"精确应验

**教训**：
- "六合"的解读不能只看"五行化合方向"，还要看**"谁合谁"的宾主关系**
- 子（日支=自己的位置）被丑（时支=外来的）合走 = 自己被外力收编
- → 这是盲派"主宾论"的精确应用场景

**建议修正**：
- 杨派 M2-Y-049 "主位被合"的解读需要细化：
  - 主位被**财**合 → 传统婚姻驱动
  - 主位被**印库**合 → 依附式婚姻（借力成婚）← **本案新增**

### L4 · "印动"的双面性验证（2020 庚子年）

**现象**：2020 年同一年内：
- 母亲正月去世（印=母亲 **亡**）
- 提拔副科（印=权力/学位 **旺**）

**教训**：
- "印动"不等于"印吉"或"印凶"，而是**"印星发生了变化"**
- 同一个"印动"流年可以同时发出两个相反的信号
- 这与 4 派共识 CON-YINGQI-001 一致：大运/流年的触发是"事件发生"，不是"吉凶方向"
- → 印星的"人事代表"（母亲 / 学位 / 权力 / 长辈）可能同时有去有来

**建议修正**：
- 在所有"印动年"预测中，增加说明："印动 = 印星代表的多个人事层面同时变化（可能一去一来）"
- 应期表中"印动"不再标为纯吉或纯凶，而标为"**变动年**"

### L5 · "反局成富"概念的重新定义

**原定义（段派）**：寅申冲坏月令 + 印化煞 = 反局成贵（自己做功方式异于常人）

**本案揭示的真实含义**：
- 命主并非"自己做功成富"
- 而是"**借外力（亲戚关系 + 妻家权力）绕过正常路径成事**"
- 寅申冲坏月令 = 正常的"杀→用→功"路径被冲断
- 印化煞 = 不走正常路，**走印（人脉/背景/贵人）路径**
- → "反局"的正确理解：**不是自己翻身，是借力翻身**

**教训**：
- "反局成富"应该细分为：
  - 反局 A：自力反转（适用于食伤制杀成功的案例）
  - 反局 B：借力反转（适用于印旺+贵人+外力的案例）← **本案属于 B**
- 段派 M1 需要增加"反局 B"的定义和应用条件

### L6 · 日柱五凶煞不一定=婚凶

**问题**：
- 本案日柱孤鸾+红艳+童子+九丑+羊刃 → 杨派/高派均判"婚凶"
- 实际：婚姻稳定（借妻家条件，双方利益捆绑）

**教训**：
- 五凶煞聚日柱的正确解读可能不是"婚姻失败"
- 而是"**婚姻形式特殊**"：
  - 孤鸾 = 不走寻常婚恋路（入赘式/被安排式）
  - 红艳 = 配偶颜值/条件好（岳父局长家的女儿）
  - 童子 = 婚姻有"前世因缘"感（命中注定式联姻）
  - 九丑 = 婚姻中自己"低位"（依附女方）
  - 羊刃 = 自己强悍但被妻宫牵制
- → "五凶煞=婚凶"规律需要加条件：**仅在无时柱吉煞 + 无强力外援时成立**
- 当时柱有天乙+金舆+天医等强吉煞时 → 五凶煞**被压制**为"形式特殊"而非"结果不好"

**建议修正**：
- 杨派 M2-Y-068 克妻口诀 → 增加**例外条件**：
  - 例外 1：时柱有金舆+天乙 → 不克妻，改为"妻贵自己低位"
  - 例外 2：妻宫被印库合 → 不克妻，改为"依附妻家"
- 高派 G-CH 车祸婚姻篇 → "日柱五凶煞"条目增加注释：
  - "若时柱吉煞 ≥ 3 个且含金舆 → 婚姻形式特殊但结果未必凶"

### L7 · 农村出身+岳父局长=典型"体强用弱借外力"格局

**现象验证**：
- 体强（壬子帝旺+辛丑+庚申=印比一片）= 能力强/承载力大
- 用弱（戊寅被冲=资源少/出身差）= 自家条件差（农村/父母农民/条件不好）
- 时柱贵人密集 = **晚辈宫/外来资源**丰富 → 岳父是"外来贵人"

**教训**：
- "体强用弱"的人生路径不是"自己挣"，而是"**等贵人/等机会/借力**"
- 时柱=晚辈宫+外来宫，天乙+金舆 = 贵人来自"婚姻/晚辈方向"
- → 完美匹配"岳父是局长，借妻家提拔"的人生模式
- **应该在初始报告中就强调"时柱贵人→婚姻方向的外来资源"这个画面**

## 五、对 v1.1 系统设计的建议

### 5.1 仲裁引擎修正

- 增加 Rule 4："精确位置神煞 > 笼统模式判断"（+15 bonus）
- 增加 Rule 5："当冲突双方各有铁应验案例时 → 不选胜负，改为条件分流"

### 5.2 婚期模型修正

- 增加"印动婚"路径（适用于体强用弱+贵人密集型命格）
- 婚期推算不再唯一依赖"财动年"，增加"印旺年"作为备选触发器
- 对于"妻宫被印库合"的命格 → 优先用"印/贵人动年"推婚

### 5.3 神煞应用修正

- 高派神煞在"配偶资源"场景升权到与杨派平级（0.9）
- "日柱凶煞"判定增加前置条件检查：时柱吉煞数量
- "金舆"单独作为"配偶/岳家资源"指标，不再归入笼统的"婚姻吉凶"

### 5.4 反局概念细分

- 反局 A：食伤制杀成功 → 自力翻身
- 反局 B：印化煞+贵人 → 借力翻身（本案验证）
- 两种反局的富贵上限不同：A 可达 L4，B 通常 L3 顶
