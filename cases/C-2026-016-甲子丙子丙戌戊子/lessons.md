## 教训清单

### CFL-C016-001 · D3"倒象成立"触发器可靠性不足 【P1·系统性】

**现象**：2014、2024、2026 三年均以"倒象成立"为主触发、★5(95%)，但 2014 完全失验、2024 完全失验、2026 暂判失验（3条中2-3条失验）。

**根因**：
- "倒象成立"定义过宽：只要流年干支与原局/大运形成冲/合/刑即判为"倒象"，未区分"有实际做功"还是"只是力场碰撞"
- 前向预测案（无 known_facts）时，三层 gate 的 L3（流年引爆）几乎等于"流年有作用力就过"，缺少事件锚点约束

**修正建议**：
1. "倒象成立"需增加**实质做功验证**：仅冲合不够，需要冲合的对象确实是该域的用神/忌神
2. 前向预测案 ★ 上限从 5 降为 **4**（已在 CFL-C016-002 确认）
3. 考虑引入"倒象强度分级"：单冲=弱倒象(★3) / 冲+合同时=中倒象(★4) / 三刑+冲=强倒象(★5)

**影响范围**：所有前向预测案的 D3 应期模块

### CFL-C016-002 · 前向预测★上限确认降为4 【P1·规则变更】

**决策**：**v1.4.1 起，前向预测案（known_facts为空或极少）三层gate输出 ★ 上限 = 4（80%）**。

**依据**：

### CFL-C016-003 · 孤鸾煞"婚凶"判断再次失验 【P2·规则降权】

**现象**：孤鸾煞挂日柱 → 预测"婚姻孤独/迟婚/独居"，实际命主25岁结婚、婚姻幸福稳定20+年。

**交叉验证**：
- C-2026-001（庚申戊寅壬子辛丑）：杨派"五凶煞=婚凶"失验（实际稳定婚姻）
- C-2026-002（壬戌庚戌戊辰丙辰）：辰戌冲夫宫 → 预测不婚，实际23岁已婚20年
- **C-2026-016**：孤鸾煞 → 预测迟婚，实际25岁婚+幸福

**规律更新**：孤鸾煞在已验证的3个案例中**全部失验**（0/3）。

**修正建议**：
1. 孤鸾煞 boost 从 -4% 降为 **-1%**（仅作微弱参考信号）
2. 在婚姻断语中，孤鸾煞不再作为独立判据，仅在"多凶煞叠加"时才考虑
3. 候选退役条件：若再失验1次（累计0/4），直接退役

### CFL-C016-004 · 婚期预测精度偏差模式 【P2·校准】

**现象**：预测2010年婚姻动象，实际2009年结婚（差1年）。

**交叉验证**：
- C-2026-001：预测2013婚期，实际2005（差8年）
- C-2026-002：预测42-52岁婚，实际23岁（差近20年）
- **C-2026-016**：预测2010，实际2009（差1年）← 本案最接近

**发现**：婚期预测在"早婚信号强"的案例中偏差较小（本案仅差1年），在"婚凶信号强"的案例中偏差极大（被凶煞干扰）。

**修正建议**：当"早婚信号强+晚婚信号弱"时，婚期窗口取**窗口前沿**而非中间值。

### CFL-C016-005 · 2012年换行业未被捕捉 【P3·gate覆盖率】

**现象**：命主2012年从工厂会计转行开学生服务中心（重大事业变动），但该年未进入三层gate输出。

**分析**：流年回溯中2012壬辰标记为【强】（冲日支戌+合大运酉化金），力量场识别正确，但gate未激活。

**根因**：gate的"候选事件"列表中2012年被归类为"工作变动/压力"而非"事业转折"，且L2（大运到位）可能未满足阈值。

**修正建议**：gate候选事件扫描时，对"冲日支+力量【强】"的年份自动升级为"事业转折候选"，而非仅标注"工作压力"。

### CFL-C016-006 · 定性画像精度高·应期精度低的分裂 【P2·系统认知】

**现象**：本案定性画像几乎全对（学历/行业/婚姻/官命/财富层级/家世），但应期命中率很低（已验5条仅中1条）。

**结论**：

**系统级建议**：

- 给命主看的report中，应期段加注"前向预测精度有限，仅供参考"

### CFL-C016-007 · "官杀旺暗引财"在教师命中不适用 【P3·规则边界】

**现象**："官杀旺暗引财=过河财/钱权交易"失验。命主为体制内教师，收入稳定但非"钱权交易"模式。

**分析**：官杀旺(力0.41)确实成立（三子聚官），但"暗引财"的应验需要**财星在局中有力**。本局财星辛藏戌中气（关库），未透天干，财路不通。

**修正建议**："官杀旺暗引财"的触发条件增加："财星须有路径可引（透干/得生/被冲开库）"，否则降级为"官杀旺=权力场强但不直接生财"。

### CFL-C016-008 · 印护官冲不凶——2014年失验根因确认 【P1·引擎bug·R-019】

**二次反馈日期**：2026-05-27

**现象**：2014甲午年，引擎给出"事业转折 ★5(95%)"（主触发：倒象成立·牢灾门）。命主明确否认：2014年无任何事业转折。

**根因深挖**：

```
流年作用链：
  甲午·午冲子×3 → 三个官杀（子=癸=正官/七杀）被冲
  → 引擎判定：官星受冲 = 事业压力/变动
  → 三层 gate 通过 → ★5(95%)

漏判的修正项：
  年干甲 = 偏印，是本命局最强护身符
  甲（印）旺 → 印生日主 → 日主丙火有根
  印旺则官杀无力克制日主，官被冲只是"体制内摩擦"而非"大变动"
  净效应：印强护官，官冲凶性衰减 ≥70%，★5 应降为 ★2-3
```

**候选规律 R-019**（引擎修复建议）：

> **印护官冲不凶律**：当命局印星强度（力量值）≥ 0.25，且流年/大运对官杀产生冲/刑时，事业变动的★上限不超过3；印星强度 ≥ 0.4时，官被冲对应"体制内小摩擦/平稳过渡"而非"大转折"。

**实现位置建议**：`engine/yingqi/threelayer.py`，在 L3（流年引爆）判定后增加"印星护官修正"系数。

**与其他案例的交叉验证**：
- C-2026-015（甲寅乙亥丙辰辛卯）：印星极旺，多个官杀冲年均平稳升迁，不见剧烈震荡 → 方向吻合
- 需再收2个"印旺官冲"案例才能升 confirmed

### CFL-C016-009 · 大运换盘年为独立人生节点——2008年漏判 【P2·gate覆盖率·R-020】

**二次反馈日期**：2026-05-27

**现象**：2008年，命主甲戌大运→癸酉大运换盘，同年民办三本毕业进入社会。引擎流年回溯将2008年标注为普通【中等】能量场，无独立应期条目。

**根因**：

```
当前引擎逻辑：
  gate 候选事件 = known_facts 里有明确年份的事件
  流年回溯 = 无 known_facts 时只给能量场，不生成独立候选

漏掉的结构信号：
  大运换盘年（甲戌→癸酉）本身就是一个独立的"人生阶段切换"信号
  任派有明确规律：大运换盘年±1年内必有人生重大节点
  本案2008年：甲戌（枭神+食神·学业/探索期）→ 癸酉（官杀+财·职业/财运期）
  换盘内涵：从"学习探索"切换到"职场发财"模式
  命主毕业进社会，完美对应
```

**候选规律 R-020**（gate 覆盖率改进）：

> **大运换盘年自动生成人生阶段切换候选事件**：检测到大运换盘年时（含前后各1年缓冲），自动生成"人生阶段切换（大运换盘·新旧模式更替）"候选事件，加入三层 gate 扫描；不依赖 known_facts，属于结构性必然触发。

**已有佐证**：
- C-2026-015：2023年庚辰运入（偏财首透），命主真实发生财务和事业变化 → 大运入运年有重大事件 ✅
- C-2026-016：2008年癸酉运入，命主毕业进社会 ✅
- 需再收1案即可升 confirmed

**实现位置建议**：`engine/yingqi/keys.py` 或 `engine/pipeline.py`，在 `_extract_candidates()` 里增加"大运换盘年自动候选"逻辑，domain = "事业"，confidence 基础值 ★3，若 known_facts 印证则升 ★4-5。

### CFL-C016-010 · 官印相生命局的职业落地路径——"财→食→官"三段式 【P2·D2输出深度·R-021】

**二次反馈日期**：2026-05-27

**现象**：引擎预测行业方向"公门/文教/管理 ★5(90%)"完全正确，但命主经历了"工厂会计（2009）→学生服务中心（2012）→教师编（2020）"的11年迂回才真正落地公门。引擎只给了静态终点，没有给动态路径。

**根因分析**：

```
命盘结构：
  官印相生（亥水官→甲木印→丙火日主）= 方向上必走公门
  食神（戊·时干）= 有一段"技术/服务"过渡

大运与职业路径的对应：
  癸酉运（2008-2018）：癸=官杀，酉=财星
    → 官杀透干但财星为主气 → 财星先发
    → 命主先做财务工作（工厂会计=财星表现）
    
  壬申运（2018-2028）：壬=七杀，申=偏财+偏印余
    → 七杀透干（强制力上升）+ 申冲日支戌（旧结构被打破）
    → 七杀逼迫命主做出事业方向调整
    → 2018年被迫考教师编（七杀"逼"进公门）
    → 2020上岸（官印相生终于落地）

中间的食伤过渡（2012学生服务中心）：
  2012壬辰·辰冲日支戌（关库被撬）+ 辰六合酉化金（财气合局）
  = 食伤泄秀找到表达出口，开了与教育相关的"学生服务中心"
  = 提前在"教育/服务"域预演，是教师路的必要准备
```

**候选规律 R-021**（D2 杨派行业输出深度改进）：

> **官印相生命局职业路径三段式**：当命局同时具备官印相生+食神，且多个大运透不同用神时，行业方向输出须附带路径说明：
> - 第一段（财星大运）：财星先发，往往先从财务/销售/商业切入，即使专业方向指向公门
> - 第二段（食伤大运或食伤旺年）：食伤泄秀期，职业摸索/副业/自营，是向公门过渡的准备期
> - 第三段（官杀大运）：七杀/正官推动，官印相生真正落地，进入体制/公门/正途

**标准话术模板**（可加入 D2 输出）：
> "行业方向：公门/文教/管理（终点）。职业落地路径：[大运X]财星先发，初期可能从财务/商业起步；[大运Y]食伤过渡，有一段探索/自营期；[大运Z]官杀推动，最终进入公门正途，约[年龄段]落定。"

**实现位置建议**：`engine/picture/matcher.py` 的行业方向输出段，增加"大运序列路径推导"模块（当官印相生+食神同现时激活）。
