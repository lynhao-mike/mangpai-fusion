## 五、重大失验分析

### 5.1 职业方向失验：民营首选 → 实际体制内事业编

**失验根因**：
- 规律"无印透干=非体制"（M2-Y-120）在本案失验
- 命主虽然印（癸）藏丑不透干，但年支丑藏印=印有根基（暗处有靠山/编制机会）
- **卫健局事业编** = 非公务员、非企业，是"准体制"（事业编=不需要印极旺，只需印有根）
- 天医神煞（年支丑）= 医疗/健康行业缘 → **卫健局**完美对应天医！

**修正方向**：
- "无印透干"判非体制的边界条件需要收紧：**印藏有根时，事业编/准体制路径不应完全排除**
- **天医神煞在行业判定中的权重应上调**：命主实际在卫健局=完全对应天医
- 高派"天医=医疗/保健/服务缘"在本案**铁命中**（之前只给了事业域+12%，未给行业首选）

### 5.2 收入失验：L6 年入30-80万 → 实际7-8万

**失验根因**：
- 食伤大运（丁未）改善效果被高估
- 群比夺财结构 + 事业编 = 薪酬体系由编制决定，非市场竞争型收入
- **CFL-C015-002 的反向案例**：体制内案例应走"薪酬体系分级"而非"市场财富7等"
- 做功0.5层的基础判没错，错在"食伤大运有望推到L6-L7"的乐观预期

**修正方向**：
- 事业编+群比夺财+做功0.5层 → **实际收入区间应压到 L2-L4（5-10万/年）**
- 食伤大运的"改善"在事业编体制下 = 工作能力表现好，但薪酬不随市场走

### 5.3 健康：手术外伤失验 → 甲亢（新发现）

**关键观察**：
- 血刃+羊刃=手术外伤 → **命主无外伤无手术**，直接失验
- 但命主有**甲亢**（慢性病）
- 甲亢 = 甲状腺疾病 = 颈部/咽喉 = **在中医五行对应中，酉金主肺/呼吸/皮肤/喉部**
- **双酉自刑 + 酉金过旺** → 金旺伤木（甲状腺属木/内分泌系统与木有关）→ 本命局"金旺克木"=甲亢的命理对应

**候选新规律（待累计验证）**：
- 坤命双酉自刑 + 金旺无泄 → 甲状腺/内分泌慢性病风险（替代"手术外伤"判断）
- 血刃/羊刃=手术外伤的判断在"无外力受伤命局"中应降级为"慢性消耗型疾病"
