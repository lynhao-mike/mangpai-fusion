## 一、4 派各自定性

### 段派（M1-D-*）
- **格局判定**：正官格（月令子，壬当令藏；月干庚正官透出 → 取月干透为格）+ 印绶辅
- **富贵层级**：中富·下等～上等。做功 2 层：庚（月干官）→ 壬（月令印）→ 乙（日主），路径完整但伤官年干透干，长远略有限制
- **关键签字**：M1-D-022（丙辛合 1:1，伤官合七杀 = 格局清纯化）；M1-D-044（身弱透官 = 官当财看）；M1-D-122/172（做功层次定级）

### 杨派（M2-Y-*）
- **主位宾位**：日主乙木为主位；时柱辛巳为宾位（七杀+正财库）
- **体用判定**：体 = 乙木日主；用 = 庚正官 + 子月壬印（生身）；做功对象 = 官印相生护身
- **调候情况**：M2-Y-011 子月乙木 = **寒木向阳**，必赖丙火调候。年干丙火透出 = **调候得位**（最关键加分项）
- **完整性评估**：8 字结构完整（无残缺、无孤立），印旺有根，伤官调候有用

### 高派（G-*-*）
- **清浊判**：官杀双透为底层「浊」，但年干丙合时干辛 → 七杀被合制 → **半清纯化**
- **神煞要点**：天乙×2 / 德秀×2 / 天德 / 月德 / 词馆 / 国印 / 天医 / 金舆 / 红鸾 / 福星 / 太极 / 天厨 = **神煞极丰**（贵人命底色）
- **命宫长生诀**：命宫庚子 = 庚金长生在巳，巳为时支 → **命宫长生归时柱** → 晚年得力 / 中晚年富贵
- **32 字诀自检**：「身印官全」、「调候得位」、「贵人扎堆」、「驿马奔波」 → 偏向贵格中等

### 任派（M3-R-*）
- **18 道法门相关**：
  - 第 16 道「富贵八字两要素」= **有势**（壬水月令印旺 = 有势）+ **有功**（庚→壬→乙官印相生 = 有功） → 富贵格成立
  - 第 1 道「主体客体」= 主体乙木有印护，客体庚官有功
- **富贵两要素**：势 ✓ 功 ✓ → 中富以上路径打通
- **阴阳划分**：阴日（乙）阳月（庚子的子为阳）+ 阴时（辛巳天干阴），整体阴阳基本平衡
- **6 触发应期**：原局含巳亥冲（已含动机），亥子合（印护），戌冲辰（流年待发） = 应期素材丰富

### 4 派一致性
- **格局**：高度一致（4 派均判正官格官印相生）
- **层级**：高度一致（4 派均落点中富区间）
- **婚姻**：高度一致（4 派均判巳亥冲 = 婚宫不安 + 晚婚倾向）
- **学业**：中度一致（杨派"重一本"+ 高派"词馆 985 顶配"，量级有差）
