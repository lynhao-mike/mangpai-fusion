## 一、本案结构特征（供同类案例参考）

### 1.1 "三比同透克单财"指纹

| 特征 | 本案 |
|---|---|
| 三乙同透 | 年干乙 + 日干乙 + 时干乙 |
| 单财透干 | 月干戊（正财）独担 |
| 体用失衡 | 体（5字）≫ 用（1字）|
| 七杀全藏 | 辛金藏丑酉酉，不透干 |
| 结构名 | **群比夺财 + 官杀藏而待发** |

**同类判断规律**：三比以上克单财 → 财来多人分，财务难独享；适合独立核算，不宜合伙分账。

### 1.2 "酉酉自刑"婚宫内耗指纹

| 特征 | 应用 |
|---|---|
| 日支酉 + 时支酉 = 自刑 | 婚宫自损，非外力破坏 |
| 七杀同时自刑 | 坤命七杀=配偶宫，自刑=内耗型感情关系 |
| 将星双现叠加 | 两个将星同自刑 = 强势与强势的碰撞 |

**同类判断规律**：自刑 ≠ 没有感情，而是"折腾"；婚姻模式可能是自主进入、自主离开，主导权在命主手里。

### 1.3 "巳酉丑待补完三合"应期指纹

| 触发条件 | 应期 |
|---|---|
| 巳年/巳运（大运乙巳已过，17-26岁）| 七杀局开（感情/官运初触发）|
| 卯年（卯酉冲）| 婚宫被冲（关系大动）|
| 丑年（丑酉半合连锁）| 财官双动 |

**同类判断规律**：半合待补命局，逢补完之年为该领域（官/财/情）的集中应期年，不可遗漏。

## 二、分析阶段工程观察

### 2.0 ❌ 错误纠正：寅酉误判为暗合（已修正）

**错误**：原分析在关键结构表、做功路径、杨派暗引、五合应事、任派道法门、EXC-3独门层中，6处将寅酉关系误判为「暗合」。

**正确**：
- **四派原文一致**：暗合仅四组 → **寅丑 / 午亥 / 卯申 / 子巳**
- **寅酉关系**：高派称为「绝」（寅酉绝车马财，金绝木衰），是断绝关系，与暗合截然相反
- 寅中藏干（甲、丙、戊）与酉中藏干（辛）之间无任何合的关系（无甲辛合/丙辛虽合但跨支不成立）

**波及修正清单**：
- [x] 关键结构表：删除「寅酉暗合」行，改为「寅酉相绝」
- [x] 做功路径：删除路径2（财生官跨柱）、路径5（寅酉暗合）
- [x] 杨派暗引第3条：删除
- [x] 五合应事：撤销寅酉暗合表述
- [x] 任派道法门：撤销寅酉暗合
- [x] EXC-3：整体撤销
- [x] input.md关系字段：修正

**经验规则**（入库避免重犯）：
> 任何时候见到「X酉暗合」「寅X暗合」的判断，必须先核查四组暗合清单：**寅丑 / 午亥 / 卯申 / 子巳**。不在此列的一律不能用暗合理论。

- ✅ 本案严格执行了 XF-001 黑名单："七杀全藏不透"**不断晚婚铁律**，改为婚路迂回三项联检（MR-005）
