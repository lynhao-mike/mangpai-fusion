## 二、核心结构特征

### 2.1 “三比同透克单财”指纹

| 特征 | 本案 |
|---|---|
| 三乙同透 | 年干乙 + 日干乙 + 时干乙 |
| 单财透干 | 月干戊正财独担 |
| 体用失衡 | 体（5字）≫ 用（1字） |
| 七杀全藏 | 辛金藏丑酉酉，不透干 |
| 结构名 | 群比夺财 + 官杀藏而待发 |

经验规则：三比以上克单财，财来多人分，财务难独享；若落入事业编薪酬体系，收入按岗位系统兑现，不宜按市场财富上限放大。

### 2.2 “酉酉自刑”婚宫内耗指纹

| 特征 | 应用 |
|---|---|
| 日支酉 + 时支酉自刑 | 婚宫自损，关系内耗 |
| 七杀同时自刑 | 坤命配偶星自刑，感情折腾 |
| 将星双现叠加 | 强势、独立、容忍度低 |

经验规则：自刑不是无婚，而是“先进入、再折腾、可分可合”。本案 2010 初婚、2013 离婚、2017 再婚，为酉酉自刑强验证。

### 2.3 “巳酉丑待补完三合”应期指纹

| 触发条件 | 实际反馈 |
|---|---|
| 巳年/巳运补完金局 | 2013 癸巳离婚，强命中 |
| 卯年冲婚宫 | 2011 辛卯生女，婚宫动转为生育 |
| 酉年伏吟婚宫 | 2017 丁酉再婚，强命中 |

经验规则：半合待补命局，逢补完之年为该领域集中应期，不可遗漏。

## 三、重大失验与修正

### 3.1 职业方向失验：民营首选 → 实际体制内事业编

失验根因：

- “无印透干=非体制”使用过度。
- 年支丑藏癸印，印虽不透但有根，足以支持事业编/准体制。
- 天医神煞在年柱，实际对应卫健局，原分析未把天医上调为行业首选。

修正规则：无印透干只能降低强公务员/高权力体制概率，不能排除事业编、医院、卫健、学校等准体制岗位。天医出现时，医疗卫生系统需列入高优先行业。

### 3.2 收入失验：L6 年入 30-80 万 → 实际 7-8 万

失验根因：

- 食伤大运改善效果被高估。
- 群比夺财 + 做功 0.5-1 层 + 事业编普通岗，应采用薪酬体系分级。
- 事业编路径中，才能提升多表现为稳定、责任、口碑，不等同市场化收入跃升。

修正规则：事业编普通岗 + 群比夺财，应把收入压至 L2-L4；食伤运最多提升岗位表现，不宜直接推高收入。

### 3.3 健康失验：手术外伤 → 甲亢慢性病

命主无手术无外伤，但有甲亢。

候选新规律：坤命双酉自刑 + 金旺无泄，可考虑甲状腺/内分泌慢性病风险。血刃/羊刃不可机械断手术，需区分外伤型与慢性消耗型。

## 四、理论错误纠正：寅酉不是暗合

原分析曾多处误判寅酉暗合，已修正。

正确规则：

- 暗合仅四组：寅丑、午亥、卯申、子巳。
- 寅酉为绝关系，不构成暗合。
- 涉及寅酉时，应按绝、金木不亲、关系断续来处理，不得取“暗中相合”。

## 六、同类案例使用边界

1. 见无印透干，不要直接排除事业编/准体制。
2. 见天医，必须检查医疗、卫健、药品、护理、健康服务等行业。
3. 见酉酉自刑，婚姻要断内耗与反复，不宜断无婚。
4. 见群比夺财，财富要先看收入制度：体制薪酬和市场收入不可混用。
5. 见血刃羊刃，健康先分“手术外伤”与“慢性病消耗”，不可一刀切。
