## 立案时观察到的"系统层"教训

> 这部分不依赖命主反馈，而是分析过程本身揭示的问题。

### L1 · 信息缺失对置信度的硬性约束

- 缺**婚姻状态** → 婚姻领域所有应期断语**置信度封顶 ★★★★**
- 缺**历史事件** → 无任何回测样本，4 派动态分都是 0
- → 本案输出严重依赖**静态分**，违背了系统"宁慢不假"的核心精神
- **建议**：未来对**纯命盘+无历史事件**的输入，应在报告封面**强制警告**："本报告无回测基础，所有结论需命主反馈才能升级"
- **是否影响 v1.1 设计**：✅ 应在 `templates/input-from-wenzhen.md` 加红色提示"历史事件 ≥ 3 件"为强必填

### L2 · 4 派对"日柱多凶煞 + 时柱多吉煞"模式的看法分歧

- 杨派：日柱五凶煞（孤鸾+红艳+童子+九丑+羊刃）→ 必凶
- 高派：时柱天乙+月德+金舆+天医+天喜 → 必吉
- → 仲裁后给"前凶后吉 / 早婚不顺晚得贵妻"的调和解释
- **建议**：此模式可能成为新的共识规律候选（"日凶时吉 = 命运分段反转"）→ 待累计 ≥ 3 案验证后晋级

### L3 · 寅申冲在杀印格中的处理

- 段派将寅申冲解为"反局成富信号"
- 杨派将寅申冲解为"用神受损"
- 任派将寅申冲解为"应期触发器"
- 高派将寅申冲解为"应期六法之冲"
- → 4 派**对同一象的解读路径完全不同**，但**结论方向接近**（都认为是大事年触发）
- **建议**：可作为"互补层经典案例"——同象不同解但方向一致

### L4 · 子丑合的"半合存疑"

- 子丑合化土（六合传统）vs 子水被丑土合住（盲派现代解）→ 4 派均倾向后者
- 本案命局印重，子丑合 = **印库合身 = 印更被动**
- **教训**：盲派对"合"的解读是"力的合住"而非"五行的化合"，这与子平/三命有本质差异
