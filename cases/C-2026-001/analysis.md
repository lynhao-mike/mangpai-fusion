**段派关键签字**：

### 1.2 杨派视角（M2 杨清娟 · 主宾体用 + 完整性）

**主位 vs 宾位**：
- 主：日柱壬子（自身），时柱辛丑（晚景 / 子女 / 印帮）
- 宾：年柱庚申（祖业 / 父母），月柱戊寅（事业 / 中年）

**体用**：
- 体：壬子（羊刃帝旺）+ 辛丑（印贴身）+ 庚申（印根）= 印比一片，**体超强**
- 用：戊寅（杀+食+财）= 用极弱（寅被冲，三藏俱伤）
- → 体 ≫ 用 = **承载有余而所获有限**（M2-Y-091 承载论）

**完整性原则**（M2-Y-014）：
- 印 ✅ 强 / 比 ✅ 强 / 杀 ⚠️ 透但根弱 / 食 ❌ 藏被冲 / 财 ❌ 藏被冲 / 官 △ 丑中己合子
- → **不完整**（财官受损）→ 富贵格局有缺

**学历判定**（M2-Y-070..072 学历 14 条）：
- 印星双透有根 + 月支寅藏甲（食神 = 学习能力）+ 词馆 + 文昌 + 德秀 → **本科以上，倾向 211/985**
- 但寅被申冲 → 学历过程**有波折**（高考二战？复读？专升本？）

**婚姻判定**：
- 日支子水妻宫被丑合走（M2-Y-049 主位被合）
- 财星（丙）藏月支被冲，**夫妻宫无财根**
- 日柱**孤鸾 + 红艳 + 童子 + 九丑 + 羊刃** 五凶煞聚 → **婚姻波折显著**
- M2-Y-064 人来印来：婚期看印星受冲合年（辰戌冲、未冲丑）
- M2-Y-068 克妻口诀：羊刃帝旺坐妻宫 → **克妻信号**（程度待回测）

**杨派结论**：印重身旺 + 财官受损 + 婚姻凶煞密集，**画面是"学问之人，妻缘多变"**。
