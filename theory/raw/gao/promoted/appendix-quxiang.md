## §Q.1 十天干完整取象

> 每节五栏速查：天象 / 人事 / 物象 / 人体 / 性情。**关键诀**单列。整节来源：`盲派命理AI分析师提示词_v1.md` 第二部分 一节。

### 阴阳成对取像总纲（v7.0 新增 · GP-XF-17 高派《象法篇》第 7~11 课晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 7~11 课各对开篇（"甲乙木 / 丙丁火 / 戊己土 / 庚辛金 / 壬癸水"五节对比）
> **铁律**：十干取象**先成对，后单字** —— 先确定该干属于五对中的哪一对、取其阳阴大方向，再进入下方具体个字速查。

| 对 | 阳干 | 阴干 | 粗细 | 明暗 | 高平 | 刚柔 | 动静 | 核心对偶 |
|---|---|---|---|---|---|---|---|---|
| **木** | 甲（栋梁）| 乙（藤蔓）| 粗大 / 细小 | 明亮（雷电）/ 含蓄（微风）| 高耸 / 匍匐 | 刚直 / 柔韧 | 动（破土）/ 静（攀附）| 甲=独立领导 / 乙=合作依附 |
| **火** | 丙（太阳）| 丁（星烛）| 大火 / 小火 | 普照 / 一角 | 高悬 / 低伏 | 猛烈 / 温柔 | 动（辐射）/ 静（内敛）| 丙=号召公开 / 丁=隐秘玄学 |
| **土** | 戊（山丘）| 己（平原）| 大块 / 细碎 | 霞光 / 乌云 | 高山 / 低田 | 厚重 / 卑湿 | 静（不动）/ 静（蓄藏）| 戊=阳刚稳重 / 己=阴柔含蓄 |
| **金** | 庚（矿石）| 辛（珠宝）| 大器 / 精细 | 白虎肃杀 / 霜露灵动 | 高（刀剑）/ 平（针线）| 刚健 / 柔润 | 动（征伐）/ 静（精雕）| 庚=武职军警 / 辛=法律精药 |
| **水** | 壬（江河）| 癸（雨露）| 大水 / 小水 | 波澜壮阔 / 润物无声 | 奔腾 / 渗透 | 刚（冲刷）/ 柔（滋养）| 动（奔流）/ 静（沉潜）| 壬=外向创业 / 癸=内敛谋略 |

**速用法**：
1. 看到某干 → 先对号入座上表 → 取其"核心对偶"大方向
2. 再进下方个字详查 → 取具体物象 / 人事 / 人体
3. 阳干张扬 / 阴干含蓄 = **铁律**（不可阴阳互套）

### 甲木
- **天象**：雷、青龙
- **人事**：领导 / 元首 / 公门 / 首领 / 第一把手 / 长者 / 栋梁之才
- **物象**：大树 / 栋梁 / 电杆 / 高楼 / 桥梁 / 铠甲 / 钉子 / 螺丝 / 钻头
- **人体**：头部 / 四肢 / 神经 / 胆
- **性情**：直 / 急 / 风风火火 / 钻劲 / 一根筋 / 不善变通
- **关键诀**：字形 = 田 + 芽（开端 / 萌发 / 第一）；带"钻头" → 钉子 / 螺丝 / 工业切削；带"铠甲" → 执法 / 武职。

### 乙木
- **天象**：风、虹霓
- **人事**：文化人 / 艺术家 / 教师 / 宣传
- **物象**：花草 / 藤蔓 / 旗帜（品牌 / 招牌 / 个人 IP）/ 绳索 / 衣物
- **人体**：手指 / 肝 / 毛发
- **性情**：柔 / 韧 / 能屈能伸 / 吃苦耐劳 / 细致
- **关键诀**："藤萝系甲" → 乙依甲存（合作 / 攀附 / 借势）；旗帜象 → 品牌 / 商标。

### 丙火
- **天象**：太阳
- **人事**：媒体人 / 影视人 / 官员 / 明星 / 宣讲 / 号召者
- **物象**：阳光 / 电器 / 电力 / 广告 / 信息 / 文化 / 神明 / 大火

- **性情**：积极 / 乐观 / 自信 / 号召力 / 光明磊落 / 扩散辐射
- **关键诀**：核心 = 扩散 / 辐射 / 传播；长生在寅、死在酉；"丙火猛烈，欺霜侮雪，能煅庚金，逢辛反怯"。

### 丁火
- **天象**：星星 / 烛光 / 灯火
- **人事**：玄学师 / 宗教人 / 中医 / 星象家 / 设计师 / 网络写手

- **性情**：温柔 / 灵动 / 付出 / 奉献 / 隐秘

### 戊土

- **人事**：中介 / 仓储 / 土地 / 地产 / 保管
- **物象**：高山 / 城墙 / 大坝 / 仓库 / 停车场 / 堤防

### 己土

- **人事**：农牧 / 文教 / 墓园 / 粉尘相关 / 化妆品

### 癸水

### 三组分类总览（v7.0 新增 · GP-XF-19/20/21 高派《象法篇》第 12~14 课晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 12 课 §一-§六 + 第 13 课 §一-§六 + 第 14 课 §一-§五
> **铁律**：十二支取象**先归组，后查个字** —— 先确定该支属于三组中的哪一组、取其组级身份，再进入下方具体个字速查。

#### 一、四正（子午卯酉）—— 中神 · 将星 · 桃花 · 沐浴 · 阳刃

> 第 12 课（"子午卯酉谓之四正…后天八卦四方位"）

| 支 | 方位 / 卦 | 五行极旺 | 核心象 | 组级身份 |
|---|---|---|---|---|
| **子** | 正北 · 坎 | 水旺极 | 流动 / 江湖 / 智 / 车船 / 文墨 | 帝座 · 智谋核心 |
| **午** | 正南 · 离 | 火旺极 | 阳光 / 战场 / 礼 / 冶炼 / 广告 | 端门 · 热烈核心 |
| **卯** | 正东 · 震 | 木旺极 | 花草 / 门户 / 仁 / 网络 / 车船 | 日出 · 生发核心 |
| **酉** | 正西 · 兑 | 金旺极 | 银行 / 钟表 / 义 / 法律 / 玄学 | 日落 · 收敛核心 |

**四正组级铁律**：
- 四正皆带**桃花 / 沐浴 / 阳刃 / 将星**四重属性（视具体日干而定）
- 四正属相（鼠马兔鸡）喜动不喜静 → 动则吉、静则郁
- 四正为中神 → 三合局将星所在 → 凝聚力最强
- **盲派 18 口诀**（选要）：子寅风流客 / 子卯离祖上 / 子辰华盖命金科有功名 / 子巳病在身吉运在龙门

#### 二、四长生（寅申巳亥）—— 驿马 · 临官 · 四门 · 动态

> 第 13 课（"寅申巳亥为四长生之地…四临官位…四驿马…四门"）

| 支 | 长生所属 | 禄属 | 门 | 核心象 | 组级身份 |
|---|---|---|---|---|---|
| **寅** | 丙戊长生 | 甲禄 | 鬼门 | 山林 / 寺庙 / 公门 / 高雅 | 春首 · 文贵动力 |
| **申** | 壬水长生 | 庚禄 | 人门 | 道路 / 车辆 / 军队 / 金融 | 秋首 · 武职动力 |
| **巳** | 庚金长生 | 丙戊禄 | 地户 | 文化 / 影像 / 闹市 / 温暖 | 夏首 · 传播动力 |
| **亥** | 甲木长生 | 壬禄 | 天门 | 池塘 / 文墨 / 科技 / 思想 | 冬首 · 智慧动力 |

**四长生组级铁律**：
- 四支同时身兼**四长生 + 四临官（禄）+ 四驿马 + 四门**四重身份
- 古驿马 = 传递动态 → 动则利（出差 / 搬家 / 调动 / 出国）
- **马星 6 断语**：①女命带马逢冲心意不定 ②男命马坐正财老婆勤快 ③比劫坐马兄弟各在一方 ④马头剑威震边疆 ⑤马奔财乡发如猛虎 ⑥庚甲寅申商途吏人

#### 三、四库（辰戌丑未）—— 墓库 · 华盖 · 刑冲发财 · 静态

> 第 14 课（"辰戌丑未墓库区别…杂气财官刑冲则发"）

| 支 | 库属（有根为库）| 墓属（无根为墓）| 核心象 | 组级身份 |
|---|---|---|---|---|
| **辰** | 木库（乙有根）/ 水库（癸有根）| 壬墓 / 辛墓 | 水库 / 学校 / 医药 / 天罡 | 春余 · 文教库 |
| **戌** | 金库（辛有根）/ 火库（丁有根）| 丙墓 / 乙墓 | 火库 / 军营 / 寺观 / 魁罡 | 秋余 · 武备库 |
| **丑** | 水库（癸有根）/ 金库（辛有根）| 庚墓 / 丁墓 | 金库 / 金融 / 监狱 / 阴暗 | 冬余 · 金融库 |
| **未** | 火库（丁有根）/ 木库（乙有根）| 甲墓 / 癸墓 | 花园 / 酒食 / 田园 / 木材 | 夏余 · 酒食库 |

**四库组级铁律**：
- **有根为库、无根为墓** = 高派核心判定（联动 §A.2.10 入墓规则）
- 古书引：『杂气财官，刑冲则发』『辰戌丑未遇刑冲，无人不发』
- **固定口诀**：丑寅好学医 / 丑卯不相逢相逢一场空 / 辰巳坐地网不利本土长 / 辰午官煞重药石连床中 / 辰未自带印华盖星坐命
- 少年逢库懵懂 / 中年逢库发财无数 / 老人逢库棺材一付（按时间宫位论）

**速用法**：
1. 看到某支 → 先归组（四正 / 四长生 / 四库）→ 取组级身份
2. 再进下方个字详查 → 取具体物象 / 人事 / 神煞
3. 组级属性 + 个字属性**叠加**才是完整取象

### 六、宫位铁律：内外环境 + 干支表里（v6.1.4 新增 · GP-XF-23 高派《象法篇》第 15 课晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 15 课 §五末段（"年月为外环境 / 日时为内环境"）+ §五倒数第二段（"天干代表人的表象…地支代表人的内在的东西"）

#### 六.1 · 内外环境铁律

```
年月柱 = 外环境（社会 / 单位 / 父母 / 公开场合）
日时柱 = 内环境（家庭 / 配偶 / 子女 / 私下心境）
```

| 应用 | 解读方向 |
|---|---|
| 年月凶 + 日时安 | 外面有难，回家有靠 → 主受外界压力但家庭可缓冲 |
| 年月安 + 日时凶 | 外面光鲜，家中有事 → 主社会形象佳但家事乱 |
| 年月凶 + 日时凶 | 内外交困 → 必须人审，应期前兆 |
| 年月安 + 日时安 | 内外双安 → 平稳期 |

#### 六.2 · 天干表 / 地支里铁律

```
天干 = 表象 / 性格表象 / 容易被外人看出的显著特点
地支 = 内在 / 内心世界 / 与家人的关系 / 比较隐蔽的思想与心志
```

| 应用 | 解读方向 |
|---|---|
| 干吉支凶 | 外表光鲜内心苦 → 形象 / 实际不一致（典型：富人之忧）|
| 干凶支吉 | 外表困苦内里厚 → 闷声发财 / 大智若愚 |
| 干支同吉 | 表里如一吉 |
| 干支同凶 | 表里如一凶（必须人审）|

> **配套铁律 GP-XF-23**：宫位判读需**同时**考虑 6 个维度 —— ①六亲（Q.3.一）②时间（Q.3.二）③身体（Q.3.三）④空间（Q.3.四）⑤八卦（Q.3.五）⑥内外 / 表里（本节）；**任意单一维度**不可铁断。

## §Q.4 取象铁律（八条 · v7.0 由五条扩为八条）

> 来源：`盲派命理AI分析师提示词_v1.md` 第二部分 + `盲派命理AI分析师提示词_v2.0.md` 第一部分『体用论命入手七步法』第三步『寻根找出处』+ 第四步『找渠道看工具』。

### 铁律 Q.4-A · 天干必找地支根源 + 发表归根定体用（v6.1.4 标题升级 · GP-XF-06 高派《象法篇》第 3 课晋级补强）

> **来源补强**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 3 课 §1+§2（"天干为头地支为肢…发表归根定体用"）

#### 基础铁律（v1 / v2.0 沉淀）

- 干无根 → 浮象，象意不兑现
- 干有根 → 实象，象意可兑现
- 根受伤（被冲 / 穿 / 合走 / 入墓）→ 兑现窗口被截
- 根再透 → 同象重见，加倍兑现
- 配套：合关系力量对比（v2.0 第四部分）—— 1:1 / 1:多 / 多:1 / 分组四态决定象意能否成立。

#### 高派补强（GP-XF-06 v6.1.4 新增）

- **干为头 / 支为肢**：天干 = 表象 / 行动 / 主动方向；地支 = 内里 / 根基 / 被动承载
- **干主动 / 支主静**：取象时干向外（社会 / 公开）/ 支向内（家庭 / 私下）
- **发表归根定体用**：地支藏干透到天干 → 干以支为根 → 此象"成立"且**可兑现**；不透出 → 浮象 / 不兑现
- **联动 Q.4-B/C**：发表方向（向我宫 / 他宫）决定象意是『命主主动派出』还是『外部送进来』

### 铁律 Q.4-B · 我宫透出 = 命主特使
- 日支（我宫）所藏 → 透到天干 = 命主亲自参与该象
- 透到时干 → 命主主导晚年事
- 透到月干 → 命主主导事业 / 单位事
- 透到年干 → 命主对外（国家 / 社会）有担当
- 透出之物 = 命主"派出去"的角色

### 铁律 Q.4-C · 年月透入 = 国家社会送来
- 年柱 / 月柱地支所藏 → 透到日 / 时干 = 外部送进来
- 年透到日 → 国家 / 远方馈赠 / 祖业
- 月透到日 → 单位 / 父母 / 同事影响
- 透出物 = 命主所"接收到"的资源（也可能是负担）
- 配合体用：透出物为用 = 资源；为忌 = 负担

### 铁律 Q.4-D · 多点成像三原则铁律（v6.1.4 新增 · GP-XF-02 高派《象法篇》第 2 课 §一晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 2 课 §一（"两点成像三点成一线方可铁口直断"）

| 原则 | 内容 | 铁断准入 |
|---|---|---|
| **一点不成像** | 单字孤象（如仅看到"丙=太阳"一字）→ 取象**不可铁断** | ❌ 必须降为"参考意象" |
| **两点成一象** | 同象在两个不同位置（如丙坐巳、又见丙寅）→ 取象**可立**但需验证 | ✅ 可断"该象成立" |
| **三点成一线** | 同象在三处或以上 + 大运 / 流年引发 → **铁口直断**准入 | ✅✅ 可断"何年兑现" |

**应用**：

```
某字断语（如"必出富贵"/"必有牢狱"/"必显学历"）需满足：
① 至少两点同象 → 才可写入 Step 3 输出
② 三点齐 + 应期星 → 才可标"★★★★★ 铁断"等级
③ 仅一点 → 必须降为"参考象 / 待验证"
```

> **联动 §3.0 铁断前提自检**：本铁律是 §3.0 铁断前提的**取象层基础** —— 任何"必"字断语必先经本铁律 3 步过滤。

### 铁律 Q.4-E · 太极点转换（一点多像反复定像 · v6.1.4 新增 · GP-XF-04 高派《象法篇》第 2 课 §三晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 2 课 §三（"一点多像 / 同一个字以谁为太极点"）

#### 核心铁律

> **同一个字在不同太极点下取象不同**：
> - 以**日主**为太极点 → 该字 = 命主自身的某面（性格 / 身体 / 事业）
> - 以**官星**为太极点 → 该字 = 丈夫（女命）/ 工作 / 上司
> - 以**财星**为太极点 → 该字 = 妻子（男命）/ 财产 / 父亲
> - 以**印星**为太极点 → 该字 = 母亲 / 学业 / 房产
> - 以**食伤**为太极点 → 该字 = 子女 / 才华 / 表达
> - 以**比劫**为太极点 → 该字 = 兄弟姐妹 / 同事 / 朋友

#### 切换铁律

```
铁律 Q.4-E.1 · 切换前必显式声明
  ━━━━━━━━━━━━━━━━━━━━
  默认所有取象以日主为太极点。
  在分析"丈夫 / 妻子 / 母亲 / 子女 / 兄弟"等他人话题时，必须显式声明：
  "现在以 X 星为太极点重新取象 ——"
  否则容易出现『把妻子的事说成自己的事』的混淆错误。

铁律 Q.4-E.2 · 反幻觉硬约束（联动主入口 §3.0）
  ━━━━━━━━━━━━━━━━━━━━
  自身（日主）的吉凶兑现永远以日主为太极 —— 这条不可被 Q.4-E 覆盖。
  Q.4-E 仅在分析他人 / 六亲专题时启用。

铁律 Q.4-E.3 · 联动 Q.4-D 多点成像
  ━━━━━━━━━━━━━━━━━━━━
  在新太极点下的"两点成象 / 三点成线"需重新计算（不沿用日主太极的统计）。
  例：以丈夫官星为太极看妻子的命定 → 不可拿日主太极的『三点齐』来铁断丈夫事。
```

> **八条铁律联用范式**：
> ① **Q.4-A** 先看根源是否健康（高派"发表归根定体用"补强）
> → ② **Q.4-B/C** 看落到我宫还是他宫，方向是『派出去』还是『送进来』
> → ③ **Q.4-D** 数同象点数（一 / 二 / 三）决定铁断准入
> → ④ **Q.4-E** 涉及他人话题时切换太极点（必须显式声明）
> → ⑤ **Q.4-F** 同一字取象时核查多重性（不止一种可能）
> → ⑥ **Q.4-G** 退一步看整体局象（包局 / 夹局 / 孤阳孤阴）
> → ⑦ **Q.4-H** 最终用宫位吉凶四柱表定方向（年 / 月 / 日 / 时）
> → ⑧ 才能给象意打分（成立 / 部分成立 / 不成立）。

### 铁律 Q.4-F · 类象多重性三条（v7.0 新增 · GP-XF-03 高派《象法篇》第 2 课 §二晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 2 课 §二（"类象的多重性 ABC"）

#### 三条铁律

| 条 | 内容 | 应用 |
|---|---|---|
| **A · 一物多象** | 同一个字可以代表多种事物（如"卯"既是树又是车也是床也是门窗也是网络）| 取象时列出 2~3 种可能 → 再用 Q.4-D 多点成像筛选 |
| **B · 同符号对多事物** | 同一个十神 / 干支可以对应命主不同层面（如"丙"既是父亲也是官也是名气也是眼睛）| 取象时先确定太极点（Q.4-E）→ 再定位哪层含义适用 |
| **C · 干主动支主静** | 天干 = 动态 / 公开 / 意向 / 趋势；地支 = 静态 / 隐秘 / 实力 / 根基 | 取象方向不同：干象看外显特征；支象看内在本质 |

**铁律 §Q.4-F-A**：任何单字取象**必须列出 ≥ 2 种可能含义**，再用上下文定位最终取象 —— 不允许"见甲就说是头"的机械单取。

### 铁律 Q.4-G · 整体取像三态（v7.0 新增 · GP-XF-05 高派《象法篇》第 2 课 §四晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 2 课 §四（"整体取像 / 孤阳孤阴包局夹局"）

#### 三种整体局象

| 局象 | 定义 | 典型含义 |
|---|---|---|
| **包局** | 年时包（大包）或月时包（夹包）—— 某类十神从外围包裹日主 | 官包局 = 男命常与官员打交道 / 女命被男人围绕 / 牢狱象；财包局 = 被财务事务包围 / 经营为主 |
| **夹局** | 两字夹中神 —— 日主或某星被两侧同类夹护 / 夹制 | 夹护 = 贵人左右逢源；夹制 = 受困进退两难 |
| **孤阳孤阴** | 独干无支（独阳）或独支无干（独阴）—— 纯阴或纯阳 | 孤阳 = 偏执 / 单亲 / 独行；孤阴 = 消极 / 被动 / 隐居 |

**铁律 §Q.4-G-A**：包局八字**看气势不看坐下** —— 气势大于单字局部判断。
**铁律 §Q.4-G-B**：判定包局后需同步确认"包的是什么十神"（官/财/印/食伤/比劫）→ 才能定性吉凶方向。

## §Q.4-F 干支原象六维取象法（v7.0 Phase 10 新增 · GP-LF-18 晋级）

> **来源**：`高派命理理论知识库/2018弟子班_理法篇_31页.md` 第 4 课 §例 2（本田宗一郎完整推演）
> **定位**：取象的"流程化模板"——对任何一个天干/地支，按以下 6 维逐层展开取象

| 维度 | 取象内容 | 示例（庚金） |
|---|---|---|
| ① **宫位** | 这个字在年/月/日/时哪一柱？代表什么六亲/身体/空间？| 庚在月干 = 父亲位/社会位 |
| ② **五行相生** | 这个字的五行生什么/克什么/被什么生/被什么克？| 金生水（生财）/ 金克木（制官）|
| ③ **状态（十二长生）** | 这个字在所坐地支是长生/沐浴/冠带/临官/帝旺/衰/病/死/墓/绝/胎/养？| 庚坐辰=养地（蓄势待发）|
| ④ **十神** | 这个字对日主来说是什么十神？| 庚对己日主=伤官（才华外泄）|
| ⑤ **干支原象** | 这个字本身像什么？（天干原始物象+地支原始物象） | 庚=刀剑/钢铁/道路/脊柱/大肠 |
| ⑥ **综合判定** | 前 5 维交叉 → 锁定最终取象 | 庚在月干+伤官+坐养地=在社会上展现刀笔才华（秘书/文字） |

**铁律 §Q.4-F-A**：6 维中 ≥ 3 维指向同一取象 → 可铁断；仅 1 维支撑 → 降为参考。
**铁律 §Q.4-F-B**：6 维逐层展开**不可跳步** —— 跳步取象容易出错（先定宫再看力，不可直接取象）。

## §Q.4-J 相合换像技法（v7.0 Phase 10 新增 · GP-XF-07 晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 3 课 §3
> **定位**：取象中"合"的象意变化规则

**核心规则**：
1. **相合双方可互用对方之象**：如甲己合 → 甲可取己之象（田/地/黄色），己也可取甲之象（栋梁/领袖）
2. **化合后不能反取**：如戊癸合化火 → 癸已变火，不再取水象（改取丙丁火象）
3. **合而不化**：仅换象，不变质——保持原五行属性但增加对方色彩

**铁律 §Q.4-J-A**：判断"换象还是化象"的标准 = 是否化合成功（看月令得否、强弱对比）。
**铁律 §Q.4-J-B**：化合成功后**严禁反取原象**——已变质的字不再代表原五行事物。

**与 §A.2.4 六合/天干五合的联动**：排盘层判"合化成否" → 本铁律决定"取象走哪条路"。

## §Q.4-K 三刑与自刑取像（v7.0 Phase 10 新增 · GP-XF-12 晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 5 课 §一
> **定位**：刑的象意层面——三刑/子卯刑/自刑各有不同取象方向

| 类别 | 组合 | 象意 |
|---|---|---|
| **寅巳申三刑** | 三字全见（或岁运凑齐） | 责难/损坏/官司/伤灾（火金交战+蛇虎猴互斗） |

| **四自刑** | 辰辰/酉酉/午午/亥亥 | 自损=自找麻烦/自残/抑郁/同类相争 |

**取象铁律**：
- 丑戌未三刑**见金多** → 六亲伤病（金=刀刃/手术/血光）
