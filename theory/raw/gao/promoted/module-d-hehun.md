# 模块 D · 合婚（双盘交互层）

> **角色**：合婚分析的"双盘交互层"。决定**两盘是否相配、配在何处、配到何级、何时同居 / 何时领证 / 何时风险**。
> **不做**：单盘排盘（→ 模块 A）、单盘格局（→ 模块 B）、单盘单年应期（→ 模块 C）、剖腹产择日（→ 模块 E）。
> **依赖**：必须先各自跑完模块 A 摘要节（双方四柱 + 胎元 / 命宫 / 身宫 + 旬空）；婚姻状态采集走主入口 §1.4；铁断前提走 §3.0。
>

### D.2.2 等级与判语对照

| 总分区间 | 等级 | 判语 |
|---|---|---|
| 90-100 | ★★★★★ 上等（命定型 / 同步型） | 强烈适配，建议在最近双窗口推进 |
| 80-89 | ★★★★ 上等 | 适配良好，按推荐路径稳步推进 |

## 【D.4 大运同步性识别（5-15 年视野）】

> 来源：合婚报告 第二部分『大运底色对比』+ 案例 002 第一卷『2.3 大运高度同步（罕见）』。

### D.4.1 四种典型同步模式

| # | 模式名称 | 命理判据 | 表现 | 强度 |
|---|---|---|---|---|
| 1 | **大运地支同序共振** ⭐ | 双方大运地支按同一节奏推进（如男 子→亥→戌→酉，女 同序） | 一生节奏同步，几乎不会"一方往前一方滞后" | ★★★★★ |
| 2 | **男偏财 × 女食神** | 乾命行偏财运 + 坤命行食神 / 食伤运 | 一个在寻找异性 + 一个想安定享受 → 目标互补 | ★★★★ |
| 3 | **比劫镜像** | 双方流年同时落入比劫 / 比肩 / 劫财 | 同时有"搭伴生活 / 合作 / 共担"意愿 | ★★★★（仅同居窗口） |
| 4 | **食伤驱动** | 一方食神 + 一方伤官（食伤混合）落在同一年 | 一方安定 + 一方推动 = 合力领证 | ★★★★（领证窗口专属） |

### D.4.2 大运地支同序共振（最强级）

```
触发条件（铁律 §D.4-A）：
  双方大运表前 5 步地支 ≥ 4 步同名 / 同冲 / 同序
  → 命定级一生同步（罕见，全人口 < 1%）

含义：
  - 不仅同居 / 领证窗口同步，连孩子出生 / 升迁 / 中年危机 / 退休都同节奏
  - 案例 002 验证：男 子运（亥子丑） → 亥运 → 戌运 → 酉运
                  女 子运（壬子）→ 亥运 → 戌运 → 酉运
                  → 4/5 同序，已通过 #CASE-002 第一卷 §2.3 应验
```

### D.4.3 男偏财 × 女食神（窗口型）

| 男方运 | 女方运 | 含义 | 持续时间 |
|---|---|---|---|
| 偏财（异性缘旺 / 投资意愿强） | 食神 / 食伤生财 | 男在寻找 + 女想安顿 → 互补 | 该 10 年大运段内 |
| 正财 | 食神 | 正缘落定 + 享受婚姻 | 同上 |
| 偏财 | 伤官 | 男异性多 + 女不耐烦 → 摩擦年（注意） | 同上 |

> 来源：合婚报告 第二部分一『大运底色对比』。

### D.4.4 比劫镜像（同居信号）

```
铁律 §D.4-B：双方流年天干 / 地支同时落入比劫 / 比肩 / 劫财
  → 同居窗口高概率（70-80%），但 ≠ 领证窗口

为什么不能领证？
  - 比劫主"非正式合作 / 平等搭伴"，缺乏官星（正式约束）的驱动力
  - 必须等到下一年食伤驱动 + 婚姻宫被冲合 才转为领证

案例锚点：合婚报告 2026 丙午年（双方比劫镜像）→ 同居 75-80% / 领证 25-30%
```

### D.4.5 食伤驱动（领证信号）

```
铁律 §D.4-C：一方食神 + 一方伤官（同年）+ 任一方婚姻宫被冲 / 合
  → 领证窗口高概率（65-70%）

为什么是组合？
  - 食神 = 想安定下来 / 主动求稳
  - 伤官 = 不满现状 / 推动改变
  - 食 + 伤 = 一拉一推 → 自然过渡到领证

案例锚点：合婚报告 2027 丁未年
  - 男：丁火比肩 + 未土食神 → "想安顿"
  - 女：丁火劫财 + 未土伤官 → "要正式确认"
  - 同时未冲男日支丑 = 婚姻宫被冲
  - 三件齐 → 领证窗 65-70% ✓
```

### D.4.7 与模块 C 应期层的交叉引用（v6.1.0 新增）

> 模块 D §D.4 是『静态识别层』（识别同步性级别），落到具体年份的应期触发权重交给模块 C。FEAT-003 在模块 C 已经把『大运地支同序共振』『官财同字呼应』『戌合卯空亡』三条规律展开成应期层规则，**双方深度推演时必须三处合看**：

## 【D.5 官财同字呼应（互为对方所求）】

> 来源：案例 002 第一卷 §2.2『官财同字（互为对方所求）』+ 第二卷论断四『男卯戌合官归夫宫』。
> 这是合婚报告未充分量化、由案例 002 实证后晋级的新维度（10 维矩阵第 4 维）。

### D.5.1 概念

```
官财同字 = 男命妻星（财星）的字 = 女命夫星（官杀）的字
        = 在同一个干支字内，男看到妻 / 女看到夫
        → 命理底层"互为对方所求"

关键：必须是"字"层面的同字，不是"五行"层面的同行
     例：男妻星癸 = 女夫星癸（同字 ★★★★★）
         男妻星癸 + 女夫星壬（同行不同字 ★★★）
```

### D.5.2 强度分级表

| # | 类型 | 强度 | 含义 |
|---|---|---|---|
| 1 | 男妻星字 = 女夫星字 + 同柱位 | ★★★★★ | 互为对方所求最强级，命定锁定 |
| 2 | 男妻星字 = 女夫星字 + 异柱位 | ★★★★ | 命定锁定，柱位差不影响本质 |
| 3 | 男妻星字 = 女夫星五行（不同字） | ★★★ | 方向相同，强度递减 |
| 4 | 男妻星 = 女命所合之字（合住） | ★★★ | 妻被合 + 夫被合 = 双向锁定（如丁壬合） |
| 5 | 双方各自妻 / 夫星均落在对方喜用 | ★★ | 加分项 |

### D.5.3 案例 002 实证（论断四完整推理链）

```
男命戊戌（戌中藏丁戊辛） · 妻星=癸（正财，年干透）
女命丁未（未中藏丁己乙） · 夫星=壬 / 癸（官杀混杂三透）

呼应铁证：
  ① 男年干癸 = 女月干癸（同字同位 ★★★★★）
  ② 男戊日合女月干癸（戊癸合化火）→ 男主动合女七杀 → 制服女命七杀压力
  ③ 女丁日双合男年干癸 / 时干壬（一丁合双壬癸）→ 女命对男方有持续吸附力
  ④ 男时支寅 + 女时支寅（伏吟）→ 时柱共振 → 子嗣同位

→ "妻在他命里有位、夫在她命里有位"双向闭环
→ 维 4 评 11/12（仅 1 分扣给寅伏吟之凶兆）
```

### D.5.4 与 §D.6 桃花归宫的关系

```
官财同字 = 命定层"互为所求"（先天根基）
桃花归宫 = 兑现层"桃花来都归家"（后天保障）

两者叠加 = 即使一方桃花重，仍因"对方在我命里有位 + 桃花归我夫宫" → 双重锁定不破婚
（详见 §D.6 案例 002 四重保障机制）
```

### D.11.1 与合婚分析的本质区别

| **输出** | 同居窗 / 领证窗 / 风险窗 | 核心协同期 / 巅峰期 / 延展期 + 边界管理方案 |
| **伦理** | 鼓励推进婚姻 | **中立客观 + 边界管理优先** |

### D.11.2 五重磁场识别法（R-004-COMPAT-NEW-01）

> 用于评估业务搭档双盘之间的命理亲密度/协同度。五重均命中 = 最高级搭档缘。

| # | 磁场层 | 判据 | 含义 | 强度 |
|---|---|---|---|---|
| ① | **日干互合** | 双方日干构成天干五合（如戊癸 / 甲己 / 丙辛 / 乙庚 / 丁壬） | 天然有引力·命主层面的互相吸引 | ★★★★★ |
| ② | **妻/夫星实人化** | 一方妻星/夫星字 = 另一方日干字 | 对方在我命中"有位"·命格实人化 | ★★★★★ |
| ③ | **大运同序共振** | 双方大运地支 ≥ 5 步同名同序 | 人生节奏高度同步 | ★★★★★+ |
| ④ | **三垣呼应** | 一方三垣干支 = 另一方关键柱位干支 | 灵魂层 / 事业层深度连接 | ★★★★ |
| ⑤ | **桃花字呼应** | 双方命局中桃花字（咸池 / 红鸾）位置互相共振 | 亲密磁场第 5 层·社交吸引力共振 | ★★★★★ |

**磁场层级评分**：
| 命中层数 | 评级 | 边界管理级别 |
|---|---|---|
| 5/5 | 命定级搭档缘（★★★★★+） | **最高级**边界管理 |
| 4/5 | 强搭档缘（★★★★★） | 高级边界管理 |
| 3/5 | 中强搭档缘（★★★★） | 中级边界管理 |
| 2/5 | 普通搭档缘（★★★） | 常规职业边界 |
| ≤ 1/5 | 一般工作关系 | 无需特别管理 |

### D.11.3 "火护神"机制识别（R-004-COMPAT-NEW-02）

> 当双盘日干互合可化（如戊癸合化火），且化神恰好保护一方日主时，构成"火护神"机制——
> 表现为：**没有对方 = 一方在某大运中"福不堪受"/ 风险极高；有对方 = 安全跃升**。

```
识别条件：
1. 双方日干互合且有化神（如戊癸合化火 / 甲己合化土 / 丙辛合化水）
2. 化神五行恰好能生/护一方日主（如火生戊土 → 女方癸为男方"安全阀"）
3. 一方某大运中日主面临"多克攻身"风险
4. 化神流通链形成保护（如金→水→火→土 完整流通）

命中 → 标记"火护神机制成立" → 在配合时长评估中该大运段评为"刚需期"
```

**案例 004 实证**：
```
女方庚正印（金）→ 生癸日主（水）→ 戊癸合化火 → 火生男方戊日主（土）
男方辛亥运（2031-2040）水气大旺 → 戊土被旺水冲倒风险
→ 有女方 = 戊癸合化火 = 安全跃升
→ 该运段为"男方刚需期·配合价值★★★★★"
```

### D.11.4 大运同序共振升级版（R-004-COMPAT-NEW-03 · 7 步全程）

> 在 §D.4.2 已有的"大运地支同序共振"（4 步）基础上，本案发现 **7 步全程同序共振**——
> 双方月柱地支相同 + 大运排列方向相同 = 起点同 + 方向同 = 序列完全一致。

**触发条件（7 步全程）**：
```
双方大运表前 7 步地支 ≥ 6 步同名同序
→ 极致罕见（全人口 ≈ 0.1%）
→ 不仅业务节奏同步，连中年危机 / 退休 / 人生转折都几乎同步
```

**识别要点**：
- 检查双方月柱地支是否相同（起点同）
- 检查双方大运排列方向（顺排 vs 逆排）是否一致
- 逐步比对地支序列
- 注意年龄差 / 起运差导致的"公历年份错位"（剧本同序 + 当下错位）

### D.11.5 配合时长综合评估法（R-004-COMPAT-NEW-04）

> 评估双方实质协同的时间跨度，分四段：

| 阶段 | 定义 | 判断依据 | 典型时长 |
|---|---|---|---|
| **核心协同期** | 同平台·同条线·日常协作 | 双方同走某地支大运 + 火护神/五行互补活跃 | 8-12 年 |
| **巅峰共赢期** | 业绩共生·产出最大化 | 火护神刚需段 + 流年共振叠加 | 3-5 年 |
| **延展协同期** | 平台/角色调整后仍有合作 | 大运切换但余热在 + 三垣余力 | 4-6 年 |
| **老友顾问期** | 非日常但保持咨询关系 | 双方均进入下一大运段 | 8-12 年 |

**输出模板**：

```markdown
|---|---|---|---|
| 核心协同期 | 20XX-20XX（X年） | ★★★★★ | [描述] |
| 巅峰共赢期 | 20XX-20XX（X年） | ★★★★★ | [描述] |

总实质日常协同 ≈ XX年 + 顾问期 ≈ XX年 = 总跨度 ≈ XX年
```

### D.11.6 边界管理铁律（双方已婚搭档必输出）

> 磁场评级 ≥ ★★★★ 时，必须输出边界管理方案。

**8 条边界管理铁律**（命定存在型）：
1. 公务往来必走正式流程
2. 重要合作必有第三方在场
3. 私下接触最小化 / 场合公开化
4. 业绩归团队不强调个人协同
5. 对外口径标准化
6. 避免特定时空场景（单独晚餐 / 出差 / 非工作时段）
7. 关键年份特别警觉（标注具体流年干支）
8. 配偶知情机制（适度透明）

**边界管理强度与磁场评级的正相关**：
```

磁场 ★★★★★  → 8 条输出 + 关键年份概要标注
磁场 ★★★★   → 输出 1-6 条 + 一般提醒
磁场 ★★★     → 输出 1-4 条概要

```

### D.11.7 关键年份双盘叠加推演

**共振判定规则**：

### D.11.8 正官能量投射洞察（深层动力分析 · 可选输出）

> 当五重磁场中"日干互合 + 妻星/夫星实人化"同时成立时，可能存在"正官/正财能量投射"现象——
> 命主对搭档的吸引力不只是桃花，而是**自己命格中某十神能量在搭档身上看到了极致兑现**。

```
识别条件：
  命主命格中某十神（如正官）仅余气 / 弱根
  + 现实配偶无法承接该十神能量
  + 搭档日主恰好 = 该十神的极致兑现
  → "正官能量投射"成立

含义：命主对搭档的持续吸引力 = 十神能量投射 + 桃花磁场 + 日干合多重叠加
     → 这是最需要边界管理的命理底层
```
