## 【B.1 体用辨析（先做这一步，必须一句话定）】

### B.1.1 五种主宾关系

| 关系 | 触发判据 | 取象方向（社会角色） |
|---|---|---|
| **我去取财** | 日主主动克财 / 财在他宫不贴身 | 主动型谋利、销售、投资、创业 |
| **财来寻我** | 财星贴日 / 财在我宫 / 财得令 | 被动型受益、继承、入职大平台 |
| **我被财困** | 财多身弱 / 财坏印 | 为钱所累、健康受损、过劳 |
| **我护他星** | 印护身、食制杀、官护财 | 庇护型职业（医、师、保、律） |
| **他星攻我** | 七杀无制 / 伤官见官 / 群比夺财 | 受外部压制、官非、被动应对 |

### B.1.2 体用一句话模板

```
体用判定：日主 [日干][月令] 生于 [月份]，[得令/失令]，[体星] 为体，[用星] 为用
        ——属"[五种之一]"格局。
```

> 这一句话**是后续所有判断的总锚**。模糊或多义会导致 B.2/B.3 全部失真。

### B.1.3 得失对称定义（v7.0 Phase 10 新增 · GP-LF-16 晋级）

> **来源**：`高派命理理论知识库/2018弟子班_理法篇_31页.md` 第 4 课 §1（"看任何一个八字，都是在看'我'得到了什么"）

**核心概念**：
- **得** = 我用什么工具（十神）→ 通过什么途径（合/生/制/墓/送/伏吟）→ 获取了他宫的什么（财/官/印/食）
- **失** = 他用什么工具 → 通过什么途径 → 获取/克损了我宫的什么

**铁律 §B.1.3-A**：工具/对象/途径三者在一个八字中往往不唯一——多条路线并存时，以"做功最重 + 路线最短"的为主线。

### B.1.4 路线优劣三段论（v7.0 Phase 10 新增 · GP-LF-17 晋级）

> **来源**：第 4 课 §1（"破坏路线有三种方法…反过也有是同理"）

| 路线环节 | 被破坏 | 被优化 |
|---|---|---|
| **源头** | 用神根被伤（如印被冲、财被合走） | 用神根强且被生（印得官生） |
| **中间** | 传递环节被截断（如食伤被枭夺） | 传递顺畅无阻（食伤生财链完整） |
| **结果** | 结果被夺（如比劫夺财、伤官见官） | 结果入库/入我宫（财入库、官合我） |

**应用**：在做 §B.4 意向识别后，接着判"这条路线通不通、哪段被破坏了"→ 决定格局等级升降。
**与模块 C 联动**：大运/流年 = 修复或破坏路线的外力（§C.2 动态分析的底层逻辑）。

### B.1.5 阳包阴/阴包阳格局识别（v7.0 Phase 10 新增 · GP-LF-22 晋级）

> **来源**：第 5 课 §例（女命壬辰丙午癸酉己未案例推演）
> **与 §D.6 桃花归宫 / §B.8 不破婚的关系**：本规律为"婚姻多变信号"，与 §D.6"4件齐=不破婚"是同一格局轴的**两端**。

**识别条件**（三特征齐 = 铁断）：
1. 原局天干分两组阴阳对立阵营（如壬癸阴 vs 丙己阳），且**互相包夹穿插**
2. 日主被一方阵营包围，但自身阵营也在包围另一方
3. 双阳干或双阴干争位，**谁也不服谁**

**铁断含义**：
- 婚姻多变 / 对象不稳定 / 感情拉锯（因两阵营无人退让）
- **区别于阴差阳错日**：阴差阳错 = 单一日柱信号（可降级）；阳包阴 = 四柱全局结构（不可降级）

**与 §D.6 的分流铁律**：
- 阳包阴 + 配偶宫被冲 + 无保护 = ★★★★★ 婚姻多变信号
- 4件齐（魁罡+合归宫+吸附力+体制约束）= ★★★★★ 不破婚
- 两者可并存于同一命局 → 需逐项对比哪方更强

## 【B.2 格局判定（先看月令，再看透干）】

### §B.2.0 月令理论基底（v7.0 Phase 10 新增 · GP-LF-01~06 晋级）

> **来源**：`高派命理理论知识库/2018弟子班_理法篇_31页.md` 第 1 课全文
> **角色**：取格的"哲学层"——为什么首看月令？为什么以财官为主？清浊从何判起？

#### 一、月令为提纲（GP-LF-01）

> "论命首重月令…月令乃提纲…天地主气的发源…命之富贵贫贱之根源"

**三层定义**：
- 月令 = **提纲**（统领全局的主气发源地）
- 月令 = **本元**（先天一气之根）
- 月令 = **富贵贫贱的根源**（格局成败皆从月令起）

#### 二、六大正格铁律（GP-LF-02）

> "月令可用之物指六样东西…就是月令除了比劫之外，其余的东西均可以取作用神"

| 可定格之十神 | 说明 |
|---|---|
| 财（正/偏） | 以财定格 |
| 官 | 以官定格 |
| 杀 | 以杀定格 |
| 印（正/偏） | 以印定格 |
| 食 | 以食定格 |
| 伤 | 以伤定格 |
| **比劫** | ❌ 不可定格 → 退看他透 |

**与现有 §B.2.1 八格的关系**：八格 = 六大正格的"正偏拆分"（财→正财/偏财、印→正印/偏印），本质等价。

#### 三、黄大陆三论（GP-LF-03）

> "为什么要从月令取用神呢…父母宫带有先天遗传信息"

1. 月令 = **父母宫** = 先天遗传信息载体
2. 看命先看"父母给你遗传了什么"（月令藏干 = 先天工具箱）
3. 再看"家里有什么东西可以为你所用"（月令本气 = 可用资源）

#### 四、盲派打开八字铁律（GP-LF-04）

> "盲人讲打开八字，专找财官…财官二字囊括了人生的事业与婚姻"

```
盲派开八字 = 专找财官
  财 = 物质 + 男命之妻 + 女命之父
  官 = 名誉地位 + 女命之夫 + 男命之子
  ─────────────────────────────
  财官二字囊括人生事业与婚姻全貌
```

**与 §B.4 意向识别的关系**：意向识别的"取财型 / 求名型"即此铁律的操作化展开。

#### 五、先天本元诀（GP-LF-05）

> "先天一气是本元，要看清浊分自然"

```
盲派秘诀《先天本元诀》：
  先天一气是本元（月令 = 先天一气）
  要看清浊分自然（格局成败从清浊看）
```

**『先天一气』= 月令**。年月为先天（社会给的），日时为后天（自己活的）。

#### 六、清纯浊杂判格局（GP-LF-06）

> "清纯代表用神的能量级别…一清到底有精神，管取生平富贵真"

| 格局特征 | 含义 | 等级影响 |
|---|---|---|
| **清纯** | 用神唯一、不混杂、得力 | 升 1~2 级 |
| **浊杂** | 用神多个互争、或忌神混入 | 降 1~2 级 |

**铁律**：独财值千金，独官掌万民；多财多官如多和尚抬水反无水。
**与 §B.3.1 五等级的关系**：★★★★★ 顶级 = 必须清纯；★★★ 中等 = 清浊参半。

- [ ] 已评估清纯/浊杂程度对等级的影响
- [ ] 来源已标注（v7.0 P10 / GP-LF-01~06 / 高派《理法篇》第 1 课）

### B.2.1 八格速判流程

```
Step 1: 看月令本气透不透干
        透 → 直接以本气定格（如"月令巳火透丙 → 食神格"）
        不透 → 看月令藏干哪个透
                透中气 → 以中气定格
                透余气 → 以余气定格
                都不透 → "杂气格"，看用神在何处

Step 2: 看格成 / 格败
        成格：用神得力、护用神到位
        败格：用神被冲克 / 用神入墓不开 / 用神远离
        中和：用神在 + 但有伤损

Step 3: 看格高 / 格低
        高：用神得令 + 透干 + 不被破 + 配印护
        中：用神得地不得令 / 透干但有微损
        低：用神不得地 + 不透 / 用神被多重克破
```

### B.2.2 月令优先取格硬约束（v6.1.7 新增 · R-004-REVISE-07 · 协议级）

> **来源**：案例 004（甲子丁卯癸卯庚申 · 坤命 1984）取格修正教训。
> **触发场景**：初始格局判定时误把强透时干（如庚正印坐申禄）当主格，忽略月令本气（卯木食神当令）。
> **状态**：协议级（从案例 004 教训直接升级为硬约束 · 不需 0.6/2.4 累积）。

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §B.2-A（月令优先）：                                                │
│  取格首看月令本气 —— 月令本气当令且不被合化空亡时 → 主格必定为月令本气      │
│  其他透干（无论多强）仅能为辅格，不得越位为主格                            │
│                                                                          │
│  铁律 §B.2-B（辅格定位）：                                                │
│  时干透强星（如正印坐禄/七杀坐刃）→ 标记为"辅格/配格"                     │
│  输出格式：「XX 格（月令主格）+ YY 配印/配杀（辅格）」                     │
│                                                                          │
│  铁律 §B.2-C（例外条款）：                                                │
│  月令本气被合化（如卯戌合化火·化神成立）→ 主格随化神                       │
│  月令为比劫（不可取格）→ 退而看透干取格（正常八格规则）                    │
│  月令本气落空亡且无填冲 → 降权看透干取格（罕见）                          │
└──────────────────────────────────────────────────────────────────────────┘
```

**判读流程图**：
```
月令本气是什么十神？
├── 比劫 → 月令不可取格 → 看透干哪个能取格（正常规则）
├── 非比劫 + 本气被合化 → 看化神 → 以化神定主格
├── 非比劫 + 本气空亡无填 → 降权 → 看透干定格（罕见例外）
└── 非比劫 + 本气当令正常 → ★ 主格 = 月令本气（铁定）
                              └── 其余透干（无论多强）仅为辅格
```

**案例 004 教训示范**：
```
命主：甲子 丁卯 癸卯 庚申
月令卯木 → 本气乙（食神）当令 → 主格 = 食神格 ✓

初始误判：庚正印透时干坐申禄（极强） → 误判为"正印格"
纠正：庚虽极强但不在月令位 → 仅为辅格
正确输出：「食神格（月令主格）+ 正印配印（辅格）」= 食神配印格（中上等）
```

**口诀**：
```
取格首看月令气，本气当令即主格
旁透虽强仅辅佐，越位取格必失真
比劫当令方例外，合化空亡三条退
```

**自检（格局判定时必跑）**：
- [ ] 已先确认月令本气是什么十神
- [ ] 月令本气非比劫 + 无合化 + 不空亡 → 已定为主格
- [ ] 其他透干已正确标记为辅格/配格
- [ ] 未出现"时干强透 = 主格"的越位判定
- [ ] 来源已标注（v6.1.7 / R-004-REVISE-07 / 案例 004 取格修正教训）

## 【B.3 富贵贫贱级别（不打数字分，只给等级 + 推导链）】

### B.3.1 五等级框架

| 级别 | 标记 | 特征关键词 |
|---|---|---|
| **顶级（一品）** | ★★★★★ | 用神得令 + 透干 + 三合/六合护 + 大运至少两步顺、忌神被合化 |
| **上等** | ★★★★ | 用神得令 + 透干，护用神在但不全；大运两步顺即兑现 |
| **中等** | ★★★ | 格成但用神有损 / 喜忌交战；靠大运补救可达上等 |
| **中下** | ★★ | 格半成 / 用神不透 + 多见破损；靠运气也只是平稳 |
| **贫贱** | ★ | 格破 + 主要五行皆受伤 / 反背 + 大运全程不顺 |

> ⚠ **此处的"贫贱"不是道德评价，仅指**"该命格在不借助外力的情况下，物质 / 社会地位的天花板很低"**。
> 时代与个人努力可改变实际兑现（→ 配合模块 C 的"时代修正系数"）。

### B.3.2 富 / 贵 / 寿 / 名 四向分判

不是所有命都"全面富贵"。一个上等命可能"贵而不富"或"富而不贵"。

| 维度 | 主要看 | 上等条件 |
|---|---|---|
| **富** | 财星 + 食伤 | 财星得地 + 食伤生财 + 财不被夺 |
| **贵** | 官星 + 印 | 官星清纯（不混杀）+ 印护身 + 官星不被冲 |
| **寿** | 日主 + 印 | 日主气足 + 印有根 + 没有大冲克日主 |
| **名** | 食伤 + 官 / 印 | 食伤透干 + 不被夺 + 名声星（驿马 / 文昌）相配 |

## 【B.4 意向识别（这个命"想做什么"）】

### B.4.1 主导意向四类

| 意向类 | 命局信号 | 取象画像 |
|---|---|---|
| **取财型** | 财星透干 + 食伤生财 | 商人、自雇、销售、投资 |
| **求名型** | 官印相生 + 食伤名声 | 学者、技术专家、官员、艺人 |
| **守护型** | 印重 + 食伤制杀 | 医、教、律、保、家族守业 |
| **冲撞型** | 杀刃 / 伤官旺无制 | 军警、运动员、改革派、极端创业 |

> 意向≠职业，而是"驱动模式"。同一个意向在不同时代会落到不同行业。

### B.4.2 意向与做功的对照（与模块 A 联动）

```
意向：取财型
    ↓
做功核心应是："食伤 → 财" 这条链路通畅
    ↓
若模块 A 显示"食伤被穿"或"财被合化" → 意向受阻 → 等级下调一级
```

### B.4.3 我宫获取财官 6 种方式（v6.1.3 新增 · 第二轴 · 来源高派《理法篇》第 3 课 §5 · GP-LF-14 ⭐⭐）

> ⚠ **触发**：高派《理法篇》提取报告 GP-LF-14 ⭐⭐（铁律级 · 候选库 0.6 / 2.4 → v6.1.3 阶段 5 晋级 → 写入本节）。
> 本节是意向识别的**第二轴 · 获取方式**，与 §B.4.1 的『主导意向四类』（取财 / 求名 / 守护 / 冲撞）**正交叠加** —— 意向四类回答『命主想干什么』，获取 6 种方式回答『命主用什么手段拿到财官』。两轴交叉得出**真正的画像**。
> 来源：`高派命理理论知识库/2018弟子班_理法篇_31页.md` 第 3 课 §5『我宫是怎么得到财和官？归纳起有 6 点』。

#### 一、6 种获取方式对照表

> 我宫 = 日时柱（日主自己 / 子嗣晚景） · 他宫 = 年月柱（祖辈 / 父母社会）。
> 获取方式 = 财官星从他宫到我宫的**作用路径**。

| # | 获取方式 | 命理判据 | 难度 | 获取性质 | 典型画像 |
|---|---|---|---|---|---|

### B.4.4 财气通门户三层铁律（v6.1.5 新增 · 来源高派《财运篇》第 2 课 · GP-CY-04 ⭐⭐）

> ⚠ **触发**：高派《财运篇》提取报告 GP-CY-04 ⭐⭐（候选库 0.6 / 2.4 → v6.1.5 阶段 5 晋级 → 写入本节）。
> 本节是 §B.4.3 获取 6 种方式的**上位判定** —— 在判定获取方式之前先看"财星与命主有无通门户"。
> 来源：`高派命理理论知识库/2018弟子班_财运篇_43页.md` 第 2 课（"本人认为财气通门户有三层含义"）。

#### 一、三层递进判定

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §B.4.4-A：财气通门户必须三层递进满足（缺一层降一级）                   │
│                                                                          │
│  第一层 · 联系：财星与我宫有"途径/渠道/办法"联系                            │
│         判据 = 年月财星与日时有合/冲/生/透/入库 任一联系                    │
│         无联系 = 不通门户 = 穷命第一标志                                   │
│                                                                          │
│  第二层 · 有情：财星组合对命主"有情"（非恶意、能为我所用）                   │
│         判据 = 联系路径为喜用组合（非忌神相克/非劫夺/非穿害破坏）           │
│         无情 = 有联系但伤我 = 财来克身 / 为他人送嫁衣                       │
│                                                                          │
│  第三层 · 旺相：财星旺相（值月令 / 受生有力）                               │
│         判据 = 财星得令或受食伤生 / 在库有根                               │
│         衰弱 = 三层满足两层 = 中小财命                                     │
│         旺相 = 三层全满 = 大财命                                           │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 二、三层与财富级别的对应

| 通门户程度 | 满足层数 | 财富级别上限 |
|---|---|---|
| **三层全通**（联系 + 有情 + 旺相） | 3/3 | 大富 / 巨富（视格局高低） |
| **两层通**（联系 + 有情，旺相不足） | 2/3 | 中富 / 小富 |
| **仅一层**（有联系但无情或衰弱） | 1/3 | 温饱 / 偏财为主 |
| **不通门户**（财与我宫无任何联系） | 0/3 | 穷命（§B.4.3 六种获取方式全不走通） |

#### 三、与 §B.4.3 获取 6 种方式的叠加

```
判读顺序（v6.1.5 升级）：
Step 1：先判三层通门户程度（本节 §B.4.4）→ 定"能不能有财"
Step 2：再判获取方式（§B.4.3 六种）→ 定"怎么拿到财"
Step 3：两轴交叉 → 如"三层全通 + 合型获取 = 稳定大财命"
```

### B.4.7 财富四级定位法（v6.1.7 新增 · GP-CY-09 ⭐⭐ · 来源高派《财运篇》第 5 课 §一）

> **来源**：`高派命理理论知识库/2018弟子班_财运篇_43页.md` 第 5 课 §一（"财富四级定位：财星<食伤<官杀<印/库"）。
> **状态**：v6.1.7-batch5 候选规则（0.6/2.4 🟡）。
> **同步源**：`盲派命理AI分析师提示词_v2.1.md` §B.4.7。

#### 一、财富四级判定表

完成财星定位（§B.4.6）后，必须进一步判定财富"级别"：

| 级别 | 核心条件 | 典型格局特征 | 年收入参考 |
|---|---|---|---|
| **Ⅰ 巨富** | 财星成局+带库+库不破+食伤/官杀为渠道 | 财局三合成化/财库双库叠加/身财两停带权 | 千万级以上 |
| **Ⅱ 大富** | 财星旺+有库+渠道通畅（合/生/送） | 独财值千金/食伤生财带库/官星护财 | 百万~千万级 |
| **Ⅲ 中富** | 财星得气+有根+渠道存在但有瑕疵 | 财有根但被合绊/财透但库受冲/身稍弱于财 | 数十万级 |
| **Ⅳ 小富** | 财星弱/无库/渠道不全 | 财虚浮无根/比劫夺财/财库全破 | 温饱~小康 |

#### 二、四级判定 5 步法

```
Step 1：财星是否成局？（三合财局/方局/透干成势）→ 是 = 起点 Ⅰ 级
Step 2：是否带财库？库是否完好？→ 有库不破 = 维持/升级；无库或库破 = 降级
Step 3：渠道是否通畅？（§B.4.3 六种获取方式至少 1 种成立）→ 通畅 = 维持；不通 = 降级
Step 4：身财力量对比 → 身财两停 = 维持；身弱财旺 = 降级（富屋贫人）
Step 5：大运是否支持？→ 有财运配合 = 维持；无财运 = 空有格局
```

```

```

### B.4.29 正偏财七重区别（v7.0 P11 · GP-CS-14 ⭐ · 来源《财运事业断法》第 5 课）

> **适用场景**：定位财星性质后判断命主求财风格、行业匹配、风险偏好。

| # | 区别维度 | 正财 | 偏财 |
|---|---|---|---|
| ① | 来源 | 正常工作薪资 | 兼职/副业/外快 |
| ② | 行业 | 稳定行业（教师/公务员/银行）| 边缘/新兴（股票/博彩/地产投机）|
| ③ | 数量 | 小而稳定 | 大而波动 |
| ④ | 风险 | 无/低风险 | 高风险 |
| ⑤ | 连贯性 | 周期性持续 | 偶得/非规律 |
| ⑥ | 合法性 | 合法收入 | 可能涉灰色 |
| ⑦ | 性质 | 劳动所得 | 脑力/资本/运气 |

### B.4.30 风险求财四特征（v7.0 P11 · GP-CS-17 ⭐ · 来源《财运事业断法》第 14 课）

> **适用场景**：判断命主是否适合偏业/风险投资/赌博型求财。

**风险求财四大格局特征**：

| # | 格局特征 | 本质 | 典型行业 |
|---|---|---|---|
| ① | 劫财制财格 | 以暴力手段取财（比劫为工具） | 期货/赌博/高风险投机 |
| ② | 杀制比劫格 | 受外力驱使竞争取财 | 军人经商/黑社会/高压销售 |
| ③ | 财旺身弱格 | 财多反受其累（贪多嚼不烂） | 负债经营/过度杠杆 |
| ④ | 财星破印格 | 为财毁名声/损健康 | 腐败/色情/毒品 |

## 【B.6 学历层次细分制（v6.0.2 新增 · v6.0.4 升级为七级）】

> 此前模型仅区分"本科 vs 硕博"，粒度过粗。
> v6.0.2 起按六级细分（来自 #CASE-001 REVISE-003）；
> **v6.0.4 起升级为七级（来自 #CASE-003 REVISE-002）**——把原 ★★★★★ 顶级档拆分为"清北/C9"与"普通 985"两档。

### B.6.1 七级学历框架（v6.0.4）

| 层次 | 标记 | 基础条件 |
|---|---|---|
| **清北 / C9** | ★★★★★+ | 印透 ≥ 2 位且夹日主 + 大运含三合拱印局 + 食伤暗藏不显 + 多贵人临命（天乙+太极+词馆 ≥ 3 项）+ 时柱有金舆/月德合 |
| **普通 985 / 顶校** | ★★★★★ | 印透干得令 + 文昌不空（或印强护卫 + 文昌空亡可豁免）+ 食神制伤 + 月令为印或官 |
| **211 / 重点 / 双一流学科** | ★★★★ | 印透干 + 文昌不空 + 杀印相生（任一减分项 → 降一级） |
| **一本 / 普通本科批** | ★★★ | 印透或印得令（二选一）+ 文昌相对清纯 |
| **二本** | ★★ | 印藏不透 + 文昌带瑕（空亡 / 被穿）但仍有用 |
| **三本 / 民办** | ★ | 印藏不透 + 文昌空亡或被合 + 杀轻伤重（学业靠"考"上不去靠"复读"撑） |
| **专科 / 中专** | ◯ | 文昌全失 + 印星彻底受伤 + 多次复读仍勉强 |

### B.6.2 减分项扣减规则

每出现一项 → 在基础层次上**下调一级**：

- 文昌空亡 + **印弱无护**（v6.0.4 新增前置条件 —— 印强护卫时不扣，详见 R-002-REVISE-001）
- 杀轻伤重（合杀失衡）
- 十恶大败 / 阴差阳错（古法减分）
- 印星不透
- 比劫夺食（食伤路被堵）
- 月令为忌神且不被合化

### B.6.2.1 文昌空亡分流铁律（v6.0.4 新增 · 来自 R-002-REVISE-001）

> ⚠ **不允许单独使用文昌空亡作铁断**。必须先评估"印星护卫强度"再决定是否扣分：

| 印星护卫强度 | 文昌空亡含义 | 学业兑现 |
|---|---|---|
| **印强护卫**（印透 ≥ 2 位 + 食伤暗藏 + 多贵人）| **学问超脱型** —— 不靠死读书而靠悟性 | **不扣分**，甚至可加 0.5 档 |
| **印弱无护**（印藏不透 + 食伤透显 + 杀轻伤重）| **学业波折型** —— 文昌真空亡 | **下调一级** |

**典型对照案例**：
- #CASE-001（印藏不透 + 食伤双透）→ 文昌空亡兑现为高考失利 + 三本民办（必败论）
- #CASE-003（双印透干 + 食伤暗藏）→ 文昌空亡反而 701 分上 C9 浙大（超脱论）

### B.6.3 高考兑现年份判定

> 仅"方向"判断，年份 ±1 弹性。和模块 C 协同。

- **顺利上岸年**：印透流年 + 文昌得用 + 大运无重大冲克
- **失利复读年**：印星受冲 / 文昌被穿 / 流年透出忌神（如 #CASE-001 的 2004 甲申，甲为正官但申冲寅破印根）
- **典型组合**："失利年 + 1 = 复读上岸年"（#CASE-001：2004 → 2005）
- **超常发挥年**（v6.0.4 新增 · 来自 R-002-ADD-02）：**食神/伤官透干 + 日主墓库填实 + 大运劫财帮身**
  - 例：#CASE-003 2018 戊戌年（戊食神透 + 戌为丙日主火墓 + 大运丁亥丁火劫帮 + 亥卯未拱印）→ 701 分上浙大

### B.6.4 顶级学霸隐形结构识别（v6.0.4 新增 · 来自 R-002-ADD-01）

> 此结构最容易被低估为"印虚透"，必须显式识别：

**触发条件**（满足 ≥ 4 项即视为隐形顶级学霸）：
1. 印透干 ≥ 2 位且**夹日主**（如月干 + 时干同为印星）
2. 大运含三合拱印局（亥卯未/寅午戌/巳酉丑/申子辰之三）
3. 食伤**全部暗藏不透**（聪明内敛型）
4. 多贵人临命（天乙 + 太极 + 词馆 + 德秀 ≥ 3 项）
5. 时柱含金舆 / 月德合 / 词馆
6. 日主在月令虽弱但有印护，构成"杀印相生"或"印护身弱"

**铁律**：满足 ≥ 4 项时，**学历底线 ★★★★（211/双一流）**；满足 ≥ 5 项时，**学历底线 ★★★★★（顶级 985）**。

## 【B.7 动态格局修正（v6.0.4 新增）】

### B.7.1 贪财坏印的劫财大运化解（来自 R-002-ADD-04）

**原规则**：年干/月干见"财合印"（如庚乙合）→ 贪财坏印 → 学业/官印受阻 → 扣分

**修正规则**：当大运透出**劫财/比肩**时，劫财克财 → **保护印星不被合走**：
- 例：#CASE-003 命局年干庚合月干乙（贪财坏印），但 12-22 岁行**丁亥大运**，丁火克庚金 → 庚被劫财制 → 月干乙印被释放 → 高考期间印星护卫到位

**应用边界**：
- ✅ 仅适用于**劫财/比肩透干**的大运
- ❌ 不适用于劫财仅藏支的大运
- ❌ 不适用于"印合财"（印主动合财而非财来合印）的情形

### B.8.1 概念

```
结构性桃花 ≠ 偶发桃花
  - 偶发桃花：流年带桃花 / 大运短暂触发的异性缘
  - 结构性桃花：原局月令咸池当令 + 月干临官 → 一生伴随的"职务型 / 社交型"桃花

结构性桃花不破婚的本质：
  - 桃花在结构层"必然存在"，但被另一组结构（合化 / 魁罡 / 合身）"必然回流"
  - 双向闭环 = 桃花来去自如，但不能落地扎根
```

```
┌──────────────────────────────────────────────────────────────────────────┐

│  铁律 §B.8-B：缺任一件 → 不能用本节，按一般桃花信号判（可能多婚 / 可能短暂）  │
│  铁律 §B.8-C：本节仅做『结构识别』；[已婚 / 未婚态] 分流与具体闭环推演 → §D.6 │
└──────────────────────────────────────────────────────────────────────────┘
```

### B.8.7 夫宫/夫星/十神三分判自检铁律（v6.1.7 新增 · R-004-REVISE-09 · 协议级）

> **来源**：案例 004（甲子丁卯癸卯庚申 · 坤命 1984）夫宫/夫星混淆教训。
> **触发场景**：坤命日支卯木（食神），误将卯木当"夫星"看待。实际：卯木只是"坐在夫宫位置上的食神"（才华工具），而真正的夫星是戊正官（仅在时支申中余气藏干）。
> **状态**：协议级（直升硬约束 · 案例 004 首例发现 + 7 条精度教训中排名 #3 严重度 ★★★★）。

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §B.8.7-A：位置 ≠ 角色 ≠ 配偶                                        │
│                                                                          │
│  夫宫（位置） = 日支（宫位定义）→ 只表示"这个位置是配偶住的地方"           │
│  夫星（角色） = 坤命看官杀 / 乾命看财星 → 十神层面的配偶代言人             │
│  日支十神（住客） = 日支地支所对应的十神 → 是"住在夫宫里的东西"            │
│                                                                          │
│  三者必须分别判读，再交叉 —— 不允许看到日支有"某十神"就直接当夫星          │
│                                                                          │
│  铁律 §B.8.7-B：输出婚姻判读时必须先声明三者各自是什么                     │
│                                                                          │
│  格式：「夫宫 = 日支[支名]；夫星 = [十神·天干]（藏于/透于[位置]）；        │
│         日支住客 = [十神]（仅代表该位置的能量色彩，不代表配偶本身）」        │
│                                                                          │
│  铁律 §B.8.7-C：夫宫住客 = 食伤 → 断为"命主才华工具放在婚姻宫"            │
│                 而非"配偶 = 食伤特质的人"（后者需额外铁证支持）             │
└──────────────────────────────────────────────────────────────────────────┘
```

**案例 004 教训示范**：
```
命主：癸卯日 → 日支卯
  ├── 夫宫 = 卯（位置）
  ├── 日支十神 = 食神乙（住客·才华工具）
  └── 夫星 = 戊正官（仅在时支申中余气·最弱根基）

正确判读：
  "命主把才华工具（食神）放在夫宫位置 → 婚姻中命主以才华主导"
  "真正的夫星戊只在申中余气 → 丈夫社会显示度低 / 无固定职业"

错误判读（v6.1.7 之前可能犯的）：
  ❌ "日支卯食神 = 丈夫是食神型人物（有才华、温和）" → 混淆住客与角色
  ❌ "卯木旺 = 配偶旺" → 混淆位置能量与配偶星力量
```

**三分判交叉判读法**：
| 判读维度 | 看什么 | 断什么 |
|---|---|---|
| **夫宫状态** | 日支是否被外来刑冲穿合 | 婚姻稳定度（安静=稳；动荡=危） |
| **夫星力量** | 官/财星透干否、有根否、旺衰 | 配偶社会地位/能力/经济水平 |
| **日支住客** | 日支所藏十神是什么 | 命主在婚姻中的主导方式/日常能量 |

**自检（涉及婚姻分析时必跑）**：
- [ ] 已分别声明"夫宫=XX / 夫星=XX / 日支住客=XX"三者各自是什么
- [ ] 未将日支十神直接等同于配偶特质（除非有 ≥ 3 重额外铁证支持）
- [ ] 夫星力量判读已独立于日支旺衰（两者可能完全不同方向）
- [ ] 三分判交叉后的综合判读已写明
