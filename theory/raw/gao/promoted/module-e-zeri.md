### E.1.4 真太阳时校正（必做）

```
铁律 §E.1-A：分娩城市经度 vs 北京时间标准 → 真太阳时偏移
公式：真太阳时 = 北京时间 + (当地经度 - 120°) × 4 分钟

主要城市偏移速查（节选 v1.8 第一部分二）：
  北京 +0 / 上海 +6 / 广州 -8 / 深圳 -8 / 成都 -16 / 西安 -14
  乌鲁木齐 -86 / 哈尔滨 +6 / 杭州 +4 / 重庆 -16 / 昆明 -28

跨时辰临界（如 23:55 ≈ 子时 vs 23:01 仍 亥时）必须明确标注；
未提供分娩城市 → 报告中醒目标注"⚠ 真太阳时未校正，时柱可能存在 ±1 时辰误差"。
```

### E.4.6 四象限分类法（盲派 vs 子平）

```
                    子平评分
                 高（≥75）   低（<75）
              ┌──────┬──────┐
       通过   │  ①   │  ②   │
盲派筛查     │ 共识优 │盲派偏  │
              ├──────┼──────┤
       淘汰   │  ③   │  ④   │
              │子平偏  │共识差 │
              └──────┴──────┘

象限 ① 双体系共识优 → 强烈推荐为首选
象限 ② 盲派偏好     → 保守选项，警示子平弱点
象限 ③ 子平偏好     → 交家属决定（盲派否决 + 子平高分，分歧场景）
象限 ④ 双体系共识差 → 淘汰
```

### E.4.7 归一化综合分（v1.7 引入）

```
综合分 = (盲派分 × W盲 + 子平分 × W子) × 置信度 - 分歧惩罚

默认权重：W盲 = 0.55 / W子 = 0.45
修正：缺父母八字 → W盲 0.40 / W子 0.60
      家属信格局不信古煞 → W盲 0.35 / W子 0.65
      家属完全信铁律     → W盲 0.70 / W子 0.30

置信度：
  全齐 + 真太阳时校正 + 万年历确认  → 1.0
  缺父母时辰（仅有日干）           → 0.85
  缺父母八字                        → 0.60（必须加粗警告）
  缺出生城市（未做真太阳时校正）   → 0.90
  日柱仅凭锚点未经万年历验证       → 0.80

分歧强度 = |盲派分 - 子平分|
  ≤10 惩罚 0 / 11-20 ×0.15 / 21-30 ×0.25 / >30 ×0.35（必须标 🔸）
```

### E.4.8 分歧标记

```
✓✓ 双体系强共识：盲派过 + 子平 ≥ 80
✓  双体系弱共识：盲派过 + 子平 65-79
🔸 强分歧：盲派否决 + 子平 ≥ 75，或盲派过 + 子平 ≤ 50
🔹 轻分歧：盲派过 + 子平 50-64
✗✗ 双体系强否决：两派都淘汰
✗  双体系弱否决：一派否决一派勉强
```

**起法口诀**：

子午从申起，丑未戌上去；寅申临子位，卯酉寅上寄。辰戌临辰地，巳亥午上居。

### §E.11.2 吉时选法（GP-MG-23 晋级）

> 原文锚点：第 15 课 §三（"如何选择吉时…口诀…吉时查寻表"）

**从日上选时口诀**（与大黄道同一口诀，改为从日支起时辰）：

```
子午从申起，丑未戌上去；寅申临子位，
卯酉寅上寄。辰戌临辰地，巳亥午上居。
```

**用法**：以当日日支为入口 → 从对应起始时辰起青龙 → 顺排十二星到十二时辰 → 黄道时辰为吉时。

**速查规则**（铁律 §E.11-B）：

```
日支为寅/申 → 子时起青龙 → 吉时：子丑辰巳未戌
日支为卯/酉 → 寅时起青龙 → 吉时：寅卯午未酉子
日支为辰/戌 → 辰时起青龙 → 吉时：辰巳申酉亥寅
日支为巳/亥 → 午时起青龙 → 吉时：午未戌亥丑辰
日支为子/午 → 申时起青龙 → 吉时：申酉子丑卯午
日支为丑/未 → 戌时起青龙 → 吉时：戌亥寅卯巳申
```

> **与 §E.3.3 时柱选择对接**：七层递进第三层"时柱与我宫"中，优先选落入本节吉时的时辰；若吉时与一票否决项冲突，以一票否决为准。
