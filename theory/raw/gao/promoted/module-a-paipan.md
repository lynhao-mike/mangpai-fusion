### A.1.1.1 起运与交运精确算法（v6.0.3 新增 · 强制项）

> ⚠ **这一节是地基中的地基**。起运算错 = 后续所有大运/流年应期全错。
> 触发：来自 #CASE-001（取整失真 11 月）+ #CASE-002（"阴女顺排"描述错 / 结果偶合对）双案例。

#### 一、阴阳判定（铁律：只看年干，不看年支）

| 年干 | 阴阳 |
|---|---|
| 甲、丙、戊、庚、壬 | **阳** |
| 乙、丁、己、辛、癸 | **阴** |

| 命主性别 + 年干阴阳 | 排运方向 | 取节气方向 |
|---|---|---|
| 阳年男 / 阴年女 | **顺排** | 取**下一**节气 |
| 阴年男 / 阳年女 | **逆排** | 取**上一**节气 |

> **易错点**：壬申、庚子、戊午、壬辰、丙寅 等"阳干配阴支"年份的**年是阳年**；
> 乙丑、丁亥、辛卯、癸酉 等"阴干配阳支"年份的**年是阴年**。
> **若在自证里写出"阴女顺排"或"阳男逆排"等组合 → 立即停下重判**（属阴阳判定错误）。

#### 二、起运岁数公式（精度到时辰）

```
起运精度 = 出生时刻 与 上/下一节气 的间隔时间 ÷ 3天/岁

精度换算表：
  3 天     = 1 岁
  1 天     = 4 个月
  1 时辰（2小时）= 10 天
  2 小时   = 10 天 ≈ 0.111 岁
  10 分钟  ≈ 1 天 ÷ 12 ≈ 0.0093 岁
```

**计算流程**：

```
Step 1: 阴阳定向 → 顺排取下节气、逆排取上节气，记节气精确时刻 T_jq
Step 2: 计算 |T_出生 - T_jq| → 得到 D 天 H 时辰（精确到分钟）
Step 3: D ÷ 3 = 整数岁 + 余数天
        余数天 × 4个月/天 = 月数（整数月 + 余数天）
        余数天 × 10天/时辰 折算 = 时辰精度
Step 4: 输出 "X 岁 Y 月 Z 天 W 时"，**不允许直接四舍五入到整数岁**
```

#### 三、交运日公式

```
首次交运公历日 = 出生公历日 + 起运精度（X年Y月Z天W时）
之后每 10 年（一个大运段）重复一次

交运节气表述 = 首次交运公历日 所处节气前的"上一个节气" + "后 N 天"
            （N = 交运日 - 该节气日）
```

> 交运表述用的节气，**不一定**等于起运计算用的节气（前者只是临近的描述坐标）。

#### 四、大运表起岁的硬约束

```
大运表第一步起岁 = A.1.1.1 第二步算出的精度（X岁Y月）
            ≠ 简单进位到 (X+1) 岁
```

| 起运精度 | 大运表第一步可写法 | 不允许的写法 |
|---|---|---|
| 5 岁 1 月 8 天 20 时 | "5 岁起" 或 "5y1m" | ❌"6 岁起"（取整失真 ≈ 11 个月） |
| 8 岁 9 月 26 天 21 时 | "9 岁起"（标准向上取整到最近年）或 "8y9m" | ❌"10 岁起"（多算 1 岁多） |
| 0 岁 4 月 8 天 | "1 岁起"（不足1岁按1岁起）或 "0y4m" | — |

#### 五、起运三连自证（**输出大运表前必跑**）

每一例出大运表前，逐条对答下面三连：

- [ ] **第一连·阴阳方向**：年干 ____ 是（阳/阴），命主性别 ____ → ____ 排
- [ ] **第二连·间隔节气**：取（上/下）一节气 ____，节气时刻 ____，间隔 ____ 天 ____ 时辰
- [ ] **第三连·首步起岁**：____ 岁 ____ 月 ____ 天 ____ 时；大运表第一步起岁 = ____ 岁

> 任一连不完整 → 大运表作废重排。

### A.1.5 空亡应用三律（v6.1.0 新增 · 来源 v2.0 第三部分补丁 B）

> ⚠ **触发**：v1 时代空亡仅作神煞速查，缺实战判法。
> #CASE-001 时支午带禄神、文昌、天厨三吉星，却落空亡，直到 2026 流年填实才兑现，暴露此盲区。
> 本节是排盘层面的**空亡三态判法**，与 §A.2.6（库 + 神煞 + 双空亡）配合，应期判读交模块 C。

#### 一、空亡三态铁律口诀

```
空亡带吉等填冲，填实冲起方显功
空亡带凶暂保身，填冲之年祸自生
吉凶混带看主导，主星填冲定乾坤
```

#### 二、三态判法表

| 态 | 触发条件 | 含义 | 兑现时机 |
|---|---|---|---|
| **① 悬空态** | 空亡支在原局，未被流年/大运填或冲 | 空亡位的吉凶**暂不兑现**，如同虚设；吉星"悬而未发"，凶星"悬而未作" | 不兑现（保留原局） |
| **② 填实态** | 流年 / 大运地支 = 空亡支（同字到位） | 吉星空亡 + 填实 = 吉事兑现；凶星空亡 + 填实 = 隐患暴露 | 该流年 / 该大运段 |
| **③ 冲动态** | 流年 / 大运地支 冲 空亡支 | **空亡反喜冲**：冲者反激活空亡位能量；吉星空亡逢冲 = 动中求得；凶星空亡逢冲 = 突发应验 | 该流年（突发型）|

> **关键术语**：v2.0 原文称之为『空亡反喜冲』，本模块沿用。
> **不要**简化为"空亡 = 不吉"或"空亡 = 不验"。

#### 三、实战 5 步（输出前必跑）

```
Step 1 找出空亡支（以日柱旬空为主；年柱/时柱亦落空亡时为"双空亡"，详 §A.2.6）
Step 2 查该空亡支的"藏干 + 神煞"组合（禄/刃/文昌/羊刃/天厨/咸池/华盖等）
Step 3 判断主导性质：
       吉为主（禄/文昌/天乙/天厨/金舆 等）→ 待填冲方兑现
       凶为主（羊刃/亡神/劫煞/华盖伴官非 等）→ 填冲之年祸自生
       吉凶混带 → 取最强主星（透干 > 本气 > 中气 > 余气）定主导
Step 4 在大运 / 流年表中标记三种事件年：
       (a) 填实年（同字）
       (b) 冲动年（正冲）
       (c) 三合 / 三会拱出空亡支的年（视同填实）
Step 5 把三种事件年回写到模块 C 的应期表，按 §C.2 四种交互打兑现度系数
```

#### 四、与"已婚-未婚态"的联动（v6.1.0 关键铁律）

> 空亡填实年的兑现内容，**完全取决于命主当前婚姻状态**，绝不可笼统判为"婚年"。

| 命主婚况 | 空亡填实年应判 |
|---|---|
| **[未婚态] 命主** | 若空亡支带配偶星 / 桃花 / 红鸾 / 天喜 → 该年判**婚期 / 成婚铁断年** ★★★★★ |
| **[未婚态] 命主** | 若空亡支只带禄 / 文昌 / 印 → 该年判**学业 / 入职 / 兑现年**（非婚年） |
| **[已婚态] 命主** | 一律判为**吉星全面兑现年**（事业 / 学历 / 置业 / 子女 / 升迁），**绝不**限定为"成婚年"或"再婚年" |
| **[已婚态] 命主** | 若空亡支被冲（冲动态）→ 优先解读为"生育 / 家庭结构变化 / 搬迁"，非婚变 |
| **[离异 / 分居态] 命主** | 单独评估：若配偶星空亡填实 + 桃花动 → 复合 / 再婚信号；其余按吉星兑现 |

> **铁律**：未拿到主入口 §1.4 的"婚姻状态"答案前，禁止把"空亡填实年"直接断为"成婚年"。

## 【A.2 进阶节】（Step 3 按需 · 用户追问"作用细节"才加载）

### A.2.1 穿的判读细则

穿不是单纯"凶"，而是**暗损 / 消耗 / 暗算**类做功。判读三问：

1. **穿到哪根**？穿日支 → 自身 / 穿月令 → 根基 / 穿时支 → 子女晚景
2. **穿用神还是穿忌神**？穿用神 → 坏功 ；穿忌神 → 反成
3. **是否有解**？被合住 / 被冲开 / 距离远 → 力度递减

| 穿的类型 | 性质 | 默认效率（无救解） |
|---|---|---|
| 子未穿 | 阴湿穿 | 中（伤情) |
| 丑午穿 | 火土穿 | 中高（伤心血/官印) |
| 寅巳穿 | 木火穿 | 中（动而生事) |
| 卯辰穿 | 木土穿 | 高（伤夫妻宫常见) |
| 申亥穿 | 金水穿 | 中（伤食伤/口舌) |
| 酉戌穿 | 金土穿 | 高（伤库 / 伤晚运) |

### A.2.2 破的判读细则

破 = 表面成、实际不成。常见两类：

- **破合** → 已合的两支被第三字破开（拆散）
- **破刑** → 已成刑的格局被破打散（救解）

破在原局往往**轻**，破在大运流年才会**重**——这一点要在模块 C 中校验。

### A.2.3 墓库判读

库支：辰（水库 / 戊土库）、戌（火库 / 戊土库）、丑（金库 / 己土库）、未（木库 / 己土库）。

| 入库情形 | 含义 |
|---|---|
| 财入财库 + 库不被冲 | 财不显但实有 |
| 财入财库 + 库被冲开 | 库门被开 → 求财动作发生年 |
| 官入官库 + 库被冲开 | 升迁 / 调动年 |
| 比劫入库 | 兄弟朋友受伤、合作破裂 |
| 印入印库 | 学历 / 母亲事 |
| 食伤入库 | 才华被压、子女事（女命） |

> 墓库的"开 / 闭"由**冲**触发，因此判读必须与模块 C 配合（看哪一年来冲）。

### A.2.4 合的判读细则

#### 天干五合
| 合 | 化神 | 化成条件 |
|---|---|---|
| 甲己合 | 化土 | 月令辰戌丑未或火土旺 |
| 乙庚合 | 化金 | 月令申酉巳或金气旺 |
| 丙辛合 | 化水 | 月令亥子或水气旺 |
| 丁壬合 | 化木 | 月令寅卯亥或木气旺 |
| 戊癸合 | 化火 | 月令巳午寅或火气旺 |

化神**得令** = 真化（合得成）；化神**失令** = 假合 / 空合（合得不成）。

#### 地支六合
寅亥 / 卯戌 / 辰酉 / 巳申 / 午未 / 子丑

判读看是否**同气相求**：
- 同气（如卯戌合化火）→ 实合
- 冰炭相犯（如巳申，火金相战）→ 空合，貌合神离

#### 三合局
申子辰水 / 亥卯未木 / 寅午戌火 / 巳酉丑金。
- 三字俱全 → 局成
- 两字 + 第三字在大运/流年 → 待引动（→ 模块 C）

### A.2.6 空亡进阶判读（库 + 神煞 + 双空亡）（v6.1.0 新增 · 来源 v2.0 第三部分补丁 B + #CASE-002）

> ⚠ Step 3 用户追问"空亡能否兑现 / 双空亡如何判 / 空亡同库怎么算"时加载本节。
> §A.1.5 是基础三态，本节是进阶组合。

#### 一、空亡 + 库的复合判读

| 组合 | 判法 |
|---|---|
| **空亡 + 财库**（如丑空、辰空、戌空、未空 而本气为财）| 财不显且能量悬空。**填实年开库**：求财动作发生年；冲动年：财库震动，得失反复 |
| **空亡 + 官库** | 官位"挂在空中"，不实任。填实年=任命 / 调整；冲动年=升迁 / 调动突发 |
| **空亡 + 印库** | 学历 / 母亲事悬而未发。填实年=证书到位 / 母亲事发生 |
| **空亡 + 食伤库** | 才华 / 子女事被压。填实年=才华显化 / 子女事 |

> **铁律**：空亡同库时，"开库"动作 = §A.2.3 的冲库动作 + 空亡填实双激活，能量兑现强度 ×1.5（具体系数见模块 C §C.2.1）。

#### 二、空亡 + 神煞的复合判读

| 神煞落空亡 | 判法 |
|---|---|
| **禄神空亡** | 禄位"挂空"，待填实方坐稳；填实年 = 入职 / 转正 / 落地年（见 #CASE-001 时支午禄空，2026 午年填实兑现副科）|
| **文昌空亡** | 才华有余功名不实，考试差一口气。填实年 = 学历兑现 / 论文 / 资格证 |
| **金舆空亡** | 富贵婚姻 / 贵车待迎，[未婚态]填实=婚期；[已婚态]填实=置业 / 升职 |
| **天乙空亡** | 贵人悬而不至，填冲方现 |
| **羊刃 / 亡神 空亡** | 凶星暂悬，填冲之年祸自生（见 §A.1.5 三律之 ②）|
| **桃花 / 咸池 空亡** | 感情悬而未发，[未婚态]填实=婚 / 恋；[已婚态]填实=丈夫桃花动（坤命）/ 异性缘动（乾命）|

#### 三、双空亡同填的罕见格局

> **触发**：日柱旬空范围内，**年柱 / 月柱 / 时柱**任意两柱地支同时落空亡。
> **典型**：#CASE-002（壬申·癸卯·丁未·壬寅 坤命）—— 月支卯 + 时支寅同为甲辰旬空，**双空亡**。

| 双空亡类型 | 判法 |
|---|---|
| **月 + 时双空亡（同旬）** | 命格"前后双悬"，月令与时柱能量都待填冲方落地。罕见格局，往往伴随母女 / 母子命格深度连接（详 §A.1.4 铁律 #6 母女空亡承袭） |
| **年 + 时双空亡（同旬）** | 起点 / 终点皆悬空，命主早年与晚景都需大运填冲方稳定 |
| **年 + 月双空亡（同旬）** | 童少年根基悬空，月令格局虚浮，需早年大运填实方真 |

#### 四、双空亡的『同步填实』效应

```
铁律：双空亡同旬空 + 流年/大运一字同时填两支 → 双激活能量倍增（×2.0）
铁律：双空亡同旬空 + 流年冲其中一支 → 单激活，但牵连另一支同步震动（×1.3）
```

### A.2.7 命宫太阳过宫修正起法（v6.1.5 新增 · 来源高派《盲派神煞应用宝典》第 6 课 §一·3 · GP-BD-09）

> ⚠ **加载条件**：Step 3 用户追问命宫/三垣/大限/小限时加载本节。
> **背景**：传统命宫排法以「月支+时支」固定对冲公式速排，误差可达 ±1 宫。高派修正以太阳实际过宫日为基准。

| 五月（午月） | 申宫 → **夏至后以寅为基准** 15 日进**未宫** | 夏至+15 日（特殊） |
| 六月（未月） | 未宫 → 大暑后第 13 日进**午宫** | 大暑+13 日 |
| 七月（申月） | 午宫 → 立秋后第 11 日进**巳宫** | 立秋+11 日 |

| 腊月（丑月） | 丑宫 → 大寒后第 5 日进**子宫** | 大寒+5 日 |

> **铁律 §A.2.7-B**：传统「月支+时支 = 14 – X」速排法仅作粗估参考；涉及大限/小限精度断事时**必须**使用本修正表。

```
Step 1：确定出生月的太阳所在宫位（上表第 2 列）

```

### A.2.9 地支六冲七法（v6.1.6 新增 · GP-XF-10 ⭐⭐ · 来源高派《象法篇》第 5 课 §二）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 5 课 §二（"六冲有七种应事法"）。
> 本节补强 §A.1.3 基础作用中"冲=冲X宫"的单一描述 → 细化为七种应事含义。

| # | 冲法 | 含义 | 判定条件 |
|---|---|---|---|
| **① 冲凶** | 冲带凶灾/伤残/破败 | 忌神坐支被冲 → 忌神发动=凶 |
| **② 冲旺** | 冲反而旺（越冲越旺） | 旺支逢冲=催旺能量（旺者冲不倒） |
| **③ 冲动** | 冲=动（搬迁/离职/出行） | 驿马逢冲=远行；时柱逢冲=门户动 |
| **④ 冲开** | 冲开库门/冲开障碍 | 财库/官库逢冲=开库得财/得官 |
| **⑤ 冲出** | 冲出暗藏之物 | 人元被冲出=暗伏显现 |
| **⑥ 冲去** | 冲走原有之物（失去） | 用神支被冲=用神离位=失去好东西 |
| **⑦ 冲破** | 冲破格局/冲破结构 | 格局关键支被冲=破格（降级事件） |

**铁律 §A.2.9-A**：同一个冲只取一种含义（以命局实际状态定）—— 不要同时说"冲凶"又说"冲旺"。
**铁律 §A.2.9-B**：判定顺序=先看"被冲方是用神还是忌神" → 再定七法中哪一种。

**判读流程**：
```
Step 1：确认冲组成立（§A.1.4 标准表核验）
Step 2：被冲方是用神/忌神/中性？
        - 忌神被冲 → ②冲旺（凶）或 ⑥冲去（吉）
        - 用神被冲 → ①冲凶 或 ⑥冲去（凶）
        - 库支被冲 → ④冲开
        - 驿马被冲 → ③冲动
Step 3：标注唯一含义 + 写入 §C.2 应期判读
```

#### 回归验证

| 案例 | 冲组 | 七法 | 事实 |
|---|---|---|---|
| CASE-001 | 子午冲（大运子冲时午） | ④冲开（午=禄神+文昌空亡→冲开=兑现） | ✓ 2026 提拔 |
| CASE-002 | 寅申冲（年申冲时寅） | ③冲动（寅=驿马→远离出生地） | ✓ 远嫁 |
| CASE-003 | 辰戌冲（流年戌冲年辰） | ⑤冲出（辰藏乙癸→冲出食神能量） | ✓ 高考爆发 |

### A.2.10 入墓规则与开关库法（v6.1.7 新增 · GP-XF-14 ⭐⭐ · 来源高派《象法篇》第 6 课 §一）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 6 课 §一（入墓 / 开库 / 关库三组完整规则）。
> **状态**：v6.1.7-batch5 候选规则 → 案例003 铁证验证（壬水入辰墓 + 戌冲辰开库 + 食神制杀） → 升级 1.2/2.4 🟢。
> **同步源**：`盲派命理AI分析师提示词_v2.1.md` §A.2.10。

#### 一、入墓 5 条件（必须全部满足方可论"入墓"）

| # | 条件 | 说明 |
|---|---|---|
| ① | **五行衰弱** | 该五行在命局无强根、失令或被克泄耗尽 |
| ② | **墓库当令或得势** | 辰戌丑未在月令/日支/时支占位，库有力量收纳 |
| ③ | **入墓五行与库有感应路径** | 通过合/生/伏吟等方式"走向"库位 |
| ④ | **无冲库之力** | 墓库未被外力冲开（冲开则不入） |
| ⑤ | **无透干牵引** | 入墓五行未透天干被他力合走（透干合走=不入墓） |

**口诀**：
```
弱而无根方入墓，库旺当令收其物
有路无冲不透干，五条齐备墓中住
```

#### 二、开库 3 法（让墓中之物"出来"发挥作用）

| # | 开库法 | 机制 | 典型应期 |
|---|---|---|---|
| ① | **刑开** | 丑未戌三刑 / 辰午酉亥自刑——刑到库位 | 流年地支刑库年 |
| ② | **冲开** | 辰戌冲 / 丑未冲——对冲打开库门 | 流年冲库支年 |
| ③ | **透干引出** | 库中藏干在岁运天干透出——人元透干即出库 | 大运/流年天干透藏干年 |

#### 三、关库 3 法（已开/半开之库重新"关闭"）

| # | 关库法 | 机制 | 实战含义 |
|---|---|---|---|
| ① | **合关** | 六合/三合将库支合住，库门锁死 | 财库合关=钱存不出；官库合关=仕途暂停 |
| ② | **墓运加临** | 大运再走同一墓库（伏吟）→ 库上加库 | 双库叠加=深藏不露，十年难显 |
| ③ | **印盖** | 印星盖头于库支天干（如戊辰·戊盖辰库）→ 封印 | 学历库被封=有才难显；财库被封=有财难用 |

#### 四、综合判断三步法

```
Step 1：先判五行是否真入墓（5 条件全检）
Step 2：再判墓库当前是开还是关（开 3 法 / 关 3 法）
Step 3：最后在岁运中找开关之年 → 即为应期
```

#### 五、与 §A.2.3 墓库判读的关系

- §A.2.3：基础速判（库支属性 + 是否成立 + 引动条件）
- §A.2.10：**精度补强** —— 提供"入墓 5 条件 + 开关 6 法"完整决策树
- 实战时优先 §A.2.10 走一遍，回到 §A.2.3 列结论

### A.2.10 天干暗合不化铁律（v6.1.7 新增 · R-004-REVISE-08 · 来源 #CASE-004 精度修正）

> ⚠ **触发**：#CASE-004 分析中模型把"日干癸暗合时支申中戊"直接推论为"暗合化火→以官身得财"，被用户精度审查指正。
> 本节解决：暗合≠化 的精度边界。
> 来源：`案例库/案例004_甲子丁卯癸卯庚申_坤命_1984_业务搭档双盘协同.md` §三.2 修正记录。

#### 一、暗合定义

```
暗合 = 地支藏干与天干 / 其他地支藏干之间存在五合关系
      但双方不在天干明面相见
      → 仅为"潜在引力"，非"已实现的合"
```

#### 二、核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §A.2.10-A：暗合存在但不化                                           │
│    → 明合（天干相见）都未必化（需月令当令 + 两干俱有根 + 无克破）          │
│    → 暗合（不在天干相见）更不可能化                                        │
│    → 禁止把"暗合"等价为"已经化"                                          │
│                                                                          │
│  铁律 §A.2.10-B：暗合的兑现需要流年/大运明合才激活                        │
│    → 暗合的字在流年天干明出 = 短暂明合兑现窗口                           │
│    → 暗合的字在大运天干明出 = 10年明合兑现段                             │
│    → 原局内暗合终身仅为"潜在引力/追求方向"，不是已兑现                    │
│                                                                          │
│  铁律 §A.2.10-C：暗合作为"意向信号"可用，作为"铁断依据"不可用            │
│    → ✅ 可说"命主有追求X的先天引力"（定性）                              │
│    → ❌ 不可说"命主已通过暗合获得X"（定量）                              │
│    → ❌ 不可打 ★★★★★ 铁断在暗合上                                      │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 三、典型错误与修正

| 错误表述 | 修正表述 |
|---|---|
| "癸暗合时支申中戊化火→以官身得财" | "癸暗合戊存在→命主有追官引力；需2028戊申年戊明出才兑现" |
| "暗合戊官全为喜用" | "暗合戊仅为潜在信号；平时主线仍是食神生偏财" |
| "暗合双根有实力" | "暗合处仅余气藏=最弱根基；需外力激活" |

#### 四、与明合/半合的强度对比

| 合类型 | 是否可化 | 铁断资格 | 兑现条件 |
|---|---|---|---|
| **天干明合**（如流年戊+命局癸） | 视条件可化 | ★★★★★ 可打 | 流年/大运天干相见即兑现 |
| **地支六合**（如卯戌合） | 视月令可化 | ★★★★ 可打 | 原局成立即兑现 |
| **暗合**（如日干癸暗合申中戊） | **不可化** | **★★★★★ 不可打** | 需流年明出才短暂兑现 |

### §A.2.11 暗合三种形式（v7.0 新增 · GP-XF-09 高派《象法篇》第 4 课 §四晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 4 课 §四（"暗合 = 见不得光"）
> **定位**：补强 §A.2.4 合的判读 —— 从"明合"扩展到"暗合"三种形式

#### 一、暗合核心定义

> 暗合 = 暗中苟合，见不得光，只能偷偷摸摸，私底下相好的关系。

#### 二、三种暗合形式

| 形式 | 定义 | 举例 | 核心象 |
|---|---|---|---|
| **A · 通合** | 两个地支**所有藏干**都能互相合 | 寅丑（甲己 + 丙辛 + 戊癸）/ 亥午（壬丁 + 甲己）| 最深层暗合 · 全面私通 |
| **B · 通禄合** | 相合的两天干，其**禄支中藏干**暗中相合 | 寅午暗合（甲己）/ 卯申暗合（乙庚）/ 巳酉暗合 / 亥午 / 子巳 | 半合兼暗合 · 双重关系 |
| **C · 干支通禄合** | 天干与非本位地支通过**禄位藏干**暗合 | 甲午（甲合午中己）/ 丁亥（丁合亥中壬）/ 戊子 / 辛巳 / 壬午 / 癸巳 | 干支自合 · 柱内暗情 |

#### 三、暗合取像铁律

```
铁律 §A.2.11-A：暗合 = 私情 / 隐瞒 / 见不得光的连结
  → 婚姻领域：暗合主外遇 / 地下情 / 未公开关系
  → 财运领域：暗合主暗财 / 灰色收入 / 私下交易
  → 事业领域：暗合主暗中合作 / 幕后推手 / 不公开联盟

铁律 §A.2.11-B：暗合与明合（六合）并存时，明合为主暗合为辅
  → 先看明合定主线 → 再看暗合定暗线
  → 明暗同指一物 = 公私兼顾（如事业与私情同归一人）

铁律 §A.2.11-C：暗合半合兼具时（如寅午既是半合火又是暗合甲己）
  → 半合取凝聚义（向中神靠拢）
  → 暗合取私情义（暗中有目的）
  → 两义叠加 = "明面合作暗含私心"
```

- [ ] 已识别原局中所有暗合组合（含三种形式）

### §A.2.14 人像物像区分铁律（v7.0 新增 · GP-XF-16 高派《象法篇》第 6 课附晋级）

> **来源**：`高派命理理论知识库/2018弟子班_象法篇_83页.md` 第 6 课 附（"扎根像法 + 人像物像区分"）
> **定位**：排盘取象时的**最顶层分流** —— 先分人物，再定吉凶方向

#### 一、核心区分

| 类型 | 定义 | 被制被伤的含义 | 断法方向 |
|---|---|---|---|
| **人像**（有生命体）| 自身 / 六亲 / 任何有生命的人 | 被制被伤被坏 → **灾**（轻则伤病，重则亡）| 不可制 = 保护对象 |
| **物像**（无生命体）| 财 / 富贵 / 学位 / 官位 / 房产 | 被制住 = **拿到 / 用到 / 兑现** | 制住 = 得到目标 |

#### 二、铁律

```
铁律 §A.2.14-A：判读时先分"此字代表人还是物" → 再定吉凶方向
  └── 人：受克 = 凶（伤亡）/ 受生 = 吉（健康长寿）
  └── 物：受克 = 吉（被我控制 = 得到）/ 受生 = 凶（失控 = 流失）

铁律 §A.2.14-B：同一个字在不同太极点下可能是人也可能是物
  └── 如"官星"：在看配偶时 = 人像（不可伤）
  └── 同一"官星"：在看事业时 = 物像（制住 = 升职）
  └── 必须先确定当前分析主题 → 再分人物

铁律 §A.2.14-C：联动 §Q.4-E 太极点转换
  └── 切换太极点时，同一字的人/物属性可能同步切换
  └── 必须显式声明当前取的是人像还是物像
```

#### 三、自检

- [ ] 当前断语中涉及的关键字已分清人像 / 物像
- [ ] 人像类断语中没有出现"制住 = 吉"的逻辑（人被制 = 凶）
- [ ] 物像类断语中没有出现"被克 = 凶"的逻辑（物被克 = 得到）
