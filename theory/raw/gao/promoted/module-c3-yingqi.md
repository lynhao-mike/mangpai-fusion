## 【C.3.14 关键字被克组合断流年（v7.0 新增 · GP-DY-20 ⭐⭐ · 来源高派《大运流年篇》第 5 课 §一.1）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 5 课 §一.1（"命局寅字为关键点…如果其他柱有丑土那么流年通常是酉年…只有无组合情况下才以申年断"）。
> 本节解决"命局某关键字何年受灾"的**精确应期定位**。

### C.3.14.1 核心逻辑

```
传统思维：关键字寅 → 被克 → 流年申（六冲）
高派纠正：关键字寅 → 先看是否有合会组合可克 → 合会克优先于六冲克
```

### C.3.14.2 判定铁律（以寅为例）

| # | 原局其他柱条件 | 克应流年 | 机制 | 优先级 |
|---|---|---|---|---|
| ① | 有丑 | 酉年 | 巳酉丑三合金局克寅 | ★★★★★ |
| ② | 有巳 | 酉年 | 巳酉丑三合中神酉为重 | ★★★★★ |
| ③ | 有申或酉 | 戌年 | 申酉戌三会金局克寅 | ★★★★ |
| ④ | 无上述组合 | 申年 | 寅申六冲（直冲） | ★★★ |

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.14-A：合会克 > 六冲克                                           │
│    原局有合会条件时 → 流年补足合会缺字 = 应期（非直冲年）                   │
│  铁律 §C.3.14-B：三合局中神为重                                            │
│    如巳酉丑合金 → 酉为中神 → 酉年应期概率最高                              │
│  铁律 §C.3.14-C：只有无合会组合时才论六冲                                  │
│    这是初学者最常犯的错误 → 见寅直接断申年 → 忽略了合会优先原则             │
│  铁律 §C.3.14-D：天干与地支理论相同                                        │
│    如命局有丑+辛巳年出事 → 天干辛透出=巳丑拱金+辛露头 → 视做成金局          │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.14.3 通用公式

```
关键字 X 何年受灾？

Step 1：找 X 的克神五行（木被金克 / 火被水克 / 土被木克…）
Step 2：命局是否已有克神五行的三合/三会部分字？
        → 有 → 流年补足缺字 = 应期
        → 无 → Step 3
Step 3：命局是否有 X 的六冲对宫字的三合/三会部分字？
        → 有 → 流年补足 = 应期
        → 无 → Step 4
Step 4：直接以六冲年论（X 的对冲字流年）
```

- [ ] 关键字周围是否有三合/三会条件已扫描
- [ ] 有合会条件时已优先判定（非直接断六冲年）
- [ ] 无合会条件时才用六冲论

## 【C.3.15 保护神阻断克路铁律（v7.0 新增 · GP-DY-21 ⭐⭐ · 来源高派《大运流年篇》第 5 课 §一.2）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 5 课 §一.2（"某字与克神之间间隔一个保护神不再适合本思维"）。
> 本节是 §C.3.14 的**例外条款** —— 当关键字与克神之间有保护神时，§C.3.14 的合会克逻辑失效。

### C.3.15.1 概念

```
保护神 = 关键字与克神之间间隔的"我生之神"（泄克神之力）
         → 克神被保护神泄/化 → 无法直接克到关键字
         → §C.3.14 的三合/三会/六冲思维全部失效
```

### C.3.15.2 典型结构

```
例：年丑 · 月午 · 日寅

分析：
- 关键字 = 寅（日支）
- 克神 = 丑（年支 → 丑含辛金克寅木）
- 保护神 = 午（月支 → 午火泄木生土，但更关键是午为寅的"我生之神"产物）

结论：
- 丑与寅之间有午（火）= 保护神
- 巳酉丑三合金局克寅 → 此路被保护神截断 → 不适用
- 永远不会是巳酉丑年或申酉戌年伤此寅字
- 改判：伤寅的方式 = 木年克保护神午（土）→ 保护神失效 → 然后金才能克寅
```

### C.3.15.3 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.15-A：关键字与克神间有保护神 → §C.3.14 合会克思维完全失效        │
│  铁律 §C.3.15-B：保护神失效年 = 真正应期                                   │
│    保护神被克/被冲/被合去的流年 → 保护神失效 → 克路贯通 → 关键字受灾        │
│  铁律 §C.3.15-C：天干地支同理                                             │
│    天干间有通关之神 → 同样阻断克路 → 判法一致                              │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.15.4 判定流程

```
Step 1：§C.3.14 判定关键字被克路径
Step 2：检查 → 关键字与克神之间是否有保护神（间隔字/中间柱位）？
        → 无保护神 → 按 §C.3.14 正常判
        → 有保护神 → 进入本节
Step 3：保护神何时失效？
        - 流年克保护神 → 保护神受伤 → 克路贯通
        - 流年冲/合走保护神 → 保护神离位 → 克路贯通
Step 4：保护神失效年 = 关键字受灾的真正应期
```

### C.3.15.5 自检

- [ ] 关键字与克神之间是否有间隔字已检查
- [ ] 间隔字是否构成"保护神"已判定（泄克神或通关）
- [ ] 有保护神时已放弃 §C.3.14 合会克思维
- [ ] 保护神失效年已识别（被克/被冲/被合去）
- [ ] 来源已标注（v7.0 / GP-DY-21 / 高派《大运流年篇》第 5 课 §一.2）

## 【C.3.16 远近组合断流年铁律（v7.0 新增 · GP-DY-22 ⭐ · 来源高派《大运流年篇》第 5 课 §二(一)）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 5 课 §二（一）（"年柱距离日柱较远…岁运的共同作用"）。
> 本节解决"年柱信息何时兑现"的距离效应。

### C.3.16.1 核心概念

```
远近效应：
  年柱 ←——距离远——→ 日柱
  年柱无论喜忌，因距离远 → 不能自动作用日主
  必须流年"再现"年柱同字 → 才能拉近距离 → 触发应期
```

### C.3.16.2 铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.16-A：年为忌 + 大运生助年柱 + 流年再现同字 = 引发凶灾           │
│    三者缺一不可 → 仅大运助年（无流年再现）→ 凶灾不发                      │
│  铁律 §C.3.16-B：年为喜 + 大运助年 + 流年是年柱 = 绝对发财                 │
│    同样三者缺一不可 → 无大运助时流年出现也只"好点有限"                     │
│  铁律 §C.3.16-C：强调岁运共同作用                                          │
│    单靠大运或单靠流年均不足以让年柱信息兑现                                 │
│    必须"大运+流年"双重引动才能跨越距离屏障                                │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.16.3 实战应用

| 年柱属性 | 大运条件 | 流年条件 | 结果 |
|---|---|---|---|
| 年为忌（官杀） | 大运=官杀（生助年忌） | 流年=财星（再生助） | 官灾 |
| 年为忌（官杀） | 大运=财星（助起官杀） | 流年=官杀（年柱再现） | 引发官灾 |
| 年为忌（官杀） | 大运=比劫（不助年） | 流年=官杀 | **不发**牢灾（大运未配合） |
| 年为喜（印星） | 大运=官杀（生印） | 流年=年柱同字（印再现） | 绝对发（学业/升迁） |
| 年为喜（印星） | 无大运助 | 流年=印 | 好点有限（不铁断发） |

### C.3.16.4 与 §C.2.4-B 岁运进入顺序的关系

```
§C.2.4-B 定"流年→大运→命局"传导顺序
§C.3.16 定"年柱远近效应" → 年柱需岁运双重引动

整合：年柱信息兑现路径 =
  流年再现年柱同字 → 通过大运（大运已生助年柱）→ 拉近距离 → 作用日主
```

- [ ] 流年是否"再现"年柱同字已核查
- [ ] 三者同时满足才下铁断（缺一不可）
- [ ] 来源已标注（v7.0 / GP-DY-22 / 高派《大运流年篇》第 5 课 §二(一)）

## 【C.3.17 人元引出三法（v7.0 新增 · GP-DY-23 ⭐⭐ · 来源高派《大运流年篇》第 6 课（一））】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 6 课（一）（"人元字平时不出来…三会三合可引出/岁运透干可出/地支相刑可出"）。
> 本节解决"藏干何时被激活"的关键问题 —— 人元不出不发作，出则应事。

### C.3.17.1 核心概念

```
人元 = 地支藏干
人元平时不出来 → 不参与命局明面上的生克
只有逢岁运"引出"才有作用 → 引出后立即参与生克 → 应事
```

### C.3.17.2 三种引出方式

| # | 引出方式 | 机制 | 示例 |
|---|---|---|---|
| **①** | **三会/三合引出** | 地支构成三合/三会局 → 合化后藏干被引出 | 辰中乙木 → 寅卯辰三会木 → 乙被引出；亥卯未三合 → 辰中乙亦出 |
| **②** | **岁运透干引出** | 流年/大运天干与藏干同字 → 透干呼应 → 藏干被引出 | 丑中辛金 → 流年辛年 → 辛被引出（天干呼应地支藏干） |
| **③** | **地支相刑/相冲引出** | 地支被刑冲 → 藏干被"震出" | 丑戌刑 → 丑中辛金/戌中丁火被震出 |

### C.3.17.3 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.17-A：人元不出不发作，出则必应事                                │
│    藏干被引出的年份 = 该藏干所代表的十神应期                                │
│  铁律 §C.3.17-B：三种引出方式有优先级                                      │
│    ①三合三会 > ②透干引出 > ③刑冲引出                                     │
│    （合局引出力最强·最持久；刑冲引出力爆发但短暂）                          │
│  铁律 §C.3.17-C：引出后必须判喜忌                                          │
│    引出的人元为喜 = 吉事应验 / 为忌 = 凶事应验                             │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.17.4 与 §C.3.10 有根论的联动

```
§C.3.10 判定"流年干支谁重要"
§C.3.17 判定"命局藏干何时被激活"

整合：
- 流年天干有根 → 天干主事 → 看天干十神
- 流年地支有根 → 地支主事 → 但地支如何"做事"？
  → 答案：通过引出命局某柱的人元来做事（本节）
- 流年天干透干引出人元 → 叠加"透干+引出"双重力量
```

- [ ] 引出方式属于①②③中哪一种已明确

## 【C.3.18 人元受克三型（v7.0 新增 · GP-DY-24 ⭐ · 来源高派《大运流年篇》第 6 课（一）1-3）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 6 课（一）1-3（"天克人元…枪打出头鸟…地支相刑人元…地支克人元只有4个"）。
> 本节是 §C.3.17 的**凶灾细化** —— 人元被引出后如何"受伤"导致应灾。

### C.3.18.1 三种受克类型

| # | 类型 | 机制 | 应期判定 | 口诀 |
|---|---|---|---|---|
| **①** | **天干克人元** | 柱内天干克本柱藏干（如丁丑 → 丁火克丑中辛金） | 大运生助主克者 → 流年人元透干（露头）→ **枪打出头鸟** = 得病 | 枪打出头鸟 |
| **②** | **地支相刑人元** | 地支被刑 → 藏干被震出受伤 | 多一弯：岁运先引动地支 → 再引出人元 → 人元才受伤 | 多一个弯 |
| **③** | **地支克人元** | 同柱地支本气克余气藏干（仅 4 组） | 吉凶判断同①：大运助主克者+流年透出被克者 | 同柱内战 |

### C.3.18.2 天干克人元详解（枪打出头鸟）

```
结构：丁丑（丁火克丑中辛金）

应灾路径 A：
  大运生助主克者（木运/丙运 → 助丁火克力）
  → 流年辛年（人元辛金露头）→ 枪打出头鸟 → 辛金方面得病

应灾路径 B：
  大运直接透出被克者（如壬寅柱 → 行丙运 = 丙透干克壬寅中甲木）
  → 也是枪打出头鸟 → 流年找生助主克者（金水年）

注意：大运与原局同字 → 无真正加力 → 不应灾
  （如丁丑柱行丁运 → 丁与丁同字=不加力→不克辛）
```

### C.3.18.3 地支克人元仅 4 组

| 地支 | 本气 | 被克人元 | 说明 |
|---|---|---|---|
| **丑** | 己土 | 克丑中**癸水** | 己土克癸水 |
| **寅** | 甲木 | 克寅中**戊土** | 甲木克戊土 |
| **辰** | 戊土 | 克辰中**癸水** | 戊土克癸水 |
| **巳** | 丙火 | 克巳中**庚金** | 丙火克庚金 |

> 其余 8 支无地支克人元现象（本气与余气不构成相克关系）。

### C.3.18.4 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.18-A（枪打出头鸟）：                                            │
│    人元不露头不受伤 → 只有流年透干或被引出时才挨打                          │
│  铁律 §C.3.18-B（多一弯）：                                               │
│    地支相刑人元需两步：先引动地支（刑/冲）→ 再引出人元 → 才受伤            │
│  铁律 §C.3.18-C（同柱内战）：                                             │
│    地支克人元仅上述 4 组 → 其余地支无此问题                                │
│  铁律 §C.3.18-D：大运与原局同字 ≠ 加力                                    │
│    同字到位是伏吟（另论），不是真正的生助关系                               │
└──────────────────────────────────────────────────────────────────────────┘
```

## 【C.3.19 交运脱运铁律（v7.0 新增 · GP-DY-14 ⭐⭐ · 来源高派《大运流年篇》第 3 课 §四）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 3 课 §四（"交运脱运两头奔命…行运换甲…大运与限运对应"）。
> ⚠ **高风险提示**：本节含"换好是人换不好是鬼"等极端表述，涉及寿限判断的部分（出生入死）需 ≥ 3 反例校验后方可铁断。日常使用仅取"交运年前后动荡"和"限运对应"两个安全层面。

### C.3.19.1 交运脱运概念

```
交运 = 走完一步大运，进入下一步大运的交接
脱运 = 旧运能量消退，新运能量尚未完全接管

古语："交运脱运两头奔命"
含义：交接期人如脱胎换骨 → 前后 1-2 年最不好过
表现：心神不定 / 性情大变 / 事业感情动荡 / 严重者耗财生灾
```

### C.3.19.2 四大关键点

| # | 关键点 | 内容 | 实战意义 |
|---|---|---|---|
| **①** | 交接之年前后两步大运兼看 | 旧运未尽+新运初至 → 两运能量交织 | 不可只看新运论吉凶，须兼顾旧运余力 |
| **②** | 行运换甲最关键 | "换得好是人，换不好是鬼" → 换甲时五行骤变 | 换甲=天干从癸→甲（水→木）= 五行属性完全改变 |

### C.4.5 流年 12 月应期推演法（v6.1.0 新增 · 来源 v2.0 第三部分补丁 A）

> **背景**：v1 仅能推演到"流年"级别，无法回答"今年几月份应事"。
> 源起：#CASE-001 用户追问『2026 年提拔的月份』，暴露此缺失。

#### 一、月份应期三层框架

```

```

|---|---|---|

| 第三层 · **5 维打分法** | 给每月打 1-5 星 × 5 维，加权累计 | 5 维评分表 + ≥20 = 高峰月 |

#### 二、第一层 · 节气月锁定（不用农历月份，用节气月）

| 未月 | 小暑-立秋 | 丑月 | 小寒-立春 |

#### 三、第二层 · 月干支推演（五虎遁口诀）

```
甲己之年丙作首，乙庚之岁戊为头
丙辛必定寻庚起，丁壬壬寅顺水流
戊癸之年何方发，甲寅之上好追求
```

> 用法：流年天干代入对应口诀 → 得寅月（正月）月干 → 后面 11 个月按六十甲子顺推。

#### 四、第三层 · 5 维打分法（v2.0 核心 · 本模块完整继承）

> **铁律**：每一个月按以下 5 项打分（每项 1-5 星），加权累计后按阈值定档。

| 维度 | 说明 | 权重 |
|---|---|---|
| ① **月干对应事十神** | 如分析提拔，则月干为印星 / 官星为强 | ★★★★★ |
| ② **月支与命局核心宫位感应** | 合 / 冲 / 刑 / 穿 / 伏吟 | ★★★★★ |
| ③ **月支与大运 / 流年地支关系** | 是否拱合、填实、冲动 | ★★★★ |
| ④ **月干是否透出命局藏干** | 人元透干即应 | ★★★★ |
| ⑤ **神煞 / 空亡是否填冲** | 空亡填实反应事（联动 §A.1.5）| ★★★ |

**综合得分对应可能性**（v2.0 原文阈值，本模块沿用）：

| 综合分 | 档位 | 含义 |
|---|---|---|
| **≥ 20 分** | **★★★★★ 高峰月** | 应期高度集中，铁断窗口 |
| 15–19 | ★★★★ 强月 | 高度可能 |
| 10–14 | ★★★ 中等 | 较可能 |
| 5–9 | ★★ 偏弱月 | 仅有迹象 |
| **< 5** | **★ 禁忌月** | 不发此事，反向规避 |

> **铁律 §C.4.5-A**：≥ 20 分才能标 ★★★★★ 高峰月，< 5 分才能标 ★ 禁忌月，**严禁跳档**（不允许 17 分直接标 ★★★★★，也不允许 8 分直接标 ★ 禁忌月）。

### C.4.7 冲合应期互补铁律（v7.0 新增 · GP-DY-26 ⭐⭐ · 来源高派《大运流年篇》第 7 课（四））

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 7 课（四）（"原局有合以冲为应…填实应吉应凶"）。
> 本节是 §C.4.6 应期六大规律第④条"冲合应期互补"的**完整展开** —— 高派定应期的第一法。

#### 一、核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.4.7-A：原局有合 → 以冲为应期                                     │
│    合住=锁定不动 → 需冲来打开 → 冲年=事件爆发年                           │
│  铁律 §C.4.7-B：原局有冲 → 以合为应期                                     │
│    冲动=不稳定 → 需合来收拢 → 合年=事件落地年                             │
│  铁律 §C.4.7-C：流年合大运 = 合动                                         │
│    合=牵引/锁定 → 大运被流年合住 → 该运能量被引动（吉凶看合方喜忌）        │
│  铁律 §C.4.7-D：流年冲大运 → 分当运/不当运                                │
│    当运（旺相有力）被冲 = 冲去（凶·正在发挥的能量被破坏）                  │
│    不当运（休囚无力）被冲 = 冲起（吉·沉睡的能量被激活）                    │
│  铁律 §C.4.7-E：六合/三合/三会/三刑 → 一字空缺 → 填实应吉应凶             │
│    如亥卯未三合缺卯 → 卯年=填实年 → 应吉应凶取决于合局的喜忌              │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 二、作用类型速查

| 作用 | 含义 | 应期信号 |
|---|---|---|
| **合**（六合/三合/天干合） | 主"到" → 牵引到位 | 事物到来/人到位/机会降临 |
| **冲**（六冲） | 主"动" → 激活/破坏 | 变动/搬迁/分离/动荡 |
| **墓**（入墓） | 主"收" → 收藏/结束 | 事情收尾/收钱/入库/病入膏肓 |
| **穿**（六穿/六害） | 主"伤" → 暗损/背叛 | 暗中受损/被人坑害/隐性伤害 |

#### 三、填实应期规则

```
填实 = 六合/三合/三会/三刑中缺少的一字被流年补上

判定：
Step 1：原局+大运已有合/会/刑的部分字 → 找缺字
Step 2：流年出现缺字 = 填实年
Step 3：填实后判吉凶：
        - 合局/会局为喜 → 填实应吉
        - 合局/会局为忌 → 填实应凶
        - 刑局填实 → 一般为凶（三刑多主灾）

特殊：墓库逢空 → 开库也无用（空亡之库未经填实前无实质内容）
```

#### 四、与 §C.4.6 的关系

§C.4.6 六大规律第④为"冲合应期互补"一行概述。本节展开完整判定流程，是定应期的**首选方法**（§C.4.6-A 已明确"最实用"）。

### C.4.8 五大凶组合速查（v7.0 新增 · GP-DY-27 ⭐⭐ · 来源高派《大运流年篇》第 8 课）

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 8 课 一~五（"羊刃聚会…财星坏印…枭神夺食…伤官见官…比劫夺财"）。
> 本节为流年断灾提供**五大经典凶组合**速查表 —— 流年形成以下任一组合即为凶年强信号。

#### 一、五大凶组合总表

| # | 凶组合 | 触发条件 | 灾类 | 轻重 |
|---|---|---|---|---|
| **①** | **羊刃聚会** | 命局羊刃 + 岁运重见/逢冲/三合刃局/刃重重 | 病伤手术·破财·官非 | 轻=破财口舌；重=死亡 |
| **②** | **财星坏印** | 流年干支=天干印/地支财（或反之）→ 地克天=重 | 失业/迁移/破财/母病/牢灾 | 地支克天干=重；天干克地支=轻 |
| **③** | **枭神夺食** | 柱有枭逢食年 / 柱有食逢枭年 / 枭食夹日 | 灾病不顺·牢狱 | 双枭/三枭夺食=灾重；食旺枭难夺=灾轻 |
| **④** | **伤官见官** | 原有伤官逢官年 / 原有官逢伤年 | 女=夫妻不和；男=惹事生非·牢灾 | 有伤逢官=灾轻；有官逢伤=灾重 |
| **⑤** | **比劫夺财** | 财星透干无官护 + 逢岁运劫财透干 | 破财 | 有官护=不惧比劫；无官护=必破 |

#### 二、各组合细则

**① 羊刃聚会**
```
《命藏经》："羊刃聚会损六亲，不入地府恐失财"
触发形式：刃见刃（伏吟）/ 刃逢冲 / 三合刃局 / 日坐刃逢冲
日坐羊刃逢冲 → 夫妻有灾
```

**② 财星坏印**
```
结构：流年干支一为印一为财 → 互克
判轻重：地支克天干（下克上）= 重；天干克地支（上克下）= 轻
忌讳点：财为忌克印为用 = 最凶 → 失学/失业/母病/牢灾
```

**③ 枭神夺食**
```
四种触发：
1. 柱有枭 + 岁运逢食 → 有灾
2. 柱有食 + 岁运逢枭 → 有灾/不顺
3. 枭食同透/枭食夹日 → 一生不顺事多；夹日=牢狱
4. 柱中无枭无食 + 岁运逢枭食 → 灾轻（无根基）

加重：双枭/三枭夺食 = 灾重
减轻：食旺枭难夺 = 灾轻
特殊：柱有枭无食 + 逢枭年 = 本人有病（不夺食但枭旺伤身）
```

**④ 伤官见官**
```
盲派口诀："伤官伤官端药罐，伤是小鬼胎中堕，旧仇新恨找上门"
男命：惹事生非/顶撞领导/打架斗殴/违法乱纪 → 严重牢灾
女命：夫妻不和/闹纠纷/情绪化

判轻重：有伤逢官（伤是旧有的·官是新来的）= 灾轻
        有官逢伤（官是旧有的·伤来犯官）= 灾重
```

**⑤ 比劫夺财**
```
条件：财星透干 + 无官星护卫 + 逢岁运劫财透干 = 破财
解法：有官星护卫（官制比劫）→ 即使逢比劫年也不惧
```

```
§C.3.14 关键字被克组合 → 定"哪个字被伤"
§C.4.8 五大凶组合 → 定"以什么方式应凶"
§C.4.6 六大应期规律 → 定"哪一年兑现"

```

### C.5.1 长生遇杀铁律

```
口诀："长生遇杀最有灾"

钥匙：这里的"杀"指长生地的七杀 —— 不是日干的七杀！
     即大运走到该五行的长生地 → 遇到该长生地的克星 = 灾

阳干长生：甲→亥 / 丙戊→寅 / 庚→巳 / 壬→申
阴干长生：乙→午 / 丁己→酉 / 辛→子 / 癸→卯
```

**判定流程**：
```
Step 1：日主五行 → 找长生地
Step 2：大运是否走到长生地？
Step 3：长生地被何五行克？ → 该五行的流年 = "长生遇杀"年
Step 4：该年为凶年强信号（★★★★）→ 叠加其他凶兆定轻重
```

甲己申酉最为愁，乙庚午未不用求

|---|---|---|

### C.6.2 空亡填实三法（GP-CS-23）

### §C.7+.7 长生十二宫六亲联动实战法（GP-MG-14 · 来源第 5 课 §三.1 释意）

> 原文锚点：「七杀丁火无制…原局三个酉充当了七杀丁的长生」

**铁律 §C.7+.7-A**：以某六亲天干为太极点 → 引到日支/时支/大运支看其十二长生状态 = 该六亲在该宫位/该运的能量态。

**操作步骤**：
1. 确定目标六亲天干（如妻星乙木/七杀丙火/正印壬水等）
2. 查该天干的长生十二宫表（阳干顺行/阴干逆行）
3. 将四柱各支、大运支、流年支分别代入 → 得该六亲在各处的能量状态
4. 能量状态 = 长生/帝旺 = 该六亲力量强；= 绝/死/病 = 该六亲力量弱

**实例模式**：
- 庚日主妻为乙 → 乙引到午为长生 = 妻到午宫有源泉（午运妻旺）
- 丙火七杀 → 丙引到酉为长生 = 七杀力量从酉处源源不绝（酉年七杀发力）
- 正财甲木 → 甲引到亥为长生 = 财星在亥处有根（亥运发财信号）

> **与 §C.7+.6 的关系**：§C.7+.6 给原理，本节给操作法；二者必须连用。

### §C.7+.12 岁运并临吉凶 6 条判定（GP-MG-28 · 来源第 14 课 §二）
