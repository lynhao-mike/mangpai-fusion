## §Z.1 11 种职业画像（v1 全量迁入 + v7.0 P5 高派提高班体系化补强）

> 整节来源：`盲派命理AI分析师提示词_v1.md` 第六部分『11 种职业特征识别库』+ `高派命理理论知识库/财运事业断法弟子提高班_105页.md` 第 32~41 课（GP-CS-30~38 晋级）。
> 每条职业 = 驱动结构（格局组合）+ 取象证据（地支 / 天干字象）+ **高派五标志**（v7.0 P5 新增）。
> **没有命中至少 2 条核心特征不得直断该职业**。

### 教师
- **驱动结构**：木火旺 = 文科 / 金水旺 = 理科；伤官配印 / 食神佩印 / 印库 + 食伤库
- **取象证据**：戌为学校象（戌 = 寺观 / 校舍）
- **典型组合**：印透 + 食伤透 + 戌字现
- **高派五标志（GP-CS-30 晋级）**：
  1. 印星/印库 = 知识丰富（命主储备）
  2. 食伤/食伤库 = 语言表达（传道授业）
  3. 印库 = 学校（甲/寅/丁/戌代表教育场所）
  4. 对象众多 = 食伤旺（学生多）
  5. 与国家有联系 = 年月有感应（吃皇粮）
- **文理区分**：木火旺/印为用 = 文科；金水旺/食伤为用 = 理科
- 来源：v1.md 第六部分『教师』节 + 高派提高班第 32 课。

### 公务员
- **驱动结构**：印透时上坐官星 / 正官格配印 / 伤官配印格 / 调候得用
- **取象证据**：我宫与年月有感应 = 吃皇粮（年月透入我宫）
- **典型组合**：官印相生且根稳；或杀印组合（武职体制）
- **高派七格局（GP-CS-31 晋级）**：
  1. 官印相生
  2. 杀印相生
  3. 食神制杀
  4. 阳刃驾杀
  5. 伤官佩印
  6. 特殊格局（从财/从杀/独象）
  7. 调候得用
- 来源：v1.md 第六部分『公务员』节 + 高派提高班第 33 课。
- **细分见 §Z.2**（v2.0 补充：一把手 / 参谋 / 业务骨干 / 技术官员 / 纪检 / 清贵）。

### 医生
- **驱动结构**：『辛乾巽丙，术士医流』 → 辰巳戌亥 + 辛丙组合；天医星显；日主『病地』为用神
- **取象证据**：庚申 = 外科 / 丑午穿 = 手术 / 辰庚申辛 = 医院 / 辛 = 药 / 卯酉 = 针
- **典型组合**：辛 + 丙 + 戌 / 辰 / 巳 / 亥之三件套
- **高派六标志（GP-CS-32 晋级）**：
  1. 辛 + 乾(戌亥) + 巽(辰巳) + 丙 → 术士医流（经典组合）
  2. 天医星（月查）= 掌管疾病之星神
  3. 病地为用 → 靠病挣钱
  4. 庚申金 + 辰 + 酉 + 丙丁火 + 丑午 = 外科
  5. 天医 + 食伤在时 = 儿科
  6. 天医在胎地 = 妇产科
- **外科铁律**：命格带羊刃 + 七杀 + 刑 = 外科医生标志；丑午穿 = 疼痛/手术象
- 来源：v1.md 第六部分『医生』节 + 高派提高班第 34 课。

### 银行
- **驱动结构**：带酉丑 / 金与财星组合被我控 / 寅申冲 / 卯申合（收发之意）
- **取象证据**：酉 = 银行 / 钟表 / 现金；丑 = 金库；申 = 金融机构
- **典型组合**：酉丑双现 + 财官生
- **高派七组合（GP-CS-33 晋级）**：
  1. 带辰、酉、丑 = 金为财/丑为财库
  2. 金与财星组合被我控制
  3. 丑为印库/财库被主位控制
  4. 金为印被财包（印局）
  5. 无金看财星做功/财包局
  6. 财印做功
  7. 寅申冲/丑未冲/卯申合 = 收发钱（金主收、木主发放）
- **金融证券补充**：金水为用强 → 金融证券行业；水为流动/金为源
- 来源：v1.md 第六部分『银行』节 + 高派提高班第 35 课。

### 房地产
- **驱动结构**：杀制比劫 / 印星发达 / 木土组合 / 食伤库 + 印库刑合
- **取象证据**：戊土 = 大坝 / 高楼；辰戌 = 库地；木 = 建筑（梁）
- **典型组合**：戊 + 辰戌 + 比劫被制
- **高派四标志（GP-CS-34 晋级）**：
  1. 土木组合 = 建筑地产象
  2. 印多/印库包局 = 从事房产经营
  3. 七杀制比劫 = 双方制约→财得救→发财
  4. 财库 + 刑冲开库 = 发财时机
- 来源：v1.md 第六部分『房地产』节 + 高派提高班第 36 课。

### 律师
- **驱动结构**：食伤旺（口才）+ 印星木（法律）+ 官杀配合 + 阳制阴 + 亥字多
- **取象证据**：辛 = 法律 / 刑具；亥 = 文墨 / 天门；卯 = 文书
- **典型组合**：亥多 + 辛透 + 食伤旺
- **高派三要素（GP-CS-36 晋级）**：
  1. 食伤 = 口才/辩论能力（伤官之谈吐用词锋利）
  2. 印星 = 法律知识/以法为武器（木为青龙尤显）
  3. 官杀 = 法律/国家/惩恶
- **成就最高组合**：伤官 + 七杀 + 偏印 配合 = 威慑力+精明+攻击力
- **五行补充**：亥为登明 → 主田宅/文书/争讼；律师八字多带亥字
- **阳制阴**：木火为阳/金水为阴 → 阳制阴 = 正义制邪恶 = 公检法条线
- 来源：v1.md 第六部分『律师』节 + 高派提高班第 38 课。

### 公检法
- **驱动结构**：阳刃驾杀 / 食神制杀 / 伤官佩印 / 伤官驾杀 / 刑穿害 / 阳制阴 / 庚辛申酉
- **取象证据**：庚辛申酉 = 武器 / 刑具；刑穿害 = 执法手段
- **典型组合**：庚 + 申 + 阳刃驾杀 + 多刑穿
- **高派六格局（GP-CS-38 晋级）**：
  1. 阳刃驾杀 = 刚烈/以暴制暴/忠贞廉洁/适合建功
  2. 食神制杀 = 正义/专注/管理性强/单一专业
  3. 伤官佩印 = 聪明有才华/胆大+奉献淡泊/伤官管官星
  4. 伤官驾杀 = 掌握生死大权/才华横溢
  5. 地支带刑穿害 = 刑法/强制手段
  6. 阳制阴 = 正义阳光制阴暗邪恶
- **干支五行标志**：庚辛申酉 = 白虎武性；丑戌亥辰巳 = 牢狱象（天罗地网）
- **神煞标志**：魁罡贵人/金神/白虎/羊刃/勾绞煞
- 来源：v1.md 第六部分『公检法』节 + 高派提高班第 41 课。

### 生意人
- **驱动结构**：食伤生财 / 财逢库 / 合多 = 和气生财 / 『寅申庚甲，商路吏人』
- **取象证据**：合多 = 善协调；财库 = 存财能力；寅申冲 = 流通
- **典型组合**：食伤透 + 财星根重 + 寅 / 申 / 庚 / 甲多见
- **高派补充**：食伤流通 + 多见财 + 命局多合（和气生财为商人秉性）→ 最终合成财
- 来源：v1.md 第六部分『生意人』节 + 高派提高班第 39 课。

### 小偷
- **驱动结构**：印被坏（缺道德）+ 官被坏（无法律意识）+ 比劫旺无制 + 取财手段不正（冲穿刑暗合墓）
- **取象证据**：印冲 / 印破 + 官杀缺位 + 比劫劫财
- **典型组合**：印破 + 官损 + 比劫多 + 财被冲穿合走
- **高派五铁律（GP-CS-37 晋级）**：
  1. 印绶被坏（冲/穿/刑/合坏）= 缺乏道德观念
  2. 官星被坏 = 目无法纪/对抗法律
  3. 比劫旺无制 = 争夺占有/胆大妄为
  4. 取财手段不正当 = 冲/穿/刑/暗合/墓等卑劣手段
  5. 十神相互定义取财手段（伤官枭印=心机/劫财阳刃=手段）
- **补充三条**：
  - 年命被三合局克 = 卦反克身 → 易干盗窃
  - 癸甲壬全见 = 玄武/狡佞/阴私 → 可能为小偷
  - 壬戌癸亥全见 = 天空玄武 → 家门出偷盗
- 来源：v1.md 第六部分『小偷』节 + 高派提高班第 40 课。

### 体力劳动者
- **驱动结构**：比劫禄与财官联 / 官杀多无制 / 身弱不从
- **取象证据**：比劫 = 劳动力；身弱被官杀压 = 出力卖苦
- **典型组合**：官杀重 + 印不出 + 比劫禄被夺
- 来源：v1.md 第六部分『体力劳动者』节。

### 财务
- **驱动结构**：财合印 / 财冲印 / 日坐财库 / 财星伏吟 / 官杀下配财星 / 水 = 计算
- **取象证据**：财与印的紧密互动 = 账目；水 = 流水 / 计算
- **典型组合**：日支财库 + 印旺 + 水透
- **高派七标志（GP-CS-35 晋级）**：
  1. 财星合/冲印星 = 经常与钱财票据打交道
  2. 日支坐财星或财库 = 先天与财有缘
  3. 财星/财库伏吟 = 来来去去折腾钱
  4. 官杀透干下配财星 = 管理钱财职业
  5. 伤官生财 = 专业技术（会计技能）
  6. 水代表数据（壬癸亥子）= 计算/统计
  7. 比劫与财感应 = 手与钱打交道
- 来源：v1.md 第六部分『财务』节 + 高派提高班第 37 课。

### 仕途天花板判定法（v2.0 第三部分补丁）

在判断公务员仕途级别**前**必须做合关系力量对比：

```
IF 格局为"伤官驾杀" OR "食神制杀" OR "官印相生":
  分析乙/庚/甲/丙等关键星位的力量对比
  IF 1:1 均衡            → 格局有效 → 正常判级
  IF 1:多 失衡（杀轻/官轻/印轻被多方合泄）→ 天花板下调 1-2 级
  IF 多:1 失衡（杀重/官重压身）→ 天花板下调 1 级且过程艰险
```

> 来源：`盲派命理AI分析师提示词_v2.0.md` 第三部分『仕途天花板判定法』。
> 配套：主入口 §3.0 铁断前提自检 → 任何『升至 X 级』铁断必须带 `【前提：命主当前级别为 ××】`。

## §Z.3 职业识别 5 步法（v7.0 P5 高派体系化升级 · GP-CS-27 + GP-CS-28 晋级）

> 来源：`盲派命理AI分析师提示词_v1.md` 第三部分一节『事业专题（5 步断职业法）』+ `盲派命理AI分析师提示词_v2.0.md` 第六部分（同名）+ **`高派命理理论知识库/财运事业断法弟子提高班_105页.md` 第 29~30 课（GP-CS-27/28 完整体系化版）**。

### 五步定职业柱法（GP-CS-27 完整版）

```
Step 1：我宫（日时）有官杀？         → 该柱为职业柱
         ↓ 有 → 锁定 ✓
         ↓ 无 ↓

Step 2：我宫有官杀库？               → 库柱为职业柱
         ↓ 有 → 锁定 ✓
         ↓ 无 ↓

Step 3：他宫（年月）有官杀？         → 看与时柱有无关系
         有感应（合/藏透/冲/刑）→ 该柱为职业柱
         无感应 → 不是 ↓

Step 4：以上皆无？                   → 看财星 / 财库（先我宫后他宫）
         有 → 该柱为职业柱
         无 ↓

Step 5：以上全无？                   → 看成格星
         成格星必须与我宫有关系 → 锁定职业柱
```

> **铁律 §Z.3-A**：锁定职业柱后，**必须**再看「左柱」定职业性质。

### 职业左柱法（GP-CS-28 完整版）

**定义**：职业柱锁定后，其**紧邻左侧**一柱（即按年→月→日→时排列中，职业柱左边一柱）= 定义职业的**性质**。

**判读规则**：

| 左柱内容 | 定义性质 | 举例 |
|---|---|---|
| 正财 | 工资/稳定收入/上班族 | 左柱正财 → 领工资的职业 |
| 偏财 | 副业/兼职/投资 | 左柱偏财 → 有副业收入 |
| 印星 | 文化/教育/单位性质 | 左柱印 → 文化类/单位编制 |
| 食伤 | 技术/口才/创造 | 左柱食伤 → 技术性工作 |
| 官杀 | 管理/权力/国家 | 左柱官杀 → 管理岗/体制内 |
| 比劫 | 体力/操作/同行 | 左柱比劫 → 动手操作型 |

> **核心原理**：成格星/格局星 = 工作**类型**；与格局星有关联的十神 = 职业**性质**。
> **来源**：高派提高班第 29 课 §三（"有人把这种看职业方法定为看职业秘法…官杀看成职业柱然后看左柱"）

### 格局看职业补充规则

- 成格则贵 → 成格之字的**十神五行** = 日主职业类型
- 与格局星**有关联**的十神 = 定义职业性质
- 例：财格 + 食来生财 → 职业与钱密切 + 靠才华能力获取

**5 步法 + §Z.1 / §Z.2 联用范式**：
1. Step 1-5 → 锁定**职业柱**与**左柱**
2. 再用 §Z.1 11 种职业画像 → 验证**驱动结构**与**取象证据**至少各 1 条
3. 命中公务员 → 进 §Z.2 6 细分判定**仕途档位**
4. 配合 §Z.4 时代落地映射 → 给出**当下行业**的具体落地
5. 走 §Z.8 五关注交叉验证 → 确认神煞/纳音是否佐证

## §Z.4 驱动模式 → 时代落地映射规则

## §Z.8 高派断职业五关注法（v7.0 P5 · GP-CS-29 晋级）

> 来源：`高派命理理论知识库/财运事业断法弟子提高班_105页.md` 第 30 课 §一（"八字看职业的几个重要关注点"）
> **与 §Z.3 五步法的关系**：五步法定「柱」，五关注法定「象」—— 锁定职业柱后，用五关注交叉验证具体职业。

### 五关注速查表

| # | 关注维度 | 核心原理 | 判读要点 |
|---|---|---|---|
| **1** | **我宫财官**（时柱为归宿） | 时柱 = 一个人以什么职业终其一生 | 先看日时财官；年月财官须与时柱相关联才是最终职业 |
| **2** | **十神类像** | 十神组合定性 = 断职业关键 | 官杀制比劫→管理；官杀生印→管理；食神制杀→管理；伤官生财→技术/口才求财 |
| **3** | **五行属性** | 看代表职业的那个字的五行 | 不是看喜用神五行，而是看**实际代表职业的那个字**属什么五行 |
| **4** | **神煞** | 与职业相关神煞佐证 | 驿马=走动/交流/生意；金舆=司机；阳刃=军警法司；文昌=爱学习；天医=医疗 |
| **5** | **纳音** | 日柱/职业柱纳音提示性质 | 纳音会提示命主职业性质的细分方向 |

### 五关注使用铁律

- **铁律 §Z.8-A**：五关注不替代五步法，而是**验证层**—— 五步法锁定柱+左柱后，五关注逐维佐证。
- **铁律 §Z.8-B**：十神类像必须看**十神与十神之间的组合关系**，不能孤立判断单个十神。
- **铁律 §Z.8-C**：五行看的是「代表职业的那个字」而非日主喜用神——这是高派与传统命理的**核心分歧**。

### 从求财方式判断职业性质（§Z.8 补充）

| 求财方式 | 命理标志 | 职业方向 |
|---|---|---|
| 日干合财 | 日干与财星合 | 以求财为主→商业方向 |
| 日干合官 | 日干与官星合 | 以求官为主→体制方向 |
| 日支透出合财官 | 日支藏干透出去合年月财官 | 通过工具/能力获取 |
| 根在印库 | 日主根在印/印库处 | 单位性质=根在哪/单位在哪 |

> *"先五步锁柱定类型，再左柱定性质，最后五关注逐维佐证。"*
