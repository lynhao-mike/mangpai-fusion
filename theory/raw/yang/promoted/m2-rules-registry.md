| M2-Y-002 | 阴阳互根对立统一 | 阴中有阳阳中有阴；独阴不生独阳不长；判命必看阴阳两股力量 |

| M2-Y-004 | 化敌为友为最高境界 | 化官化杀无官灾；制官得官有官灾；化=最高层次 |
| M2-Y-005 | 天干五合做功效率最高 | 合=轻松得到；克=辛苦取；层次差距=天地之别 |
| M2-Y-006 | 谁生的财谁说了算 | 食神生财归属论；不生时财就走了 |
| M2-Y-007 | 大运定乾坤流年定应期 | 大运决定10年方向；流年触发具体事件 |
| M2-Y-008 | 命中有的终须有 | 命中有的字=认识的人/具备的技能；命中无=不认识需公关 |
| M2-Y-009 | 清娟易理9要素体系 | 家里家外/力量平衡/追根溯源/气质辨别/真假旺衰/宇宙法则/转象换象/八字风水/高维探索 |
| M2-Y-010 | 敬畏之心慈悲之心 | 算命须有敬畏与慈悲；不做风水营销；不恐吓客户 |
| M2-Y-011 | 调候改运基础原则 | 调整时候=提前引好驱走坏；方法=化敌为友+制伏 |
| M2-Y-012 | 万物有形必有灵 | 物品气场影响命主；命局喜的五行物品=助运；忌的=损运 |
| M2-Y-013 | 五行非仅质勿忘五行气 | 五行既代表有形的质也代表无形的气(意念)；学习为了与人和谐 |
| M2-Y-014 | 完整性原则 | 完整=不受穿/破/克坏；财不完整不大；官不完整不长 |
| M2-Y-015 | 凶神大于用神怕用神出来 | 原命凶>用时用神出来=灾；用>凶时凶出来=发 |
| M2-Y-016 | 在劫难逃vs漏网之鱼 | 大运+流年双具备=在劫难逃；仅流年坏=漏网之鱼 |
| M2-Y-017 | 人生三宝印食禄 | 可坏两个不能全坏；全坏=贫贱多灾 |
| M2-Y-018 | 旺神冲衰神拔衰冲旺发 | 旺的冲弱的=弱的被拔根；弱的冲旺的=旺的更旺 |

## §20 · 基础篇补全（M2-Y-049..063）

| rule_id | 规则名 | 核心结论 |
|---|---|---|
| M2-Y-049 | 主位宾位正式定义 | 日时=主位(自己+子女)；年月=宾位(长辈+兄弟)；大运流年=旅途所遇 |
| M2-Y-050 | 体用正式定义 | 体=取财官的工具(印枭/比劫/食伤/禄)；用=人生追求(财/官杀) |
| M2-Y-051 | 定旺衰三旺点法 | 月令50%+时支45%；得两旺点=必旺；两旺点不得力余全帮=仍弱 |
| M2-Y-052 | 做功效率排序 | 化官杀>食制杀>食伤生财>比劫取财(最低)；官杀级别>财=过河财 |
