## §3 · 天干五合完整体系

### §3.1 五合化合产生物（M2-Y-024）

| 合 | 化 | 产生的实物象 |
|---|---|---|
| 甲己 | 化土 | 药/化妆品/面粉/饮食 |
| 乙庚 | 化金 | 钞票/武器/首饰/工具/钢铁 |
| 丙辛 | 化水 | 酒/化学品/化工/发电/流通 |
| 丁壬 | 化木 | 文化/文字/木材/教育/印刷/传媒 |
| 戊癸 | 化火 | 西药/药物/颜色/信仰/名声 |

> **天干五合做功效率最高**：合=得到(轻松)；克=辛苦取(层次低)。
> 合到主位=自己得；合在宾位=环境象。

### §3.2 化合3状态（M2-Y-025）

| 状态 | 条件 | 结果 |
|---|---|---|
| **化成** | 地支有化神+一方有力 | 真合得实物 |
| **合绊** | 力量相当互牵 | 都做不了事 |
| **搅局** | 第三干介入克合首 | 合被拆/转向 |

### §3.3 搅局星判定（M2-Y-026）

> 克合首者为搅局星 → 合被拆 → 失去做功。

- 搅局星力量>被搅者时 = 100%拆
- 搅局星力量<被搅者时 = 仅减效
- 搅局星无根无气时 = 搅不动

6组经典：庚甲己/辛乙庚/壬丙辛/癸丁壬/甲戊癸/乙己甲

### §3.4 天干五合应事表（M2-Y-027）

| 合什么 | 应事 |
|---|---|
| 合财 | 得财/得妻/合作 |
| 合官 | 升职/得权/考试 |
| 合印 | 得学历/得房/得母助 |
| 合食伤 | 得技术/得口才/子女缘 |
| 合比劫 | 得朋友/得助力/合伙 |

> 合的财官为忌神时，合到反为灾。

### §3.5 天地一气原则（M2-Y-121）

- 天干地支做功一致 = 天地一气 = 表里如一 = 专一成功
- 天干地支做功不一致 = 表里不一 = 内心想法与表面不同
- 天干合官+地支也合官 = 有啥说啥做说一致
- 吃皇粮不一定当官，天地一气才当官

### §3.6 同柱干支互动规则（M2-Y-028）

- 天干与天干作用 / 地支与地支作用
- **同柱**干支互作用
- **异柱**天干不直接作用地支
- 同柱同五行地支逢生天干旺（如丙寅，寅逢生丙旺）
- 同柱不同五行地支逢生对天干产生力量（如戊寅，寅逢生戊受制）

## §4 · 十神系统

### §4.1 十神暗引5组公式（M2-Y-029/030）

> 旺而不受伤则暗引对应十神。

| 显象（旺点不受刑穿） | 暗引 | 应事 |
|---|---|---|
| 食伤旺 | 暗引比劫 | 有贵人/良好人际关系 |
| 印枭旺 | 暗引官杀 | 得权/制印得权 |
| 比劫旺 | 暗引印枭 | 劫财旺=学历/比肩旺=技能 |
| 官杀旺 | 暗引财 | 过河财/钱权交易 |
| 财旺 | 暗引食伤 | 口才/技术=赚钱的根 |

> 比劫暗引印枭定学历方向：劫财重→正印=有学历；比肩重→枭=报考技能类学校。
> 金水=理科(金融/法律/医)；木火=文科(教育/文化)；木火库=与佛有缘。

### §4.2 十神条子运用原则（M2-Y-138/139）

| 十神 | 旺时性格 | 过旺/无制 |
|---|---|---|
| **食神** | 大度不记仇/奉献/包容 | 食伤旺=浪漫；不生财不制杀=无所事事 |
| **伤官** | 聪明/记仇/钻空子 | 过旺=传销/找事打架/混蛋 |
| **正印** | 善良守法/有正义感/慈母 | 化官=守法；伤官见官有印=逢凶化吉 |
| **偏印** | 敏感/对号入座/学易最适合 | 领悟力极强能挖掘看不见的；起坏作用=狠毒自私 |
| **禄** | 喜配印/忠诚/专注 | 过旺不生食伤=打架 |
| **阳刃** | 身弱见刃=能干有头脑 | 过旺=赌场/暴力 |
| **官杀** | 人生发展动力/事业来了 | 七杀无制=坏蛋/暴力/不满足 |

> **正印条子四句诀**（M2-Y-139）：慈母/善良守法/制伤官保官/有印看护无凶险。
> 找谁解救看印五行：木=找证据；其他=花钱找贵人。

### §4.3 比劫9条断语系统（M2-Y-137）

1. 比劫本是无知物，油盐酱醋记得住 = 底层劳动
2. 比劫见到食伤势必归 = 有出路
3. 财在家里身又旺，比劫帮我把活扛 = 扩大经营
4. 财落他乡比劫搭伴上 = 竞争
5. 财源头不在我，比劫生 = 我扛活 = 破/夺/口舌
6. 比劫不能伤，伤了比劫就破财
7. 谁有印谁是老大
8. 财是我的落他乡，比劫和我搭伴上 = 争财
9. 比劫有印=销售；无印=体力（M2-Y-093/137）

### §4.4 枭印人才铁律（M2-Y-131）

- 只要有印就有特殊技能 = 人才（不管夺不夺食）
- 枭印在家里 = 某领域技能好
- 不在家里 = 有一技之长
- 枭印领悟力极强，能看见别人看不见的，心细
- 枭起好作用 = 慈悲心/母爱；起坏作用 = 狠毒/自私
