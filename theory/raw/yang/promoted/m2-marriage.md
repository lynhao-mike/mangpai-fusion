## §9 · 大运流年与应期

### §9.1 气数判定（M2-Y-037）

| 概念 | 定义 |
|---|---|
| **气数犹在** | 前好运走完 + 流年同神 = 还好一年 |
| **气数已尽** | 前好运完 + 流年克神 = 彻底转坏 |
| 指导意义 | 气尽前收场止损；好大运第8年坏流年来=叫人家赶紧退 |

### §9.2 大运流年作用关系（M2-Y-047）

| 大运 | 流年 | 实际效果 |
|---|---|---|
| 好运 | 好年 | 一发不可收拾 |
| 好运 | 坏年 | 坏不到哪去=挫折非失败=有贵人 |
| 坏运 | 好年 | 虚像=看别人赚钱自己投资必败 |
| 坏运 | 坏年 | 大凶 |

> 坏的大运投资基本是伪劣产品或钻法律空子（M2-Y-154相关）。
> 命好运差=龙盘虎卧。

### §9.3 缘分年限=大运长度（M2-Y-136）

- 大运该字走几年，缘分就几年
- 大运变字 = 人走/领导换
- 新旧大运同五行阴阳不同 = 兄弟交接仍是贵人
- 大运不是命局喜的字 = 小人环绕怎么做都错

### §9.4 在劫难逃vs漏网之鱼（M2-Y-132）

| 条件 | 判定 |
|---|---|
| 大运来了 + 流年触发 | **在劫难逃**怎么都躲不过 |
| 大运未到 + 坏流年 | **漏网之鱼**只是坏事不断不会丧命 |

> **断灾一定是大运流年两方面条件具备。**
> 没有辛巳大运就没有灾，顶多丙申年摔坏了不会成植物人。

### §9.5 命中有的终须有（M2-Y-135）

- 命中有的字 = 认识的人/具备的技能
- 命中没有的字 = 不认识的人/不具备的技能
- 年月有 = 过去认识的，这个字出来 = 过去的人来找你
- 不认识的需要公关，看力量能否制住

## §10 · 婚姻与六亲系统

### §10.1 婚姻看法三步（M2-Y-038）

| 步骤 | 内容 |
|---|---|
| 第一看宫位 | 夫妻宫静否（喜静忌刑冲穿） |
| 第二看星 | 配偶星强弱（虚透=不稳） |
| 第三看应期 | 印来了(家到) + 配偶星到(人到) = 结婚 |

### §10.2 结婚应期完整版（M2-Y-141）

- 印来了 + 人(配偶星)来了 = 结婚年
- 印来了人没到 or 人来了印没到 = 第二年结婚
- 两个条件挨着前一年后一年中的一年
- 条件不具备 = 无奈/有人反对/急匆匆
- 条件具备 = 得到所有人祝福
- 结婚应期不看夫妻宫的刑冲克害

### §10.3 星虚透理论（M2-Y-039）

- 星虚透 = 感情不稳/不易找到如意/易变心
- 见根年 = 人来结婚
- 根走年 = 离婚
- 若还犯比劫则需同时解决比劫问题

### §10.4 犯桃花定义（M2-Y-142）

- 家里有财官跑到外面去了 = 我追别人
- 外面财官跑到我家里来 = 别人追我
- **犯桃花就是财官丢了**，和破、穿、生等都没有关系
- 别人追不追你老婆，看你的财喜欢谁

### §10.5 离婚条件（M2-Y-040）

| 类型 | 条件 |
|---|---|
| 比劫见财型 | 比劫力>财力 + 宫位受冲 |
| 食伤生财型 | 食伤泄身太过 + 财星外露 |
| 官杀混杂(女命) | 基本都离(比劫争夺日主) |

### §10.6 六亲看法定位（M2-Y-041）

- 真正的六亲以**宫位**定（月柱占50%承上启下）
- 印=母/财=父/比劫=兄弟/食伤=子女/官杀=配偶或上司
- 命中没有妈妈就在爸爸的宫位安上妈妈的星
- 枭神夺食按宫位断：时支女命=子女灾/月干=兄弟灾/日支=自身灾

### §10.7 不孕判定（M2-Y-143）

- 命局趋于一方 = 多数不生小孩
- 巳火戌土一派坏了巳中金和戌中金 = 不孕(输卵管/卵子)
- 金水太旺(亥水) = 不易怀孕
- 木在寒冬无法存活 = 卵子成活率低
- 泄体过度 = 恐惧 + 生育困难
