## §1 · 核心心法

### §1.1 真假虚实

| 概念 | 定义 | 应用 |
|---|---|---|
| **真** | 有根有气=地支有本气/中气/余气 | 真财=长期稳定收入 |
| **假** | 无根无气=天干虚透 | 假财=才华虚名不是钱 |
| **实** | 坐实落地能直接做功 | 官真实=实权 |
| **虚** | 漂浮不落地仅显象 | 官虚透=是财非官 |

> **十神虚透断语**（M2-Y-108）：
> - 比劫虚透=不实在/行事低调
> - 印星虚透=吹嘘/善良（反之）
> - 财星虚透=虚荣心/有才华（反之）
> - 官星虚透=是财不是官

### §1.2 完整性原则（M2-Y-014）

完整 = 不受穿/破/克坏的状态。
- 财不完整 → 财不大不长久
- 官不完整 → 当不了官/官有灾
- 印不完整 → 文凭低/母弱/丢权
- 官命要求官完整独立存在

### §1.3 化敌为友（M2-Y-032）

> **化官化杀无官灾；制官得官有官灾。**

| 层次 | 方式 | 应事 |
|---|---|---|
| ★★★★★ | 化杀生枭+枭生身不见食神 | 官最大(行政最高层) |
| ★★★★ | 化官生印 | 官大有名气(行政官/党务) |
| ★★★ | 财生官+合官 | 官大但低于化杀 |
| ★★ | 食神制杀 | 技术官/企业官 |
| ★ | 伤官伤尽 | 当官但条件=伤官暗制官 |

> 正者要化，偏者要制（M2-Y-003）。

### §1.4 财官取得层次排序（M2-Y-021/024）

| 层次 | 方式 | 映射 |
|---|---|---|
| ★★★★★ | 天干五合+合到主位 | 阳干合财/阴干合官=富贵命 |
| ★★★★ | 用印取 | 脑力工作者/管理者 |
| ★★★ | 用食伤取 | 技术/销售/中介 |
| ★★ | 用比劫取 | 劳动者 |
| ★ | 用禄克财/禄冲财 | 最底层/贫贱卑微 |

> 「禄可以合财合官就是富贵命，禄冲财冲官就是贫贱卑微命。」

## §2 · 算命流程体系

### §2.1 算命切入点五步法（M2-Y-119）

```
第1步：先在家里找财官（日时=家里）
  → 家里有 → 弱制就能得到
  → 家里没有 → 去外面找（年月=外面）→ 看力量

第2步：看财官从哪里出来（寻根基）
  → 从我坐下出 = 完全是我自己的
  → 从比劫宫出 = 两人的/合作
  → 从年月出/无出处 = 国家的/外面的

第3步：用什么取财官（定层次）
  → 用印/食伤取 = 管理者/白领（层次高）
  → 用比劫取 = 劳动者（层次低）
  → 禄克财 = 最底层

第4步：天干透什么定吃皇粮/民营（M2-Y-120）
  → 天干透官/印/食伤 = 端国家饭碗（吃皇粮）
  → 天干透比劫 = 端比劫饭碗（民营）
  → 财从比劫坐下出 = 民营/合作

第5步：地支做什么定具体职业
  → 地支是载体=实际做的事
  → 天干是表象=外在身份
```

### §2.2 寻根基找真假看出处观动静（M2-Y-162）

| 步骤 | 内容 | 要点 |
|---|---|---|
| **寻根基** | 有根(本五行比劫/生字/本库)+有气(余气库) | 戊己土特殊：午巳未戌为根，辰丑不论根(M2-Y-163) |
| **找真假** | 天干在地支有没有气能不能立住 | 立住=真，立不住=假 |
| **看出处** | 判断吉凶祸福力量大小与源头 | 家里出来的弱制就得/外面的看力量 |
| **观动静** | 大运定乾坤流年定应期 | 合/穿力量最大 |

### §2.3 家里家外铁律（M2-Y-019/023）

| 原则 | 内容 |
|---|---|
| 人生秩序 | 家里有肯定先拿家里的，外面再好也不确定 |
| 家里=自信 | 拿自己家里财官=自信稳定 |
| 家外=自卑 | 拿别人的=不自信/犯口舌 |
| 弱制得 | 家里的财官弱制即得（不需要力量大于） |
| 外面看力量 | 外面的须体力>财官力方得 |
| 命中有=认识 | 命中有的字=认识的人/具备的技能(M2-Y-135) |
| 命中没有=不认识 | 不认识就需公关，看力量能否制住 |
