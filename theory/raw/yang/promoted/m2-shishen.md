## §5 · 做功方式与食神体系

### §5.1 食神生财归属论（M2-Y-123/125/126）

| 配置 | 结果 |
|---|---|
| 食神是我的 + 生我家里财 | 全是我的 = 发财 |
| 食神是别人的 + 生到我家里 | 别人给我机会 |
| 食神在比劫宫 | 两人的 = 合作 |
| 不生时 | 财走 = 破财 |
| 我的食神生外面的财 | 我给别人资源 = 我老大 |
| 别人的食神生我家里的财 | 别人给我机会 = 我受摆布 |

> **谁生的财谁说了算**。有印的是老大。

### §5.2 食神生财=能力 / 食神制杀=成就（M2-Y-124/033）

| 食神方向 | 功能 | 判定 |
|---|---|---|
| 食神生财 | 能力（办事能力强到处能办成事） | 90%自己做/上班不长 |
| 食神制杀 | 成就（能办大事当大官） | 力量相当=求官；食伤>杀=当官；食伤<杀=求财 |
| 食神泄秀 | 聪明伶俐 | 比劫寒冷有火=人才 |
| 既不生财也不制杀 | 无所事事找不到方向 | 靠大运 |

> 食神代表：中介/技术/销售/管理。巳火午火同现=两方面都做（M2-Y-127）。

### §5.3 食神生财破财应期（M2-Y-126）

| 条件 | 应事 |
|---|---|
| 食神被合住/被制 | 不生财 = 破财年 |
| 寒水(丑/子)生 | 生不起来木 = 能生一点赚小钱 |
| 库(未)来 | 认祖归宗财被收走 |
| 火来暖局 | 水流通能生财 = 发财年 |

### §5.4 食伤当财制食伤条件（M2-Y-153）

| 身旺/身弱 | 能否制食伤 | 结果 |
|---|---|---|
| **身旺** | 不能制（人生三宝） | 制了有灾 |
| **身弱** | 可制 | 制了发财(食伤旺=大财/弱=小财) |
| 身弱+制过 | 可制但不能制过 | 发财但身体有灾 |

> 灾是自己还是六亲，以宫位定。

## §6 · 格局判定（杨派版）

### §6.1 格局=十神有对应配置（M2-Y-031）

- 体有对应制化神 = 有格局 = 勤奋
- 体闲无对应 = 懒汉
- 用闲无制化 = 没能力
- 杀无制化 = 没格局的人，到哪都受欺负
- 大运补来对应神时临时形成格局（运过则散）

### §6.2 官命9种取法层次排序（M2-Y-042）

| 排序 | 取官方式 | 层次 |
|---|---|---|
| 1 | 化杀生枭 | 官最大（行政最高层） |
| 2 | 化官生印 | 官大有名气（行政官/书记） |
| 3 | 合完整官 | 官大 |
| 4 | 财生官+合官 | 官大但低于化杀 |
| 5 | 食神制杀 | 技术官/企业官 |
| 6 | 比劫生食伤制杀 | 吃皇粮 |
| 7 | 劫财制财 | 武职 |
| 8 | 伤官伤尽 | 当官（条件：伤官暗制官） |
| 9 | 制印得权 | 条件：印不化官杀+印与体打架 |

### §6.3 吃皇粮判定三条件（M2-Y-120）

| 配置 | 判定 |
|---|---|
| 天干透官/印/食伤 | 端国家饭碗（吃皇粮） |
| 天干透比劫 | 端比劫饭碗（民营） |
| 财从比劫坐下出 | 民营/合作 |
| 合的官是国家的 | 吃皇粮 |
| 合的官是比劫坐下的 | 民企 |

> 吃皇粮不一定当官。当不当官看官能否立得住（完整独立存在）。

### §6.4 官大官小判定

- 官大官小看官的旺衰（官旺=级别大/官弱=级别小）
- 食伤制杀看力量：能制住坐下的杀=部门经理；能制月令的=总经理
- 制不住=级别小/欲望小
- 宫位距离：离得近的能制了，离得远的制不住=机会

## §7 · 力量体系

### §7.1 承载财官本质（M2-Y-089/133）

| 体vs财官 | 结果 |
|---|---|
| 体 > 财官 | 承载得到 = 稳定发展 |
| 体 ≈ 财官 | 基本平衡 = 能承载 |
| 体 < 财官 | 承载不了 = 想放弃(事业/家庭) |

> 用什么取财官，该字必须旺（力量>财官）。取财官的字坏了 = 什么都干不了。

### §7.2 体旺财官弱=穷命两种（M2-Y-134）

| 配置 | 性格 | 大运帮扶后 |
|---|---|---|
| 体旺+财官弱 | 大材小用/懒 | 能赚到但在本领域最低 |
| 体弱+财官旺 | 好高骛远/光说不做 | 同上 |
| 体旺到极致+财官弱 | 也懒 | — |

### §7.3 身旺身弱vs体旺体弱四种组合（M2-Y-152）

| 组合 | 现实映射 |
|---|---|
| 身旺+体旺 | 最好，有格局做大事 |
| 身旺+体弱 | 能力有但做不大 |
| 身弱+体旺 | 有资源但本人撑不起 |
| 身弱+体弱 | 什么都拿不起来 = 穷命 |

> 体旺财官弱 = 无事可做或懒。

### §7.4 份额分配以宫位力量定（M2-Y-020/122）

| 位置 | 力量占比 | 说明 |
|---|---|---|
| 月令（中神） | ~50% 最旺 | 谁在月令谁拿最多 |
| 时支（旺点） | 次之 | 时支也是旺点 |
| 年上（弱点） | 最小 | 弱点占比小 |

> 中神力>隅神力>墓库力。谁旺谁拿多。
> 禄(本气) > 根(中气/余气) > 库(入墓) > 气(天干虚透无根)。

### §7.5 禄喜配印完整条件（M2-Y-140）

| 配置 | 结果 |
|---|---|
| 印旺生禄 | 富 |
| 印弱生禄 | 小富 |
| 禄无生无帮 | 普通命甚至穷命 |

> 不是有禄归禄就好，要看有没有生和帮。
> 有禄喜走印运；有印喜走禄地 = 走禄地就富贵着。
> 印禄相随 = 房子多/事业稳。

## §8 · 开库与财富级别

### §8.1 开库原则（M2-Y-034）

| 原则 | 内容 |
|---|---|
| 我的根参与开库 | = 发财 |
| 根不参与 | = 看别人发财 |
| 财库喜在坐下 | 坐下的库最容易开 |
| 开官杀库 | = 德才兼备有能力(既发财又当官) |
| 有库不能开 | = 有才无德不可培养 |
| 中神在旺点 | 三合局/三会局中神必须在月令或时支才能合成(M2-Y-036) |
| 墓中物喜透出 | 透出来才能用 |

### §8.2 财富级别7等排序（M2-Y-035/089）

| 排序 | 级别 | 说明 |
|---|---|---|
| 1 | **官杀库** | 最大（开了=亿级） |
| 2 | 食伤库 | 次大 |
| 3 | 旺杀 | 制杀第一大=制和化一样 |
| 4 | 财库 | 中等 |
| 5 | 旺官 | — |
| 6 | 食伤当财 | 比财大但有限 |
| 7 | **财（最小）** | 直接以财为财最小 |

> 同八字不同地区财富数字不同，按当地经济能力定。
> 财库在两旺点=级别大；坐下=级别小。
> 印库>财库>官杀库（Y-17新排序，特殊条件）。
> 制杀第一大>制官化官>制食伤>制财（Y-14进阶）。

### §8.3 四步坏大运（M2-Y-089相关）

四步坏大运 = 所有灾难来源：
1. 劫财见财
2. 枭神夺食
3. 伤官见官
4. 财坏印
