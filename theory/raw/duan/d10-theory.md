# D-10 · 《盲派命理开山之作》深度提炼

> D-10 是段派对外**定义体系坐标**的奠基讲义。它对**宾主 / 体用 / 功神 / 能量效率 / 贼神捕神**五大签字概念给出**首次正式定义**——这是后续所有段派著作（D-01/D-02/D-03 等）默认引用而不再重述的"定义层"。
> 本文件按 theory-extraction-template §1 的 9 节强制结构产出。反 dump，做 architecture。

---

## §0 · 本书独特贡献摘要

D-10 在 D-01/D-02/D-03 之上的独特贡献（与既有规则的 delta）：

1. **五大签字概念的正式定义**（§第五章 第二节）— 此前 D-01/D-02 是"使用层"，D-10 是"定义层"。每个概念给出 What / Why / How 三段式表述。
2. **盲派 vs 子平的根本立场**（§第五章 第一节）— 段派**主动废除"用神"概念**的理论解释（"手拿杯子喝水，无法定义谁是用神"比喻）→ 转而用"做功"+ "功神"统摄。
3. **六做功方式的完整工程化**（§第六章 第二节）— 制用 / 化用 / 生泄用 / 合用 / 墓用 / 复合 — 每种给出 ≥3 案例 + 适用条件 + 失败模式。**合用结构内含 4 子情形**（身旺财旺 / 身弱财旺 / 身旺财虚 / 身弱财虚）是 D-10 独有细化。
4. **富贵贫贱三档判定模型**（§第六章 第一节）— 富贵 = 有势 + 有功；普通 = 有功无势 / 有势无功；差 = 无功且废神多。**势的 12 种典型组合**（木势/火势/金势/水势/燥土势/湿土势 + 6 种两党组合）首次列全。
5. **干支配置三原理**（§第五章 第三节）— 干支生克原理 / 干支互通原理（禄与原身互通）/ 干支虚实原理。**互通是应期的最重要工具**——本书首次明确化。
6. **应期判定 4 条主线**（§第六章 第三节）— 字出现 / 遁藏透干 / 冲合刑穿 / 当运不当运（冲去 vs 冲起）。**"原局合则以冲为应，原局冲则以合为应"**首次成文。
