## §0 · 本书独特贡献摘要

D-05 在 D-01/D-02/D-03/D-10 之上的独特贡献（与既有规则的 delta）：

1. **子平 vs 盲派根本立场对照**（§算命三步曲 + §测象 + 多处卜文解评）— 指出子平派常见 5 个判定漏洞：宫位机械化、用神僵化、衰旺为唯一标准、神煞滥用、应期单线。
2. **卜文反驳模式 = 信息复证原则**（§痛失麟儿 line 207 显式给出）— 关键事件（损子/死亡/婚姻/官非）的判定不能只凭 1-2 条信息，**必须 ≥3-4 条信息复证才能下铁口断**。这是段派对"过拟合 / 单线归因"的最强约束。
3. **风火集独特断语 5 类**（§痛失麟儿 / §生离死别 / §老兵的凋谢 等）— 过旺虚倾、子星空亡看湖中、夫星临长生拱局不入墓、羊刃见岁运冲合必死、食神死地丁壬合而去之必死。
4. **暗合 5 子类工程化**（§第三者插足 line 357 完整列出）— 寅丑暗合 / 亥午暗合 / 卯申暗合 / 亥未半合中丁壬甲己暗合 / 巳丑半合丙辛暗合 / 巳酉半合丙辛暗合 / 子辰半合戊癸暗合。比 D-02 §第七章 "合象"原则更细化。
5. **应期看法多元集成**（§痛失麟儿 + §第三者插足 + §无法捉摸的八字 等）— 三合六合定婚期 / 岁运合杀去一留一 / 流年合大运为合动 / 寅午戌合入夫宫等多源应期判定。
