## 1 · 三大法则（贯穿全模块）

```
理法（理解）→ 命局 = 文章 / 剧本，AI 任务是读懂
  ↓
象法（细化）→ 干支象 + 宫位象 + 十神象 + 神煞象
  ↓
技法（操作）→ 父母 / 婚姻 / 寿命 / 应期 等具体技巧
```

### 1.1 理法
- 让你能**读懂八字**：富贵贫贱、可能职业、每一步大运吉凶
- 核心概念链：**宾主 → 体用 → 做功 → 能量/效率 → 贼神捕神 → 判读层**
- 段派理法**不重视旺衰**，重视**做功**
- 段派论命的两层时间观：**先天垂象**（原局决定命格）→ **后天应期**（运年触发事件）

### 1.2 象法
- 段派象法 = 把抽象做功**翻译成现实事物**（如"巳火制金"→"刑警"或"医生"）
- 十干物象 + 十二支物象 + 六十干支自象 = 完整象法体系（M1-D-063..065, 286..288）
- 象的应用 7 大原则：共象 / 合象 / 化象 / 墓象 / 制象 / 带象 / 借象（M1-D-071..077）
- 段建业言明象法（D-24）= 干支→职业映射操作系统

### 1.3 技法
- 段派 18 大专题命题独立做成**操作手册**
- 每个专题 = "看法 → 做功要点 → 应期" 三步
- 已完全操作化：学历 / 婚姻 / 六亲 / 求财 / 职业 / 疾病 / 车祸 / 牢狱 / 伤官

## 3 · 命局判定层概览（§17 详展）

段派看命的第一步 = 命局结构识别：

```
Step 1: 宾主 4 层定位（谁是主、谁是宾）
Step 2: 体用判定（工具是什么、目标是什么）
Step 3: 做功识别（6 种方式 + 效率评分）
Step 4: 势/党判定（12 种组合 + 主神/辅神/废神）
Step 5: 贼神捕神（是否成立 + 引爆条件）
Step 6: 制法分析（5 大制法 + 夹制/远制）
Step 7: 判读层校验（6 大要诀 + "有vs没有" + 信息复证）
```

**5 大基本结构类型**（M1-D-179）：
1. 去用 40%+（制用结构）
2. 化用（杀印格等）
3. 泄用生用（食伤生财等）
4. 合用（合财合官）
5. 无用（废命/从格候选）

## 4 · 格局判定层概览（§18 详展）

```
正格 12 个：
  官印格 / 财官格 / 杀刃格 / 杀印格 / 财杀格 /
  食神制杀格 / 合杀格 / 官杀假混 / 官杀真混 /
  伤食生财格 / 身健胜财格 / 食神格 /
  伤官泄秀格 / 伤官配印格 / 官禄格（段独创）

变格 4 大类：
  从格（4 子）/ 顺格 / 化气格（5 子）/ 专旺格（5 子）

组合判定：
  过河拆桥 / 反局 / 做功层次计量 / 富贵贫贱三档
```

**核心判定原则**（M1-D-122）：
- 格局成败 = 看做功是否成功
- 不成格的命也能富贵（功大效率高）
- 成格但破格的命反差（"贵气被打"）
- **段派优先看做功，其次看格局**

## 7 · 富贵层次定级（M1 核心输出）

| 层次 | 做功要求 | 现实映射 |
|---|---|---|
| **L0 平庸** | 无势 / 有势无功 / 废神多 | 普通工薪 / 体力劳动 |
| **L1 小富小贵** | 有势 + 功 1 处，效率中等 | 个体户 / 科级 |
| **L2 中富中贵** | 有势 + 功 ≥2 处 + 效率高 | 中型企业主 / 处级 |
| **L3 大富大贵** | 制财 + 财源同制 / 食制杀彻底 | 大企业主 / 厅局级 |
| **L4 富贵双全** | 财官印全 + 主位有功 + 反客为主可成 | 行业领军 / 省部级 |
| **L5 国之栋梁** | 极特殊命格（贼神捕神成势 + 三家共识极贵） | 历史级 |

**做功层次计量**（M1-D-173）：
- 1 层功 = 百万级
- 2 层功 = 千万级
- 3 层功 = 亿级
- 4 层功 = 百亿级

**财富 4 级别**（M1-D-172）：财 < 食伤当财 < 官杀当财 < 禄当财
