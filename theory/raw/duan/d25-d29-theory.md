# D-25/D-26/D-27/D-28/D-29 深读理论提炼

## §0 · 本书独特贡献摘要

D-25~D-29 是段派 5 大专题独立小册子（总计 76KB），每本聚焦一个签字核心概念：

1. **体用六亲分论法**（D-25 独有）：宫位为体/星为用，父母体用同坏方死、妻子体多现=多婚、子女出生后体不重要只看用；天干生地支=干支同象；合用财星的身强身弱分论
2. **作功核心方法论完整版**（D-26 独有）：功神/废神概念定义、日主→日支→延伸物三级降格路径、能量与效率（物理学比喻）、有势+大功=大成就、反局定义（日柱与势对抗）
3. **应期三法则完整工程化**（D-27 独有）：出现应/引动应/干支对应表（10 干→支映射）+ 大限分段 + 大运体用 + 流年只管应期不管吉凶
4. **过河拆桥命格组合 8 例系统**（D-28 独有）：主位财重→生宾位官杀→制住官杀=发大财/升大官的完整模式 + 原局弄住功大/大运弄住功小
5. **入墓判定 + 穿的方向 + 禄神五忌**（D-29 独有）：入墓二条件（多而墓之/寅申巳亥见墓）、中神见墓以拱合论非入墓、穿与被穿看势、禄喜静五忌（不喜财/冲刑/重见/官杀/穿）、日禄归时条件严格

**与既有 M1-D-001..215 的 delta**：
- D-25 深化 M1-D-003(体用)、M1-D-041(合财) 但给出六亲分论的完整操作规程
- D-26 深化 M1-D-005(做功) 但首次给出"效率"量化维度 + 三级降格路径 + 反局精确定义
- D-27 深化 M1-D-042/043(应期) 但首次给出完整干支对应表 + 大运干支体用翻转
- D-28 是全新命格组合模式（过河拆桥），D-19 §3 官命虽涉及但未系统化
- D-29 深化 M1-D-036(墓库) 但首次给出入墓判定条件 + 穿的方向判定 + 禄五忌
