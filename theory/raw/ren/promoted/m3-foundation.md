# M3 · 任付红盲派十八道法门 · 操作手册

> **核心心法（先于一切方法）：**
> 「八字是已存在的事实，AI 的任务是 **解读**，而不是 **强加**。
>   旺衰是工具不是目的；用神是参考不是真理；能解释命主现实的方法 = 对的方法。」

> **使用方式：**
> AI 在 Step 3 深推时，按 §1 → §18 顺序过一遍 checklist；
> 每一条法门后都跟一个【AI 必查项】（boolean），全部过完才算完整推演。

## §1 · 定时论命

> 「生辰八字分乾坤，看命先把时辩清。年月日时四柱排，时辰不清别论命。」

### 要义
时辰错则全盘错。AI **必须**先确认时辰。

### 操作清单
1. 用户报"我大概是某时" → 默认时辰存疑；
2. 用 §宫位歌 + §兄弟个数 + §长相 + §头胎性别 反推 2-3 件**已发生**的事；
3. 让用户确认；不一致则换候选时辰；
4. 三次反推都对不上 → 标记 **"时辰不可定"**，进入降级模式（只做骨架定性，不做铁口断、不做应期）。

### 反推工具速查
| 工具 | 用法 |
|---|---|
| 兄弟个数 | 子午卯酉 1–2 个，寅申巳亥 3–5 个，辰戌丑未独 1（参 §8 兄弟歌） |
| 脸型 | 子午卯酉脸方尖；寅申巳亥脸长；辰戌丑未圆脸 |
| 睡姿 | 子午卯酉仰睡；寅申巳亥侧睡；辰戌丑未俯睡 |
| 头胎 | 男命：年/月/食神/伤官/官/杀位置 → 反推头胎性别（参 §盲派十排歌） |
| 父母现状 | §8 父母歌 |

### 【AI 必查项】 时辰是否锁定？ □ 是 □ 否（拒绝继续 Step 3）

## §2 · 纳音论命

> 「旺忌丧门衰怕吊，运年遇到防穿孝。不是至亲遭凶死，必是自身凶灾缠。」

### 要义
- 60 甲子纳音是**民间合婚 + 丧门吊客 + 穿孝**的快速判断器。
- 任付红派**不**用纳音定五行旺衰（与某些古法分道扬镳）。

### 操作清单
1. 报年柱 → 查纳音五行（如 1990 庚午 = 路旁土）；
2. 用纳音五行判定该命**旺/衰倾向**（口诀：纳音逢生比和吉，纳音被克逢库凶）；
3. 合婚：双方纳音相生比和吉，相克凶（仅作参考，最终以全盘宫星为准）；
4. 丧门吊客：年柱后第三字为丧门，前第三字为吊客，流年到位 → 至亲或自身有灾。

### 【AI 必查项】 纳音 + 丧门/吊客已查？ □

## §3 · 格局论命

> 「命中五行分清浊，八字格局看杂纯。纯粹轻清定富贵，驳杂重浊主夭贫。」

### 清纯歌诀（必查）
- 伤官生财不见官 → 印伤不连为清纯
- 印星化煞不见财 → 财印不连为清纯
- 官杀阳刃无食伤 → 官伤不连为清纯
- 财星去印无官杀 → 官印不连为清纯
- 食伤制杀无财印 → 印伤不连为清纯

### 五类伤官格（必查）
| 格 | 喜 | 忌 | 备注 |
|---|---|---|---|
| **土金伤官** | 见印（权力）/ 见杀（伤官制杀大权） | 见官（"局中有官下命看"） | 大运见官命归天 |
| **金水伤官** | 见官（群阴要阳） | 原局无官怕官运 | 金水伤官最好色；伤官入库库为印 |
| **水木伤官** | 财官两见富贵断 | 不见财不见官 → 满腹经纶无处展 | — |
| **木火伤官** | 活木用伤吐秀必荣昌 | 死木用伤把官伤、木焚定遭灾 | 木火伤官最傲慢 |
| **火土伤官** | 宜伤尽 → 入公门 | 伤官未尽 | 伤官佩印执掌刑律；伤官佩印军宪武职 |

### 操作清单
1. 数原局有几个克 → 越少越清；
2. 套五大伤官格 + 清纯歌诀，看是否成格；
3. **不成格不必强套** → 普通命也能富贵，关键看做功（参 §16 动门）。

### 【AI 必查项】 格局已定（清/驳）+ 是否成 5 大伤官格？ □

## §4 · 宫位论命

> 「年月日时宫位分，祖宗父母和自身。还有儿女身后事，从年到时要看清。」

参 M0 §3 宫位划分（已写）。

### 任付红派特别强调
1. **主位/宾位**：日柱（日支）= 主位，余三柱 = 宾位；
2. **吉神在主位** = 自己直接享用；**吉神在宾位** = 别人享用 / 自己间接得；
3. **凶神在主位** = 自身灾；**凶神在宾位** = 六亲灾；
4. **宫位象**也用于看身体（参 M0 §3.4）和器物（年=鞋杖、月=祖产、日=房子、时=车门）。

### 【AI 必查项】 主位 / 宾位已标 + 各柱十神归位？ □

## §5 · 十神论命

参 M0 表 + M3 §4 宫位象。

### 体用划分
- **体**：比肩、劫财、印（自身一族）
- **用**：财、官杀（自身追求的）
- **食伤**：体用兼具（既是身体延伸又能换利）

### 好的组合（吉象）
| 组合 | 含义 |
|---|---|
| 财制印 | 命局不用印时，财制印 → 求财、开店 / 制印得权、文化类 |
| 食伤制官杀 | 显贵掌权，多公检法 / 当老板 |
| 官杀制比劫 | 管理能力强、领导、军宪武职 |
| 官印相生 | 公职、国企、领导 |
| 财生官 | 富命，与政府打交道；投资金融 |
| 食伤佩印 | 文化人、学者、教师、技术人员 |
| 印制食伤 | 文化部门官员 / 企业老板 / 教师 |
| 食伤生财 | 自助创业 / 技术求财 / 企业家 |

### 坏的组合（凶象）
| 组合 | 含义 |
|---|---|
| 财坏印 | 没上进心、贪图享受、好色 / 用印时坏印克父克子犯官刑 |
| 枭印夺食 | 思维不正常、六亲不利、思想堕落（按枭金/枭木/枭土/枭水/枭火各有伤灾对应） |
| 伤官见官 | 不服约束、放荡、犯罪意识 |
| 官杀混杂 | 多病、多事、多反复 |

### 【AI 必查项】 十神好坏组合已罗列？ □

## §6 · 旺衰论命

> 「日弱财重无法弃，财又坏印无所依。哪堪日干又合财，财星伤身一生贫。」

### 要义
- **活看，不死看**：针对具体事看旺衰；
- 同一命：看财时论"财旺否"；看官时论"官旺否"；看寿时论"日主有无生扶"；
- **旺 ≠ 好，衰 ≠ 坏**。旺有旺的玩法（须制伏对手成大事），衰有衰的活法（须从势随波）。

### 旺衰 4 种典型形态
| 形态 | 处理 |
|---|---|
| 身强 + 财官印强而成势 | 富贵命；制伏对手能量越大命越高 |
| 身强 + 财官弱无气 | 普通命；强众而敌寡，势在去其寡 |
| 身弱 + 财官强而旺 | 顺从从势 → 从财从官；不顺则灾 |
| 身弱 + 印比有救 | 中等命；用印比转身 |

### 【AI 必查项】 旺衰已按当前问题分项判定？ □

## §7 · 用神论命（活用神）

> 「我们所说的用神不是固定不变的，而是针对不同的命所取用神也不相同。」

### 多用神原则
| 看什么 | 用神 |
|---|---|
| 看财 | 财星 / 食伤（财源） / 禄（无伤食时） / 官杀（被制时） |
| 看官 | 官杀 + 印 + 做功 |
| 看寿 | 日主、日支、印、禄 |
| 看学 | 印 / 食伤 / 官杀 / 财（虚透时） |
| 看父 | 偏财 / 月柱壬癸或父位 |
| 看母 | 正印 / 月柱癸壬或母位 |
| 看妻 | 正财 + 日支配偶宫；男命无财 → 食伤 |
| 看夫 | 正官 + 日支配偶宫；女命无官 → 财 / 印（特殊） |
| 看子 | 男：官杀；女：食伤；备：财（特定情况） |
| 看兄弟 | 比肩劫财 |

### 【AI 必查项】 当前问题对应的用神已锁定？ □
