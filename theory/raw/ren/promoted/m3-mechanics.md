## §15 · 作用方式法（断命的"语法"）

> 这是 M3 中**最具操作性**的一节。所有"事情"都是通过作用方式发生的。

### 15.1 生 / 克
- "生" 不一定吉，"克" 不一定凶；
- 旺生衰 = 帮扶；衰生旺 = 被泄垮；
- 旺者克衰 = 制服；衰者克旺 = 反伤自己；
- "水多木漂" = 表面相生实则成负担。

### 15.2 天干五合（合 ≠ 化）
| 合 | 现象 |
|---|---|
| 甲己 | 占有、结亲、合作、亲近 |
| 乙庚 | 同上 |
| 丙辛 | 同上 |
| 丁壬 | 同上 |
| 戊癸 | 同上 |

5 种合的子类：
- **合制**：双方旺衰悬殊，强者克去弱者（合去 / 合死）
- **合伤**：力量不悬殊，被伤之字身体不好或事不顺
- **合留**：流年合留住八字中之字 → 婚姻应期常用
- **合绊**：力量相当，紧贴相合 → 双方失原性，逢冲解
- **换象**：参 §10

### 15.3 地支六合（六组）
| 合 | 性质 |
|---|---|
| 子丑 | 闭金库（丑中辛闭气） |
| 寅亥 | 助木气；亥中壬反克寅中丙（"无情之合"） |
| 卯戌 | 戌为财官库时 → 闭库凶 |
| 辰酉 | 闭水库；酉克辰中乙木 |
| 巳申 | 长生合制效率最大；力量大者制服力量小者 |
| 午未 | 火生土旺 → 闭木库 |

口诀：「冲为冲跑，合为合克，暗中相害。」**合中容易出大事，冲中反而没事。**

### 15.4 三合（仅 4 组）
- 申子辰 → 水
- 亥卯未 → 木
- 寅午戌 → 火
- 巳酉丑 → 金

要点：
- 中神最重要（中神受制 → 局不成）；
- 中神多现 / 双现 → 不成局，转为入墓论；
- 三合无合绊；六合可解三合；
- 三合局成后**怕冲**——一字逢冲、紧贴 = 破局。

### 15.5 暗合（盲派仅认 4 组）
- **寅丑**：寅戊 + 丑癸 / 寅丙 + 丑辛 / 寅甲 + 丑己
- **午亥**：午丁 + 亥壬 / 午己 + 亥甲
- **卯申**：卯乙 + 申庚
- **子巳**：子癸 + 巳戊（注：子巳还是绝）

> 暗合在盲派 ≈ 一种隐性"做功"方式。

### 15.6 干支自合（"害死伍子胥" 8 + 2 柱）
| 柱 | 自合 |
|---|---|
| 丁亥 | 丁壬合 |
| 己亥 | 甲己合 |
| 辛巳 | 丙辛合 |
| 癸巳 | 戊癸合 |
| 壬午 | 丁壬合 |
| 甲午 | 甲己合 |
| 戊子 | 戊癸合 |
| 丙戌 | （戌逢未刑时）丙合戌中辛 |
| 壬戌 | （戌逢未刑时）壬合戌中丁 |

要点：
- 自合是合中带制 → 做功方式（不是"绊住"）；
- 支被冲则干支不能自合；
- 力量大者反制力量小者。

### 15.7 干支虚实（盲派关键概念）
- **实**：天干有根有生 → 落地、坐实
- **虚**：天干无根无生 → 漂浮、虚透

实干怕坏根；虚干怕通禄上运。

虚透含义（重要！）：
- **财星虚透** → 不主钱财，主**才华、才气、能说会道**；时上 → 能花钱
- **官星虚透** → 不为官职，主**名气、名望**

### 15.8 冲（六冲）
| 冲 | 性质 |
|---|---|
| 子午 | 午被克坏，子劳累 |
| 卯酉 | 卯被克坏，酉劳累 |
| 寅申 | 申克寅为主，寅丙、申壬同时受伤 |
| 巳亥 | 亥克巳为主 |
| 辰戌 | 土冲变虚，辰乙癸、戌辛丁同时受伤 |
| 丑未 | 土冲变虚 |

冲的属性：可破合、可冲起、可冲动、可冲开（库）；
近冲冲坏，远冲冲损；
运冲为强，岁冲为弱（动）。

### 15.9 三刑（仅 2 组）
- **寅巳申**（寅巳为主，论穿；巳申为合；寅申为冲）
- **丑戌未**（丑未为冲；丑戌、戌未论刑）

要点：
- 三刑全见时才论"刑"；
- 财官入墓库逢三刑 = 开库吉；
- 子卯归 §破，辰午酉亥归自刑（盲派一般**不**重视自刑）。

### 15.10 穿（最厉害的伤害方式）
| 穿 | 性质 |
|---|---|
| 子未 | 强 |
| 丑午 | 强（火土关系反） |
| 卯辰 | 强（穿坏卯木根） |
| 寅巳 | 弱（穿带生） |
| 申亥 | 弱（穿带生） |
| 酉戌 | 强（盲师用得最多） |

要点：
- **穿不可制，只能坏**（冲合克都可制，唯穿是不可调和的仇）；
- 穿倒 = 反——正印当偏印看、七杀当正官看；
- 酉戌穿是断肺病、断婚姻、断丢官的高频组合。

### 15.11 破（仅 3 组）
- **子卯**（病魔缠）
- **卯午**（血光难）
- **午酉**（旺午能克酉，做功）
- 子酉 **不**破。
- 破不做功（仅卯午、午酉特殊情况做功）；表无情、破坏、漏洞、破耗。

### 15.12 墓库
**核心原则：**
1. 寅申巳亥见各自墓库 → **入墓**（寅入未、申入丑、巳入戌、亥入辰）。
2. 子午卯酉见各自墓库 → 一般**不**入墓（论拱局）；除非该字极弱受伤才入。
3. **多而入墓**：同一柱或同一字出现两次以上 + 见其墓 = 入墓。
4. 天干坐墓 = **不**入墓（如辛丑不论辛入丑墓）。
5. 土墓特别：辰为水墓也为土墓；丑入辰墓、未入辰墓；丑未冲见辰则**不**入。
6. 辰戌冲 = 论冲，**不**论戌入辰墓。

> **关键：墓库不仅看五行表面，更要看库中藏干的十神**——库 = 仓库，藏的是什么"货物"决定富贵贫贱。

### 15.13 绝
- 子巳、寅酉、卯申、午亥 = 4 组绝；
- 寅酉相绝见血光；卯申相绝路途伤；午亥相绝犯口舌；子巳相绝情谊断。

### 15.14 患 / 调候 / 关财门 / 禄与原身
- **患**：患夫患妻 → 重配重娶；患父患母 → 必克。
- **调候**：大雪后雨水前出生的金木水土 = 寒，须火调候。
- **关财门**：女命关财门 → 伤夫离婚；男命关财门 → 破财伤灾。
- **禄与原身**：天干在地支为禄，地支在天干为原身。
  - **半禄关系**：丁未、癸丑两柱（丙戊共禄于巳，丁己共禄于午）；丁未、癸丑当一家看。
  - **藤萝系甲**：甲乙木为活木时可共根。

## §16 · 十二道门法（断命的"配方"）

> 法门 1–15 是"语法"，§16 是"句子"。AI 在 Step 3 输出深推时按以下门类组织。

### 16.1 动门
> 「先看日干做何事，再查日支有何意。升天入地加伏吟，刑冲合穿为动门。」

- **看日干合什么** → 阳干合财 / 阴干合官；
- **看日支动什么** → 与谁合冲穿；
- **合财合官分衰旺** → 身材两停好命相；身弱财强 = 财困主（穷）；身弱官强 = 欺主（伤灾、牢灾）。
- **绝技**：日支为家。日支透到天干 = 我派出去的人在控场。

### 16.2 格局门
（见 §3）

### 16.3 天地门
> 「一字为门天地开，五行之气地天连。」
- 一字（天干 or 地支）若能联通天地之气 → 可大用。
- 透干 + 通根 + 与日主联系 = 此字"打开了门"。

### 16.4 十天门
（见 §13；活木死木戊己庚辛壬癸各自喜忌）

### 16.5 十二地门
12 个地支各有"地门"，结合阴阳燥湿燥湿和墓库门联看。

### 16.6 统领门（富贵层次跃升）
- **官统财**：官星统了财（财少官多） → 官当财看 → 富命（级别高）
- **财统官**：财统了官（官当财看，财生杀走主位）→ 富命
- **杀统财**：制杀有功 → 财富级别高
- **一统二，二统三**：统的层数越多，级别越高
- **统领要作财王看**

### 16.7 墓库门
- 日坐墓库 = 收财官 = 上命；
- 收官 → 官贵；收财 → 富命；
- 财在年柱 → 看是财皮（年干）还是财肉（年支）；日干合到皮 / 肉 → 富贵分清；
- 墓库不开 = 死墓；冲刑 = 开库 = 活库。

### 16.8 夹拱门
- 拱禄拱贵格、拱库拱印、夹禄夹马 → 大富贵格；
- 天干同字为拱（层次好），不同为夹（次一点）；
- 这种格局**一般不论生克**——直接看结构层次。

### 16.9 旬空门
- 命主日柱旬空 → 心灵手巧、上进、漂浮无定、夫妻缘薄；
- 四柱有几柱空 → 一空 = 平庸 / 二空 = 官职不大 / 三空 = 反致富 / 四空 = 短寿；
- 用神空 → 学艺术 / 玄学 / 气功；
- 仇神空 → 凶事消融。

### 16.10 鸳鸯门（婚姻 — 详 topics/marriage.md）
**核心法则：**
- 男命财星 = 妻；女命官星 = 夫；无 → 看食伤 / 财 / 印；
- 宫星相配为美；宫坏星伤为劫；
- 阳刚阴柔相配 vs 阴阳失位婚不美；
- 财（情缘线）一断 → 男女皆是半路缘。

**典型坏组合：**
- 劫财合财 → 配偶跟别人跑
- 伤官身旺无财 → 一对鸳鸯两分开
- 阳刃克妻；阴刃妨夫
- 禄合亡劫 → 阴婚（梦中和鬼睡觉）
- 食伤包围日 → 杀夫刀剑

### 16.11 寿元门
- 用之用神不可破；
- 虚干通禄入地府；支神虚透升天堂；
- 主位衣食跑到运上 = 寿到；
- 春天万物怕午运；酉金不能走子运；
- 火炎土燥柱无水，运到水地是归期；
- 贪财坏印天元尽；众劫分财气数尽；
- 旺忌丧门，衰怕吊客；
- 四大空亡命中犯，寿元最多 33。

### 16.12 伤残门
- 年月干头克父母；日时干头伤自身；
- 日干根气坏 → 伤肢体损寿元；
- 日支化忌 → 坐轮椅；
- 丙丁逢壬癸 → 眼目不光辉；
- 子卯破病磨缠；卯午破血光难；
- 寅酉绝出血光；
- 印枭重重 + 食神运 → 灾殃。

### 16.13 牢灾门
- 日干合官看仔细（皇门 vs 牢门）；
- 日弱官强 → 官制身 → 牢门；
- 日干左右逢官杀；三合财局气数尽；
- 阳刃见五鬼 → 牢狱；
- 亡神劫煞犯官符 → 砍头；
- 金沉水底；丙遇丑地 → 徒配；
- 食伤官杀两相敌 + 柱运见财 → 牢门；
- 比肩抗杀 / 贪财坏印；
- 伤官合杀怕见财；
- 劫财伤官临官符 → 吃牢饭。

## §17 · 直定应期法

> 「命中之字到大运 / 流年出现就是应期。」

### 应期 6 大触发
1. **本字到** — 命中某字（含藏干）到大运 / 流年出现 = 该字代表事件应期；
2. **伏吟引动** — 流年伏吟原局某柱 → 该柱所代表人事应；
3. **合冲引动** — 命中有合，流年逢冲 = 应；命中有冲，流年逢合 = 应；命中既不冲也不合，流年见合 = 应；
4. **墓库开闭** — 财官入墓不发，逢冲刑开库即应；
5. **藏干透出** — 藏干通过大运 / 流年透到天干 = 应；
6. **倒象成立** — 大运或流年使原局形成倒象 → 应凶。

### 婚姻应期细分
- 配偶星合配偶宫 + 流年触发 = 应婚；
- 三合局逢拱、逢冲中神等。

### 牢狱应期
- 用之用神被破之年；
- 三合局成而冲日 / 主位之年；
- 大运合官官被印化 + 流年合制官星之年；

### 死亡应期
- 寿星合绝之年；
- 用神见劫 / 见枭夺食之年；
- 岁运并临 + 天克地冲 + 三刑齐之年；
- 主位移位、主位衣食跑掉之运。

## §18 · 命局大运流年定事法（三层叠加）

> AI 输出"未来 5 年最可能发生的事"时**必须**遵此三层结构。

### 三层
1. **原局** = 命中本来要发生什么（性别决定 + 命主一生潜在事件库）；
2. **大运** = 改变命局结构（10 年一段，开启 / 关闭某条事件线）；
3. **流年** = 引爆点（具体年份）。

### 叠加规则
- 原局**有**该事 + 大运**到位** + 流年**引爆** → **必应**（铁口断 ★★★★★）
- 原局**有**该事 + 大运到位 + 流年未到 → **窗口已开**，待引爆（★★★★）
- 原局**有**该事 + 大运未到 → **暂不会发**（★★★）
- 原局**无**该事 + 大运 / 流年仅有相似五行 → **轻微影响 / 不应**（★★）
- 大运 / 流年带新字进入命局 → 看新字与原局产生何种作用（合 / 冲 / 入墓 / 透出）

### 三层每层都查
| 层 | 查 |
|---|---|
| 原局 | 18 道法门全过一遍 |
| 大运 | 当前大运 + 前后各 1 步；伏吟、合绊、解合、引动 |
| 流年 | 眼下年 + 未来 5–10 年；应期 6 大触发 |
