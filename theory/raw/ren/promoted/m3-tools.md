## §8 · 十排歌论命（民间口诀大全）

> 这些是任付红派"铁口断"的弹药库。**用 ≠ 套用**。先用 §15-§16 通理路，再用 §8 配口诀加分。

### 父母歌（节录）
- 月财合年支犯穿，父定把双姓占（年时两头成，出生父母少一人）。
- 年月印星透到时，定是离家随母儿。
- 比劫旺相印空亡，父死母再嫁。

### 寿元歌（节录）
- 八字看寿有多端，祖师另加生死诀。
- 头重尾轻犯夭煞，求神拜佛世无药。
- 四大空亡命中犯，阳寿最多三十三。
- 回头破碎来夺命，克身命难过三十。
- 马倒禄斜：天生禄马定生死。

### 牢役歌（节录）
- 劫财伤官两格成，官杀运中犯牢绳。
- 一重破军是张口，两重破军犯徒流。
- 勾陈包局杀夹日，肩枷带锁犯徒配。

### 婚姻歌（节录）
- 望门寡：命见夫星两个墓，临上轿前哭丈夫。
- 招门女婿：月令正官合日宫，坐堂招婿来传宗。
- 独身命：月令正印临 XX，一生不婚是独身。

### 子女歌（节录）
- 食伤入墓不生育，家中有子是抱养。
- 子嗣宫绝后单一，坏了妻宫妻无子。
- 女命伤官临空亡，不生儿子只生女。
- 命中子星落空亡，亲生儿女防早丧。
- 子星值年去抱养，假子假女来传香。

### 兄弟歌（节录）
- 命占华盖兄弟孤，影只形单缺帮扶。
- 月令劫财五兄弟，月令比肩四兄弟。
- 月上正财兄弟四，月令偏财兄弟三。

### 职业歌（节录）
- 戊临戌酉做先生，己临巳酉郎中命。
- 劫财七杀两相连，早离家门去戍边。
- 伤官入库库为印，此命多为戍边人。
- 辛与丙合化成食，从魁登明开酒店。
- 食神合印百家饭，行医育人细分论。
- 戊土日元见酉戌，十有八九当老师。

### 伤残歌
- 年时两头挂伤官，困在家中行走难。

### 大运歌
- 谁到大运就说谁，缘来缘去在此运。
- 春天万物怕午运，走到南离凶事追。
- 酉金不能走子运，走到子运是死运。

## §9 · 神煞论命（任付红派精简版 · 仅 6+ 神煞）

> 「世传神煞混杂多，少是真来多是讹。门师只存六种煞，用来如神细揣摩。神煞配五行，断事有神灵。」

### 9.1 伏吟（一字重现叫伏吟）
- **核心断法**：伏吟之字所代表的人事物 = **移位** / **双姓** / **重复**。
- 例：年柱伏吟到时柱 → 此人爷爷（年柱代表）有过迁移、抱养、双姓。

### 9.2 空亡
| 类型 | 用法 |
|---|---|
| **六甲空亡** | 按日柱旬空查；阳干为"空"（轻），阴干为"亡"（重）；空亡之字代表谁，谁就有问题；空亡 + 受破 = 残疾或寿不长 |
| **四大空亡** | 甲子、甲午（水绝），甲寅、甲申（金绝）；命犯阳寿最多 33 |
| **截路空亡** | （日干 + 时支组合）专项查 |
| **互换空亡** | 年日空亡的字相同 → 最厉害 |

> 关键诀：「命有空亡多不祥，阳为空来阴为亡。空亡之字又遭破，此命生来带残伤。」

### 9.3 亡神 / 劫煞
| 三合 | 亡神 | 劫煞 |
|---|---|---|
| 亥卯未 | 寅 | 申 |
| 巳酉丑 | 申 | 寅 |
| 寅午戌 | 巳 | 亥 |
| 申子辰 | 亥 | 巳 |

诀：「亡神劫煞犯官符，合克身命遭砍头。亡劫喜冲不喜合。」

### 9.4 将军箭煞（小儿关煞）
| 季节 | 凶时 |
|---|---|
| 春 | 酉戌辰时 |
| 夏 | 未卯子时 |
| 秋 | 午寅丑时 |
| 冬 | 申亥巳时 |

诀：「一箭三岁，两箭六岁，三箭九岁，不死自损爹娘。」

### 9.5 自缢煞（破身凶亡煞）
- 戌、巳、亥、辰、未、寅 等字大忌；
- 配空亡 / 墓鬼 / 官符 / 大耗 → 凶亡（古为自缢，今为车祸 / 凶死）；
- 例：寅入未墓 + 亥入辰墓 → 短寿凶亡。

### 9.6 孤鸾煞
- 木火逢蛇：乙巳、丁巳；金猪：辛亥；土猴：戊申；木虎：甲寅；
- 渊海派加：甲寅、丙午、戊午、壬子、壬寅、辛亥、乙巳、丁巳；
- 日时同时出现 → 孤鸾。

### 9.7 华盖
- 三合局最后一字 = 华盖（实战中辰戌丑未都论）。
- 临印日时晚孤独；临印被坏精神有问题；临食伤被坏思想有问题。

### 9.8 孤辰寡宿
- 顺时针为孤、逆时针为寡。
- 男怕孤、女怕寡。

### 9.9 灭门煞 / 丧门吊客 / 病符
- 灭门煞：辰戌六十二、丑未三九月（破解：拜干亲）；
- 丧门吊客：年柱前后第三字；
- 病符：年柱后一位。

## §10 · 象数论命（盲派核心断细节工具）

> 「象就是相似之意，把有共性又曾相似的东西归为一类。」

### 7 种象法
1. **带象**：天干生地支的柱子，干支同象（12 组：甲午、乙巳、丙戌、丙辰、丁丑、丁未、戊申、己酉、庚子、辛亥、壬寅、癸卯）。
   - 必须**与日柱有关联**（合 / 生 / 禄换象 / 伏吟）才论带象；
   - 最重义：**官杀带财帽** + **印带官杀帽**。
2. **共象**：同五行 / 同十神可借象用。
3. **合象**：合中分 → 合绊 / 合动 / 合去；合中看克是真机。
4. **化象**：合而化（如 戊癸化火 等，盲派**少用合化**，重视合而不化的"绊住"含义）。
5. **借象**：午巳同火可借看；午也代母（与卯破时）等。
6. **换象**：天干五合换象 — 一方定义另一方（如劫财合财 = "不劳而获之财"；伤官合印 = "学术"；伤官合杀 = "权力"）。
7. **倒象**（最关键）：「又制又生又合又冲成矛盾 = 倒象。命运倒象必应凶，不见伤病进牢中。」
   - **原局倒象** / **大运倒象** / **流年倒象** 都凶；
   - "又制又生"= 同一星既被制又被生，自相矛盾，必有不祥。

### 象数取数字
- 看到老人 → 联想 60–70 岁；看到小姑娘 → 11–12 岁。
- 一切预测落到数 = "观其象，知其数"。
- 数字用于：身高 / 体重 / 配偶距离 / 兄弟个数 / 牢狱年限 / 应期年月。

## §11 · 三元论命（藏干必用）

> 「天元为主首，地元还复是冲刑，人元有无相战斗。」

- **天元** = 天干 8 个字
- **地元** = 地支 4 个字
- **人元** = 地支藏干（必查；不查则永远看不出隐藏的财官）

### 关键应用
1. 八字无明财 → 看墓库藏财（财库 + 制伏 = 富）；
2. 八字无明官 → 看藏干透出 → 何时透 = 何时应官；
3. 应期：藏干透到大运 / 流年 = 该字"到位" = 应事。
