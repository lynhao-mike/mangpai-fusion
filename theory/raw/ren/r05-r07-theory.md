### §2.1 聚类A · 象法取象操作体系 (M3-R-061..068)

```yaml
- rule_id: M3-R-061
  title: 十干类象取职业/疾病法
  trigger: 需要判定职业方向或疾病部位时
  conclusion: |
    每干有固定物象/人体/人性/形态4维类象：
    甲=雷/大树/首领(直) 乙=风/花木/柔情(曲)
    丙=太阳/权力/传媒(大) 丁=星星/文化/玄学(小)
    戊=大地/建筑/政府(方) 己=云/田园/自我(平)
    庚=霜/铁器/军警(棱角) 辛=月/珠宝/法律(致密)
    壬=云海/江河/贸易(无规则) 癸=雨露/泉水/谋略(圆润)
    取象时必须看与其组合的字——组合决定具体含义。
  source: R-05 §一 十干类象
  confidence_init: 4.0
  falsifiable_by: 用十干类象推断职业，实际从事完全无关行业
  cases:
    - "癸坐卯+卯生巳火+癸巳自合 → 用墨描绘色彩 = 绘画专业"
    - "丁×2透+月丁坐卯被辰穿+时丁坐亥自合被伤 → 双眼受伤 = 盲人"
  cross_ref: [M3-§10, M3-§13]

- rule_id: M3-R-062
  title: 十二支类象取行业/场所法
  trigger: 需要判定具体行业、场所或学科方向时
  conclusion: |
    地支阴阳分类决定基调：木火为阳(寅卯辰巳午未)，金水为阴(申酉戌亥子丑)。
    关键特殊规则：
    - 丑酉=阴中之阴 → 犯罪/黑社会/牢狱/坟墓/玄学/淫邪
    - 辛酉日主见丑时=黑社会(阴见阴又入墓)
    - 辛丑日主≠黑社会(浊中见清)
    - 戌亥=计算/数学(入乾门)；丑=玄学；木火=文；金水=理
    - 辰见子=化学；申酉=法律；申=金融(酉不代表金融)
    - 辰=运动/机器/车辆
  source: R-05 §二 十二支类象
  confidence_init: 3.5
  falsifiable_by: 按支类象判断学科方向，实际完全相反
  cases:
    - "辛酉日见丑时 → 黑社会"
    - "戌亥组合 → 数学/计算"
  cross_ref: [M3-§10, M3-§14]


- rule_id: M3-R-063
  title: 干支组合取象法——组合决定含义
  trigger: 单字有多种象，需要确定具体取哪个象时
  conclusion: |
    每字都有多种象，具体取哪一个由与其组合/关联的字决定：
    1. 看该字在命中表述的十神含义
    2. 看与该字组合(合/生/克/穿/冲)的另一字
    3. 两字象的交集 = 最终取象
    例：丁丑组合——丁若为官杀+丑 → 丁=盗贼；丁若取眼象+丑 → 近视/眼盲
    例：甲申组合——甲=头+申=骨/脊椎 → 颈椎
  source: R-05 §一/§二 (案例解析通则)
  confidence_init: 4.5
  falsifiable_by: 按组合取象推断具体事项，实际完全不符
  cases:
    - "丁(眼)+丑(浊) → 近视"
    - "甲(头)+申(骨) → 颈椎"
    - "癸(墨)+巳(色彩)+自合 → 绘画"
  cross_ref: [M3-§10, M1-D-204]

- rule_id: M3-R-064
  title: 带象取职业法——天干生地支柱
  trigger: 判断命主职业或配偶职业时
  conclusion: |
    带象 = 天干生地支的柱子(12组)。
    带象的关键应用：
    - 印带官帽(印星头上坐官杀) → 有职务/在单位工作
    - 官带财帽(官星头上坐财) → 与财有关的官/企业
    - 财带官帽 → 行政官
    - 带象入库 → 控制/管理该类事物
    带象必须与日柱有关联(合/生/禄换象/伏吟)才论。
  source: R-05 §一 + R-06 提示
  confidence_init: 4.0
  falsifiable_by: 用带象判定职业，实际从事完全无关
  cases:
    - "时柱印带官帽+入库 → 开门店做生意(控制财源)"
    - "年柱印带官帽+巳火印化官杀入库 → 银行票据工作"
    - "乙巳=印带杀帽入坐支库 → 管理金融类(兽医=父业传承)"
  cross_ref: [M3-§10.1, M1-D-204]

- rule_id: M3-R-065
  title: 十神组合取象法——对应关系决定含义
  trigger: 确定十神在当前命局中的具体含义时
  conclusion: |
    十神象因对应关系不同而变化：
    - 食伤+印 → 学问/学术
    - 食伤+财 → 财源/技术/赚钱手段
    - 官杀+印 → 权力/职务
    - 官杀+财(官统财) → 富命(级别高)
    - 比劫+财 → 竞争/合作/分财

  falsifiable_by: 按十神组合推断，实际表现完全相反

    - "食神入辰库+控制 → 学校管后勤"
    - "禄当财+头带印+时柱门户 → 开门店做生意"
  cross_ref: [M3-§5, M3-§16.6]


- rule_id: M3-R-066
  title: 同五行借象法
  trigger: 原局某字无直接象时，需要找替代取象
  conclusion: |
    同五行可借象使用：
    - 庚辛同属金 → 辛可当庚看(需有申存在时酉也可当根)
    - 午巳同属火 → 可借看
    - 丑辰同属土 → 可借宫位
    前提：必须在命局中有逻辑关联(同透/同根/相邻/有合冲关系)。
    应用：当禄神/印星/财星在原局只有一个，但需要看两个时(如双腿)。
  source: R-05 §三 禄神类象 + §七 宫位类象案例
  confidence_init: 3.5
  falsifiable_by: 借象推断后实际不符
  cases:
    - "壬通根在申+酉也可当根看(有申存在) → 申酉=双腿都残"
    - "辛坐未=庚坐未(同金可借) → 看桃花对象家庭状况"
  cross_ref: [M3-§10.5]

- rule_id: M3-R-067
  title: 伏吟取象法——一字重现定移位/双姓
  trigger: 命局中同一天干出现在不同柱位时
  conclusion: |
    伏吟(同字重现到他柱)取象铁律：
    - 所代表的人事物 = 移位/双姓/重复
    - 年柱字伏吟到时柱 → 祖辈迁移/抱养/双姓
    - 宫位伏吟 → 该宫代表事物有两次/两个
    - 辰伏吟到时 → 两次手术(两辰穿两次卯)
    伏吟+坐支状态(旺衰)可判断大小/先后。
  source: R-05 §七 宫位类象案例(卵巢手术例)
  confidence_init: 4.0
  falsifiable_by: 伏吟判为重复/移位，实际只有一次且未迁移
  cases:
    - "乙伏吟到年+年乙坐丑与酉合 → 母也在年柱体现"
    - "两辰穿卯 → 两次卵巢手术"
  cross_ref: [M3-§9.1]

- rule_id: M3-R-068
  title: 十二长生取象——旺衰定大小
  trigger: 同一十神出现多个，需判断哪个大/哪个小时
  conclusion: |
    用十二长生位判断同类字的大小/强弱：
    - 坐长生/帝旺 → 大/强/有气
    - 坐绝地 → 小/弱/有形无气
    应用：年乙坐酉(绝地)=小，时乙坐亥(长生)=大
    → 判断耳朵右大左小(年=左，时=右)。
  source: R-05 §一 十干类象(耳朵大小例)
  confidence_init: 3.5
  falsifiable_by: 按长生位判大小，实际相反
  cases:
    - "年乙坐酉绝=小耳+时乙坐亥长生=大耳 → 右耳大左耳小"
  cross_ref: [M3-§14]
```

```yaml
- rule_id: M3-R-069

  conclusion: |
    禄神3层类象：

  conclusion: |

- rule_id: M3-R-071
  title: 墓库方向法——从库中出来vs入库

  conclusion: |
    墓库不仅看"入墓"，更要看"方向"：
    1. X入Y库 → X被收藏/控制(常规)
    2. X从Y库中出来 → X的来源是Y库代表的人/事
    应用：
    - 日支杀入丑库 → 可理解为杀从丑库出来 → 工作来自父(丑看父)
    - 辰土水库+水多入辰 → 自己控制这些水(=控制财)

  source: R-05 §五 墓库类象

  falsifiable_by: 按墓库方向判来源/归属，实际来源完全不同
  cases:

    - "水多入辰库+丑财库随入 → 银行(管财)/接父班"

  title: 空亡类象——有气无形/务虚/减半
  trigger: 命局中某柱逢旬空时

    - 日支空 → 虎头蛇尾/夫妻缘薄

  source: R-05 §六 空亡类象

  cross_ref: [M3-§9.2, M3-§16.9]

- rule_id: M3-R-073
  title: 十神正面/负面双性——有用为吉无用为凶
  trigger: 判断命主性格、适合职业时
  conclusion: |

    - 正印有用 → 公务员/教师/宗教/慈善；过重 → 依赖/惰性/无主见
    - 偏印有用 → 技术/医生/玄学/律师；过重 → 自私/冷淡/不通人情

    - 偏财有用 → 生意/企业/证券；过重 → 贪欲
    - 比肩有用 → 运动/中介/司机；过旺 → 固执/刚愎
    - 劫财有用 → 运动/军人/券商；无用 → 赌徒/骗子
    - 食神有用 → 学者/医生/演艺；不怕旺只怕克
    - 伤官有用 → 艺术/设计/律师；太旺 → 放荡/犯罪意识
  source: R-05 §八 十神类象(①~⑩)

  cross_ref: [M3-§5]

- rule_id: M3-R-074
  title: 半禄关系——丁未/癸丑当一家看
  trigger: 命局中出现丁未或癸丑柱时
  conclusion: |
```
