# 交叉验证操作手册 · Cross-Validation Manual

> **核心原则**：每个结论都必须经过至少两个维度的审视。
> 不是"M1说完M2再说"，而是"同一组合三家各出结论，再比对"。

---

## 1 · 何时触发交叉验证

对八字中**每个关键组合**都必须触发：

| 组合类型 | 示例 |
|---|---|
| 天干五合 | 甲己合/乙庚合/丙辛合/丁壬合/戊癸合 |
| 地支六冲 | 子午/丑未/寅申/卯酉/辰戌/巳亥 |
| 地支三刑 | 寅巳申/丑戌未/子卯 |
| 地支六穿 | 子未/丑午/寅巳/卯辰/申亥/酉戌 |
| 制克关系 | 食制杀/印克食/财坏印/官克比 |
| 墓库 | 辰戌丑未的开/闭/入/出 |
| 拱夹 | 拱禄/拱贵/夹贵 |

## 2 · 交叉验证三步流程

### Step 1: D1视角（M1段派问）

```
对这个组合：
→ 这是什么做功方式？（制/化/生泄/合/墓/复合 中的哪种？）
→ 效率如何？（高/中/低 — 距离/力量/有无废神阻挡）
→ 能量级别？（做几层功？对应什么量级？）
→ 对命局结构是好还是坏？（有功=好/无功=废/反噬=坏）
```

### Step 2: D2视角（M2杨派问）

```
对这个组合：
→ 产生什么画面？（合=得到/冲=变动/制=制约/穿=暗伤…）
→ 对应什么人/什么事？（看十神属性→人物关系）
→ 暗引什么？（显象→暗引→真实背景）
→ 调候方向？（需要什么五行来平衡？）
```

### Step 3: D3视角（M3任派问）

```
对这个组合：
→ 属于§15作用方式14法中的哪种？
→ 属于§16十二道门中的哪个门？
→ 什么时候激活？（§17应期6触发中哪个适用？）
→ 三层叠加是否齐备？（原局有✓/✗ + 大运到位✓/✗ + 流年引爆✓/✗）
→ 如果是穿：不可调和？如果是倒象：必凶？
```

### Step 4: 比对与聚合

| 比对结果 | 处理方式 | 置信度 |
|---|---|---|
| 三家结论一致 | 直接采用，进铁口断区 | ★★★★★ |
| 两家同指+一家不反对 | 采用，标高置信 | ★★★★ |
| 各说各的但不矛盾 | 合并为立体结论（级别+画面+时间） | ★★★★ |
| 各说各的且部分矛盾 | 查CF-*裁定，有则按裁定执行 | ★★★ |
| 三家明确分歧无CF | 降级+入rule-conflicts | ≤★★★ |

## 3 · 典型交叉验证示例

### 示例1：丙辛合在日时柱

| 维度 | 问 | 答 |
|---|---|---|
| D1 | 做功方式+效率 | 合用结构（第4种），丙辛合化水——效率取决于是否真化 |
| D2 | 画面+人事 | 丙辛化水→酒/化学品/化工/流通；合财=得妻得财 |
| D3 | 分类+应期 | §15.2天干五合第X子类；逢水旺年引动合化 |
| **叠加** | | D1定能量级别 + D2定画面人事 + D3定时间节点 |

### 示例2：日支与时支相冲（如子午冲）

| 维度 | 问 | 答 |
|---|---|---|
| D1 | 对命局好坏 | 冲去废神=好（做功）/ 冲去用神=坏（破格） |
| D2 | 什么画面 | 动象——搬家/换工作/婚变/走动（看冲什么星） |
| D3 | 什么时候引爆 | 逢合年解/逢冲年加倍/三层叠加定是否真应 |
| **叠加** | | D1定好坏 + D2定什么事 + D3定什么时候 |

### 示例3：食神制杀格

| 维度 | 问 | 答 |
|---|---|---|
| D1 | 做功效率+层次 | 制功+身旺有势→L2-L3；效率取决于食杀距离 |
| D2 | 职业画面 | 五步法→技术管理；暗引→有副业/第二收入 |
| D3 | 成格验证+应期 | §3格局门清纯歌诀验证；§17制杀到位年=发达年 |
| **叠加** | | D1定中富/大富 + D2定什么行业什么人 + D3定哪年发达 |

### 示例4：戌中辛金被冲

| 维度 | 问 | 答 |
|---|---|---|
| D1 | 库被冲=做功? | 若辛是配偶星→冲开配偶库→见配偶；但若辛是财→冲散财 |
| D2 | 画面 | 若辛=正财=配偶：冲年见人/结婚；若已婚→配偶出问题 |
| D3 | 应期 | §16.7墓库门：冲开=应事；§17本字到+逢冲 |
| **叠加** | | 需结合已知婚姻状态判定画面→D1定好坏→D3定何年 |

### 示例5：穿（酉戌穿）

| 维度 | 问 | 答 |
|---|---|---|
| D1 | 做功？ | 穿=不做功的伤害，效率低但持续，属于"暗伤" |
| D2 | 画面 | 持续性慢性病/暗中矛盾/同事关系暗斗（看穿的星属性） |
| D3 | 特殊规则 | §15.11穿的不可调和性——冲合克都可制，唯穿不可调和 |
| **叠加** | | D1定"不是做功是暗伤" + D2定"什么类型的暗伤" + D3定"不可解只可避" |

## 4 · 交叉验证的输出格式

每个关键组合验证完毕后，内部记录格式：

```
【组合】XX合/冲/穿XX
  D1结论：____（效率/级别/好坏）
  D2结论：____（画面/人事/暗引）
  D3结论：____（分类/应期/三层叠加）
  比对结果：一致/互补/矛盾
  聚合置信：★★★★★/★★★★/★★★/...
  最终结论：____
```

此格式为内部推理链，不直接输出给用户。用户只看到聚合后的结论+推理链摘要。

## 5 · 特殊处理规则

### 5.1 时柱存疑时

- D1/D2/D3中涉及时柱的结论一律降级1星
- 不在时辰未定时对时柱组合下铁口断
- 明确标注"本结论依赖时柱，时辰存疑则可能不成立"

### 5.2 同一字参与多个组合时

- 分别对每个组合做交叉验证
- 最后综合判定"这个字在命局中的核心角色是什么"
- 多重身份的字=命局关键字，需要特别关注大运流年何时引动

### 5.3 大运/流年引入新组合时

- 原局组合+大运新组合 = 同时做交叉验证
- 看新组合是"加强原局做功"还是"破坏原局结构"
- D3的三层叠加此时最关键
