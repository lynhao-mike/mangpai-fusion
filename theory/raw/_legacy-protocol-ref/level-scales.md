# 层级标尺 · Level Scales（学业 / 事业 / 财富）

> **定位**：本文件为分析报告中「层次定性」的唯一量化参照标尺。
> AI 在 Step 2（M1 骨架定层次）和 Step 7（输出校验）时**必须**参照本标尺给出精确层级。
>
> **与 M1 L0-L5 的关系**：
> - M1 L0-L5 是段派的「做功效率」抽象层级
> - 本标尺是「现实映射」精细层级（5级15档制）
> - 两者互补：M1 定骨架 → 本标尺定具体档位

---

## 一、学业精细层级标尺（5级15档）

### 特高级（顶尖高知）

| 子等 | 对应学历 | 命理典型配置 |
|---|---|---|
| 上等 | 清北名校、常春藤名校、博士后 | 官杀配印极清纯+食神制杀彻底+印做功效率极高 |
| 中等 | 国内顶尖985博士、名校统招博士 | 官杀配印清纯+印星有力做功+食伤秀气 |
| 下等 | 顶尖名校全日制硕士 | 印星做功+食伤泄秀+官杀有制 |

### 高级（优质学历）

| 子等 | 对应学历 | 命理典型配置 |
|---|---|---|
| 上等 | 985/211重点本科 | 印星做功明确+学历星（官杀/印/食）得力 |
| 中等 | 省属重点一本、双一流本科 | 印星有根有气+食伤不破印 |
| 下等 | 普通一本、专业硕士 | 印星虽有但不够清纯，或大运补学历星 |

### 中级（大众高等学历）

| 子等 | 对应学历 | 命理典型配置 |
|---|---|---|
| 上等 | 公办二本 | 印星存在但力量不足/位置不佳 |
| 中等 | 民办二本、三本院校 | 印弱或被财克/学历星受损但仍有残余 |
| 下等 | 公办全日制大专 | 印星虚透或入库/学历星受制明显 |

### 初级（中等学历）

| 子等 | 对应学历 | 命理典型配置 |
|---|---|---|
| 上等 | 重点普通高中 | 印星微弱/比劫夺印/财坏印但有食伤 |
| 中等 | 普通普通高中 | 无有效印星做功/学历星全无力 |
| 下等 | 职高、中专、技工学校 | 比劫或食伤为体无印/从格无印 |

### 无学历（低层学历）

| 子等 | 对应学历 | 命理典型配置 |
|---|---|---|
| 上等 | 初中学历 | 印星被完全制住/财坏印彻底/从财格 |
| 下等 | 小学及以下、早年辍学 | 无印无食伤/比劫独旺无教化/七杀无制身弱 |

### 学历判定规则映射

| 规则 ID | 规则名 | 对应层级影响 |
|---|---|---|
| M1-D-241 | 学历三星判定法（官杀/印/食 vs 财/伤/比） | 三星齐→高级以上；缺一降一级 |
| M1-D-242 | 印星做功=有学历/不做功=懒笨 | 印做功→至少中级上等 |
| M1-D-243 | 文理区分法（金水=理/木火=文） | 不影响层级，影响方向 |
| M1-D-244 | 官杀配印=学业有成标准结构 | 标准结构→高级中等以上 |
| M1-D-245 | 伤官配印做功=高学历 | →高级上等~特高级 |
| M1-D-246 | 大运补学历星=考学应期 | 大运补可升1-2档 |

## 二、事业平台权力精细层级标尺（5等15档）

### 一等：顶级掌权格局

| 子等 | 对应平台/职位 | 命理典型配置 |
|---|---|---|
| 上等 | 省部级高层领导、集团董事长、行业龙头领袖 | M1-L4~L5 + 贼神捕神成立 + 过河拆桥 |
| 中等 | 厅局级干部、跨国企业总裁、头部企业创始人 | M1-L3~L4 + 杀印相生极清 + 食制杀彻底 |
| 下等 | 实权市级领导、大型集团总经理 | M1-L3 + 官命5结构之一 + 做功层次3+ |

### 二等：中层管理格局

| 子等 | 对应平台/职位 | 命理典型配置 |
|---|---|---|
| 上等 | 正处级、大厂副总、大型企业股东老板 | M1-L2上~L3下 + 官命结构成立 + 印护有力 |
| 中等 | 副科级/正科级、企业部门总监、中型企业主 | M1-L2 + 杀印或财官格 + 吃皇粮/合官 |
| 下等 | 基层实职干部、门店总负责人、小型企业老板 | M1-L1~L2 + 有官但力量不够大 |

### 三等：专业技术格局

| 子等 | 对应平台/职位 | 命理典型配置 |
|---|---|---|
| 上等 | 主任医师、资深律师、高级工程师、资深公职骨干 | M1-L2 + 食伤做功为主 + 印护 + 金水/木火伤官格 |
| 中等 | 普通医师、执业律师、中级工程师、在编办事人员 | M1-L1~L2 + 食伤有力但非极清纯 |
| 下等 | 基层技术人员、普通技术岗、合同制专业岗 | M1-L1 + 食伤存在但做功效率偏低 |

### 四等：普通从业格局

| 子等 | 对应平台/职位 | 命理典型配置 |
|---|---|---|
| 上等 | 资深白领、稳定正式岗、优质个体户 | M1-L0上~L1 + 有一定做功但层次有限 |
| 中等 | 普通上班族、本地小商户、稳定外勤人员 | M1-L0 + 有功但无势/或有势无功 |
| 下等 | 临时职员、外包人员、零散自由从业者 | M1-L0 + 废神多功神少 + 财弱无根 |

### 五等：漂泊无业格局

| 子等 | 对应平台/职位 | 命理典型配置 |
|---|---|---|
| 上等 | 工作不稳定、频繁跳槽、勉强糊口 | 无势无功 + 驿马被冲 + 官印全破 |
| 中等 | 无固定主业、打零工、短期务工 | 从格不清/破格 + 比劫独旺无出路 |
| 下等 | 长期待业、无所事事、无稳定收入来源 | 用神全破 + 禄克财 + 七杀无制身弱极 |

## 三、财富财运精细层级标尺（5级15档）

### 巨富级（亿级资产）

| 子等 | 对应财富量级 | 命理典型配置 |
|---|---|---|
| 上等 | 十亿以上身家、顶级资本大佬、商业巨头 | M1-L4~L5 + 过河拆桥 + 做功4层 + 财官印全 |
| 中等 | 3-10亿身家、上市公司大股东 | M1-L3~L4 + 做功3层 + 官杀库开 |
| 下等 | 1-3亿身家、知名企业掌舵人 | M1-L3 + 做功3层 + 食伤库开 |

### 大富级（千万资产）

| 子等 | 对应财富量级 | 命理典型配置 |
|---|---|---|
| 上等 | 5000万-1亿身家、资深实业企业家 | M1-L2~L3 + 做功2层+ + 旺杀制住 |
| 中等 | 2000万-5000万身家、高净值中产老板 | M1-L2 + 做功2层 + 财库开+身旺 |
| 下等 | 千万左右身家、优质企业股东 | M1-L2 + 做功1-2层 + 食伤当财旺 |

### 中富级（百万资产）

| 子等 | 对应财富量级 | 命理典型配置 |
|---|---|---|
| 上等 | 500万-1000万资产、城市中产精英 | M1-L1~L2 + 做功1层+ + 官杀当财/食伤生财 |
| 中等 | 200万-500万资产、家境殷实小康上层 | M1-L1 + 有功有势但效率中等 |
| 下等 | 100万-200万资产、高薪稳定富足家庭 | M1-L1 + 有一定做功 + 印护财不散 |

### 小富级（安稳富足）

| 子等 | 对应财富量级 | 命理典型配置 |
|---|---|---|
| 上等 | 年收入30万以上、体制内高薪、家境安稳无忧 | M1-L0上~L1 + 有食伤生财或官印格稳定 |
| 中等 | 年收入15-30万、普通优质小资生活 | M1-L0 + 有一定做功 + 财星有根 |
| 下等 | 年收入8-15万、温饱有余、日常安稳度日 | M1-L0 + 体用平衡 + 无大破坏 |

### 贫困负债级

| 子等 | 对应财富量级 | 命理典型配置 |
|---|---|---|
| 上等 | 收支持平、无存款、勉强维持生计 | 无势无功 + 财弱无根 + 比劫夺财 |
| 中等 | 收入微薄、常年拮据、少有结余 | 禄克财 + 财无源 + 食伤不生财 |
| 下等 | 常年破财、入不敷出、背负外债 | 禄冲财 + 劫财当道 + 财库被合住无法开 |

## 四、层级输出格式标准

### 静态层级标注（用于骨架定性）

```
【学业层级】：高级·上等（985/211重点本科）★★★★
【事业层级】：二等·中等（正科级/企业总监级）★★★★
【财富层级】：中富·上等（500万-1000万资产）★★★★
```

### 动态层级标注（标注大运阶段性变化）

```
- 25-35岁（甲寅运）：事业二等·下等 → 基层实职干部
- 35-45岁（乙卯运）：事业二等·上等 → 正处级晋升期
- 45-55岁（丙辰运）：事业一等·下等 → 市级实权领导
```

## 六、使用规则

1. **每次分析报告的骨架定性（Layer 1）必须包含三维层级标注**
2. **动态层级标注至少覆盖当前大运 + 下一步大运**
3. **层级判定必须有规则依据**（标注 M1-D-XXX 或 M2-Y-XXX）
4. **层级可升可降**：大运补星可升1-2档；大运破格可降1-3档
5. **地区修正**：同一命局在不同经济区域，财富绝对值有差异（一线城市 vs 三线城市），标尺以全国中位数为基准
6. **时代修正**：本标尺以2020年代中国经济为参照，历史命例需按时代购买力折算
