# 报告模板 v1.4 · mangpai-fusion

> v1.4 变更说明（修复 C-2026-016 发现的 12 项格式/内容问题）：
>
> 1. **Report vs Analysis 分离**：Report（client版）= 精简中等版（核心结构+五维+铁口+画像+共识）；Analysis（master版）= 完整技术版（含全部推导+反馈位）
> 2. **表格紧凑**：所有表格行之间禁止空行
> 3. **十神暗引**：去除阿拉伯数字序号，直接以"·"列出
> 4. **神煞分布**：必须以年柱/月柱/日柱/时柱横向四列表格排列
> 5. **体用结构**：用表格形式展示（体/用分行，加粗标题）
> 6. **五维定位层次标尺**：
>    - 学业：初中未毕业/初中/高中·中专/大专/民办三本/二本/一本/省重点·211/985/C9·清北
>    - 事业：给出具体方向/职业
>    - 婚姻：早婚/晚婚/二婚/多婚/幸福美满/婚姻不顺
>    - 官命：事业单位/普通公务员/副科/正科/副处/正处/副厅+
> 7. **应期总表强弱符号**：用【强】【中】【弱】替代 ▲●▽
> 8. **铁口断语**：每条必须写明"应期事件：YYYY年 命主N岁，具体事件描述"
> 9. **旁证补强（高派）**：必须以表格列出神煞+挂柱+事件指向+置信度
> 10. **命主画像**：用清晰表格展示（禁止╔╗框架）
> 11. **应期共识总结**：必须列出共识最高年份+事件+吉凶可能性（无论好坏）
> 12. **候选事件表格**：事件列需给出具体描述（不能只写"事业转折"，要加括号说明）

## 格式硬性规则清单（v1.4 起强制执行）

| # | 规则 | 违反后果 |
|---|------|----------|
| F1 | 表格行之间禁止空行 | 表格渲染断裂 |
| F2 | 十神暗引禁止阿拉伯数字序号 | 阅读混淆 |
| F3 | 神煞分布必须横向四柱表格 | 信息松散 |
| F4 | 体用结构必须表格化 | 信息不突出 |
| F5 | 五维定位必须用具体层次描述 | 抽象无意义 |
| F6 | 强弱符号用【强】【中】【弱】 | 符号不清 |
| F7 | 铁口断语必须写明年份+岁数+具体事件 | 应期无着落 |
| F8 | 旁证必须表格化（神煞+挂柱+事件+置信度） | 信息混乱 |
| F9 | 命主画像禁用╔╗框架，用表格 | 显示混乱 |
| F10 | 必须有应期共识总结（含吉凶+可能性） | 核心信息缺失 |
| F11 | Report ≠ Analysis（Report精简，Analysis完整） | 内容重复无意义 |
| F12 | 候选事件表格事件列必须加括号具体描述 | 信息模糊 |
