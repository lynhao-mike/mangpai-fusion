# 八字输入模板 · 问真八字 APP 信息提取规范 · v1.0

> 命主八字从问真八字 APP 排盘后，按本模板整理信息再贴入会话。  
> 该模板**最大化保留信息密度**，避免 Kiro 来回追问。

---

## 使用说明

1. 打开问真八字 APP → 排盘
2. 按下方"模板正本"逐项填写（不知道的字段填 `[未知]`）
3. 把整段文字直接贴入对话
4. 在"分析重点"末尾说明你的核心问题（决定走策略 A/B/C）

填写时**遵循 GIGO 原则（Garbage In Garbage Out）**：信息缺失会直接降低置信度。
能查到的字段一定要填全。

## 模板正本（直接复制使用）

```
# 八字盘信息

## 基础信息
- 性别：[男/女]
- 公历生日：[YYYY-MM-DD HH:MM]（精确到分钟最好）
- 农历生日：[YYYY 年 X 月 X 日 X 时]
- 出生地：[省 + 市 / 国家]（用于真太阳时校正）
- 是否真太阳时：[是/否]（若不是请说明用了什么时区校正）

## 四柱（问真八字 APP 排盘结果直接复制）

- 月柱：[天干][地支]
- 日柱：[天干][地支]
- 时柱：[天干][地支]

## 大运排布（必填）

- 文昌：[字 / 位置]

- 是否在体制内 / 公务员：[是/否]，若是 [级别 / 职级]
```

## 必填项 vs 选填项

### 必填（缺一项 → 置信度上限 ★★★★，无法到 ★★★★★）

- 性别
- 四柱（年月日时全部）
- 公历或农历生日（至少一种）
- 大运起运岁数 + 至少 3 个大运排布
- **婚姻状态**（极其关键，影响"已婚态分流"判定）
- 分析重点

### 强烈建议（缺失会大幅降低应期置信度）

- 出生地（用于真太阳时）
- 历史重大事件（≥3 件，用于回测校准）
- 职业 / 学历 / 健康现状

### 选填（增加判定维度）

- 神煞详细列表
- 父母兄弟八字
- 配偶八字（合婚专题用）

## 信息隐私

本系统**默认对所有命主信息保密**：
- 案例归档时姓名 → 化名（命主 1 / 命主 2 / ...）
- 出生地 → 仅保留省级
- 公开报告中 → 隐去精确生日（仅保留四柱 + 年份）

如命主希望完全隐私 → 在分析重点末尾标注 `[隐私模式]` → 报告不归档到 cases/ 公开目录，仅存于 `cases/_private/`。

## 反馈采集（分析后请回填）

```
## 反馈
- 关于 [断语 X]：[应验/失验/部分应验]
  详情：[命主原话或事实]
- 关于 [断语 Y]：...

## 新增信息（命主对话中提供的额外信息）
- ...

## 命理师评价（你的判断）
- [对本次分析的整体评估]
```

详见 `templates/feedback.md`（M7 阶段产出）。

## 质量自查清单

贴入前请自查：
- [ ] 性别 ✅
- [ ] 四柱完整 ✅
- [ ] 生日精确（误差 ≤ 30 分钟）✅
- [ ] 大运排布 ≥ 3 个 ✅
- [ ] 婚姻状态明确 ✅
- [ ] 历史重大事件 ≥ 3 件 ✅
- [ ] 分析重点 1-3 句话 ✅

7 项全打勾 → 期待报告置信度可达 ★★★★★  
缺 1-2 项 → 置信度上限 ★★★★  
缺 3+ 项 → 建议补全后再分析
