## 注意事项

1. **只读**：本目录文件不应被修改。如发现错误，请记录于 `META/source-trace.md` 的勘误段。
2. **追溯**：每条 `theory/` 中的规律必须能在本目录中找到原文出处。
3. **隔离**：子平格局派、滴天髓调候派资料应分别投放到各自子目录，不与盲派资料混放。
4. **提取**：从原始教案提取规则时，优先使用 `templates/theory-extraction-template.md`。
5. **隐私**：原始文档不含个人八字数据，可公开。

详细来源记录见 [`../META/source-trace.md`](../META/source-trace.md)。
