# 第一章：应期

看八字要先看原局，原局定富贵贫贱，大运流年看应期。

原局中年、月、日、时表示时间发生的顺序，称原局应期，也为大限应期。

一般而言，大运表吉凶，流年表应期。大运以生克作用为主，要注重大运对原局的作用，当大运作应期时，一般是原局的东西在大运出现，被原局反作用。

流年看应期。命运中一个重大事情的发生首先在原局有体现，然后在大运中又有体现，这才可以在流年中找。依据流年应期规律确定其发生时间。

# 大限应期：

年：1—18岁，月：18—35岁，日：35—55岁，时：55岁以后。

大运应期:

■ 大运干支生、克、刑、冲、穿、墓、破、三合、六合、五合，作用于八字定吉凶。

八字中字在大运中以禄、原身、藏干形式出现，被八字作用，定吉凶。

大运干支有体用的关系，行天干运，以天干为用，地支为体；行地支运，以地支为用，天干为体。

流年应期：

■ 流年只管应期，不管吉凶，吉凶由原局与大运管。

流年与八字作用有生、克、刑、冲、穿、墓、破、三合、六合、五合，这种作用只是引动，必须配合大运与原局定吉凶。流年出现原局的禄、原身、藏干，也是引动。

# 应期的三种方式：

出现为应期：原局的东西，在大运流年出现，表示这个字到了，这个字的

吉凶会在本运或本年出现。

引动为应：流年冲合可以引动八字中的字；通过合局，与墓之气到位，还可以引动墓库。

(申、酉可引动丑，亥、子可引动辰……)

干应支，支应干（后文有述）

四、命局出现申、酉组合而为官者，多数与执法有关的官职。有寅丑组合容易是公安。

# 第四章：财命的看法

# 第一节：财富的取法

命有无官者，但没有无财者。凡命或多或少都有自己的财富。除财星之外，可以当财富看的星还有很多。

1、官杀可当财富看。官杀的财富级别要比财星的多一个数量级，比如，财星是百万级，官杀就可以到千万级。不是所有的官杀都可当财，只有日主有用的官杀才可当财。

2、伤官、食神可以当财富看。它们是财的原神，当财看时，其财富级高于财星。但不及官杀。

3、比肩与禄神可以财富看。八字无财、无官杀时，比禄可以财富。喜禄印相生，必有福禄。

# 官杀当财看的几种情况：

1、官杀制不尽当财看。食伤制官杀局，官杀制尽可当权力看，制不尽可当财看。

2、阳日主，七杀当财富看，如戊土日见甲之类。

3、官统财、杀统财，官杀当财富看。

官杀当财富，级别一般会达千万。

# 官杀当财富例1

乾造：戊己戊甲

# 第二节：取财方法

1、经营取财。指商贸、生产、经营、开发型企业；开门店及销售型企业。

经营取财的组合较多：伤食生财取财、制官杀取财、制财或财的原神取财、还有的印星参与做功，表示所经营企业有相当的规模。

商贸型企业多数是有相冲做功，相冲是往来互换之意。生产型企业多数有内食神或伤官。开门店多数是在年柱或时柱出现印星与财星做功。制官杀或杀星当财，多数经营项目较多。而地产与矿产企业，则与土、金相关。

2、风险取财。指投资股票、期货方面。也有是金融风险投资。还有博彩或赌博。

■ 风险取财一般做功之神要有比肩、劫财参与。

3、智力取财。一般是指靠人的思想、技术、知识来取财，包括艺术家、作家、设计师、咨询师、律师、教师等。

智力取财一般作功之神是伤官、食神参与。

4、体力取财。体力劳动者或运动员都属于此。

体力取财一般做功之神有比肩、劫财或禄神。运动员还有伤官参与做功。

5、还有一种职业是半体力半脑力型的，如司机、护士等。做功之神与伤食与比劫相关。

# 第七章 车祸专辑

车祸的八字，都会在原局有所体现，首先要在原局中找到车象。
