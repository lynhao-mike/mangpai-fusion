# 法无定法 唯变是法

命理的推演绝非数学演算，可以套用公式。“年年岁岁花相似，岁岁年年人不同。”两句诗词道出一个永恒的真理，即宇宙中、天地间的万事万物都是变化的，唯一不变的就是“一直在变”，所以学习八字预测一定要具有一种“变”的思维。《易经》的“易”字就是日月交替，表达的就是不停的变。世界上没有任何事物是绝对静止的，有“变”即为“动”，有“动”即为“活”。所以《易经》是“活”的，易理也是“活”的，知识也是“活”的，技能也是“活”的。故“活学活用”不单单是把掌握的知识与技能能够灵活的运用，更重要的是要用“变”的思维来学习“变”的知识，掌握“变”的技能，最终应用到“变”的事物上。这就是“法无定法，唯变是法。”

# 寻根基 找真假 看出处

寻根基：主要看每个天干在地支有没有根，有没有气。

有根？有根指的是天干在地支有本五行的比劫和禄，或生本五行的那个字。例如，甲木乙木在地支见寅木卯木，或见亥，（亥为甲木的长生之地，亥中有甲木藏干），或者地支有木的本库未；壬癸水在地支见申金，亥水、子水，（申金是壬水的长生之地，申中有壬水藏干），或见辰库，（辰为水的本库），庚金辛金在地支见申金或酉金，或者地支有丑库，是金的本库，这些都是有根。

有气？还是以甲木为例，甲木在地支见到辰为有气，辰中有木的余气。以此类推，丙丁火在地支午火、巳火为有根，见未土为有气，因为未中有火的余气。壬水，癸水在地支见丑土为有气，因为辰为水的本库，丑中只有水的余气。庚金在地支见戌也是为有气，因为戌中有金的余气。戊己土是比较特殊的，在地支见午火、巳火、未土、戌土为有根，见丑土和辰土不论根，因为辰为水库，丑为金库，都是湿土，一般情况下不能作为土的根气。（附练习）

# 正才，偏财

我克者为财，正财偏财都一样。但又有所不同。命局中有正财之人，代表有正义、诚实、明辨是非，有正知正见。正才代表的是正常的，合情、合理的财、合法的收入，稳定的收入，代表一份长期正式的工作、正式事业，男命正财也代表妻子。

*正财最好是有根有气，正财无根无气，代表一个人工作经常变动。财富来源不稳定。正财代表感情时，也代表不长久的感情，偏财也一样。偏财：偏才代表的是意外的收入，例如股票收益，彩票中奖，偏才不是长期的，持久的稳定的，偏财比正财大很多。偏才除了代表财富之外，在命局中还代表父亲，代表感情，妻妾。

*取财方式1、合财，2、制财，也就是用比劫克制财，克财的效率是最小的，得到的财也是最小的，与合财相比，和的财来的容易，来的大。

*命局中有穿财时，代表不择手段的取财。

*用比劫或禄去直接克财是最低效的做功方法，效率低，得到的也少。

*用财生官，官生印，或财生官，官合身，是最好的取财方法。
