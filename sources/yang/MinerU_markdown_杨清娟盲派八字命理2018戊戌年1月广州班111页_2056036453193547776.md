# 八字中遇到以下组合产生的现象

正印是生我的，食神是我生的，都是我的体，体是获得财富的工具。想要获取财富，必须行动起来，这个动指思想和行为方面。通过思考指导行动，用行动去实现自己想要的财官。如果自己的思想和行动相互矛盾，换句话说想要获得大量的“财富”，但是想的多做的少的哪种，（即体和体合）这样取财官的力量必定降低甚至得不到想要的财官。当我们看到这种情况的人出现在我们的面前的时候，我们就知道对方是一个什么样的人。

如果你要和这类人相处，自己心里要有个准备，要学会大度，学会包容。对方的言语和行动确实让你过意不去，你能真正的包容乃至容忍的了，你可以和这样的人相处。如果忍受不了，趁早远离这类人。
