# 正印合食神

正印合食神做事优柔寡断，稀里糊涂，不知进退，说话没层次没深度肤浅，不知好坏，和他合作走不下去，合作不愉快肯定分手，还要看组合，不管怎么样的组合，他就有这方面的心性，因为他的思想观念不正常，不是正向思维，是逆向思维的人，我们小心说话，他也能挑出毛病，很多矛盾是天长日久积累的，这种矛盾有两种现象，一种现象是不在沉默当中死亡，就在沉默当中爆发，当一个人找我们算命，述说她的婆婆对她如何不好，她的先生对她如何不好的时候，我们一定要一分为二的看待问题，当向我们倾诉的这个人各方面做的比较好的时候，她的先生她的婆婆不能这样对待她，这时候我们就不能完全听客户的，当命主有这种组合的时候，一定是抱怨对方对她不好，事情出在她身上，食神是制杀的，代表快乐之神，代表技术，有的人我说他样样通样样松，因为他的命局食神旺被制了，所以我说你是搞技术的，你样样通样样松，站在宫位上这一个食神哪一个食神样样通，食神不被制，你一定有一技之长，食神被合制了也是样样通样样松。

食神又是快乐之神，当食神被合制的时候，他不快乐，他认为自己没有错，错都在对方，这就叫抱怨。

食神是制杀的，也是生财的，每个人都有做好自己的本分，做好自己的本分天下太平，自己本分不做好你去管闲事，你说是为他好，你不知道他喜欢什么，他快乐就好，对她好她不快乐那叫自我。

# 财官和体的关系
