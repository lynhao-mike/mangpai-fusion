# 干支统一看，五行平衡观

技法：大运干支通根连体，对命局都起好的作用，或都起坏的作用时，可以干支统一看，如果干支对命局的作用不同，刚不能统看。

2，定日干旺衰：

戊土生在未月，得地，坐下午火印星，时支辰土同党有一根，时干丙火坐辰生戊力小，又有月干辛金泄戊土之力，故，日干以旺而不从，以旺论（有承载财官的能力）。

# 3，找财官：

纵观全局，财官星不现，那么，无财官能否是富贵之命呢？

局中无财，官当财看，财官全无，禄当财看。日干戊土坐下午火，火土一家，午为戊之半禄，那么，只要禄旺命主也能发财，这里以禄当财看，禄在这里不能被坏，是人保护的东西，只要禄旺，则财源不断。午火生在未月，午禄自旺，财也旺象。

# 大运定乾坤，流年观动静

大运干支对局中影响的吉凶趋向一致时，可能看，不一致，不能统看。

一步大运管十年，这十年中的好坏是由代表这步大运的干支来决定的，如果这步大运对命局的影响是起好作用的，那么这十年大运就是以吉去断，反之，以凶而断，大运吉，即是不好的流年，发生不好的事情级别也是小的或是会有贵人相助，度过不好的事情，好事俱喜俱喜，反之，不好的大运，如果不吉的流年上，发生不吉的事情级别就大，好的流年发生好事级别也小了。
