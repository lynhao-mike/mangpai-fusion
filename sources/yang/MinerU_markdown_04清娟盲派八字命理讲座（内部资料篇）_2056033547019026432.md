# 第一部分：天干五合产生的物质详释

天干五合：

甲己合化土，乙庚合化金，丙辛合化水，丁壬合木，戊癸合化火。五合特点：阳天干合财，阴天干合官。

合财合官层次高，克财克官层次低。

从这个五合的角度去看人生，财官才是人生去要追求的东西，财是人生俱喜之物，是养命之源，官是奉禄之神，得之荣华富贵，所以，财官是命中要取之物，如何才能得到财官，如何取财官才是算命的重点。天干五合，代表了一个人在家庭之外的活动形式，在社会上打拼的情况，用五合换象分析全局，可以看出一个人取财取官的路线及事业的成就，命中的五合情况可以决定一个人的富贵分贱，定出其在社会中的层次和地位。

重点：天干五合，是得到的意思，天干五合是做功效率很高的，不管日主的承受能力怎么样，只要合到的财是完整的，命主就得到的多，只要合到的官是真官，是没有被破坏的官，命主就一定是当官的。

# 天干五合有明象和暗象。

在分析一已经发生或即将发生的事情时，出现了天干五合，我们要从两个方面去看，一是五合的表面现象。二是要看五合后所合成的物质所代表的象。

明象：只要日主合成了官，我就当官，合成了财，我就发财。

暗象：乙庚合，乙顺从了官，我特别听领导的话，特别守规矩，甲己合，合化为土了，己为财星，我特别听老婆的话。

# 第四部分 十神暗引

# 什么是十神暗引？

十神之间的作用关系不单独是相生相克，还有一种暗引，有的人命中食伤很旺，这个人的人际关系好，因为食伤他在旺点上，不逢生，但是他没有受克，那么他暗引来了什么呢？那一定是暗引来了“比劫”。暗引来了兄弟，比劫是兄弟，朋友，同事，食伤旺自然是比劫所生，只是比劫未现命局而矣！

# 十神暗引：

食伤暗引比劫 （枭 印）

印枭暗引官杀 （比 劫）

注：所有十神都要旺，不能受伤（刑，穿，破）。

# 十食暗引的必要条件：

所有能暗引的十神，都必须是完整的，有根有气，或者在旺点上，而且很旺，才能暗引。

# ①食伤暗引比劫

有的人命中食神很旺，食神在旺点（月支、时支）上，不逢生，但是又没有受伤（不能受到刑、穿），就不用看命中有没有比劫出现，有没有比劫的配置，他就会暗引来比劫，会有贵人有朋友帮助，所以，当一个人的食伤旺的时候，只要食伤在月支或时支，只要不被刑，不受穿，那么这个人就会有良好的人际关系。

食伤很旺，人际关系好，当遇到困难时，就会有人来帮，食伤旺就是有贵人的命。

命中有食神透出，再出现比肩，这样组合的人就有福，有贵人。

# ② 印枭暗引官杀

同样的道理，有的人命中印枭很旺，印枭又在旺点（月支、时支）上，不逢生，但是又没有受伤（不能受到刑、穿），就不用看命中有没有官杀出现，有没有官杀的配置，他就会暗引来官和杀，因为印和枭是官杀生的，制印得权，就是根据这个道理来的。

# 附：论禄神：

人生有三宝，印、食、禄。

禄是奉禄，寿禄， 是人生最好的东西，所以命中之禄不可坏。

原局禄被晦被克：一是命苦福浅，劳碌，二是行运让禄减力，或克禄最晦禄的五行加力时，主身体疾病，工作事业不顺。如果禄和禄的原神严重受伤时，发生的伤害危及生命。

禄虽是好的东西，但也不是所有的命局都喜见禄的，要结合八字的组合而定，如果八字中官杀一片，则见禄不吉，如果命中禄被坏严重，如果见禄为大凶。命中晦禄者，禄到命主会生气，受气，诸事不顺。命中有克禄的，禄到身体病灾，工作事业不顺，收入减少，破财等。命中有穿禄者，岁运见禄，主应大灾。命中有禄入墓，运行墓地，主寿到。

1，命中有禄，怕见财官。

2，命中有坏禄之神，大运流年怕见禄到。

3，命中财官旺者，不喜见禄。

4，原命禄坏，禄到寿到。

5，冲禄穿禄，轻则疾病灾厄缠身，重则危及生命。

引伸：食神是衣食奉禄，是人生所喜的东西，是人生三宝之一，如果命中有两个枭神，或枭神旺极，则不喜见食，见食则夺，食到则应凶。

枭印夺食：遇坏的大运，投资不吉，多为严重亏损，甚至血本无归。
