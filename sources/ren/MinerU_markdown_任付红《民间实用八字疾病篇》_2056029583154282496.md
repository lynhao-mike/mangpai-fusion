# 疾病

从命理来看疾病，首先要熟悉以下几点。

一、阴阳。无论哪类易学或中医书都会提到阴阳之理，因为阴阳是万物变化的起源，生长的过程，消逝的原因，也就是破解万事万物的刚领。因为各门各式，所用之不同，其所解释的阴阳之理也不尽相同。但是只要能掌握了阴阳的本意，无论用它哪个行业其理都是适用的，这就是人们所讲的万变不离阴阳之理。因此，八字中阴阳是至关重要的，不仅看病要用到，就是看毒、看牢狱、灾祸性情，职业等等都要用到阴阳之理，推而广之，通过风水的阴阳之理可以做到解灾，调病，催财，催官等之功效，在这里仅为八字看病作一介绍。读者如想全面了解阴阳之原理及阴阳的具体运用可看今年改版的“函授教材”，内有较详细的讲解，包括在八字、六爻和风亦调解文病，催财、催官等如何使用都有讲到。

从八字干支五行来讲阴阳

木火为阳 金水为阴

天干为阳 地支为阴

地支寅到未为阳，申到丑为阴。

阳表燥热，阴表寒湿。“阴阳”要相交，“五行”要流畅，人体“气血”才能畅通。只有阴阳五行和顺，气血通畅才会不生病或少生病。

只要掌握了上述内容，并能理解活用，八字一排出（出生年月日时要准确）按照干支，宫位所对应的人体类象、部位，结合阴阳之理及作用方式，立刻就能确定此人一生最容易患那方面的疾病，那方面最容易发生灾祸，早夭还是凶亡、长寿还是短寿等等吉凶信息。

关于各作用方式可参看函授资料或有关各作用方式的讲解，命中的冲、穿、刑破、合等方式是否影响到了身体的某一部位等等。

虽然前面讲述了干支及宫位对应八字人体的情况，但各干支是不能独立来表示它们显示的脏器功能和作用的，就象人体自身的器脏，不能独立存在而发生作用产生功能一样。所以，看疾病，关键要注重八字干支组合，各作用方式作用后的具体情况而定。

2、辰酉合金、肺病、各种哮喘、咳嗽病。

3、丑午穿、心慌、心悸、心脏病

4、火克金、易有血液病，重者成白血病

5、木克土或土重刑冲者，易有脾胃，消化系统病

6、火被辰丑晦重者：易有血液病。

7、破：表伤破出血，或手术刀伤、重者脑出血、心梗。

如何具体从八字中看疾病

前面我们已把从八字中看病有关的阴阳，干支在人体五脏六腑类像的具体部位及干支对应的各种类象，组合关系等都讲了，接下来重要讲的，就是如何把干支类象，恰如其分地运用到实践中去。方法只有一个，那就是多实践，多总结。只有经过大量的实践和总结才能把干支类像的因素，熟练的从八字中提取出来，技术的东西是“懒”不得的。

辰卯午亥

命透双丁，月丁坐卯被辰穿午破，时丁被坐下亥合吉，丁在干支象中表眼的象，故此命是个盲人。

未酉寅子

此命行辛卯运因脑瘤成植物人，为何如此？因大运与原局之月令成阴阳倒置。明此理则病因一目了然。

病类多科，难一一详述，但无论什么病都无非阴阳五行之理，把前面所讲干支类象及对应人体部位熟悉了，再通过大量实践都可做到合理推断的。方法与原理都已讲明，就看自己如何去掌握运用了，包括六亲患何病也在此理。
