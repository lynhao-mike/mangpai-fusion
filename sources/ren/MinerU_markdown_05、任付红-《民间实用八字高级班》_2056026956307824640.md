# 象 的 运 用

象法有多种，有宫位象、十神象、干支象、五行象、合象化象等等，可以说事无巨细，有事即有象，象就是事物的表现，也可以说事物是通过象来表述的。

对于象的运用，需要建立在对基础的熟练加以丰富的，思维性联态，和严谨的逻辑推理，这样取出来的象方可与命局表述的一致。

为使学习者对象的表现形式做进一步的熟练，我们把前面讲过的或没讲过的象的表现形式重新写在下面。

# 二、十二支类象

十二支类象与十干类象一样，是看一个人职业或从事行业的重要依据，也是分析八字时的核心内容，可以使我们在分析事情时找到具体的依据。

地支分阴阳，木火为阳，寅卯辰巳午未为阳，辰为阳中阴。金水为阴，申酉戌亥子丑为阴，戌为阴中阳。丑酉为阴中之阴，大多数丑酉与犯罪、黑社会有关，主牢狱、黑社会、坟墓、玄学、淫邪。亥子不是阴中阴，亥是阴之位，但中含阳，子为阴，但位在阳。辛酉日主见丑时为黑社会，因阴见了阴又入墓。辛丑日主不是黑社会，因为浊中见清，人性是好的。戌为阳乱，如歌厅，丑为阴乱。

看学习的文理，哪门考的好坏都可以看。戌亥为计算、运算的意思，表数学，入乾门。丑主玄学。木火主文，金水主理。辰见了子为化学。申酉为法律，申还代表金融，酉不代表金融。辰有运动的意思，表机器、车辆。

子象：水、冰、河流、泉水、井、流动、流转、车船、贸易、江湖、文墨、玄学、圆润、聪明、技能、淫邪；于人体为肾、耳、膀胱、泌尿、血液、精、腰、喉咙、耳朵；动物为：蝠、鼠、燕。

例：乾

辛 癸 癸 癸

未 巳 卯 丑

这是今年（己丑）高招前江苏电话求测儿子的命问儿子适合学什么样的专业？

我当时在电话中先断他几句：这个孩子长相秀气，性格文静，象个女孩，睛睛带点近视，平时爱好绘画对吧！

电话那边马上传出兴奋与激动的语言：任老师，你断的太准了，我的儿子长相的确象个女孩子，眼有些近视，平时不好出门，也不很与别的孩子们一起玩，就是自己每天呆在屋子里画些什么或上会网的。

然后我就先测他，你的儿子将来是个搞绘画或是图案设计类的，你就让他选择这方的专业吧，未来的前途是很不错的，不必为他的未来的事业担心。

他己接受了我的建议，让儿子报这方面的专业了。

为何我们断他是画画而不是别的专业呢？

就用我们上面讲的干支类象，就可把事情解读出来。

看癸水有墨的象，而癸坐卯可表花草，也表丝线，看卯生巳火，而巳表色彩，火又表漂亮，而巳火又透出癸墨的象，癸巳又自合，表示癸墨要去为巳火服务，也表示巳火表的色彩漂亮要用这个癸墨，这样一组合了联想，就可表示用墨去描绘美丽漂亮的图案的意思，这也就是绘画的意思。

我们还可以推理出他将来的绘画风格是笔墨浓重而感观抽象，很易引发人们想象力的那种图画。

在使用干支象时，一定要灵活运用，每一字都有多种象，具体如何取那一个，有时可看自身象，但大多情况要看与它组合的字或有关联的字，这样取出的象更为丰富具体。

如：丁丑组合，丁和丑各自都有多种象，当组合到一起时如何用，这就要取决于各自在命中表述的是什么了。

如果丁火是官杀与丑的组合，丁就可以表盗贼，如果丁火在原局取眼睛看，与丑组合就可表示，眼近视或眼盲。

辰 卯 午 亥

原局有丁，在天干类象中丁有眼的象，但不是所有的命只要有丁就都可以表示眼，此命有两个丁，真接可表示双眼的象，月令丁坐卯被辰穿伤，而时上的丁坐亥，自合也被伤，这时就可直断他眼盲，实际也正是个盲人。

己 癸 庚 壬

巳 酉 戌 午

这是去年冬天电话测的一命，断她有个工作，企业性的，搞与文字有关的事，但命中婚姻不好，命硬克夫，丙子运时对夫不利，丁丑运时，要注意破大财，命犯小人，特别是今年（己丑）要有盗贼出现。结果今年三月又打来电话说二月份因自己工作失误，使单位财务被盗。为何丁丑运时犯盗贼？

原局官杀制羊刃及食伤，丁丑运时，丁即原局的午或戌，原局实大运虚透坐个丑，表示官杀的性质转变了，丑为阴，又晦火穿午，可理解成此丁是个坏的官杀之意，即盗贼的象，丁壬合原局食伤看财，表示丁要取这个财，即盗的意思，己丑年是应期到即丁到，丑也表应期到。

实际她是某单位的会计，一天下班后忘记取下财务室的钥匙，给盗贼提供了可剩之机，致使财务被盗，好在单位放的钱不多，损失的不大。丁丑运时，因参与合财赌博把房子都卖掉了。丙子运时丈夫因病已亡。

所以在用各字的象时，要先从原局组合中分析出各自所表示的意义，因发生作用或与相关联的字不同，每一个字的意义也都在进行不同的变换。

如：甲申组合，甲表头的象，申表骨、脊椎的象，这样一组合就表示支撑头的骨头，可表颈椎等，但当甲不是与申组合时，甲也有头的象。

例：乾

乙 甲 己 乙

酉 申 丑 亥

这是丙戌年爷测孙子的命，当时我就断这个孩子是一个耳朵大，一个耳朵小，是右耳大、左耳小，且克父严重，因当时小孩不足两岁，克父的事应期还没到（后父与戊子年亡）但他爷承认是明显的右耳大、左耳小。虽然说断出这些事情时对命主的富贵吉凶意义不大，但在实战时可以使客户感到震服的效果，也说明八字取象意义的重要性。

如何看出，如果不点破，有的人会一生也理解不出来的，而如果点破了，又会感到这么简单的理，其实往往看似深奥的东西从理上讲都是很容易的，关键是没有遇到给你讲这个理的人，所以有的人会用一生精力，最终也没有任收货，这就是首先自己选择学习的路是错的，因此就会有越学离成功反而越远的现象出现。

日主己土合甲木，甲表头的象，甲申一柱更表头的象而日主己土是自己这个是不会变的，那道会出现自己在日柱而自己的头跑到月柱上去的现象吗？这是不可能的，因此，不能去这样理解，那么怎么理解呢？这时我们可以把月柱与日柱合并当作一体看（这个只能从理解的角度上去体会）那么年上的乙和时上的乙就可看作头部两边的东西，具体当什么看，要从象上来分析，首先，乙的形状是弯弯曲曲的，可类象耳朵的形状，从木的含义上讲木主风，而风有耳的象，这就可以肯定是耳朵了，再看两个乙各自所处的状态，年上乙坐绝地，有形无气可表示小，时上乙坐亥长生地，统根得生表示大（提示：这是用到了十二长生的用法）而年柱表左时表右，这样一联想，就是八字在告诉我们这个命是右耳大、左耳小特征了。

提示：只要学习的路是对的，很多东西自己是可以悟出来的！

# 三、禄神类象

我们在前面资料中讲过禄刃的概念，即：甲禄在寅、乙禄在卯、丙戊禄在巳、丁己禄在午、庚禄在申、辛禄在酉、壬禄在亥、癸禄在子。它是日主的延伸，代表日主到地支行使权力，其心性含义有：亲为、独立、主张、自我意识、尊贵、独占、主宰、享受之意；禄之物象为：权力、封地、财富、供养；于人体表示身体、肢体、女人的肉体、寿命。

提示：关键是要找准禄神在原局中代表的意义，这样方可把代表的象取准确。

坤：辛 庚 癸 壬

未 寅 丑 戌

这是去年测的命，母女同来测女儿上学情况

断她学习一般，不会有很好的发展，现在国家普及教育，最多也就是个高中生，甚至会上不完。我说到此问她在上什么学，母亲说，今年（戊子）考高中，可能考不上好学校，计划让她去读职高。我说她今年（戊子）可能会不上学了，是她不想上了。有心外出打工挣钱的。然后当着母亲的面，我说她女儿命中犯桃花，今年（戊子）就到，明年（己丑）还要有，我提醒你们，也好有个心理准备，她的桃花不是男朋友类，而是比她大的多且是有家庭的。母亲听后说会严管女儿的。事隔半年多，女儿又一人来找我测，并主动告诉了我以上来测过（如非她提醒我们一般是记不住来过没来过的）说是确实遇到一男的对她不错，问我此男的将来会不会对她好，我给她讲了此男的长相及性格，然后说此男人有家庭，她说断的都对。

看学习不好是卯星虚透又正偏印混杀表示学习不专心，故断她学历不会高。

如何看她的桃花运及桃花运是有家庭的？

桃花是看命中的丑为日主的半个禄，禄表女人的身体，丑去合寅木伤官，这个寅就表丑的桃花，寅木透庚是日主癸水的印表示此桃花比命主大的多，为何有家庭呢？看寅的库在年，可以理解成这个寅木桃花是从未库中出来的，那么这个未库就可理解成寅木的家庭（库有家的象） 从天干来讲庚是寅的外在象，那么辛也可当庚看，同五行有时可以借着来看（这个以后再讲），辛坐未也就是庚坐未，未库就表示庚的家，未中财、官、印俱全，有示寅木桃花，父母、妻儿都有更表家庭。

坤：戊 癸 丁 庚

申 亥 亥 戌

# 七、宫位类象

提示：在实际断命时，宫位的用法很重要，因为它包涵的内容很广泛，人生所有事情都是在宫位的基础上表现出来的，试想一下，如果没有宫位，干支往哪里放，如果没有干支，那就不是八字了，因其所含内容庞大，在实战使用时就必需要具备高度的灵活性，针对不同的问题找到在宫位中相对应的不同的象，这样方可把象取准。

因为宫位之象关系任何一个八字的全局，我们不可能把所有有八字写出，选用几个有代表的例子作说明，当你在看任何一个八字时，都会有宫位用法的出现。

具体可划分以下几个内容：

① 六亲类象

首先四柱宫位含有自己与六亲的信息，除日干代表着自己外，其它都代表自己亲人的位置：如年柱代表祖上、父母、外戚长辈；月柱代表父母、兄弟；日支是配偶宫，代表妻子或丈夫所在的位置；时柱是子女宫，代表着自己的后代。父母即可以在年上体现，也可以在月上体现。

丈母娘在年上找，因年上表外戚，和妻宫发生了关联的印。

伤食都是丈母娘。伤无感情，食重感情。如和妻宫无关，出现的伤可能是奶奶，意思就变了。

月柱表同学、同事，若夫妻象现在月上可能是同学。

日支：表最近的，如住宅、妻子等。

门户：表学生、晚辈。

根据宫位可判断找的夫（妻）大或小。

四柱宫位还代表不同年龄与不同关系的人，年柱代表外人、长辈或老人；月柱代表同学、同事、领导；日支代表与自己非常亲近的人；时柱代表晚辈、学生、朋友、下属。

② 时间类象

四柱宫位还代表着人一生从小到大的时间顺序，即：年柱代表童年与幼年时期，大约 1-18 岁；月柱代表着青年时期，大约 18-35 岁；日支代表着中年时期，大约 35-55 岁；时柱代表着晚年时期，大约 55 之后的岁月。除此之外，从年到时，还是从大到小或从先到后的顺序，比如一个人命中有三次婚姻，那他三次的排列顺次也是从年到时依次来找。

# ③ 空间类象

四柱宫位还代表一个人生活中空间所在，比如你要出远门，命中对应的宫位就是年柱或者时柱，因为年柱代表远方，而时柱代表门户。月柱是代表祖籍，日支是代表现在的居所。

# ④ 身体类象

四柱宫位还代表人身体的不同部位，年柱离日柱远，代表腿、足、四肢；月柱代表人体的躯干部分，如脊、肩、背等；日支代表人体最重要的部位，如五脏、六府、心、脑、髓；时柱是代表与外部相通的人体器官，如头、面、手、眼、嘴、耳、鼻、生殖器、排泄器等。

# ⑤ 物品类象

四柱宫位还代表人的使用之物，如看车，那就在时柱上找，因为时柱代表门户，故出门的代步工具车子就在时柱上看。其它还有：年柱代表鞋子，拐杖、或别人之物；月柱代表祖产、家业、单位、学业；日支代表房子、卧室、私产；时柱代表车、门、衣服、帽子、眼镜、化妆、进出之钱物。

# ⑥ 情感类象

四柱宫位还代表人的心志与情商，有的人容易受外界环境的影响，有的人却

容易影响别人，从四柱的宫位也可以看出这些特点来。

# ⑦ 表理类象

四柱中，天干代表人的表象，如外表特征、性格表像，容易被外人看出的显著特点；而地支代表人的内在的东西，如内心世界、与家人的关系等，往往是人比较隐蔽的思想与心志。

# ② 偏印

偏印又是与日主同性之生，是不情愿的生，“生我且斥我”，就如同跟继母生活在一起一样，日主的心性就会变得非常敏感，其心性含义为：

有思考力与领悟力、敏感、灵活、机智、心眼多、精明、有谋略、脑子好、创意、发明、不爱学习教课书、不随大流、严肃、孤独、有宗教心。

如果偏印过重，也表现为负面的心性：自私、懒惰、冷淡、福薄、挑剔、呆头呆脑、不通人情、无福享受。

如命中偏印有用，则可能从事的职业有：技术性的职务、医生、艺人、五术业、宗教、咨询师、律师、记者、编辑、情报员、侦察员、设计师、技术人员等。

③ 正官

命中七杀为凶神，有制化则显示其吉的一面，无制化则显示其凶的面。

如命中七杀有用，则可能从事的职业有：警察、法官、纪检、督察、军职、执法部门之官员、政治家、谋略家、律师、教师、医生等。

正官与七杀共俱类象：职务、官职、权力、名望、管理、法规、官司、法院、牢狱、忌恨、官害、财富；人物为：上司、老师、长辈、父亲、丈夫或情人（女命）、子女（男命）、敌人、小人、恶人、盗贼；于人体为：神经、外伤、疾病。

这是温州易友让电测的命，问是做什么工作的？

和上例一样如果对象的认识不到，单凭用作用方式也是很难解读此命的。

断：从事与经济有关的帐目或手续类的内部工作。对方马上回答所：是在银行，专门从事银行对外的票据性工作，你说的帐目或手续类不很对。

我向他做出解释，在我们的类象中，帐目、手续或票据都是属于一类象，我们不可能划分那么仔细，在说你说的票据具体是什么，我至今也不清楚，因为我也没搞过，也没见过的，当然他能理解我对他的解释，并非是狡辩性的。然后就又断了她后面的具体性事情。

解释此命，有两个重点象必须用：一、年上为带象为印带官帽的象，因为有这个巳火印的存在，可以理解把这些官杀化制了，但制的不很好，二、巳印带官帽入到了自己的坐支库，表示自己要控制管理这些东西。然后就看干支象，乙木有书象，坐支巳火可表文化类，又印也可表帐目类，但因原局官杀没制尽也有当财看的象，坐支巳火可表文化类，又印也可表帐目类，但因原局官杀没制尽也有当财看的象，表示此印（即帐目）与财有关，入了库表示有很多这样的东西（如果不是有官当财看的象也可理解成图书馆等）那就说明归自己管的这些帐目或票据上记录的都是与财有关的东西（而非户口档案类）所以在银行做票据工作。

看来人生已有命定，该你干什么，就是干什么，只不过我们还没有那么高超的技术解读出来而已！

提示：因为象法在断命时至关重要，我也曾打算把象法整理归类，但在整理过程中，发现了一个不得不面对的现实“就是说这是不且合实际的”，因为象无处不在，可以说有事就有象。关键还有就是它是有灵性的存在，而并非单纯的套用，就像电脑与人脑一样，人脑也可以改变事物，而电脑是做不到的，只有在人脑的指控下电脑方显出它的灵动性。如果我们把象归类了就形同电脑一样，不具备灵活性和多变性，这样断出的命一定不会准确的。

“思维决定行动”而我们断命的要讲“思维决定结果”，只有你分析的思路对了，这个结果才可能正确，而良好的思维需要一定的灵性和悟性，而悟性又来自良好的心态和德行，德到了，功力自然就来了，灵性也就出来了。

因此我们在学习时从不强调死记硬背，首重修德开始，然后才是多思、多想、多实践，只要你选的路是对的，总会有收获的。

申 卯 辰 子

这是福建某面授学员送给我的礼物，问：老师你看命主的儿子将来成不成材？我一看此命，虽杀星两透，但己都坏，且妻宫是穿，又三合，子女宫与妻星破，哪来的子和妻，我问他你是不是记错八字了，此人是个单身，且无儿子的命，即使抱养也抱不来的，何来的儿子。说至此，他说我是想考考你的真实水平的，此人的确是个光棍。我说他，你这样做是不对的，既然来了就要虚心学习，如果你对我存在疑虑，那么我在给你讲解时你就会有抵触情绪，这样对你是很不利的，在说这个命我是断对了，万一我断错了，你心中一定会有看法，这样对你的学习更不利。我也不是活神仙，一定会做到百发百准，即使断错了也很正常，只要事后我们能通过理象的合理解释，找到答案，也是很满意的。

我们学习就是要学习这个断命的理和思路，掌握了这些，随着自己的实践进展，很多东西自己都会悟出来的。

我告诉他，学习是个漫长的时间过程，不会一下子就达到自己想得到的水平，这是自古至今不变的真理，不需要去找什么近路，只有踏踏实实地去做，才能做好，关键是首先学习的路和方法要先对，否则也不会有收获的。

最终他听了我的建议，学习收获还很不错。

此命不用再作解释，我相信你看过了前面的内容，解读此命已不是什么难事了。

乾：癸 辛 壬 戊

丑 酉 子 申

这是丙戌年在安阳测的例（中级讲过，可联系起来看），断：你的母亲改嫁来的吧！

他说是的，我又说你母亲改嫁过来时给你带了一个哥哥，这个哥哥对你很好，能帮上你忙，是不是？他说是的，自己的工作就是哥哥给帮忙安排的，现在对他还很好。

分析：月柱辛酉看母，申也看母，但酉丑一拱，丑就表母的家（这里是宫位的时间用法）因丑在年为先，这个丑就是母原来呆过的地方，也可以理解成母是从丑中来的，而丑透癸，癸就表母在丑那个家时生的儿子，而癸戊合把癸合到了时，表示带儿过来了，申子合，申也看母，申从丑中来，申是自己的亲生母（月令是象上的母，因酉不很生子水，可看作有其象，而无其实的意思）而戊癸一合时柱也当母带的儿子看，对日主而言时柱表示自己的工作，因印带官帽且合入了自己的坐下，就表示自己要得到这个工作，但因癸戊合，也可理解成，癸在先要先得到，自己后得到，而癸是当哥看的，戊也当哥看，这样这个工作就有哥帮忙的意思了，又申子合表助力大，感情好。

反馈：母改嫁给父的，过来时带了一个哥哥，哥招工进了某国企，当自己到参加工作的时候，哥当上了个小科长，就把自己也安排进这个企业，现在他们在一个厂上班，感情上的确很近，哥哥一直帮他。

# 结 束 语

讲的例题再多，如不能做到理解应用，也是没有用的，如果能做到理解应用也就算有收获了，学习应该把心态放低，这样你能有最大的收获，如果你能全部掌握理解，那是最好不过了。

因八字涉及人生方方面面，内容极其庞大，我们不会做到一步到位的全面讲解。如想全面了解，可看系列资料及例解等资料里面的例子都有所涉及，我会用最大的心力去讲解给你们。

希望你们能认真对待和理解，在学习过程中要多反思勤实践，这样你会有很大收获的。

因写的比较匆忙，定有不到之处，但可肯定书中的每个例子都是真实的，并且是经过我的分析整理出来的，其中原理不必怀疑，你尽管放心去领会就行。但在使用时千万别忘了“活用”二字。
