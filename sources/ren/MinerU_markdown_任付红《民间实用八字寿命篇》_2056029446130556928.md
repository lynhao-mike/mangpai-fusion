# 寿

从八字里看寿命，首先要知道都是什么可以表示寿命，这些道理如果搞不清，那么即使你已知道这个八字代表的人已经不在了，但也找不出他不在的原因的。因此从八字看寿命，首先要定准八字里哪个字代表寿命星。

再就是要对各种作用方式的熟练，分析命中通过各种方式的作用，对这个代表寿命星的字的好坏，作用方式包括：冲、合、刑、穿、反吟、伏吟、颠倒、空、破、墓、绝、阴阳关系等，因这些内容我们在别的相关资料中都已讲到过，这就不再作专门讲解。

当命中代表寿星的这个字在命中处于被破坏的状态时，就会直接影响到这个八字的寿命。如行大运再对这个字不利或大运引动原局对这个字不利时，就要考虑这步大运是否有寿命危险，一旦有，就怕流年引发应期到位而应凶。

关于应期可参考别的资料中所讲用法，此处不再专门论述。

如去年（庚寅）冬天时，南阳的函授学员小刘给我发来一个八字：

坤：戊乙丙己

辰卯寅亥

大运壬子

小刘当时问我：任老师，请看看此命今年会有什么事情发生？

我看后即回复他：此命今年寿数已到，没什么好看的了。

他马上就打电话过来说：是啊，这是我一朋友的亲戚，也是我的病人（小刘在他们当地是很有名的医生），前几天刚去逝，也没查出死亡的原因（有些病单凭仪器是查不出来的）。我们几个易友，想通过从命中找出她死亡的原因来，最后达成一致认同，是因为命中木多，大运流年木又加旺，因木多火塞而亡，但不知此理对不对？

我说道：这和你看病一个道理，并非什么木多火塞，而是一个阴阳之理，此命原局阴灭阳气，就是短命的象，大运，流年一引发寿数就到了，必亡的。

小刘听了感到似懂非懂，虽然他医术上常用阴阳之理，但一旦把医术的阴阳道理转化到八字上还是有些摸不着头绪的。然后我就又把资料中所写到的阴阳道理，通过此例给他进一步讲解了如何具体应用，他才算有所明白此中之理。小刘为了借助自己医术的平台把八字看病看寿等事搞明白，春节一过就不停的打电话要求面授学习了。

注：此例后来又再在学习班上做了详细讲解。

此例寿短就是阴阳原理，原局阴坏了阳命不长。

命中丙日坐寅木，而时见亥水，寅亥一合灭了寅中火，表示丙的阳气没有了，阴阳分离就表死亡。运年是引发。

在八字中看寿有如下几种。

一、阴阳，命中一旦阴坏阳或阳或阴时，就有寿短象。

二、五行：各五行的性质不同，因此它们各自代表自身寿命的因素也不会相同。

三、十神：命中的食、财、印可看寿，有时比劫羊刃也可看寿，但要分清如何取用。

四、禄：禄表身体，也可表寿看。

以此四条看，命中什么都可表寿看了，但具体到实战时，该怎么取舍，学者就需要灵活变通，并多实践总结，才能达到铁口直断的效果的。针对不同的命局，该拿什么当寿看，定位一定要定准，定错了位就什么也不用说了。当你看我们的讲解多了，再加自己多实践，就会体会到其中奥妙的。

当然，看寿如细分还可分出很多死亡类型的如：是水里死，还是火里死，是凶死还是正常死，死家里还是死外边，死床上还是死地上，是穿衣死还是光身而走，包括死后灵魂去向，何时灵魂再回来看望亲人等等内容，都在看寿的范畴内，如因基础知识未熟练，讲这些内容就容易扰乱学习思维，因此，这些内容就不属本篇所讲范围。本篇讲看寿的粗线条看法。

下面用实例讲解。

乾：甲丁癸甲

辰卯未寅

此命日主癸水，年见辰为根，时见甲寅为流绝，表寿不长，后在丙戌年突然意外而亡。

水通根怕绝，行壬中运，水得根，丙戌年，引动寅木见绝寿到。

乾：甲庚壬辛

寅午辰亥

此例是丙戌年测的，在《技巧》中曾有讲过，命中亥水为禄可表寿星看，亥入辰墓寿不长，丙戌年，冲辰慕应期到，突发心梗而亡。

坤：丁壬癸辛

丑子巳酉

此命巳火被子、丑、酉、寒湿之势而制，巳为阳，巳又为财，本命财就丧寿看。巳阳被坏不吉，巳被坏的年运就会夺去生命。而命有巳酉丑合局，已被酉坏，酉表肺，必亡于肺病，果在丙戌年，因肺病无治而终。

乾：乙丙乙壬

未戌丑午

命中丑阴被刑又被午穿坏，寿不长，午借未成之势而穿丑，午表心血，庚寅年，合动午穿丑，突发心梗而亡。

坤：戊乙庚丁

辰卯辰丑

命有乙庚合，乙坐卯被辰穿坏，乙卯财可当寿看，于庚辰流年，穿坏卯木而亡于肝病。

坤：癸壬丙丁

卯戌戌酉

命中丙日主局有卯戌，时见酉阴，酉被穿坏，寿不长的象，行丙寅运，火燥得势，寅合动戌，而去制酉阴，表此运寿到，流年癸未，刑动戌而穿坏酉，此年命殒。

乾：甲丁己丁

寅丑卯卯

干透双丁枭神，局有丑，丑中有辛，最怕食透干被丁夺，运行辛巳，辛食到，食表寿，表此运寿到，丁亥年，表原局丁到夺辛食，此年病亡，何病？辛表肺，死于肺病。

乾：丁壬乙壬

卯寅未午

日主活木，怕坏印星，命透双壬，月上壬坐寅绝地，时上壬被午合坏，寿不长，庚子运，印通地支被原局坏，寿到。丙戌年，引动原局制子，此年死亡。

乾：辛戊戊乙

亥戌子卯

此命财看寿，子被卵破成土克，又卵成合坏子水就表寿短，行乙未运，穿坏子水，丙戌年，引动命局克子，此年凶亡。

坤：丁丙乙辛

卯午未巳

全局一片火势，日主死木怕火多焚木，寿短之象，时透辛杀克日主，要遭凶亡，申运，甲申年被人杀害。

乾：丁壬癸甲

酉子亥寅

日主癸水，年通根月见禄，水在年月见源通根怕流绝，命主时见寅木绝地，行未运，食伤入墓，寿到，见子年禄到应绝，脑出血而亡，水主血液。

乾：癸丁丙乙

未巳戌未

原局一片燥火，戊中辛看寿，土是脾胃，火主炎症，戊中辛看瘤癌，运行辛亥，流年乙酉，都表命中寿到，此年亡于食道癌。

坤：乙 乙 庚 丙

未酉寅子

命中木多入墓，寅表头，必然头上病，子水看寿，全凭酉金生，行辛卯运，阴阳颠倒。酉金虚透不生子水，卯又破子，此运寿到，戊子年，子到被破又被未穿坏，因脑瘤而成植物人。

乾：乙 甲 辛 壬

丑申巳辰

命中已申合，已丑拱，辰晦巳，都对这个巳火不利，一旦大运巳火出现，就表已被坏的时间到了，这个巳火不能破坏的，因在此命中已当寿看。此命偏偏第三步运就行到了辛巳，此运必然应凶，流年丁亥，把关键的巳坏了，这年命主被水淹死。

看命中什么表寿，要根据具体八字组合来定，一般取整个八字中最关键的这个字较多，因为命局不同取的字也就不同，要具体情况具体对待。

坤：己甲甲丙

未戊辰寅

此命甲木坐辰为活木，时上见禄透丙，命必高寿，丙戌年时寿已80多岁，身体尚健康，生活可自理。最怕大运透印见墓，这样代表寿的字就会受伤，运行癸未，印墓同现，此运寿到。当时断她，己丑年寿命交关，寿高90岁，后果于己丑年寿终。

坤：己癸己丙

亥酉丑寅

日主已上，月见酉金食神，日支丑拱酉，使酉金表示的寿增加很强的生命力，命必长寿，丁亥年时，已寿高达108岁，身体尚健康。

取寿命星，如基础不熟练，确实有些难取，因为命局千变万化，而取寿星之理也就要随命的变化而变化，但无论怎么变化也就是这几个原则，关键要“活用变通”，不可死套。

乾：甲 甲 癸 癸

戊戌丑亥

怎么取寿星，看日主癸坐丑，丑为半禄表身体，而丑为印库，就可直接定丑中辛金为寿星，原局有戌刑丑，这时就最怕大运见辛金出现，因为辛金一出现就表丑被刑的时间到了，行运进入辛巳，此运命有关口，己丑年，丑到被坏，死于肺癌。

如去年庚寅春天时，湖北几个客人专门开车来我家找我看命，其中一个人首先报了这个八字：

乾：己癸壬壬

未酉寅寅

问我：你看这个命是做什么工作的？

他们的意思我明白，是在考试我，如果我断错了，那么他们可能就会不再往下看命了。但这样的命短寿的信息一眼就可看出来的。我就回他们一句话，“这个命是睡觉的”。他们装做不理解似的说，“师傅说睡觉是什么意思，我们是问此人是做什么工作的。”我说：“到地下睡觉了还能做什么工作呢？”这时另一人开口说：“师傅看对了，此人已经不在了，但不知师傅你是怎么看出来的，并且看的这么快。”我说“如果你懂得真正的看命方法，这个道理说来就很简单了。就看一个食神入墓就行了。但因为他们不懂易学，讲了这个原理他们也不会明白的。

乾：己 甲 己 丙

未 戌 酉 子

命中以子水财看寿，有酉紧贴生，不会寿短，运行丁卯，卯冲酉破子，不利寿，辛未年，酉不生子，未穿坏子，此年病亡。

乾：丁 戊 戊 戊

酉申申午

以午火印星看寿寿，原局火见金为死地，最怕大运再来坏午，行辛丑运，丑坏午，此运寿到，己酉年酉丑引去坏午，此年病亡。

乾：戊 甲 乙 丙

午寅卯戌

丙戌年，客户测了自己的命后又问这个命的情况，我断：今年要生死之灾。

问：可有救吗？

我说：按命中运年看，已无救了。

对方说：这是我一亲戚，前些天与别人打架把别人打死了，已被捕，我省里有关系想花钱把他的命救下不死就行，看来是劫数难逃了。

命中一片木火，日主死木最怕火焚，运行丁巳，流年丙戌，命局成了一片火海，还往哪跑呢？

在这里我们讲的都是通过八字看本人寿命的原理，其实从任何一个八字中都可看出与此命有关的六亲的寿命的，只有先把有关本人寿的原理搞通顺了，关于看六亲的寿，少作变通就可看出来的，具体如何变通，我们在其它《民间实用八字》系列资料都有所讲到。
