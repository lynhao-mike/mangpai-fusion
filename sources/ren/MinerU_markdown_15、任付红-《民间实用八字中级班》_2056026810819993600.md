# 一、正确理解使用各种作用方式

提示：①我们针对任何一种作用方式都称之为：“制”，表示“制伏”之意。即一方要制伏另一方。

②任何一种作用方式都不会一方产生，至少要两种以上的五行方向发生作用，这样就产生了双方的作用，或冲，或刑，或克，或墓等等，但无非都为了一个目的，即制伏对方，控制对方或获得对方。（人生即如此）

③相制的双方都可互制，看谁的势力强大就可制伏另一方。

具体地讲，双方互制有以下几种类型：

# (一): 比劫制财: 这种情况的特点是原局中比劫得势或旺, 制去财星。

例1、乾造 丙 辛 丁 壬

午卯卵子

分析：丁日主年上见午禄，禄上坐劫丙，丙合制了月干辛财。局中木火有势，辛财被制得很干净。午为禄，丙午柱同象，丙制财等于日主自己取财。但时上官杀壬子无制，是局中的毛病，虽有年上午禄冲子，但相隔太远，制之无力。走甲午运，午可直接冲子，去了局中的病，此运发财。

例 2、乾造 丙 辛 丁 丙

午卯卯午

分析：此造原局为木火之势，但月干辛财无根虚透。所以丙火劫财合制辛财的作用不大，命局结构不好。此与上造只差时柱不同，但本造午运并没有发大财。

原局是劫财制财作用，劫财代表合伙人、同伴、朋友等，所以此人靠合伙求财，而且得财非常容易，不用自己费劲。

例3、乾造 壬 癸 己 己

寅丑未巳

分析：原局火和燥土有点势，但水与湿土也不弱，两势对抗。丑中含辛癸，是财和财的原神，丑也当财看。时支已生未，未冲丑意在制丑中的财和食神，寅丑暗合也为制丑中的金水，财与其原神均被制，命局的意思在求财。但壬癸水透出年月，丑也制得不好，食财在原局没制干净。此造要结合大运分析，才能看清全局的财势，单看命局是无法断其为发大财的。

行丙辰大运，与中用火势，但丙在大运虚透，辰又为湿土，晦原局的巳火和未土，与原局制丑的意思反了。此运不吉，命主破财又坐牢。

丁巳运原局的火势到，丁运第一个流年为丁丑，丁己一家，丁到为己到，丁合壬制了壬水，丑在原局被制，流年见之为应期。命主在此运发财几百万。后运连走火和燥土，一路顺风。己未运原局燥势未到，财运更会锦上添花。此人是搞装璜的，因为原局结构为火与土木。

# （八）印星制伤食星：

例 坤造 己 戊 壬 癸

酉辰申卯

分析：印星申酉金和湿土成势，申卯合、酉冲卯、辰穿卯，卯被制了。原局有月令戊合制癸，卯的原神也制了，作用的主要力量是卯申合。女命印制伤官不主权。此造八字无财，伤官当财看，制伤得财，是发财的命。此女现有几百万家产。但婚姻不吉，因为卯申暗合做用，不喜卯辰又穿。曾有再次婚姻，两任丈夫都破她的财。

提示：1）不管任何一种制法，相制的双方都可以互制，谁得势谁旺就会制去弱的对方。

2). 印制伤官有时也主权。有官杀，则以官杀当权。此造官杀无用，所以不主权。

3) 制的作用中，日主得了根，其根不能被制，制之则伤寿。日主在年月的禄可以被制，有的时支之禄可以制，但坐支的禄一定不能被制。

例 乾造 庚 乙 癸 庚

辰酉卯申

提示：在制的过程中，各作用方式产生的力量效果也不尽相同，力量比较高的有以下几种：

# 一、合制

指地支六合、暗合、半合局的制法。

1、已申合的作用最高，既可以金水有势，申制已，也可以木火有势已制申，都是将原神藏干全能制死。

2、卯戌合是卯木制戌土：卯申合是申金制卯木；午亥合，火有势可以制亥，金水有势可以制午。这三者的作用次于巳申合，但也是很高的。巳酉半合，当局中木火旺里，可以制去酉金，效率也很高。

3、子丑合是丑土绊住子水，固定子水；寅丑合是寅木将丑土制住，此二者的效率略差一些。

4、寅亥合是亥中水制寅中的火；辰酉合是酉金制辰中乙木；午戌合，午中火制戌中金。此三者作用则较低；卯未与亥未半合局，如制未土，其效率相对要低。

5、午未合，是两字相生，不制。

有关合制的例子，我们前边已列举过许多，不再重复。

# （一）冲制、刑制

指地支六冲与未戌刑、丑戌刑。

以丑未、辰戌冲库与丑戌刑库作用最高。金水与湿土成势，丑可坏未与戌库，辰可坏戌库；木火与燥土成势，则未可以坏丑库，戌可以坏辰库与丑库。其他冲的作用都赶不库冲，一定要有一种强有力的势才行。

寅申、巳亥冲的作用也很高；子午、卯酉冲的作用略低于前者，其作用的条件是一方之势一定要大于另一方之势，否则这些冲都不能作用。

# 十、分八字的富贵贫贱

# (1) 富贵八字:

八字一般具备两点，一是有气势，二是有制伏或控制的一方。

八字的气势有许多种情况，有木势，有火势，有金势，有水势，有燥土势，有湿土势；许多八字往往是两种势成势，而不仅仅是只有一种势，如：木火可以一势，金水可以一势，水木可以一势，火与燥土可以一势，金与湿土可以成一势，水与湿土也可以为一势。

成势之后，就必须有所制之神，如木势可以制丑，也可以坏金；火势可以制金，也可以坏水；金势可以制木；燥土势可以坏水，也可以坏金，还可以坏湿土；湿土势可以坏火，也可以坏燥土。还有，木火有势制金（若金中含水也同制），火与燥土有势制水（水中含木也同制）或制金水或制湿土（湿土中所含的东西也同制）；金水有势制火（若火中含土也同制）或制燥土（燥土中的含的东西也同制）；金与湿土有势制木火；水与湿土有势制燥土或火；水木有势制土（土中含的东西也同制）。

# 十四、大运，流年与应期

一般而言，八字讲贵贱，大运讲吉凶，流年看应期，流年应期的看法分以下几类：

①八字中某字在流年（大运）中出现或见其禄神与原身，则此年（运）要发生与该字有关的事情，流年（大运）中该字产生什么样的作用，就如同原局中该字发生的作用一样，这叫此字在流年（大运）的应期。

坤：戊 壬 甲 甲

申 戊 子 寅

分析：以子为夫宫，申为夫星，申子三合，是夫到夫宫之意，结婚时间应在壬申年，此年是夫星出现，拱入夫宫的应期。当大运到己未运的未运时，未穿了子，婚姻当出现问题，应期当在丙子年，子年时夫宫子水在流年出现见穿。

当一个八字到手，通过仔细分析评判，先找到八字的敏感点，由此重心逐渐引伸出别的事物。

一片金水之势，原局有几个重点，金多入丑墓，年干透刃被戊杀合，时上印带官帽，日支合丑。

断：1、宜当兵，回来要有工作（癸戊为配杀表当兵）

2、有个小职务权不大（时上印带官帽表职务）

3、文化不高，不爱学习（印入墓不开）

4、工作在企业，不会到政府、大型企业（丑看工作在年）丑生的是金，对丑来

说是食伤表企业。戊申也看工作戊土杀也表企业。

5、婚姻较迟，妻为同单位，否则较远。（年透刃表婚迟，看子与时上合，申可看婚，申带戊表示同一单位，如不是则远就是只看申在时表远了。）

6、头胎男孩，但有伤胎流产之事发生。（戊杀看儿，但戊坐申被泄严重，表伤胎或流产了。）

7、夫妻常吵闹，但离不了婚，妻想管你，但管不住。（申看妻带戊克壬，管夫之意，但坐壬子不怕戊克表管不住）

8、财运一般，工资财（官杀看财，戊虚了表不聚财）

断 到此，反馈正确，曾当兵，现在某大企业工作，小组长，妻远方人，两人好吵，婚较迟，头胎儿子，曾流过几次产。

然后就让命主关心什么事可问，这样我们在针对具体情况详细分析。（这样做，对我们来说，好有个重点去看，否则各人关心事情不同，你测别的事准确，因为不否合命主现在的想法，他也会认为一般，因此，我们就让对方来问，这样也就把他测算的重点解除了）。

这是丙戌下半年测的命行丁巳运，问：是否适合投资能否挣到钱？

这时回到原局看，戊表财，已运即戊到有投资之象但已在原局三合化局。（已酉丑）把巳财转化成了印，（这是巳火的特殊性，巳火可变性）明显的破财之象，并且是合作投资，（三合表合作）丙戌年，丙即戊到。非投不可，看到此，我断他不适合投资，更不适合，合作，否则要破财的。可我说了你也不会听，现已在运作投资。他承认为是，但又不想收手。一直到戊子年除夏天在见到他时，才知他的那次投资失败，损失较重。

提示：在实践时，注重抓重心，由重心引发慢慢的把别的事物引出来，关键是要分析到位，以一牵百，有的事物是在分析别的事物时引发出来的，并非一眼就都可看透彻，特别是测全家卦时。如是单纯问一项或两项事，这样目标明确，只要找准代表此事物的字做到仔细分析还是容易得多。
