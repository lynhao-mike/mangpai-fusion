# 干支互通在实际断命时的使用条件和使用方法

我们在前几期的资料中分别介绍了生、克、冲、刑、合（天干五合、地支六合、暗合、三合、半合、拱合）相破，伏吟以及六穿（害）地支四墓库在实际断命时的使用条件及使用方法。

本期我们重点讲天干与地支如何作用在实际断命时的使用条件及使用方法。

还像以往我们讲的一样，在实际断命时，我们不管使用到了哪种作用方式，首先要对使用的方式，有一个清晰的认识，要认识清楚这个作用方式的本质意义。它都是包涵什么内容？我们应该深刻的去理解它？这样当我们使用起来时，才能得心应手，不出偏错。

天干与地支到底该如何作用？古今书中各具一词、各说其理，但谁也没讲明白。而这些内容又是八字实战中的一个重要环节，如果你不能把这些道理搞清楚，至少说明你的八字水平还是一个很初级的水准，即使你有时给人看命断准了对方，那也只能说是你"猜"准的，而非真正算准的。

因为这些内容几乎涉及每一个八字的每一个断事环节，而我们又不可能把每一个命都写出来作讲解。因此，我们只能选择部分有代表性的例子，从不同的角度作解说，望学习者能认真领会理解，做到举一反三之效果。

八字中天干与地支的关系，从大的方面来讲分四个部分。

一、天干与地支是相生的关系

二、天干与地支是相克的关系

三、天干与地支是相同的关系

四、天干与地支是虚实的关系

因为第四条关系我们已在别的资料中讲过，此处不再讲解，只讲前三个关系。

具体如何理解运用这三个关系，下面我们一一来作讲解。

# 一、天干与地支是相生的关系

在天干与地支相配的六十个单柱中：甲午、乙巳、丙戌、丙辰、丁丑、丁未、戊申、己酉、庚子、辛亥、壬寅、癸卯，只有这十二个单柱为天干生地支的柱子。

天干生地支的柱子，天干与地支可以理解成一个象来看，针对不同的命局组合，可把它们理解为

1、印带官帽

2、官带财帽

所谓带的帽是指天干而言，天干相似一个人的头，古人是按万物类象而来定命理，我们可把这些带帽对应到现实生活中，如果头上戴了警察的帽子（指正常性工作上的，而非乱戴的哪种）你就知道这个人是干公安的，而如果一个人戴的是军人的帽子，你就知道此人是个当兵的，而一个八字中他的官带了财帽就可理解为他的工作大多与财方面有关等等，这些统称为带象。

这些仅是从干生支为带象的角度来说明这个事理，而命中有了这些带象的某个柱子，到底会不会被自己得到，还要看命局的组合情况来决定。

下面用实例讲解：

乾：丙 乙 辛 癸

辰 未 酉 巳

分析：此命年柱丙辰，年丙生辰，为带象，对日主而言，丙是官而辰是印，地支是印带官帽，丙辰与日柱天地合表日主得到了这个象，是个当官的命。实际也正是，在中央某部委。

坤：丁 壬 戊 乙

卯 寅 子 卯

分析：此命月令杀带财帽，工作与财有关，实际此命在某大企搞财务工作。

乾：壬 庚 丙 己

寅 戌 戌 亥

分析：此命年上见壬寅，对日主丙而言，是印带杀帽，日主坐下戌与寅木是拱局的关系，这样日主可得到这个壬寅，为印带杀帽，命主有职权，实际命主是个近万人的村书记。

有人会问年柱与月令的戌不是相拱吗？怎么理解？在这里就要多方面考虑命局的组合问题，首先月上见戌可理解成兄弟，哪么月戌上透庚金为财，又有年柱壬寅带象的拱局，（看合的资料讲过什么是拱局）这样就表示他的兄弟中有从事与财有关的工作的，实际也正是这样，一个哥哥从事企业管理有数万千资产。

提示：看命是个牵一动百的事，分析一个整体命局每一个环节都要用到，如果断章取义的去理解运用是很难看准事物的。

命中即便有了干生支的带象柱子，也不一定就表示这个柱子与命主有关，但至少要考虑到这个柱子是否会与日主有关。

乾：己 壬 癸 戊

酉 申 酉 午

前几天去江苏，客户报此命让看。

我断他是个单位里的副职，或是负责某一方面的主管。

反馈：是某港口主管车船的调度。

断此命是个主管职务，并非是看癸日主合了戊土官，关键是年上有个带象，印带杀帽，这里年上的酉与日支的酉是伏吟关系，哪么年干上的已就可理解为随年支的酉一起来到了日柱，这样年干的杀就表自己的权看。

提示：任何一个作用方式的存在，都可能影响到一个命局的整体，而存在的作用方式又都有各自代表的含义。

坤：戊 甲 丁 壬

午 寅 未 寅

此命，时柱印带官帽，丁壬合而寅入未象，命中木火一片，可表文化类的职务。

实际命主是某大学里的某方主管和讲师。

如果此命没有这个壬水的存在则一无是处，既不能表为文化方面

有成，更不能定为有职务。

提示：很多时候看命就是这样，一个字的不同就会影响到整个命局。

坤：戊 甲 丁 辛

午 寅 未 丑

此例我曾在前期资料中讲过，是广西一易友让测的命：我断此命是个开店铺的，与吃的方面有关，且财运不错。

反馈是：开一家风味小吃店，生意兴隆，财运好。

此例与上例：仅就一个时辰是寅时，一个是丑时，而命运就会有如此之差别，因为第一例有了个印带官帽的象，而二例确无此象的存在。

现今流行剖腹产选择日时，试想如果你对这些都不熟练，给人选的时辰就差这么一点点就会是命运天地悬隔。这两个命相对而言还都不错，有的会因选择时辰错误而导致使人终身残废甚至死亡，这都是很可怕的事情。

记得去年有人曾让我看过一个刚出生不到一礼拜的命，我断他会因血液病夭亡。实际是当时在医院急救抢救（后反馈夭亡了）说是请一个算命高手选的剖腹产，按看命高手说此时辰出生会富贵双全的。像这样的情况作为一个看命者，你如何向客户交待？又如何为自己作辩解呢？

提示：看命就在微妙之间，一个很小的失误就会影响别人一生。

像这样仅时辰差一点或命中组合差一点的命例我们还有很多，并且都是经过我们亲手测过且有准确反馈的。当讲到某种作用时我们就会通过例证对比的方式讲出来的。

乾：癸 丙 丙 辛

卯 辰 戌 卯

大连人，电话测：

断：命中有个工作，与财有关，看似有个职务，因自身无心求官的原因好像又没得到。

反馈：在税务局工作，领导一直想提拔他，但自己因无心提职务而拒绝了。

为何断他有工作？就看年柱印带官帽，与日合了，就表工作。

为何断他工作与财有关？

就看时柱印上透财合了自己（此处不为带象，但因有卯木伏吟都可表自己的看）

如果你能对这些方法熟练掌握，看命是非常直观的，根本不需要去讲那么多没用的道理。

乾：壬 戊 乙 丙

辰 申 巳 子

北京易友电话问此命：

断：是个贵命，当官的，可达厅级以上，主搞行政方面的工作。

反馈：是个厅级干部。

命中月令为官带财帽，巳申合自己得到了官，故命贵。搞行政而不主财有关是看戊通巳（这是干支通用，后会讲到）。

至于看官的级别层次我们在官贵资料中已进过，其实，从命理来看官有多大或财有多大这些问题都不关键，看命的关键问题是首先要把道理搞清楚，试想你一生能算上几个部级的命或上亿身价老板的命，即使是最普通的命，解释原理与富贵命也是一样的，当你把这些看命道理搞明白了，思维稍微一转换，看富贵命的问题轻易而解。

提示：学看命首先要从最关键处入手。

乾：乙 辛 丁 壬

巳 巳 亥 寅

命中时柱印带官帽，又有寅亥合，丁亥合的关系，可定为官命，具体是哪方面的官，就要用到别的理论来解释。

命中木火一势为阳，对亥水包围而制，亥是阴，命中阳制阴的组合可理解为公安，实际正是一位公安官员。

提示：看命时不要一见有带象就认为是富或贵具体情况还要结合别的作用方式来定论，带象仅是一方面的信息提示。

看命是个有机的整体，任何一个八字中的任何一个单独作用方式都不能表示此八字的全部。

# 二、天干与地支是相合的关系：

在八字体系中，天干可以克地支，而地支是不可以克天干的。

地支如果想克天干是有条件制约的，是什么条件呢？

只有在天干与地地支相合的情况下，地支才可以克天干。

在六十个单柱中：只有甲午、丙戌（戌要开库时）丁亥、戊子、己亥、辛巳、壬午、壬戌（戌要开库时）癸巳。共九个天干与地支相合的单柱，其中，丁亥、己亥、辛巳、癸巳，可以是地支克天干，别的柱也各有其意，请看例讲。

下面用实例讲解：

乾：甲 甲 乙 辛

辰 戌 卯 巳

此命时柱见辛巳是干支自合的关系，辛金是杀，巳是伤官，为伤官合制了杀表权力，但因辛杀虚透，根不通地支又在时上故权不大，是个小官。

注：看官大官小可看官运专讲，此处不作讲解。

乾：乙 辛 己 壬

卯 巳 巳 申

此命月柱辛巳自合，辛金当母看，早克母的象，丙戌年丙辛一合，辛则去，（此为合去的用法）此年丧母。

再看命中木火成势，巳申合制申金，申为财源头又在时柱命可小富，实际在外发展财运不错。

乾：乙 辛 戊 已

卯 巳 寅 未

此命是月柱辛巳自合，不仅财运不好，还住了牢狱，为何？命中伤官表自由星，制了伤官没自由了，关键还有一个象，日主坐七杀与巳火禄相穿，就表自己与官方对抗的意思，表不服官方管束而犯了牢灾。

提示：对比此二例，虽然都有辛巳自合的方式存在但组合不同，再有其它作用方式的制约，出现的结果就截然不同，因此分析命局时要面面具到，方不出偏差。

乾：癸 壬 辛 辛

卯 戌 亥 卯

易友电话问此命：

断：命不为官，但与官方有关系，他结交的都是厅部级以上的官员。

反馈：正确，他是搞企业的，但他高层官方关系很广。

命中月柱见壬戌自合，壬为伤官，为想法，合了戌中丁官就表示与官方打交道了，但因原局戌库不开故他认识的官对他都帮助不会大。

提示：命中有了干支自合关系，要理解相合的意思。

坤：丙 甲 庚 壬

寅 午 子 午

此例：时见壬午干支自合，表示什么意思呢？

女命食伤合了官就表桃花看，再看月柱又是官带财帽，靠男人挣钱的象，实际此命是个妓女。

注：要理解多种方式的组合运用，及象的提取。

坤：壬 甲 甲 乙

戌 辰 子 丑

我断此命有个工作，在企业工作，应该是负责写材料的。

她说：我在厂办公室，担任董事长秘书工作。

命中年柱壬戌自合，为伤官合印，日壬活木，伤官配印，表文类，就断她是写材料的工作。

乾：乙 辛 壬 乙

巳 巳 申 巳

此例：月柱辛巳有合为财星制印，表权力，辛虽虚透但地支见申表权力大，实际命主为某市国土局长。

提示：命中有了自合的柱子，要把它表示的含义提出来。

# 三、天干与地支是相同的关系

在八字体系中，天干可以当作地支看而地支也可当作天干看，天干与地支是可以相互通用的，也就是禄与原身的道理。

天干：甲 乙 丙 丁 戊 己 庚 辛 壬 癸

寅 卯 巳 午 巳 午 申 酉 亥 子

如：八字中天干有个甲，而地支见了寅，这个甲就可当寅看而寅可当甲木看。

还有就是地支所藏当天干出现时也可当作地支这个字来用。

如：八字中有亥水，行运见甲木，可表这个亥水到。

再就是：癸见丑，丁见未，为半个禄的关系，而丙与戊，丁与己是同宫的关系，因此两个同用一个禄。

干支通用关系，在实战时不仅可提供事物发生的来龙去脉，是断事更具体详细化关键还是应期的一种表现形式。

这些一定要多看实例，多看讲解，多思考！

因为这种情况很普遍，我们仅选部份有代表的例子作实际讲解，你们可通过这些例子作举一反三的去理解。

乾：戊 癸 丙 己

申 亥 申 亥

命主面测：

断：你要离两次婚，第一次婚是你的亲人应该是哥哥给你坏掉的。

他说，我已离过两次婚了，第一次确实是哥哥给坏的，你看我什么时候会再有婚姻？

命中离两次婚信息明显，为什么断他第一次是哥坏的呢？坏婚的原因是命中的申亥穿，而年柱戊合月柱癸水地支申亥穿，因丙戊同宫有兄弟象，这个戊就可当哥哥看，戊坐申与亥穿，就表哥给他坏的婚。

乾：己 辛 庚 辛

酉 未 寅 巳

断他是个在大型企业是当管理的命。

反馈：是某省联通公司副总

命有职权当管理是因时上辛巳自合，官杀制了辛金羊刃看，为何是大型企业？就看这个辛金通了年上的酉金，也就是辛就是酉或酉就是辛，因命中合制了辛就等同于制酉金，年到时的关系表远也可表大，就为大型企业之象。

辛巳年升职，辛巳就表原局制辛的时间到了，这是应期看法。

乾：丙 乙 辛 癸

辰 未 酉 巳

此例在带象中讲过，因当时还没讲到干支通用就没去解释这方面内容。

年柱带象为官命，而丙通时支的巳，这个巳也可当丙看，就表示这个官管理的范围大，故在中央某部任要职，而时上的癸巳自合，又表示他的工作性质是具体做什么工作的。

提示：看干支互通时，要结合象来提取信息，这样表述的更清析。

乾：壬 戊 戊 丁

寅 申 申 巳

日主坐下食神财源，被年上寅，时上巳包住跑不掉，戊通巳又巳申合，表都是自己的，是个大财团老总。

乾：己 丁 己 庚

酉 卯 丑 午

日主已通时上午，已坐下丑穿午禄，坏了身体就表有病，已丑年，丑穿午的应期到，突发病住院一个月。（懂了此理，看应期非常直观）

坤：丙 己 辛 戊

寅 亥 巳 戌

年上丙通日支巳，表婚早也表婚自谈（通了婚姻宫就表丈夫看而不当工作看了，丙戌年（21）岁丙到合辛自谈成婚。）

命有亥巳冲，母反对不愿意，但有辛巳自合，命主坚决要成。丁亥年母女同来测问，会不会离婚，我告诉她们离不了，只是母亲不喜欢这个对象而已。

提示：这里不仅有干支互通，还有年柱带象，又有辛巳自合，如

果你对这些关系搞不清楚，看命是无法看准确的。

坤：癸 乙 丁 丙

丑 卯 巳 午

命中丁通时午，原局有巳丑拱局，表丈夫到了家里，（丑到也表癸到了，因癸与丑是半禄关系）但时上通了午禄与丑穿，就表此婚不会到头，后果然又离婚了。

提示：真正的看命方法就是这么简单直观，根本不需要那么复杂的道理，如果你能认真理解，学会看命就不是什么难事情。

坤：乙 丙 戊 壬

巳 戌 戌 戌

前段时间去广东，朋友拿此命问我。

断：命中还有个小官当的，但晚运会有癌病发生。

反馈：是个街道居委主任，前段时间查出患有癌症。

为何有个官当？

日主戊通年上巳火，巳火带乙官，就表自己带了个官象。这里乙巳一柱既可当作印带官帽，又可当作自己带了官帽看。

乾：甲 癸 戊 丁

年柱印带杀帽，但地支子午冲如何得到这个权呢？看午通时干丁，而丁坐下巳，巳就是日主戊，通过这样转换就把年上这个带象归为自己所有了。

实际是个厅级领导。

提示：干支通用很关键，如不细加分析，是很容易忽略的。

# 如何才会学好命理

首先要明白什么是"命理"，我们讲"命理"即论命的道理，你只有把"理"学会了，才可谈得上入门了，否则，你所学的连"命理"的门也没进入，这些，当你看我们讲解的多些时，慢慢就会明白的。

所谓的"命理"即是命（八字）在告诉我们，此命该如何如何……而非我们人为的去强力与他（她）该如何如何……

也就是这个八字所包涵的一切吉凶都是已先天注定好的，我们只不过是利用我们所学的知识在解读这个八字而已，当然解读的正确与否，关键还在于你掌握的解读八字的理论及方法正确与否。

我发现很多易友拿到一个八字总是习惯性的去看旺衰，取用忌，其实这是非常错误的。

就像工程师解读图纸一样（只不过图纸可以人为的自行设计，而命是天意定好的），图纸中的每一环节都已设计定好的，你只不过是通过你所学把图纸解读出来就是了。如果你没学过解读图纸或学的技术不到或学的方法不对，那你就不易把这图纸解读出来或解读的某些方面会与原图纸表述的意思不一致。而根本不需要我们去评论此图纸某方面设计的不好或需要更改某个方面（这就是评旺衰，取用忌了）。

好与不好不是我们的事，我们只管解读出来就行，而你只会看好或不好（评旺衰找用忌）而问题的关键是你连根本内容都解读不出来，又有什么用呢？

因此，学习命理关键在于把学习的"路"选择对，"路"选对了，成功就只是个时间的问题，"路"选错了，最终会越学越远，直到一事无成，连自己的命也不会解读，还如何去给别人看呢！

所以要想学好八字命理，就要多从理解命理上去下功夫，当然对各种作用方式的正确理解也是很关键的，任何一个作用方式的理解错误，就有可能会产生分析一个命局的整体错误。

这里戊到就表巳到就表一种应期的看法。

# 下期资料预告各种作用方式的综合运用

我们通过九期的资料讲解，使学习者明白了每一个作用方式在实际看命时的重要性，但还有部分人讲一个方式怎么用还可以，一旦一个八字里出现了多个作用方式时就又摸不着头绪了。

为了使学习者能尽快学好本技术，从下期开始，我们全部采用实例，把八字中同时出现多个方式时该如何理解提取信息，以及一个八字看所有包涵的内容怎么区分提取作讲解。
