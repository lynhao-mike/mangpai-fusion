# 一、如何正确认识命理

首先要重新改变一个观念“是八字在告诉你此命该如何如何……，而非我们人为的或硬性地去确定他该如何如何……”

每一个八字都是确定性的，我们学习断八字就是在解读这个八字，每一个八字的干支都有他一定的含义，我们通过各种作用方式以及干支自含的“象”“意”等把他表述出来。可以把八字想像成是一幅画或一幅图纸，通过我们所学的知识把这幅画或图纸所包涵的内容描述出来，这就是断八字。只不过画或图纸是可人为的自行设计，而八字是自然形成的，也就是我们常说的天意或天命。

如果你能用这样的思维去看或理解命理，那么你就去解读他的时候就容易的多了。当然解读的细致与否和自身掌握的理论工具及自身悟性有很大关系。

# 二、阴阳的划分

从五行上划分，木火为阳，金水为阴。

从天干上划分，甲丙戊庚壬分阳，乙丁己辛癸为阴。

地支从寅到未是阳，从申到丑为阴。

在使用时要结合后面讲到干支的各自类象。

从大的理解方面说：阳表正义，阴表邪恶，如果八字中是表现出阳制了阴的字，一般表示公、检、法或执法有关,因为阳表正义的象;而如果命局体现阴制了阳的字，一般是表示坏的一方面，如恶势力等，因阴表示邪恶之就。

例：乾 戊 乙 壬 壬

# 午 丑 寅 寅

分析：原局寅午拱，午穿制丑寅合克丑，木火为暗阳、丑为阴。丑在月令表工作，丑阴被制，工作与执法有关，实际是一位刑警。

在平时的学习中，很多易友都不重视阴阳，主要还是不知道阴阳的重要性或不知具体的使用方法，因此我们一在强调，看似简单的基础性知识，在八字实践时就变得高深莫测了。关于阴阳的基本使用方法在后面的例题中我们都有讲例，请读者留意。

# 三、天干的自然特性：

提示：这点要结合自然现状去理解，把十天干类象成我们现实生活中的实际物体然后在比喻作人的特性来理解。

1．木：可联想成自然中的花草树木之类，甲木表树木，成材之木。乙木表花草或禾苗，因此逢日干是木时要看是活木还是死木，活木有两个条件，一有根，二有源即为活木，当天干见甲乙而地支只有寅或卯时可共用根，活木喜见火，表示木开花表成就，类像到人就是有作为。如果是死木，喜见金用金劈木，更利于生火之用。

活木表树木，花草类，死木表示柴火类。

2．火：丙火表太阳，一般可类象成名气，无私阳刚、热心肠等。丁火表月亮、柔和、秀气，也表热情等，火有奋发向上的精神，一般表示有斗志。

3．土：戊表大地之土，因此一般喜见甲木来克方有用，己土表田园之土，喜用乙木或用金泄。

4．金：庚金表顽固之金，粗糙之金，喜用火煅成材。辛金，表首饰、细微之金，不怕弱，喜癸水。

5．水：壬水，江河奔腾之水，喜见甲木泄。

癸水、雨露、杯中水、喜见乙木。

五行、干支的其它特性。

# 四、辰戌丑未土的特性：

辰为温湿之土，戌为干燥之土，丑为寒湿之土，未为热燥之土。

对于火来说，戌与未土基本上不晦火，辰与丑土晦火，辰土易晦巳火，丑见午火，晦之更甚。

对于水来说，戌与未土克水力大，但戌主要克亥水，未土主要克子水，辰土与丑土原则上不克水，但辰见亥为墓水，子丑合为合绊水，如壬辰或癸丑单柱时，反为得水之根为水旺。

对于金来讲，戌土与未土不能生金，反而脆金制金，而辰土与丑土则能生金。

对于土来讲，戌土与未土帮土力大，辰丑土帮土力小，几乎不帮土。

辰戌丑未土四墓库，如寅申巳亥，见各自的墓库，都以入墓论，而子午卯酉见各自的墓库因半合的关系不论墓。

木的死活

木分死活，活木为树木，草木。死木为柴火或为木材由木材加工的东西，活木有根又有生这两个条件必须具备，无根无生为死木。

活木与死木遇水（包括大运之水）情况又不同，活木喜水，尤其是生在夏天或比较干燥的活木更喜水生，相反，死木见水则坏了，一种情况是水多易腐烂，

另一种情况是水湿而木不能生火了。

木遇空亡时，像木之心空了，如为活木，则长的不会很健壮，如为死木，则难以成才。

活木之根可以阴阳互根，死木之根则不可互用。

巳火的特殊性：

巳属性是火，但又是金的长生，故它兼有双重性，当火旺有木生时，巳火表现克金的性质，当火弱时，巳见丑和酉就会归顺于金，合金局助金。是十二支中唯一的性质随环境改变的一支。八字中遇到巳火时，要看局中是木火旺还是金水旺，以确定它的性质，但有时它还能随着大运的改变而变化，遇之要细心辨别。若原局中巳火占有主要位置的人，头脑灵活，变化快，思维快，性格方面多有易变心理。

见了巳火一定要分清是要用还是要制，岁运作用时要注意。

水的特性：

子水壬水为流动之水，流荡奔腾，不是很生木，癸水亥水为静水，雨露之水，能生木。

# 五、如何看待干支虚实

干支的虚实与我们通常所说的衰旺不是一回事，传统命理讲衰旺是参照月令来定的，民间我们讲的虚实只就一柱干支而言，与周围的生克关系没有联系。虚与实本身就有象，在四柱中表示一定的含义：更重要的应用是看八字的应期。

天干之气弱，地支之气强，天干要依附于地支才能得旺，地支不生扶天干，则为虚。

虚实的基本原则是：天干无根无生者虚，天干有根有生者实。

# 六、如何理解藏干

① 地支中的藏干一定要用，关系着八字事物或事情定性及定量的问题。

例：乾：乙 己 己 庚

巳 丑 未 午

分析：原局火与火土有势，制丑土及丑中辛癸，原局无财，丑中辛及癸表示财富，财与财的原神同被制，表示很富有，故是个大富命。

例：乾：丁 庚 己 庚

亥 戌 未 午

分析：原局火燥之势，制年上亥及亥中甲木（制财、官了）官与官的原神同

被制，表示大富贵，此为蒋介石之命。

通过以上二例。如果不用地支藏干是很难从八字中看出这些内容的。

② 藏干还表示应期的问题

例：乾：丙 庚 丁 辛

寅 寅 未 丑

分析：原局金多入丑库，金为财表示克父，行癸运时，表示丑到了，应父之灾，丁亥年，丁即未，大运癸与流年丁冲表示原局未丑之冲，当年父亡。

例：乾：乙 乙 庚 丙

未 酉 寅 子

分析：原局木多入未库，乙木坐酉为绝地，要有伤的象。行辛卯运，辛即酉到列，卯即乙到，乙辛冲，卯酉冲，此运对乙不利，己丑年，丑即酉到，丑酉拱冲卯木，表示手术，当年开刀手术。

如上例蒋介石之造，行甲辰运时兵败，为何？因原局制亥中甲，甲运时甲透干无制了，表示制不住对手，反而被对手制自己了，故兵败。

在实际断命时，一定要注意看藏干，这点也很关键。

# 七、长生决的正确理解

① 与别的所讲不同之处，我们讲阴阳干同长生，如木都是长生在亥，而不分阴阳之木，余类推…

② 火与土、同长生。

③ 天干土长生在寅，地支土长生在申。

④ 一般常用，长生、帝旺、死墓绝、五种状态。

⑤ 长生一般表示事物的开始或新的事物。

帝旺一般表示事物已成熟或达到顶峰。

死墓绝，一般表示事物的衰退或陈旧，如果此三点一应时，也可表示事物的死亡。

# 八、对宫位的认识

# 宫位类象

首先四柱宫位含有自己与六亲的信息，除日干代表自己以外，其它都代表自己亲人的位置，如年柱代表祖业、父母、外戚长辈；月柱代表父母、兄弟；日支代表妻子或丈夫所在的位置、时住代表子女后代。父母则既可以在年上体现，也可以在月上体现。

四柱宫位还代表着人一生从小到大的时间顺序，即：年柱为童年与少年时期，大约 1-18 岁；月柱为青年时期，大约 18-35 岁；日柱为中年时期，大约 25-55岁，时柱为晚年时期，大约 55 岁以后的岁月。除此之外，从年到时，还是从小到大或从先到后的顺序，比如一个人命中有三次婚姻，那他三次的排列顺序也是从年到时依次来找。

四柱宫位还代表一个人生活中空间所在，比如你要出远门，命中对应的宫位就是年柱或者时柱，因为年柱代表远方，而时柱代表门户。月柱代表祖籍，日支代表现在居住的地方。

四柱宫位还代表不同年龄与不同关系的人，年柱代表外人，长辈或老人；月柱代表同学、同事、领导；日支代表与自己非常亲近的人；时柱代表晚辈、学生、下属、朋友。

四柱宫位还代表人身体的不同部位，年柱离日柱远，代表腿、足、四肢；月柱代表人身体的躯干部分，如脊、背、肩等；日支代表人体最重要的部分，如五脏六腑、心脑、脊髓；时柱代表与外部相通的人体器官，如头面、手、眼、嘴、耳、鼻、生殖器、排泄器官等。

四柱宫位还代表人的使用之物，如看车，那就在时柱上找，因为时柱代表门户，故出门的代步工具车子就在时柱上找。其它还有：年柱代表鞋子、拐杖或别人之物；月柱代表祖产、学业、单位、学业；日支代表房子、卧室、私产；时柱代表车、门、衣服、帽子、眼镜、化妆品，进出之钱物。

四柱宫位还代表人的心智与情商，有的人容易受外界环境的影响，有的人很容易影响别人，从四柱宫位也可以看出这些特点来。

四柱八字中，天干代表人的表象，如外表特征、性格表象、容易被外人看出来的显著特点；而地支代表人的内在的东西，如内心世界，与家人的关系等，往往是比较隐蔽的思想与心志。

# 宫位吉凶

一般而言，年柱为吉神得力，则主出生家境较好，祖辈有过荣耀之人，若被月柱冲克，则祖辈遭受破产、败落、迁变之打击，多代表出生前的信息。

吉神在月柱得力，则主自己可受长辈庇荫，可承父辈的产业，若被日柱冲克，则代表背祖离乡，六亲缘浅，同胞不和等信息。

吉神在日支，主妻贤夫贵，中年可振兴家业，若被克破或化合为忌，则代表

好而不久，婚姻中途抛离断弦再续。

吉神在时柱，则主儿孙才能出众，教敬自己，被冲克则好而不久或远离自己，若时柱为忌，则代表子女不孝，凶顽不尊，晚景凄凉。

除上述内容外，在宫位的认识上一般还有年，月表示别人或与自己比较疏远的关系，而日支和时柱表示自己或自己比较亲密的关系。这样当命局中有财或有官时，具体应该在自己身上或应在别人身上体现，这时就要通过宫位来划分。

寅 亥 辰 卯

为何断他一生是在别人手下挣钱的小老板？

分析：原局水为财，而财在月。年月又寅亥六合表示亥财被寅控制，这样自己要想得财就只有从寅木手中去取财，故断他是个在别人手下挣钱的命。实际反馈也正是如此。

因此看命时不要一看命中有财，就断会富，一看命中有官就断有贵，这是很片面的，关键要分清这个财或官是不是你的，如不是你的财，有时候官反而会起到反面效果。但这要根据具体情况来定。

# 九、“禄”“刃”的作用

# 禄神类象

即：甲禄在寅，乙禄在卯……，它是日主的延伸，代表日主到地支行使权力，其心性含义有：亲为、独立、主张、自我意识、尊贵、独占、享受之意；禄的物象为：权力、领地、财富、供养；于人体表示身体、肢体，女人的肉体、寿命。

# 羊刃类象

甲刃在卯或乙，丙刃在午或丁，戊刃在未或己，庚刃在酉或辛，壬刃在子或癸，只有阳干有羊刃，阴干没有羊刃。羊刃虽是劫财的一种，但比劫财之心性更凶悍，其心性含义为：胆大、勇敢、凶恶、狠毒、不计后果、不顾面子、占有欲、侵害、不利父、妻等；其物象为：刀、枪、剑、手术、兵器、武装、政法、执法；于人体表示四肢、身体。羊刃喜制服，制之则用正，可能从事的行业有：军人、警察、执法人员、外科大夫、运动员、武人、炒作者等；羊刃无制服则用偏，则可能从事：匪徒、赌徒、打架斗殴、杀人越货、非法谋营等。

禄在断八字时很重要，在实际使用时要针对不同的八字组合去认真的分析对待。

# 十、不论旺衰与“用神”“忌神”如何看八字

要明白这些问题，首先要从道理上来理解，要搞清楚八字的真正本质是在于表述人生，而非我们人为的去强加与他的好或坏。我们测八字就是在解读这个八字，如果这个八字是存在的，那么他所包涵的内容也就是一定的，我们只不过是通过我们所学的知识把这个八字解释出来而已。

如果这个八字是存在的，那么他的干支组合就已自身决定了他的所有状态，至于旺或衰那是先天就决定好的，当然也就体现出了他所包涵的意义。

如果这个八字是存在的，那么他的干支组合就已自身决定了他所代表的意义，我们只不过是把他解读出来而已。

就象一个人，因为他眼睛长的什么样，就决定了眼睛所代表的含义怎么样是一个道理，一个人既然能富贵是因为他的相貌表现出来就是富贵相，只不过你要会解读他，这是从相学的角度来分析，其实八字也是一个理，只不过使用的分析方式不同，但他们道理还是一样的。

所谓的取“用神”或“忌神”说穿了就是一种假设，是毫无意义的。还以看相来说，当你看到一个人的鼻子和眼睛长的都接近于好而非好时，你就会说，如果他的鼻子和眼睛长的怎么怎么样时，此人就会很好了。这里的“如果”就是你在取“用神”了。而事实上他不会按你的意愿去长成你理想的模样。因为就是这样的模样就正好能体现出他一生的状况与模样表现的状况是一致的。这就是相学里讲的命。

看八字就象看相一个道理，八字里的某干或某支的出现不是我们人为的，而是先天决定的，而这样出现就是在告诉你他们所代表的含义。而我们就是解读他，并非找什么“用神”或“忌神”。换个思维，即使你找了“用忌神”又能说明什么问题呢？即使你认为是“旺”或“衰”了又能表现出什么？这一切都解决不了看八字的根本问题，也可以说，看八字和旺衰用忌神是毫无关系的。

例：坤 乙 戊 戊 己

断：

① 命要离婚，一婚不到头。

所断这些一都是使用到基本的各种作用方式。与旺衰或用忌神是毫无关系的。其实我们自己也根本不知道此命是旺还是衰，更搞不清什么是用神或忌神。

而是直接把原八字中所表达的意思提取出来，告诉给对方，这就是真正好断八字的方法。

如果你能用这样的思维去看八字，那么你就会进步很快的。

# 十一、如何看待神煞？

神煞问题只要从使用意义上去理解，是个毫无根据的名称。

记得山西一易友让测女儿工作问题，根据八字所显示的信息，我断工作能成，与文化类有关的工作，时间在申、酉月份。事实反馈也是如此。然后此易友又问此事是否有贵人帮忙。因女儿命中有个贵人星。我说：此工作已成是从别的作用关系上来决定的，而非看有个贵人是来定的。假如此事通过分析不会成功。哪么那个贵人还能有用吗？事情的成功与失败与贵人是没有关系的。

# 十三、对八字“象”的认识

“象”是断八字时至关重要的东西，是八字的细化，是八字的最高境界，象有多种，有阴阳象，干支象、五行象、十神象、宫位象、禄刃象、墓库象等等。

在使用象时要针对不同的八字组合，不同的作用方式，从多方面多角度去分析，有的八字组合“象”的表现较明显，而有的较难把握，还有的可不看作用方式直接取象就行，这要具体对待。

“象”就是相似之意，古人把有共性又曾相似的东西归为一类，称之为“象”。因此，“象”也称类象。因他们在某一特性或形状或用途上有相似之处。

# （三）十二地支六合

六合共有六组：子丑、寅亥、卯戌、辰酉、午未、巳申

# ⑴地支六合的作用意义：

地支六合是八字中最紧密的合，与天干合也是一样，象征结亲、和好、亲近、羁绊之意，使用时要仔细分辨。

1．助合：辰与酉合，酉金逢助，辰土因合而泄，自身作用减弱；

寅与亥合，寅木逢助，亥水因合而泄，自身作用减弱；

午与未合，未土逢助，午火因合而泄，自身作用减弱。

2．克合：卯与戌合，戌土受克，巳与申合，申金受克。

3．闭气：子丑合，丑中之金闭气，辰酉合，辰中之水闭气，午未合，未中之木闭气，卯戌合，戌中之火闭气。闭气则不能流行，不能引拔，成无用之物，只有逢冲时才可用。

4．合伤：辰酉合，辰中之木被酉金克伤；寅亥合，寅中之火被亥水克伤；巳申合，巳中之火被申中之水克伤。

# （六）地支六冲：

六冲是位置相对应，五行有相克，有冲突，相战之意。除了位置对冲还有支中所藏冲克之意。子午相冲，子中癸水克午中丁火，午中己土克子中癸水。丑未冲，丑中辛克未中乙木，未中己土，丁火克丑中辛金癸水等，余仿此……

六冲中，子午冲，子水冲克午火，而午火对子水有冲无克、卯酉相冲，酉金冲克卯木，而卯木对酉金只是有冲无克；寅申冲，申金冲克寅木，而寅木对申金只是有冲无克；巳亥相冲，亥水冲克巳火，而巳火对亥水只是有冲无克。

相冲在八字中有相邻冲和隔位相冲，有顺冲，有倒冲，因此，相邻近冲则力量大，隔位相冲则力量小。视其情形有不同的作用。

1．相冲可先坏去掉，相冲以相克为主，其伤害要胜于一般相克。

子午冲，午被克坏，子则劳力；卯酉冲，卯被克坏，酉则劳力；寅申相冲，以申金克寅木为主，寅中丙，申中壬同受伤；巳亥冲，以亥克巳为主，巳中戊，亥中甲同受其伤；辰戌冲，土冲变虚，辰中乙癸，戌中辛丁同受其伤；丑未冲，土冲变虚，丑中癸辛，未中乙丁同受伤。以上冲克作用，都是两支相邻力量对等的情况，如受冲克之神因得八字势，力量大于冲神，这样受冲克的一方反过来胜了冲神一方，此为倒冲。

2． 相冲可破局破合：相合而逢冲则大伤其合，并非贪合而忘冲。若冲神在合神中间则破合，论冲不论合（如子午丑三字的关系，因午在子丑合中间，使子丑因冲而不能合），若冲神在合神之外紧邻，则伤合，相合与相冲兼论（如子午丑三字的关系，子午冲合子丑合同时都论），若冲神在合神之外，相隔不紧邻，则为贪合而忘冲，论合不论冲（如子丑午三字关系，子丑论合，午不能坏其合），以上是论六合，但三合局的情况也是如此。

3．相冲可冲动冲开，近冲则克破，隔冲则伤损。单有运冲则冲破冲坏，岁冲则冲动冲开。所谓隔冲，是指八字中的年与日或月与时相冲。运冲是大运来冲八字中的字，岁冲是流年来冲八字中的字，动为发动，开为开墓。墓近冲则坏墓中之物。隔冲为损失，遥冲为动，也为冲开之意。

# （八）十二支相穿

冲是正面的，当敌不过对方里可以跑掉，故冲还有冲走的意思。但穿就不同了，穿是从侧面而来，跑不了，是仇敌，力量太可怕，要拼个你死我活，而且防不胜防，躲不了的。穿也为制了，但必须要旺，命局中不用的东西被穿论吉。穿有两种结论：一、为穿倒，为穿反了，正的东西可以穿偏，偏的东西也可以穿正。“倒”表示立不住、破坏、消亡；故也常将穿说成是“穿倒”。二、主凶灾，八字有穿多主凶灾或死亡，看被穿的字在八字中代表的六亲是否穿坏，如穿坏了就有可能死亡。

以相克又带穿的情况最严重，如：子未、卯辰、酉戌（戌为燥土，坏酉）、丑午（丑为湿土，晦火）这几种穿的最厉害的；而申亥、寅巳穿是穿中带生，被生的一方穿时还有动的意思，且杀伤力不严重。

流年来穿制八字中主位的字，主心情不好，办事不利或受阻，如配偶宫受大运来穿，此大运如没有结婚就不能结婚，如结婚了就会出问题，但不一定离婚。也见过在穿的运上结婚的，但恰好是流年制了穿的字，且夫妻星动了。

# 十一、看八字如何入手

在分析八字之前，首先要把前面所讲的各个基础性环节掌握熟练，对任何一个环节的 都会直接影响实际断命的发挥，这样就直接导致准确率下降，只有对基础越熟练，分析八字才能越透彻，相应的准确率就会越高。

通过前面内容的学习，我们知道看八字是与旺衰 、用忌神没有任何关系的。我们只从作用方式与十神、宫位及各种象的提取，把这些内容有机地联系在一起就是分析八字。

我们看命，首先注重原命局分析，因为原命局中就包含了这个八字所有一切信息，任何事只有先从原命局中看出是必须存在的，然后，大运流年才是事情发生的应期。

因此，在看八字入手时，先从分析原局开始。

具体如何操作呢？

我们都知道，在八字中日干代表命主，因此我们就要先从日干入手，先看日干是否有生或合，因为日干所生者，就表示日主的想法了。然后看日主生的东西是去干什么了？是生财还是制官杀，这就是日主思想，如果日主没有生，就看日主是否有合，或合财或合官，日主的东西也就表示日主的追求了。

如果日主又无合，就弃日主不看，看日支是否有生或合或其它作用方式，因为日柱一柱又都代表命主，故日支的作用也就代表日干的作用。

如果日支也没有作用方式发生，我们就看是否有日干的，禄或比劫在作用，因为禄比也都与日干同类，也可代表日干的作用来看。但这种作用方式层次都不会高。

如果日干有生或合，而日支也有作用方式发生，这时，我们就要以日柱为主来看。

例：乾 甲 壬 辛 甲

寅 申 巳 午

分析：日干辛金生壬水，壬水去生甲木财星，从这一点来看日主的意思是要去求财。再看日支巳火，首先辛巳是自合的关系，巳火官星与日主合表示日主要用这个官去做事，然后再看，巳申合，表示巳要制申金劫财，这时巳就表示工作看，且这个工作是制申劫的，以相来看应该与执法有关的。

实际也正是这样，命主有工作是个警察，同时又自己开有酒店，这样分析出的结果与命局表现出的事情是相符合的。可以这样理解，是八字在告诉你这个命该如何，而非我们人为的去决定八字该如何……

所讲这些都是分析八字的基本方法，随着知识的提高和后面讲到的思维的转变，这些框框都可不用，但开始时还必须要用的，否则分析时就没有个方向了。

# 十二、如何具体分析原命局？

分析原命局是分析八字的第一步，也是很关键的一步，只有把原局分析透彻了，后面的事情才会有结果。

分析原局就是把八字中的包括宫位、十神、干支类象，五行阴阳知识，结合八字，找到它们所对应的各自位置，和各自在八字中所代表的意义，然后通过各个作用方式，把作用的结果提取出来。这样说起来感到有些麻烦，其实当你对基础性知识熟练后，看到八字时马上就会找到对应点，当然，这也需要大量的实践过程。

下面我们就用实例来说明：

乾：壬 辛 戊 乙

寅 亥 辰 卯

这是今年（丁亥）7月3日，某妇夫面测的命例：

# 原命局断：

① 你精明能干，头脑灵活，但自己的才华都是为别人创造了财富。

② 你一生是个在别人手下挣钱的小老板，不为富命。

③ 想独立干还不行，可心中又不服，如独立干了定破财。

④ 43 岁前，有一次因事业上的大破财，可能会使你倾家荡产。

⑤ 妻子为同乡之人，心直口快，是个贤惠的家庭主妇。

⑥ 命中头胎男孩，且有伤胎之事。

⑦ 儿子不在身边，要到远方去。

⑧ 如果你自己开车易出车祸。

⑨ 先送父的命。

解：①月透伤官主技能、才华等，伤坐亥财，亥透壬坐寅杀，寅亥合，寅把财留住了，意思是自己通过智慧想创造财富，但自己的努力确被别人控制得到了。我们在前期资料合的用法中讲，六合大于别的合，故自己的才华为别人服务了。

②财被寅合主了，寅为别人，但亥又半合卯，亥入辰墓，为自己想得到财，就只能从寅木那里去取财，故是个靠别人大老板挣钱的命。因亥卯为半合力小，辰墓不开无用。（看下期资料地支墓库的用法）故不为富命。

③辛冲乙，想通过辛的手段制住乙官，自己当老板干事，故不服气，但辛根本制不住乙官，为官不为自己所用，一但用官自己干事，定出事破财。

④43 岁前，是按大限来定出的，因干事业破大财，会倾家荡产，亥卯半合自己一直不死心，想独立大干，官主事业，而卯与辰穿，不相容的象，坐支辰为家，故有倾家荡产一说。

⑤亥财为妻，合寅合卯，故妻心慈善，因亥在月令，月为同乡同事故近。

⑥男命看杀为儿，在年柱故生男，卯辰穿故有伤胎。

⑦寅杀为儿在年为远方，被合住，要留远方的象。

⑧时柱为门户（看宫位用法）门户看车，与日相穿，有因车致灾的象。

⑨看财为父，父在月令，不喜虚透年干，亥入辰库，为财临墓库父当早死，故先送父。

这些信息完全符合命主的现实生活。

其中日主提到看那一年破的财。这时我们再看大运找流年，查运，4 岁起运，34 岁交乙卯大运，为命中的坏事来了，当定在此运。看流年，42 岁，癸未流年，癸财透合日主，为想挣钱之意，亥卯未合局想要干大事了，谁知一为癸财虚没财，二为三合局不成立（看三合局的用法）卯去穿辰，处处破财之象。

实际：一直跟人家大老板干的很好，癸未年想跳出大老板的控制，独立自己干，结果大破财，他说直到去年（丙戌）才把外债还清。

他说：在二十左右时为司机，一次意外车祸，差点丧命，到现在也不敢开车了。

儿子远在福建当兵，长期不在一起。

别的事他说了一句话，用毛泽东与周恩来的关系来比喻他的处境。看来头脑聪明，无所不能，他是肯定的，不服气也不行。最后我劝他，你还是老老实实跟别人干，当你的小老板吧！这就是命。他已经接受了我的建议。

一九六九年十月六日 卯时

乾：己 乙 甲 丁

酉 亥 午 卯

这是今年（丁亥）8月8日面测的例子：

# 原命局断：

① 命硬克父，先送父。

② 文凭不高，很早就出来挣钱了。

③ 命犯离祖，出外创业。

④ 兄弟无靠，自打江山自创业，一切靠自己。

⑤ 干的是自己亲自操作的技术活。

⑥ 是个小老板，收入还可以。

⑦ 一次婚姻不到头，要二婚。

⑧ 妻无助力，家里家外都是自己。

⑨ 头胎女孩，对你没有帮助力。

⑩ 自己小脾气，心中顶不住事。

解：①看己土为父，己虚透坐酉杀，己的根在午，亥午合克，卯破午，（看地支破的用法）己父的根伤，故克父在先。

②印起了坏作用无文化，年上见财，很早就出去挣钱。他说十六岁就出门打工。

③年月为祖，时上卯冲酉合印，又时透丁火为吉，时为远方，故出外创业。

④月令看兄弟，透乙克他的己财，亥又合克他的午火，此命日主为活木，喜见火（看十干的特性一文），火表示活木的成就，而亥来合克午火，故兄弟无靠，反来讨他的便宜。

⑤活木喜火，时上丁火为技艺，丁坐卯劫，比劫为手脚，卯冲酉，破午合亥，意思为用手脚做事。因见丁故为技术性的。他说自己在城里开了一家汽车修理店。

⑥甲合己，己坐酉官，酉冲卯，故为小老板，财生官，官制不尽时当财看（财

运看法有讲）所以收入还可以。

⑦看婚姻宫午中藏己丁（看地支藏干的用法）透天干，年上己时上丁，又宫被亥合，卯午又被宫坏。故要二次婚。

⑧婚宫己坏，宫为家坏了，故妻无助力。

⑨看酉官为女孩，酉生亥，亥合克午，对自己无助力了。

⑩小脾气，因伤官用事，伤官有心胸狭隘之意。

随后命主又问今年运气如何？查运，3 岁交运，现行辛未运之未运。

我说：你今年（丁亥）工作一直不顺心，心烦的很，想改行，但又改不了，不改行生意明显减少。我问：你带徒弟吗？他说有，我说您最烦心的事是你的徒弟引起的，是徒弟给你作乱生意吧。他大吃一惊！完全认可。

解：看丁为伤，应原局的午、丁虚透即午虚了，丁要走。丁伤在时柱代表学生、下属、徒弟等，丁亥年，为丁伤到，带来一个亥水，而原局有亥午之合，午为生意，丁用亥把他的生意合伤了，没了生意。午为思维被伤故烦恼。

实际是：今年五月他的一个徒弟学艺毕业，自己开了一家修车店，徒弟不仅带走了他很大的一部分客户，反而四处宣传他的坏，给他坏了生意。

看来人生之事都是命运安排好的，一个小徒弟能给师傅带来如此烦恼，且在命运中有所显现，一切都是注定了的。

# 十三、原局、大运、流年的关系。

一般来讲，原局定层次，大运定吉凶，流年定应期。

人生的层次等级及重大事情，都在八字原命局中有显示。比如：一个人的富贵贫贱，职业学识，脾气性格以及是否早死父母、早婚或晚婚，几次婚姻，子女几个是否孝顺等，甚至一生能当到多大的官或赚到多少钱，都在原命局中有显示。而大运、流年只是应验其重大事件发生时间，所以，分析原命局才是看命的关键所在。

只有先从原命局中分析出某件人生大事是必然发生的，然后才可能断出在某运某年中发生此事。

因此，我们在看命时先不管大运流年，就要直接从原命局中说出许多事情，且简练到位，切中要害，直入客户内心，如果细问，再推断具体流年。

当你从原局中分析出事情后，再看大运，大运直接和原局发生作用，以决定某件事情的吉凶，然后在看流年，流年和大运发生作用，然后在作用命局，以决定某事的应期。

乾 癸 戊 辛 壬

丑 午 卯 辰

分析：原局丑看母、癸也看母，而戊癸合丑入辰之墓，有早克母之象，这是从原局分析出的结果，但具体应在什么时间克母，就要去看大运和命局的作用来决定在哪步大运了，这时一般要结合原局来看大运，行丁巳运，虽然有癸丁之冲，但己丑又拱局，表示丑有生意，故此运无妨。行丙辰运。①丙即原局之戊列，表示戊要合克癸的时间到了，而辰即原局之辰到，因为原局有丑入辰墓应凶的象，也表示这个辰的应期到了，故定克母在此运。然后找流年，因为流年表应期，所以一般要找代表此事物的字出现的时间，戊辰年交的丙辰运，虽然戊与辰都到了，但原局是丑表示母，但关键的丑没有到位。故此年也无妨。下边的己巳年也不到位，庚午年，先与大运作用，丙克庚，辰泄午，使午无力穿丑，也无妨，辛未年，先与大运作用，丙合克辛，使辛无法生癸，辰不泄未，未作用与原局，午未合，表示因未的合使午动了起来去穿丑，未又冲丑表示丑的动。而天干之辛表示丑到之意，这里的穿和冲都表示促使丑去入辰墓，即丑代表的母的象应凶的时间到了，实际也正是此年丧母。

提示：这里需要注意，如果是大运与原局发生作用时，原局中的字易受伤，但也要看力量的对比去具体对待，后面的例题讲到时要多加注意。而流年与原局发生的作用，一般都表示动的意思，这是原则性的讲，针对特殊情况时，也会表示别的意思，如：年与命是合绊的关系时就表示此字被绊住不发挥作用了。看后面的例题时要留意。

# 十四、应事应期的原则

原局有什么字，就应什么事，只是原则上讲的，但在实际使用时还要注意到藏干以及禄与原身的关系，因为地支中的藏干也可以代表这个字的本身来使，而禄与原身就更表示一个字的意思，只不过是在天干与在地支所处的位置不同而异。

例：乾 丙 庚 丁 辛

寅 寅 未 丑

分析：原局庚看父（因庚在月令）而庚坐寅为绝地（长生用法，绝有死亡之意）这里表示有克父之意。时干见辛金为金多入库了（墓库用法）同样表示克父之象。行癸巳运，癸表示丑库到，因癸见丑为禄，（禄与原身）地支寅巳穿，庚坐的寅受伤，（表示要应父之事）此运要克父。丁亥年，丁即日主到也表示未土到（因丁即未）亥冲已动，巳去穿寅，寅受伤，亥在参与原局作用。

寅亥一合寅即去，而庚坐寅，庚随寅而去（只可理解上这么去讲）未丑冲开库，金入库了（表示父亡）此年丧父。

乾：庚 己 庚 丁

午 卯 午 亥

分析：原局卯木看父亲，卯午双破（破的用法）父伤之意，这是从原局中看出的信息。行庚辰运，庚表日主到，而原局之亥入大运之辰墓（用法）表示亥不生卯了。辰又去穿原局之卯，表示父要有伤在此运，戊子年，子辰拱局，（大运与流年的关系）表示辰加力又表辰动之意。子在冲原局之午，表午动去破卯，又子去破卯木（流年再与原局作用）

原局有不利父的信息，大运表父受伤的时间，流年到位了。

此年父盖房从高处摔下来，死里逃生。
