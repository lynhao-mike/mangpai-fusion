# 《官运篇》

# 一、看官运的基本原则：

1、官命的核心必须有官、杀及印星，官杀表职务，印表权。官职的大小看做功的效率如何。

2、官命的官杀及印星不管如何做功，都必须有根。官杀虚透表示名气。

3、官命组合中，如出现申、酉两字做功，一般是与法律有关的官职。如出现申、酉、寅丑字有关的组合做功，一般与公安有关的官职。

4、出现羊刃与七杀组合，一般表执法部门，实权派或军队部门的官，如出现伤官与官杀组合，一般为实权或记检部门。

5、食神合印为带有思想文化性的行业或文官。

乾：庚 壬 乙 丙

寅 午 酉 戌

分析：原局三合火局，木火之势，酉戌穿制了酉杀，日主乙庚合，表示日主要得到庚官。壬午自合为去了壬水印星，为伤官制官。丁亥运，丁壬合把印合住，地支寅亥合，午亥暗合，都表示合印的意思，此运升职，省政法委书记。酉表法律。

乾：壬 丙 壬 丙

子午辰午

分析：日主坐辰为七杀，又有财生杀。年上壬子水都入了辰墓中(为多而墓之的原理), 年柱子水羊刃又合辰土七杀 (但是辰土杀统了局中之财, 为杀统财) 是个官命, 因杀统了财故与钱有关, 是个法院经济庭的庭长。而羊刃合杀有执法的象, 故也表法律有关, 为法官。

乾：丁 己 癸 丁

未酉巳巳

分析：局中火土成势，制酉金印星，而酉印头上带了己土杀帽，为印带杀帽子，印表权力，此人当官，未土七杀制酉金，七杀做功主公检法。酉在象中表阴，表示坏人，犯罪分子，制酉就是抓坏人的象，阳制阴表示枪象，火也有枪象，巳火两重。为刑警队长，辛巳年升，因辛到即酉到，辛巳自合制辛金，升职。

乾：戊 壬 乙 己

辰戌未卯

分析: 原局燥土势大, 制去辰中之水及天干壬水 (辰戌冲未戌刑, 壬戌自合制去壬水印星) 制的好, 时上卯木为日主的禄, 卯戌合到自己主位, 表示自己能控制这个戌去做功, 是财库制去印库, 印表权力, 制印得权, 因为是用财制的, 故是管财的大官, 行戊辰运辰到被制, 升职, 行己巳运, 已给戌加力, 制辰又升。

提示：看是什么性质的官，要看主要功神是什么十神或在象中表示什么来定。

乾：乙 己 庚 庚

巳丑午辰

分析：日主庚金坐正官，但是庚金日主合年上乙木财星，而乙木财星坐巳火正官，为官带了帽子，因时上庚坐辰，辰为食伤库表示学校，坐下午官生辰，辰又生身，为学校里管后勤的副处长。坐正官被丑穿倒，合巳杀但杀在宾位，所以是副职，七杀坐偏不为行政之官。丁丑年，丑到穿制午，丁即午透坐丑有制，升职。

乾：庚 己 甲 己

子卯辰巳

分析：日主甲木坐辰为财也是印库，年柱庚子，印带杀帽，子辰一合，此杀有用表官职，卯辰一穿是制印的意思，制印得权，所以是当官的。行壬午运癸酉年升副县长，大运午冲去子水印为去印得权的意思，行癸未运壬午年升常务副县长。子辰拱局中神在宾位不在主位，所以是副职不是正职。此人当了二十年县长，一直是副职。

提示：如正官在主位被穿表副职，因正星一穿即偏了，如合局中神在宾位也表示副职。子水印在宾位可制。

坤：壬庚丙辛

寅 戌 午 卯

分析：原局三合羊刃局，中神在日支主位，丙辛合财，地支有木火之势，关键看年柱，印带杀帽合入日主。表示自己带了个杀帽子，所以此人是个官命。七杀表权利，官职，木火相生成势，又丙火日主也表示文类，月上戌土为食神合局，食神表教育，故此人是教育部门的一个校长。

提示：火表文类，食表教育，与文化教育类有关的官。

乾：壬己壬甲

# 寅酉申辰

分析: 原局申酉金与辰湿土成势, 用主位的申印去冲制年上的庚食做功, 印在主位有功表权力, 食神表财原神、食神可表银行, 申金有金融之象, 月令为印带官帽子, 表职务, 此处是银行的官员, 八字中用印时怕大运来坏印, 坏印则表示丢官罢职之意。行甲寅运, 甲申年, 甲己合先把官帽摘了, 寅申冲在把印权交了, 丢官。

注：此中用到“用申金印冲寅做功”的话，此处这个“用”表示使用之意，并非“用神”之意，从概念上要分清楚。

乾：甲 癸 辛 己

午酉卯亥

分析: 日主辛金生癸食, 癸生甲, 甲坐午生午官杀, 为杀带财帽。但这个午如何与日柱有联系呢? 如无联系就表示这个官不是自己的官。再看时透己土印星, 己通禄在午一个象, 己坐亥, 亥卯一合, 这样, 这个年柱的官就归自己了。原局辛坐卯, 卯不生午 (卯午是破的关系), 金水有势制午官杀, 亥午合制杀做功, 行丑运, 丑午穿制杀, 升财政局长。

提示：只有和主位发生了关联的，做功，方可当官职看。

注：此例用到了天干与地支互通及原身禄的关系。因官杀带了财帽，是表示官与财有关的性质。

乾：癸 癸 庚 庚

卯亥午辰

分析：金水伤官，伤官生财，午亥暗合为伤官制官，有权，伤官又入辰库，把重要的做功之神入库，用庚控制，辰为印泄午生身，印表单位，为国有企业的官，月令亥合制午火官，月令为大，为大企业，某国有企业副总。

乾：辛 戊 庚 辛

丑戌寅巳

分析：原局木火燥土势，制年上丑土比劫羊刃库，为官杀库制比劫羊刃库，比劫羊刃库有制表示武，而丑库也表示很多的意思，可理解成集团军的意思，哪不就表示一个集团军的首领，一个元帅之意吗？这是元帅徐向前的命。七杀配刃表军队。

提示：看官的级别要看做功效率的大小。如此例制库。就有多大等意思，故级别就高。

看官贵的性质，要看象，还有带象，如官带了财帽一般表企业或与财有关，而财带官帽或印带官帽一般表行政。

以上讲的都是有级别的官例：下面这几例都是很一般的，也就是我们讲的村官，只不过级别低了，经做功方式都是一样的。

乾：乙 甲 庚 丁

未申申亥

分析：原局金水成势，要制未中丁官，但没有很好的做功方式，故制的不好，好在未中丁透到时上，被亥水合制，但丁官虚透不表权，行至巳运，巳申合，巳亥冲制官，升村书记。

提示: 时上亥与年未有拱合之意, 拱出虚卯透年上乙, 时表子女官, 儿子不争气, 败家, 儿住牢。

乾：辛癸巳辛

亥巳酉未

分析：原局金水势略大于火土，亥冲巳，癸巳自合，制月令的巳火印星，月令印表权，巳酉一合拉过来（制了再得到，归自己所用）已丑运，己为日主到，巳酉丑三合，拉过来，戊子年，加强了金水之势，戊即巳到，戊癸合留，竞选上村主任。

提示，在寅运时，也连续参加两界竞选都没成功，因寅巳穿有生巳之意。

乾：乙 戊 丙 己

卯寅申亥

分析：用宾位的体，寅卯印星制主位的用，申金财星，亥寅合用寅印化官。但申亥穿，生亥官，申也没有得到很好的制伏。但印配官也表权，故有职不大，为一车管站长。申表执法。

乾：癸 癸 乙 庚

卯亥亥辰

分析：原局水多，都入了时上辰墓，日主乙木合时上庚官，庚坐辰被生，表示日主用庚控制了辰。但时上卯为日主的禄与辰穿，庚无根，行申运，庚通根，申生亥，印化官生身，升村书记。这种穿中带生的的制法都不吉。

提示：穿制时，表示申也受损，再原局庚不通根，故职不大。卯禄在年表祖，被穿无祖业或祖业很穷不得祖力。

坤：壬癸丙壬

# 子丑辰辰

分析：水多入辰库，丑也随癸入库，辰为丙的食神，也表官杀库丙控制辰，辰伏吟到时透壬杀为自己的杀。断：有工作与金融有关。（丑为财库入辰了）自己搞与数字有关的工作（子辰合象数字），有个职务（看壬坐辰透，因辰与辰都表日主的）。实际：在某银行工作为会计。

提示: 看工作性质时要多分析象的原理, 我当时断她以上几条外, 还说你的工作是父亲给你安排的或接父班的。她说是接父的班。如何看出, 看杀表父, 而丑财库也表父, 癸坐丑也表父, 都入辰墓, 表承父业了。

# 《民间实用八字断命例题解》

选用我们平时给客户断命时的原始实例记录，稍作改动（主要是把当时为何会断出事物及事情的原因及分析思路加了进去）。在整理编排时没有分类，保持了原来的风貌，为的是更好地体现“民间八字实战”精髓。

选用的两百多个实例，内容涉及人生方方面面，不仅具有应事及名期，更重要的是把应事应期的思路分析过程合盘托出。

只要你是一位细心的读者，都会从中得到自己想知道的内容，当然收获的多少于读者本身的基础及悟性有关，读者基础越熟练，收获就会相应越多。

分上下两部分，没有顺序上的划分，而是为了用尽量多的实例来说明我们的断命方式，因为学命的关键就是要多看命例（特别是分析思路过程及方式），这样才能进步很快。

内容看仅平淡，但处处是“实招”，绝招来自熟练和思维方式的改变，因此，处处又是“绝招”。
