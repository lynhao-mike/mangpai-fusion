# 《财运》

# 一、看财运的基本原则：

我们讲看财富，并不仅仅只看财星，八字中其它的神在特定情况下也可以当财论。

# 1、禄神可以当财富

禄：就是衣禄，吃喝，享受等意思，古代还将它当作俸禄，是自己享用或占有的东西，可以理解为财富。

原则：命中占禄，无伤食泄时，或八字无财时，禄可以当财看。

乾：丁 丙 庾 丁

未午申丑

分析：八字无财星，以禄看财富。时上丑为禄库，在门户拱禄，不怕官杀克身。此人很能干，开车搞运输挣钱，因申在“象”中表传送，有车之象，又时丑门户拱，表往外边跑，癸卯运财星合禄，发财十年。以禄神当财富看时，禄神制财星可发财。

注：禄作财富看时最怕见劫财，因劫财有分禄之意。

坤：戊 丙 己 己

午辰未巳

分析：已见年午为禄，午禄紧贴劫财，生了劫，表示她祖上有祖产也会分给兄弟们，与她无缘。以午禄看财富，忌辰泄禄。甲寅运，寅木生午火以吉看发财。但逢庚辰年。辰土劫财泄禄为凶。此年被盗破财，好在大运吉，破案后追回部分财物。

乾：癸 戊 己 甲

# 卯 午 酉 戌

分析：月令见午禄，午与时支戌拱禄局。坐下酉食神为财源，禄和劫财制食神。八字是制食神为吉，行乙卯运开始发财。卯戌合引动戌土又和火土结党来制食神酉金，所以此运发财，至今已成百万富有。走甲寅运应该有千万的。

劫财戊土为坏东西，因为戊坐在自己的禄上，所以他的钱曾被一表兄弟骗过两次。其中戊寅年，戊土到了，为兄弟损过财。庚辰年又是劫财年份，为兄弟担保的三十万元，贷款有去无回，银行找他还款，给他惹了很多麻烦。

注：命局制财的原神做功者，为大富，一般起步百万级。如做功能量大者可达千万到亿级。

# 2、伤官食神可以看财富：

原则：八字无财星，但有伤食星，以伤食当财富看。八字有财星，伤食是其原神，可以当投资之财。

乾：甲 戊 癸 己

辰 辰 卯 未

分析：日主坐下卯木食神，时上卯未拱局加强了木的能量，并且年上透出甲木坐辰，为甲木有根有力克制辰土之意，地支卯辰穿，卯木制辰土官星，虽年月有两个辰地，但年上甲木已控制一个辰，卯有未加力来穿制辰杀，日主合戊，表示日主要控制辰土杀星，食神制杀表权。八字无财，卯是财源当财看，是生财的地方，所以财的原神也表银行之意。时上未土是食伤之库，表示装钱的地方，故此人是个银

行行长。

注：食伤制杀表权。又有财源，又有财源库，就表银行了。这点要从象上去理解的。而上例也是制食伤就仅表生意了。

乾：乙 乙 辛 已

巳酉未亥

分析：财官临库喜刑冲，日坐财库不开，时上亥水伤官藏财，亥未拱木，透到年月，乙木财生了官，官星也当财看。但是亥水不能坏，走巳运冲了亥水不好。庚辰年做生意关门倒闭，亥入辰墓设法在做生意了。庚劫合乙财为劫财合财，是破财的意思。因为亥未拱出财表示自己的财，所以不喜劫来合的。已运到处奔波，一事无成，一分钱也挣不到。因为冲了亥水财源。日坐财库不开，表示老婆抠门，只进不出之意。

# 3、官杀星可以看财富。

原则：1、官统财或财统官，官杀当财富看。

2、官杀有制，但制服不太好，官杀可当财富看。

注：官杀当财富看时，其财富级别会很高。

乾：辛庚丙丁

卵子申酉

分析：满局财星，庚财坐子官，申子又合，官星只有一位，为官少财多，官星可统财，官星可当财富看。制官杀时可得财，当运行乙未，未穿倒子水官星，发财数百万。

乾：乙 乙 戊 甲

# 巳酉子寅

分析：日支坐子水财星，财星只有一位而官杀重多。时上甲寅七杀受主位子水财生，因为是主位的杀，表自己的财富，为财统官杀，（此处可理解成由因为财生了杀，而杀又在主位表示自己的，是财富的一种转换方式）制财星时可发财。行至午运，午冲去子水财星，引通寅木杀星，发财数亿。

乾：戊 甲 戊 丁

子寅辰巳

分析：日主坐辰财库，但子辰拱到年上，表示自己的财拱到了别的地方，而子财又生月令甲寅，是财杀相连，七杀在月令宾位也可以表财富，只是表示此财不是自己的，因月令表公家的象。行戊午运，午冲去子水，引通寅木，自己是经理。掌管经营，风光一时，到己未运，己运一到，甲己合表示甲木官方之财，被别人合去了，结果在经营中被骗，公家损失惨重，自己惹上官司，破财又丢职。

注：此中不仅讲到了发财应期也讲到了破财应期，而有关官杀看财，需要从理解上下功夫，财生杀，也就是一种思维的转换看法，因财杀相连之关系，此杀就可表财富。

提示：戊午运风光，此戊即表示原局之戊到了，而原局戊坐辰是不制子财的，到戊午运表示戊坐了午，也可理解成戊控制者午去制子财，然后寅午一合，寅生午，午生戊，日主就得到了杀，表得权又得财。而已未运，己为劫表别人已甲己一合，表示甲寅一柱被己控制，故受骗损财。这些答案只有从理上去理解，方可答出正确结果。而不是单纯的作用方式能说明的。己为劫表别人，别人骗了他之意，这是原因。

# 巳辰未亥

分析：此命如果从做功的角度来分析，就一无是处，因并无一方气势，也没有制局存在，但从象的角度来分析却大有玄机。首先是财多官少，官又带财帽的象，是官统财，这里的官星就表示很大的财富之路，再看亥入月令之辰墓，月令之辰墓就表示一个巨大的财富集团之意；这个辰墓如何与主位发生关联呢？也就是说如何才能被日主控制住呢？再看辰土头上带庚财星，而有乙庚之合，表示庚金被乙木控制，控制庚也表示就控制住了辰。而乙木正好入墓于日支主位的未墓，并且就有亥未拱出乙木之象，这样乙木控制的庚辰一柱因乙入墓未就表示是日主控制住了庚辰一柱，因丁控制了未又丁未一家的象，所以局中整个财富都在自己的掌控制中，实际他是一家大型上市公司的董事长，掌管数十亿财富。

注：似此类命局的看法，必需对各种基础的熟练。

# 二、取财方法：

看命首先要确定一个人的职业。发财命要定出他如何取财，这样才对人生有更准确的指导意义。

1、经营取财：包括范围比较广，如做商业贸易，开店面，生产与经营性企业，开发型企业等。凡是通过伤官，食神做功或财星做功的类型都是适合经营取财。

地支食神做功者，适合于做企业经营。

乾：庚 辛 甲 癸

子巳辰酉

分析：原局佥水湿土有势，月令巳火食神被克泄，食神又合官星，（辛巳自合的关系），食神是生财的，表示财的原神，可产生财富，就表示企业的意思，原局制了财原表示从根本上制住了财。但辛巳自合，食神带了官的象，表示不是他本人的企业。应该是企业的管理者。实际是一家公司董事长。

坤：癸 甲 癸 癸

巳 子 卯 亥

分析：日支坐了食神。卯木生了年上宾位的巳财，表示自己的头脑与辛苦为他人赚钱之意，因卯木食神表自己的才华。坐下食神有功，表示做企业。年柱癸巳自合，巳中戊官合癸，表示有管理的意思，因年癸也表日主癸，即表示官合日主之意。故有管理的意思，适合做企业经理。实际是一家大型商场的经理。

乾：辛 癸 癸 壬

丑巳卯戌

分析：日支坐了卯木食神，卯木生巳火财星，又合了时上戌官也表财库，原局木火燥土势，制年上丑印库，也表示官，但因有巳丑之拱局不表官，内食神有功表示做企业的。卯木食神表财原也表自己的才智，先去生了月上的巳，而已透癸自合，表示已被月令的癸控制，表别人的财，可理解成，自己与月令癸合作经营巳财，但别人比他有权之意。(巳中之戊官被月上之癸所得)，实际与别人合作经营一个厂子，自己只有股份而没全权。然后，再看卯木合戌财库，巳财入戌库，而戌透壬劫，壬戌自合（壬水合戌中丁）表示戌是壬劫的，但卯木也合戌，表示戌也有卯的份，可理解成日主又与别人（壬水劫）合作，经营者更大的企业（因戌库表火）但更有权，因卯戌合，表示用卯控制住戌之意。实际：原来是跟别人合作办一小厂，自己入有股份但无实权。而后又与另一人合作搞一更大的厂子（看戌）自己任总经理，妻子管财务。戊子年厂子不景气。戊即原局之巳，虚透天干，表示钱要跑了，地支子破卯，卯木受伤，有点控制不住戌的意思。丙戌年财好，因戌库到被合住了。

提示：看命时要重视“象”的运用，因很多问题只有从象的角度去分析，方可理解。

2、凡是财星落于时上或年上做功，适合于做生意。因为年时都表门户，若财星配印时，适合开门店，因印表房屋之意。

坤：己庚甲戊

酉午戌辰

分析：时上戊土财星坐辰土，辰既表财，也表印之库，与日支冲，原局午戌拱火燥势略强于水。戊财与午火伤官联手，制辰印库有功。因为印在时上，可以表示房子，墓主表示多之意，可以表示多家店面，所以适合开店经营。果然开多家店面，资产百万。为何有百万级呢？因年上为官统财，提升了财富的级别。

乾：癸 壬 癸 庚

# 卯 戌 巳 申

3、官杀看财富，凡伤食与官杀做功者。伤食制官杀或官杀制伤食的组合，适合多种或规模性经营。

注：制财与财的原神成功者，财富级别会很高。

4、智力取财，一般是靠人的思想，技术知识来获得财富，所做功之神是伤官食神，且伤食星占了主位，或跟主位的星做功。

# 申 未 辰 寅

乾：丁 壬 庚 丁

未子戌亥

5、体力取财，体力取财做功之神是比肩劫财，禄神，当然不同的职业八字有不同的组合，效率与做功方式都有所不同。

酉 辰 寅 辰

提示：1、看财运之大小，要看原局被制之神是什么，一般制财的原神，财与原神同时被制，官统财，财统官，或官当财看时，都表示为大财，级别都高。

1、若以制局中之财星而看财，仅表示我们生活中的零花钱，表示小财。

2、“制”即做功，要结合做功的效率及能量大小来综合分析。

3、得财或破财的原因要看制与被制之神的十神类象，有时也要结合官位来看。

# 《民间实用八字断命例题解》

选用我们平时给客户断命时的原始实例记录，稍作改动（主要是把当时为何会断出事物及事情的原因及分析思路加了进去）。在整理编排时没有分类，保持了原来的风貌，为的是更好地体现“民间八字实战”精髓。

选用的两百多个实例，内容涉及人生方方面面，不仅具有应事及名期，更重要的是把应事应期的思路分析过程合盘托出。

只要你是一位细心的读者，都会从中得到自己想知道的内容，当然收获的多少于读者本身的基础及悟性有关，读者基础越熟练，收获就会相应越多。

分上下两部分，没有顺序上的划分，而是为了用尽量多的实例来说明我们的断命方式，因为学命的关键就是要多看命例（特别是分析思路过程及方式），这样才能进步很快。

内容看仅平淡，但处处是“实招”，绝招来自熟练和思维方式的改变，因此，处处又是“绝招”。

注：书中选为为数不多的大富大贵命例是从别的书上摘录，但全部采用民间实用方式分析解读。为的是让读者理解富贵命与普通命的相同分析方法。

一个命例不足一元的代价，或许能引发你无尽的断命灵感。
