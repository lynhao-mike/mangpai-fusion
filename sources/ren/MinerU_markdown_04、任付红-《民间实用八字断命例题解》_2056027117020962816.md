# 前言

此资料是针对具有“民间实用八字”函授基础者所阅读的辅导性资料。

资料中没有解释的断语是为了使学习者通过断语来进行反推演练所用，因为反推断语也是一种很好的学习方法。当你知道了结果而能反推出断语的来处也是一个很大的进步！

对已解释的断语，自己要多理解原理，为什么会有这样的结果，要多问为什么？这样看得多了，原理就明白了。

用文字表述讲话的内容很难写到位，因此所写例题都是不同程度的修改，但所写结果都是正确的。
