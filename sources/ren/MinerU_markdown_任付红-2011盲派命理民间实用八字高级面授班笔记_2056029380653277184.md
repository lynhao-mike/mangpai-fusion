# 河南·安阳

本人不惜重金，高价引进了最新任氏盲派命理民间实用八字2011河南安阳高级面授班手写笔记，是因任氏理论的研究已走在了盲派理论的前沿，值得借鉴。

该高级面授班为防资料外流，没有发任何文字资料，全靠学员弟子自己用笔记录，学习班结束，任老师还对每个学员的笔记做了拍照存档，来堵截学员笔记的流失。为了绝对保密，对学员负责，本人只得日夜辛苦，尽最大的努力对这些珍贵的资料进行全文、换位打印，并做了个人注解，对重要的字句做了加色加粗提示。因本人水平有限，对错误的注解或无法解释的命例，欢迎感谢各位同好和大师给予指正和谅解。

该整理后的资料用 A4 纸打印 52 页，虽页数不多，但句句是真言。任老师的资料就是少而精，应用任老师的话说“我们虽然写的内容不多，但每一个讲解都是很实用的，望你能做到认真理解！话不在多，到位就行”！

用民间八字理论断命实际就是一个思维的过程，不用看旺衰，不讲用、忌神。论命以阴阳、宫位、十神为主，通过各种作用方式来解读表述命理。断命过程是活的，必须放开思维来准确地解读命理。原局决定人一生的富贵贫贱，大运决定吉凶，流年断应期。

●阴阳：

任何一个八字都包含阴阳，阴阳实际上就是一种气。

寅--未为阳，主上升之意，申--丑为阴，主下降之意。

阴阳能看人品、职业、疾病等。

●寅：为公门为阳气的开始；

●辰：为阳中之阴；医院、殡仪馆等

●戌：为阴中之阳，表示晚上灯光亮的地方：夜总会、色情场所；

●申、酉：从象上看为刀，从职业上看表示执法象，如公检法等，阳为正义，阴为邪恶。

财为官的源头。地支火与躁土成势，制亥水阳制阴表执法工作。亥午暗合，午禄暗合财表示不光明的象，合财为贪财，引为贪官了。

(帅哥学易学易：戊午同柱，阴阳错位，午此时可以为禄看，如不在一个柱子上为羊刃看。癸亥财同柱干支一气，为实财为钱财，如合的财虚透，不为钱财而为才华了)

●财：制财源与库为大财。

乾：己己己庚(子丑戌亥)

巳丑未午

原局无财，但丑未冲制住了财的源头食伤库为大财。

乾：庚乙癸庚(辰巳申酉)

辰酉卯申

大富命，制住了财和源头。

命局不见财星，但坐下卯木食伤为钱的源头，地支辰卯穿，酉卯冲申卯合，制住了卯木为大富命。

坤：癸己己癸(午未戌亥)

西未丑酉
大跨国公司老板。

原命局无财星，但酉金食伤为财的源头，日支丑库为金库，食伤库，表示把财都控制在丑中。酉财在年时表示发远方的财。

(帅哥学易学易：丑未互冲制约为开库是发财的关键点)

●官杀制不净时可当财看为大财。

坤：癸庚乙癸(寅卯辰巳)

卯申巳未

活木喜制金。地支木火燥土成势制申金官杀，但庚申官杀制不净当财看，为大财，天干乙庚合为控制之意。

●命中官少财多为官统财，或财少官多为财统官，都为大财。

乾：癸丙乙乙(午未辰巳)

卯辰酉酉

财统官富命发大财的。

命局辰为财，酉为官，财少官多属财统官发财的命。财统官，官都当财看。大运遇辰酉，或生辰酉的字，必发财。

乾：辛癸癸壬(辰巳辰巳)

丑巳卯戌

地支卯戌合，成为财库，制住了财库为发大财。

(帅哥学易学易：戌透壬劫财应该为合伙求财)

●食伤生财为智能求财。

乾：丁己癸乙(午未辰巳)

酉酉巳卯

月令带象表示规模大。

月令己酉为印带官帽当企业看或公司看，卯木为食伤生巳火财为智力求财，酉为印冲卯表示做企业取财。

乾：辛庚丁己(申酉寅卯)

亥寅丑酉

天干年月财虚透表示才华，丁己同宫地支酉入丑库，酉为财，丑为财库有财。

●财在门户远方求财或开店。

乾：辛己辛庚(申酉寅卯)

亥亥亥寅

寅为财在时上表开店，或远方求财。地支亥为食伤，伏吟生财，奔波跑动生财。

乾：壬戊辛乙(戌亥寅卯)

子申未未

财在时上开店的，时为门户为远方。

●比劫、禄为体力，比劫、禄求财为体力求财，辛苦。

乾：戊己己丁(午未戌亥)

辰未丑卯

地支为食伤库表技术，未禄冲丑表比劫禄求财，辛苦。

乾：戊壬庚戌(子丑寅卯)

申戌申寅

地支申寅冲，申为禄寅为财，寅在时上表开门店的，辛苦。

(帅哥学易学易：禄多也为辛苦之象)。

乾：甲戊乙甲(寅卯子丑)

寅辰巳申

地支辰为财，巳火为食伤，巳生辰为智力取财。日主坐食伤表自己要动脑筋求财。

●婚姻：

①婚姻星要与日柱发生关联。

坤：丁甲戊丁(寅卯子丑)

卯辰戌巳有三次成婚。

地支卯戌合，卯为婚姻星合入日支，但夫星卯被辰穿表找了一个有家的男人，卯被辰穿出去了，辰为卯戌合成婚的阻力，第一次不会成。

第二次甲辰，辰戌冲与日柱有关联，成婚还要失败的。第三次成婚宫透丁坐巳，表示进了家，巳火为戊日主的禄表示戊土自己，才能稳定住。

②宫制住了星为好婚。

坤：戊丁丙壬(午未寅卯)

申巳戌辰

夫宫戍土透丁坐巳，巳申合制住了申星。

地支巳戌辰为一体，制申星，宫制星好婚，天干丁壬合控制了辰，日主坐戌，戌透丁坐巳，巳为丙禄。

③星宫同被制死配偶。

乾：辛甲戊己(午未申酉)

巳午子未

地支巳午未成势制子水，星宫同被制，妻必死。

(帅哥学易学易：子被未穿、午冲、巳绝，又无生源，无处逃生，必死之象。如子同午或未任换一个位置，那就是无婚之象了)。

④宫多易多婚

乾：庚己庚丁(戌亥戌亥)

午卯午亥

宫星多现为多婚。

地支午、午、己、丁都为妻宫，亥卯是坐下都为妻星。

坤：甲丁丁甲(寅卯子丑)

寅卯未辰

女命八字看婚即无官又无财，只能看宫位未土。未把周围的木印都收收到库里，说明男人多。

是个交际花男人遍天下。满局宫、星，未土为印库，辰为官杀库，未辰都为食伤。交的男人都有钱。

坤：丙己己壬(戌亥戌亥)

寅亥巳申

寅官为第一个老公，第一次婚不成，巳透丙坐寅，寅穿巳不吉(帅哥学易学易：寅官是丙的寅，也是巳的寅，现寅巳穿，是自己同自己过不去，所以婚不行)，最后一次嫁个远方的巳申合成婚大吉了。

⑤女命无官杀看财，男无财看食伤，如全无看宫位。

坤：戊乙甲丁(申酉戌亥)

辰卯戌卯

二婚的命。原局无官看财，年支辰，日支戌都为夫星，第一婚难成，卯辰穿所致。第二次夫宫戌透丁坐卯合戌，成婚。

坤：丁己丙乙(申酉子丑) 同下例比较

已酉子未

离婚命。子官星坐正位，子被未穿、被已绝，但子有酉原神生，夫不会死亡，夫配不了离婚而已。

乾：己辛戊戌(午未寅卯) 同上例比较

酉未子午

子为妻星，未穿午冲必死。

坤：辛己庚丁(辰巳寅卯)

亥亥子丑

原局丁火夫星太弱，丑为印又为庚墓，断本人一生都不想结婚。地支亥子丑都是阴，坏了丁火的阳气。

乾：己丙己甲(寅卯子丑)

未寅酉戌

原局无财星，食物酉金为妻星，地支木与躁土成势制酉金，戌穿酉最凶，一生难成婚。

(帅哥学易学易：本人认为，妻星坐正位也该婚是有的，不过配不到头而已。己日主配甲寅是为连体，己日主要用戌，戌是日主自己的，酉为食伤也为日主自己的，现酉戌穿，自己同自己过不去，也就是在婚姻上反局了)。

乾：丙戊丁辛(辰巳戌亥)

寅戌酉丑

丁酉日柱，酉妻坐正位。地支寅戌拱，戌有力穿酉，酉受伤入丑墓，表示酉跑掉了。第一婚看酉金，第二婚看辛丑，丑酉拱合成婚。
