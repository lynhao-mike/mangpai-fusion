# 学历

八字里定学历星：以印、官杀、食、伤、当学历星看，理论上是这么讲，但在实战运用时，还要结合具体命局组合来定，需要通变理解。

如：财星表不利学业，但当命中印星太重而坏局时见了财克去印星反而是好事，其实这时的财星反而有利学业看了。

如：伤官，为好动，不听话，表学习不专心，但伤官又为聪明的象征，如命中官杀或印星与伤官配合到位，这对伤官反而可表有利学业看等等。

因此我们一再强调变通强解，针对不同的命局组合要灵活理解运用才不失偏差。

文理区别：

阳主文，阴主理，木火为阳表文，金水为阴表理。

细划分：庚、申、亥、子表数学，申子或申辰结合可表化学

庚、申、午、酉可表法律 金加丑也可表法也可表物理

食神可表传统文化或外语，官杀可表理科或军警

学历一般指的是文凭，但有很多聪明的八字确没有多高的文凭，这时就要具体情况具体分析，如某人在某一方面很有成就，但确没有相对应的文凭，因此判定一个人的能力或成就大小不能单纯以文凭来定论，还有种情况就是本人没多少文化知识，但可以通过某种关系搞到较高的文凭，这样的情况是区分文凭高低的一个误区。

我们所讲的学历不是单纯以文凭来衡量的，而是以八字中显现在某方面有专业性特长来衡量的。再一种情况，即使有很高的文凭也不代表就会有一定的事业成就，而即使没有文凭确干成了很大的事业，这些情况也是看命时区分文凭高低的缺陷，一句话，有文凭或没文凭说明不了这个八字的能力大小，文凭仅可作为看命好坏时的一个参考，而不起决定作用！

八字中以印星、食伤、官杀、财（虚透时）当文凭看时，因各种命局组合不同，还有各五行特性不同，因此具体到实战时，到底该取什么为文凭看，是要有一定的功底才可定准的。

关于文凭的分类，文科、理科、法律、数学、外语、化学等等，这些内容与八字的取象有一定的关系，当命中某字代表的象与文凭职业有关又有作用时，这个字代表的象就可定为学历分类中的哪个方面来确定。这与定职业有很大关系，可参看象的讲法内容，一般常用的代表学历划分方面的有如下几种情况：

木火可表文化类，文科类、英语、语文、历史、哲学类等，金水可表数理类，理科类，数学、数字、几何、化学类等，而命中的申、酉金可表法律类，金又可表物理类。

水与火的组合可表玄学类，有时食伤也可表玄学，金或劫刃又可表执法、武类，这是与文凭有关的大概划分，如具体到职业情况还需要针对干支类象的熟练方可。

他听了后说，我原以为她是个从格的命呢？经你这么一讲，原来看命是不看这些的。看来我要改变原来所学的方法了。
