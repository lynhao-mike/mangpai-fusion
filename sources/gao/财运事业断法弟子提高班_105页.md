# 第1课、盲派八字论命的入手点

打开八字从哪下手？

学习八字要先学习理法，什么是理法？理论是用来指导实践的，是论命的指导思想，理法是两个概念，理念和方法，每一种论命方法都要有一定的理论作为指导思想，如传统命理讲究平衡，平衡就是理论基础，旺衰是方法，也就是以日主为中心，定出日主及各十神的旺衰强弱，然后以平衡法确定喜忌、断吉凶。所以打开八字从旺衰入手。不同的论命方式有不同入手点

格局法论命，以命局中的天地主气（月令司气之神为天之主气，天干中最活跃的十神为地之主气）为中心，立向、定格局论成败，成格者贵，成局者富，不成格局者均为普通之人。所以从月令入手，看是否形成格局。

像法论命，是以想象、比喻、引申为手段，抓住八字干支五行及十神体现出的特点对应命主的生活现象，他是从八字的特点入手。

体用论命理法，重八字的组合结构优劣，八字论命核心在体用得失；现代学命者，对于一些命学的体系不熟，往往管中窥豹，没有在自己的头脑中构成体系框架，所以打开八字不知从哪说起。

有理论才能产生方法，有理论有方法才能思路清晰，这点非常重要。

今天重点给大家讲讲盲派的入手式：

盲派论命先分体用，体用之说古代就有，最早的见于《荀子》。所谓体指一物的形体，所谓用指一物的功用。开始主要用于哲学范畴，后来引用到易学领域。

# 一、什么是体用？

一个四柱是一个太极，太极分两仪，四柱是由年来推月，由日来推时，四柱本身就将四柱分成了两部分，日时代表体的部分，年月代表用的部分。关于体用地不同叫法，现代命理专家段先生，把体用换成，主位宾位，后起之秀杨青娟，把体用说成家里家外，换了一个名词，似乎更接近生活，便于理解；终南命理命传人惭愧学人把体用叫做我宫，他宫；把体用关系说的更明白。

八字预测是一门人文学科，主要探讨人的生命轨迹，探讨人与自然的关系，人与社会的关系，人与家庭的关系。体用这一概念引入八字命理，可以进一步展开描述人与自然的各种关系。体用是阴阳关系，而相对的阳动阴静。

我们都知道，八字是以日元代表“我”，那么显然我是体，而其它七个字代表的是客观环境为用。因此围绕着我这一“体”通过阴阳生克关系来定义出其它七个字的内含，用“十神”来体现其涵意。日主代表我，天干代表我的外在形体，地支代表我的内心世界，天干代表我的精神气质，地支代表我的身体实质；更具体的说，日干代表我，（我个人）日柱代表我家，（包括，妻子、或丈夫）日时代表我宫，（包括老婆孩子，也包括我的财物，包括我的思想。）体用法看小家与大家的关系，我与社会的关系。看自己这辈子追求什么，自己的能力，能否得财以及得财的方法。一个太极分两仪，就是我宫和他宫。四柱生成，先定体用。这是看八字的第一点思维。也就是打开八字先分体用

# 第 3 课、预测财运的思维框架

# 一、预测财运的理念思维（首先要明确几个观点）

1、人生所追求的终极目标只有两个：一个是财运，一个是官运，这是人进入社会，穷其一生所追求的目标。

2、财为养命之源，不可或缺，财，不要仅仅理解为钱财，他是人门生存的资源，其实说白了每个人都有财，只不过有财大财小，财多才少的区别。

现代人们又把富人分成三六九等，我们要明白，所有的预测，求的都是一种概率，没有必要划分的那么细致。

有人把财富级别分为四个等次，小富（百万以上千万以下）中富（千万以上，亿万以下，）

大富（亿万以上十亿以上下）巨富（十亿以上）千万注意这只是概数标准，不是绝对值。

3、打开八字首先要定位用神，我说的用神不是平衡命局的用神，也不是格局用神，而是测项目标，就是测什么事就以什么事为用神，测财运就以财星为用神，侧父母就是父母为用神，测婚姻就以配偶为用神，说白了就是预测什么事，就要盯住能代表那件事的用神。一定要做到心中有数，目标明确，然后给以定位，不能模棱两可，胡乱猜测。

为什么要有一个明确定位呢？古人告诉我们，在命理规定中我克者为财，这是基本理念，也是根本法则。这是不变的，但是八字的组合千变万化，从十神的角度说，食伤可以代表财富，印绶可以代表财，特殊组合的情况下官煞也可以代表财，所以看财富菜单定位就非常重要。

# 第 5 课、对正财与偏财的认知

# 一、对正偏财的认知

1、就钱财获取的来源而言，正财是指个人的正常工资收入之财，企业的主营业务收入所获之财；偏财则是指个人的兼职收入之财，企业的附加业务收入所获之财。也可以说，正财是财的主要来源，偏财是财的辅助来源。

2、就钱财获取的行业而言，正财是指具有国家属性，或具有稳定性较强集体企业或个人企业所获取的劳动酬劳；比较固定的工资收入；偏财则是指从边缘性行业，就是偏行所获得的钱财，比如中介、心理咨询行业、周易预测行业等属于不固定的收入。

3、就钱财获取的数量大小而言，正财是指相对数目较小的而且是比较固定钱财、其中包括祖产祖业；而偏财则是指数额相对较大的钱财，又叫横财等。

4、就钱财获取的风险大小而言，正财是多指上班族的固定收入，一般来说没有风险；而偏财是指经营投资风险性相对比较大的项目所获得钱财，比如证券、期货等之类投资所获取的钱财。

5、就钱财获取的连贯性而言，正财是周期性得财、连续性得财，如个人的工资收入，企业的营业收入等；而偏财是意外之财、偶得之财，如彩票中奖、遗产继承等。

6、就钱财获取的合法性而言，正财是合法之财，如合法的工资收入、合法的营业收入等；而偏财则有些属于非法之财，比如走私、贩毒、偷税、漏税、受贿、盗窃等获取的钱财。

7、就钱财获取的性质而言：正财是通过付出劳动获取的钱财，属于辛苦求财；偏财多是脑力求财，计谋求财，的性质。

八字中的正财和偏财在一定条件下是可以相互转化的。只有正财的八字命主不一定在现实生活就没有偏财，也不一定不适宜求偏财。

# 第6课、对断财运歌赋的理解

古今命书包括盲派秘诀，对财富的的断语很多，而且是从不同角度去下的断语，并没有系统的专题论述，归纳起来大致可分为以下几个侧重点：

# 一、强调身旺财旺是富命。

财多不富身弱穷，身旺有财独得生。

独财能招千斤鼎，身弱无根财局成。

这两句是古代命书比较典型的断富命歌诀。

民国时期的命理大家徐乐吾对“财气通门户”的诠释是：财星当令，得气得地，配合有情也。我认为这个解释虽言简意赅，却仍然没有到位。仍然没有点破他的真意。本人认为“财气通门户”有三层含义：

第一层含义，财星与命主有某种必然联系。

“门户”是与外界联系的通道，是人进出的路径，那么是否可以引申为财气通身的途径、渠道、办法？我们都知道年月之财是公众之财、国家之财；也就是说八字的财，不一定都是命主的财，年月的财星，只有与命主发生必然联系，或是有途径送到命主家门口，或是命主有办法拿到，才算命主的财，才有发财的可能。那么这种联系、途径、办法才是“财气通门户”的真意。

第二层含义，财星组合与命主有情。

通，就是没有阻碍，通达、通畅之意；没有阻碍就是一种有情。也就是说在财星与命主产生联系的组合中，是有情的组合，而不是恶意的组合，换句话说，财星给命主带来的是财富，而不是灾难。

第三层含义，财星要旺相。

最好是值月令旺相，如不值月令也要受生而有力。这是判断财大财小的重要依据，而不是判断能否发财的依据。有的学者将财星旺衰与日主旺衰相比较来判断命主能否发财，认为身、财两旺才能发财，两方有一方不旺相就不能发财。这种观点笔者不敢苟同，我们知道，八字中有财，不一定是命主的财，首先要看财星与命主是否有联系，也就是说要看命主是否有取财的途径和办法，这是关键的一步。财星与命主没有任何联系，身与财再怎样两旺、平衡也不会发财，也就是说这个财与你没关系。

在确定财星与命主有联系的基础上，要仔细推敲财星与命主有联系的组合中是否有情，能否为我所用，这是判断命主能否发财的第二步。最后再去考虑财的大小，即财星旺衰的问题。与日主的旺衰没有必然联系。

# 第9课、以官杀当财看技法点窍

什么情况下以官杀当财看？有两种情况官杀当财看：

其一：官统财或财统官，官杀当财富看；什么是官杀统财或财统官杀？就是命局中官（或者杀）多，而财星少，财可统官（杀）；说白了财统官：就是一个财星生多个官，叫财统官。官统财：就是多个财星来生一个官叫官统财。

还有种情况是干财支官（或者杀），为官杀带财象，官杀可统财。

（注：少指只有一个，且财官必须相连了，即财生官了，此时才为财统官或官统财。且只论原局，大运出现不算。）
