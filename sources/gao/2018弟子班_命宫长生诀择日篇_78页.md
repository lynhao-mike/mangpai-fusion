# 第1课、胎身命的认知

关于胎身命，盲派各家叫法，用法不尽相同，首先从叫法上说，有的叫，暗八字（胎元、命宫、身宫，胎息）三垣（胎元、命宫、身宫）直接叫胎身命，在用法上，有的只用胎元，命宫。

# 一、胎元的含义及起法

什么叫胎元？在母亲体内最原始的一种状态。叫胎元。说白了就是父母怀胎的那个月，月令向后倒退9个月。

定胎元口诀:

胎元起法记心中，生月干支要记清。

月干后位胎元干，月支后三支配成。

# 乾造 乙卯 庚子 庚寅 壬午

生月为庚子。按口诀：庚后一位为辛，是胎元的天干。地支子后三位是卯，是胎元的地支。将干支连起来辛卯就是胎元。

# 二、命宫的含义及起法

# 1、命宫的重要性：

不管四柱也好，星宗也好，均重视“命宫”的作用《三命通会》论坐命宫：“神无庙，无所归。人无室，无所栖。命无宫，无所主。故有命宫之说，不然，流年星辰，爲吉爲凶，以何凭据”。万育民这里所说的“流年星辰”就是十二流年限，《渊海子平》论起通法亦要以命宫爲依据推算。万育民说的作用太小了，命宫主要功能是以命宫位排出人生的其他十一个时空和具体方位。连命宫共计十二，台湾钟义明先生说“命宫在人体中爲大脑，管一个人的思想、行为，非常重要，近代研究子平法的人大都不重视，这会故步自封，不能深入命理堂奥”。它也是“七政四余”法的基本点。有此重要，我们就要讲究命宫的推算准确性了。

# 2、命宫的起法:

历代盲人传有命宫秘诀：“欲查命宫要记清，子起正月逆上行，生时落到月支上，顺查卯上为命宫。”也就是从子上起正月，逆转到出生月，看落宫，再在落宫起出生时，顺转到卯，落宫就是命宫。

如：申月未时出生命宫在寅。

盲派定命宫口诀：

若问命主何命宫，子起正月逆查行。

生月支上起生时，顺查至卯是命宫。

生月天干两头查，顺逆宫支天干停。

例如：申月未时出生命宫在寅。

# 第 5 课、长生十二宫命理求真上

引言——

宇宙间万事万物的发展变化都有一个过程，从规律来看都有一个孕育、出生、成长、壮大的过程；都要经过衰败死绝的折磨，又重新获得新生，周而复始，循环不已。生生不息的过程，就是生旺墓绝的辨证精髓。

# 二 “犯重丧” 及其化解方法

# (一)、何谓犯重丧？

重丧，重，重复双重之意，意思是不止一起丧事，指的是一个人去世后近则百日内，远则一年左右，家人或者亲戚中又有人相继去世的情形。

# (二)、重丧的种类

重丧根据死者之间的关系，分为外重丧、内重丧。

# 1、外重丧：

指的是前后逝去的死者之间是亲戚关系,重丧涉及到的是六亲关系,诸如姑姑、姨姨、舅舅、丈人、女婿、外甥之类的。

# 2、内重丧

指的则是指的是前后逝去的死者之间是家族关系,内重丧涉及到的是家人关系,诸如祖父母、父母、伯父伯母、叔父婶母、夫妻、儿子儿媳、侄儿侄媳、孙儿孙媳、兄弟、姐妹等。

另据《天师通书》记载，重丧又有所谓小重丧、真重丧之说。

小重丧的歌诀为:

正七连庚甲，二八乙辛当，五十一丁癸，四十丙壬妨。三六九十二，戊己是重丧。(数码指亡故的月份；天干指亡故的日子)。

真重丧以亡人年为主：虎忌甲，兔忌乙，马忌丁，龙狗忌戊，牛羊忌己，猴忌庚，鸡忌辛，猪忌壬，鼠忌癸。

《阴宅秘旨》曰，“重丧有人死，双棺立堂前”。

# 三、重丧的含义

犯重丧有两重含义: 一重是指死者去世的日子犯了重丧; 另一重含义则是指送葬的日子 (举办葬礼的日子) 犯了重丧。这两种情形都可能引起重丧的发生。

# 四、什么情况下犯重丧？

内重丧和外重丧发生的情形不同，具体来说：

# 1、什么情况下犯外重丧？

如果死者去世或者送葬的日子（举办葬礼的日子），其日干正好与月建同行，就容易发生外重丧。具体重丧日的规律为：

正月甲日、二月乙日、三月戊日、四月丙日、五月丁日，六月己日，七月庚日，八月辛日，九月戊日，十月壬日，十一月癸日，腊月己日。

比如说，逝者如果逝于正月的甲子日、甲寅日、甲辰日、甲午日、甲申日、甲戌日，就都是犯重丧的日子，可能会有重丧发生。其他月份类推。

# 2、什么情况下犯内重丧？

巳日、亥日，天门地户都开了，所以每个月的这些日子都犯重丧，而且是典型的内重丧。

如果死者去世或者送葬的日子（举办葬礼的日子），正好是巳日、或者亥日，就有可能发生内重丧。

# 五、犯重丧怎么办？

注意犯重丧主要指死者去世日子、举办葬礼的日子。也涉及择日问题。重丧关系到逝者亲属的生死问题，不可轻视，要请专人进行化解，以确保家人、亲属平安。

# 一、汉字的五行属性的认定原则

# 1、汉字有五行吗？

按周易哲学的观点，宇宙万物皆有阴阳，有阴阳必然有五行。汉字作为表达万物形象、意义的文字，当然有阴阳五行了。

# 2、汉字五行属性认定的主要依据是什么？

主要依据两点一是【汉字六书】：象形、指事、形声、会意、转注、假借。二是《说文解字》

所谓的六书，即指中国人造字的方法有六种：象形、指事、会意、形声、转注和假借。六者中，前两项是造字法，中两项是组字法，后两项是用字法。真正与汉字五行有关的是前四项，它们都是由表意象形字根，如金、木、水、火、土等偏旁，像积木一样组合而成的。所以，汉字从造字之初，就有了五行属性。其中，象形字的五行，指事字的五行，会意字的五行，形声字的五行，都好认定。比较难的是转注字，假借字。

《说文解字》主要是解读汉字的含义，从含意上来确定汉字的五行。如：李，“果也。从木子聲”。李指李木，李树，李果，它的性质“从木”，属木性，五行为木才是最正确的，下面的“子”是子声，而不是五行的代表。

再比如 “淡” 字，个别 “砖家” 认为 “淡” 字是水火两种五行的组合，把这 “谈” 字搞的水火不容。《说文解字》云：淡，“从水，炎声”。本义是 “其 水淡而清”（《世说新语·言语》），字意也是冷淡，冷漠，冷清，不热烈，一点儿没有火热的意思。其五行就是水。水是火的天敌，不配两片火，让水灭了，怎么 能显出水的厉害来，这就是古人造字的妙境。因此，要真正明白汉字的五行属性，必须知道中国人造字的方法和汉字的起源。

以上汉字五行的认定方法，是最古老最可靠也最广为人们接受的一种汉字五行划分法，原理源于汉字的象形文字特性。因为汉字是表义文字，从汉字的造字方法来看，无论是象形、指示、会意和形声，其字形与字义都有着紧密的联系，特别是汉字的部首，它其实就是表义的符号，所以，通过汉字的部首，可以对大部份汉字的字义五行作出准确的划分。

由于汉字的复杂性，少数字虽有表示某种五行的部首，但字义经“假借、转注”等已发生了很大的变化，很多字形不包括木、火、土、金、水这五种偏旁部首的汉字则难以明确划分其五行属性，必须按新的字义来划分其五行；而对一些无部首的独体字和有相同部首却字义演变较大的字，须根据字义划分。

# 2、从字的含义判断字的五行属性。

许多汉字可以从字本身的意义上来确定字的五行。

1) 表示具有火的意义或表示日出、光明等意义的字为火五行。如晖、旭、星、春、明、昆、晔、烨、炎等。

2) 表示具有山、石意义的为土五行。如岚、岩、研、岳、岑、峰、峥、碧、琛、玲、琦、琳、璘、珊、璇等

3) 表示具有草木含义的为木五行。如艺、芳、芬、芸、芹、花、萱、笛、蓝、蘭、策等。

4) 表示钢铁刀剑含义的为金五行。如钢、利、剑、列、刘等。

5) 表示鱼、雨含义的归为水五行。如鲜、雯、雪、霄等。

# 3、从字的音韵来判断字的五行属性。

声母为 b\p\m\f\u----属水五行。如博、普、斐、等。

声母为 d\t\n\l\j\q\---属火五行；如强、宁、东、等。

声母为 g\k\h----属木五行；如凯、涵、开、敢等。

声母为 x\zh\ch\sh\r-i---属金五行；如雄、众、润等。

声母为 a\o\e\ai\ei\ao\ou\ie\an\en\ang\eng\er ---属土五行，如傲、安、英、爱等。

从三个方面来判断，有两项一致，就可以确定该字的真五行属性。若找不到两项统一，则按着1本义2形体3音韵的顺序来确定。另外有些字在演变中凝聚了两种或三种五行，就像地支藏干一样，如卯支只藏乙木一个，而寅支藏甲丙戊三个，那就要选主要五行用之。

# 二、五行起名的具体步骤：

# 第一步确定命主八字喜用之五行:

1、首先以八字格局喜用为主，或补格局相神之不足，或补助偏弱之日主。从而达到以名字五行扶抑日主之某十神不足之弊，从而有利于命主财、官、印、食之运也。此法必须要认真评析八字格局。举例说明：

# 坤造: 癸巳 乙卯 丁丑 庚申

此造年干七杀，月令印星，格成杀印相生。时支有财，格局不低矣。但嫌七杀无根，也就是说印旺，官轻，反有影响命主将来官运之虑也。此命应以补水为好。故此命主取名“季洁”，后读书有成，一步迈出“农”门，先进机关当科员，后再提升为某县妇联主任，后享受副县待遇，不得不说此五行带水之名字，对命主官运亨通起了一定影响。

# 2、(八字无格局)以八字组合利弊确定喜用五行, 重点看财官印是否得用

# 乾造：壬子 丁未 丙寅 戊戌

从格局的角度说，有杀先论杀，为劫财合杀，月令伤官，制杀太过，没有了贵气，常言说：财为养命之源。无财何以养命？八字无财或财弱终是此命局之病。因此取名时，当以救“财”为要。此人名字中宜取带“金”之字以救财。但此人原名“朱贵平”，外出闯荡十余年，非但不贵不富，三十过了还没见过女朋友什么样儿呢。后听笔者之言，改名“朱贵鑫”，恰这“鑫”字属金，乃命局日元丙火之偏财，自改名后，此人生意渐入佳境，日子渐渐好起来了。

姓刘 男命：公元2002年3月13日5时-7时出生

# 第二步、根据福主要求设计起名方案

一般福主起名都提出一些要求，如要求起两字名，有的要求三字，四字命，有的要求按祖上辈分起名，有的要求加进父母的元素等等。所以首先要问清福主的要求。其二要问清福主的基本情况（如父母姓名、有何忌避）

# 四、制定方案的四项基本原则：

1 以八字的真正用神为主前提。(直接选八字用神五行或生用神五行。)

2 以姓与名的组合为中心。姓与名的搭配组合要有美感，有气势。确定姓氏的五行，其姓与名的关系可用天、人、地来表示，一般都为三字名。那么组合关系如下：

天生人，地生人：上等名

天生人，地比人：上等名

地生人，人生天：上等名

天生人，人生地：中等名

地生人、人比天：中等名

天克人，地生人：中等名

地生人，人克天：下等名

天克人，人克地：下等名

人克天，人克地：下等名

3 以用字的音、形、义为重点。起名选字的重要依据。

4 以五格剖象及 81 数理为辅。

# 五、起名的五条测评标准

# 1、关于字形的宜忌

宜用笔划数适中的常用字，忌用笔划数太多或太少的冷僻字。宜用偏旁部首不同搭配的字，忌用偏旁部首相同或笔划悬殊太多的字。

# 2、关于字义的宜忌

字义包括字的本意和寓意两方面, 而一个人或一个单位的名字, 往往体现出某个某单位领导人的愿望和追求。因此在取名时, 对字义的要求就要讲究有寓意, 有回味余地, 能清楚体现取名者或被取名者的志趣、爱好和理想。

所取名字名称的字义宜健康高雅，忌所取名字名称流于俗气土气。一个好的名字不仅让人好识好记，还会让人产生一种高雅和崇敬的情感，使人从社会交往中不知不觉也受益非浅。

还应注意体现不同性别、不同个性特点、不同行业、产业特色的名字，切忌取那种不分性别、无个性特点，相互雷同、一个名字千人可用。如“胜利”

# 3、关于字音的宜忌

宜名字名称的四声组合有差别,忌名字名称的四声平仄无变化。汉字读音分为平、上、去、入四种声调,取名必须符合音律。名字要响亮,要有音韵美。

# 4、搭配的要求

1）必须与姓氏搭配得当。名与姓氏组合，能产生非常恰当且丰富的创意。例如：张开翔（任您飞翔在宇宙）苏展越（苏展而超越）任翔宇、甘雨泽、吴阶廷等。

2）必须符合性别要求。让男性名字充满阳刚之气，让女性的名字产生阴柔之美，既符合中国的传统习俗，又具有新世代的象征。男子偏重于有理想、有抱负、立大志、成大业，可分为文用名及武用名。

3）要有创意，避免雷同。

4) 要符合心理暗示导引。当今社会中，人人都希望实现自己的理想，体现其自身的价值。父母更是望子成龙。

# 5、关于数理的喜忌
