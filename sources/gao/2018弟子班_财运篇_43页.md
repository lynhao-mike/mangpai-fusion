# 第 1 课、怎样理解古人断财运富命歌诀？

我们在学习如何用八字看财运的时候，先要了解古人对财运、富命批断的主要思路和基本规律。古今命书包括盲派秘诀，对财富的的断语很多，而且是从不同角度去下的断语，并没有系统的专题论述，归纳起来大致可分为以下几个侧重点：

# 一、强调身旺财旺是富命。

财多不富身弱穷，身旺有财独得生。

独财能招千斤鼎，身弱无根财局成。

# 这两句是古代命书比较典型的断富命歌诀。

第一句强调日主与财星的平衡，也就是说财多身弱，或身旺无财，都不可能富贵。只有身财两停，才能成就富命。

第二句是从另一个角度强调，原命局的财不在多而是要旺强；八字财星组成格局，不在于身强身弱都能发财。大多后学者对身弱无根财局成。这一句理解不够深入。

我个人理解，凡有格局的八字，不必强调身强身弱，能不能发财，是不是富命，与身强身弱没有关系。所谓格局代表一个人的事业，无论是在国家，企业干事，或从事个体经营，都是事业。没有格局的八字，或格局被坏的八字，需要看身强身弱，没有格局的八字犹如一个人，没有事业，没有单位，靠体力赚钱，靠打工，靠做点小生意赚钱，这种人只有身体强壮，岁运见财代表有活干，有事做，才能有钱赚。

# 第 2 课、何知其人富财气通门户

何知其人富？财星通门户

何知其人能大富，格局会财通日主。

原文出自《滴天髓*何知章》关于“财气通门户”，古今命书多有解释，其重点都在突出：“门户者，月令也”。说的就是财星居月令门户为用，财格成而无破必然富足人间之简单含义。还有学者强调日主与财星的平衡，即身旺财旺就能大富；身弱财旺多为富屋贫人；身旺财星微弱，难得大富等等。

民国时期的命理大家徐乐吾对“财气通门户”的诠释是：财星当令，得气得地，配合有情也。我认为这个解释虽言简意赅，却仍然没有到位。仍然没有点破他的真意。本人认为“财气通门户”有三层含义：

第一层含义，财星与命主有某种必然联系。

“门户”是与外界联系的通道，是人进出的路径，那么是否可以引申为财气通身的途径、渠道、办法？我们都知道年月之财是公众之财、国家之财；也就是说八字的财，不一定都是命主的财，年月的财星，只有与命主发生必然联系，或是有途径送到命主家门口，或是命主有办法拿到，才算命主的财，才有发财的可能。那么这种联系、途径、办法才是“财气通门户”的真意。

第二层含义，财星组合与命主有情。

通，就是没有阻碍，通达、通畅之意；没有阻碍就是一种有情。也就是说在财星与命主产生联系的组合中，是有情的组合，而不是恶意的组合，换句话说，财星给命主带来的是财富，而不是灾难。为什么有的人一发财，就有灾祸了？就是因为组合无情，比如七杀贴身，年月财星有渠道能生到七杀，那就是财生七杀克身，有灾。财星组合是否有情？”展开来说比较复杂，简而言之就是财星能为我所用。（后面举例说明）

# 第三层含义，财星要旺相。

最好是值月令旺相，如不值月令也要受生而有力。这是判断财大财小的重要依据，而不是判断能否发财的依据。有的学者将财星旺衰与日主旺衰相比较来判断命主能否发财，认为身、财两旺才能发财，两方有一方不旺相就不能发财。这种观点笔者不敢苟同，我们知道，八字中有财，不一定是命主的财，首先要看财星与命主是否有联系，也就是说要看命主是否有取财的途径和办法，这是关键的一步。财星与命主没有任何联系，身与财再怎样两旺、平衡也不会发财，也就是说这个财与你没关系。

在确定财星与命主有联系的基础上，要仔细推敲财星与命主有联系的组合中是否有情，能否为我所用，这是判断命主能否发财的第二步。最后再去考虑财的大小，即财星旺衰的问题。与日主的旺衰没有必然联系。

# 一、首先我们了解一下正财与偏财有什么区别？

正财——属于正常、固定的财源收入，例如：工作赚取的工资、补贴、销售收入或利润等，此乃数目亦较为固定，不会大起大落，因此，命中有正财而没有偏财者，适合以“上班一族”的方式来求财，以积累储蓄的方式来增加财富的收入。

这种类型的人，普遍属于工薪阶层和农村人士比较多，其如果是自己经营企业则往往采取薄利多销的原则，一般的商场和小店之类的经营都属于正财类型。女性八字逢正财，则多是贤妻良母型的伴侣。

偏财——表示投资、或投机，不定时之收入。偏财由于是同性关系，相克无情，属于意外的收入，例如，中彩票，经商，承受家产等谓之偏财。

这类人比较豪爽，口才出众，可以把稻草吹成金条，有很好的策划才能和整合能力，钱来得快也去得快，不执着钱财，比较喜欢享受和追逐物欲。这类人在城市比较多，尤其是发达的城市。活跃在广告策划、创意整合、市场营销等行业的精英分子里，有很多是这样的类型，偏财越旺这个方面的特点也越明显。

# 第5课、如何判断命中财富级别？

# 一、首先我们从财的感念来区别一下，

以财星看财，财的级别较小；财星就是现实中的钱，取财工具主要是禄，或比劫。禄就是自己亲力亲为啊。比劫是合作者，或是你的老板。你无力取财，他们帮你取财，但要付出代价的。这样一来，财的级别就不会太大了，说合作好听一点，其实就是打工，给老板打工能赚几个亿吗？能达到一亿就是极品人物了。

# 第6课、如何从八字看求财方式？

# 1、基础财富看财星

财星是最直接的财富信息，也是最基础级别的财富标志。八字中的财星代表一个人对金钱的欲望和掌控能力，偏财的财富值大于正财。财星落在月令最强，虚浮天干就比较危险。

命中只能通过财星来断财富级别时，财富的级别相对比较小，这种情况属于小富。

# 2、赚钱能力看食伤

食伤是生财的源神，当命中有食神直接生到财星时，生财渠道畅通，这样的财富等级就会提升一个层次。

食伤是智慧才华之星，代表一个人赚钱的能力，食伤能生到财星，说明命主可以依靠自己的智慧才华去创造财富，生财能力超强。

食神生正财，伤官生偏财，因此伤官的谋财能力远大于食神。有伤官生财之人必须要创业经商才能发财。

食伤生财的财富级别属于中富。

# 3、高级财富看官杀

官杀是财富的升级版本，首先官杀是财星所生，其次官杀可以卫财。命中组合以官杀论财富时，财富等级是最高的。现实中，有钱之人要做大格局，最终都会谋求权力，权力和金钱的结合才能带来最高利益。个人财富总会有上限，但权力的财富几乎可以说是没有上限的，权力可以带来一切也可以剥夺一切。

官杀是正官和七杀的统称，七杀有制化时，权柄大大超过正官。因

此命中有七杀一定要先看七杀的作用。

# 4、存财能力看财库

命中有财库才能存住财富，财库冲开之时就是发财之时。命中没有财库，运中遇到财库同样可以发财存财。通过风水调整也可以催旺财库。

除了财库之外,禄神也可以当做财富信息来看。禄神代表福禄、财禄和衣禄等等,禄神的财富是一种享福享乐之财富,幸福指数比较高。命中禄神最忌被破坏,不仅会破财,身体健康也会受影响。

# 第 10 课、一夜暴富与大起大落的命理特征

一、八字中日坐偏财，年月见偏财，与命主有强力感应的可以一夜暴富。命主与大财横财结有善缘的人能够暴富。大财就是偏财，因此，无论偏财是命局喜忌，有总是胜于无。偏财、大财为善，为喜，这说明自己有得到大财的善缘，是自己应得的，能够得到的大财。而偏财为忌者，则意味着和大财、偏财有恶缘，即使偶然能得大财，也很容易失去，否则恐有灾祸临头。

二、八字中有发如猛虎条件的人能够暴富。何谓发如猛虎？就是马奔财乡。八字中的财星为寅、申、巳、亥四马星。不过，要得到偏财、横财需要的条件一般是：如果命中四马星为喜用神，又遇到四马星的财星大运；逢流年引动马星、财星的时候，就是得到横财的暴富机会。如中奖、博彩、投机获利而暴富的可能性很大。如果命中的财星为寅、巳、申、亥四马星，但也是本命的忌神，那就需要大运逢冲制寅、巳、申、亥四马星，也就是马奔财乡，发如猛虎的机会。

三、命主自带开库的钥匙，八字年月见财库，或大运流年见财库，就有一夜暴富的机会。

大运中含有财库是指出生日天干是甲乙的，大运中有走“戌”运的；出生日天干是丙丁的，大运中有走“丑”运的；出生日天干是戊己的，大运中有走“辰”运的；出生日天干是庚辛的，大运中有走“未”运的；大运中有走“戌”运的。

常言道，“人无横财不富，马无夜草不肥。”特别是现今社会，钱财的多少往往是衡量一个人成功与否和地位高低的标准。上面我们总结了三条一夜暴富的基本规律，我们来看下面的几个案例，是否符合一夜暴富的条件？
