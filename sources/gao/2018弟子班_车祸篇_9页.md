# 高德臣盲派命理

# 车祸篇

# 从八字看车祸命理求真（专题）

引言：

我们为人预测的主要目的是趋吉避凶, 其中避凶是人生之重点。凶有大凶小凶。所谓大凶即指车祸、伤残、疾病、夭亡、官非、牢狱等。古人讲: 天有不测风云, 人有旦夕祸福。那么从八字中能否看出人生大凶吗? 研究表明, 人的生辰八字中储存着人生大量的吉凶祸福的信息符号, 关键是看我们能否破译出来。为了深入研究, 我将人生之大凶分为四个专题来探究: 即《从八字看伤残车祸命理求真》、《从八字看疾病带残命理求真》、《从八字看官非牢灾命理求真》《从八字看死亡寿夭命理求真》人的精力是有限的, 不可能把所有的问题都研究透彻, 但我走的是一条捷径, 即对前人的研究成果进行整理、归纳, 并寻找规律。本专题重点探讨伤残与车祸。受伤致残是人生一大凶事, 而受伤致残的原因很多, 有打伤、摔伤、工伤、意外伤, 有人统计现在伤残比例最高的是车祸导致伤残。因此如何根据八字, 结合岁运, 找出车祸标志, 警示命主趋吉避凶是禄命理论必须研究的课题。

这里把车祸与伤残分开来讲

# 第一课、八字看车祸的一般规律

# (一) 总诀:

八字车祸信息显，掌握规律很简单；

寅申巳亥午是马，金水湿土像车船；

印星金舆代表车，冲破穿绝车要翻。

灾祸大小不好断，原局之中看寿元。

注：寅申巳亥是天马，午是真马，申酉亥子辰丑代表车像。
