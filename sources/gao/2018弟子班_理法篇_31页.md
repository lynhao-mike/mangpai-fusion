# 高德臣盲派命理

# 理法篇

# 第1课、月令是八字论命的关键

# 学习要点：

无规矩不成方圆，无理法难以论命，理论是用来指导实战的。

在古人论命的歌诀中有：有杀先论杀，无杀看月下。

在命学典籍《四言独步》中先观月令,论格推详.以月为

主,专讼财官,区分贵贱,妙法多端.

古人为何主张论命先看月令？看月令看什么？

论命首重月令，这是任何命理门派所公认的，月令是八字的核心点，月令是天地主气的发源，月令是取用定格的要件，这是不争的共同认识。古人在典籍中对月令反复强调不惜笔墨，现代人论命同样视月令为重要。如旺衰生克法，把月令视为五行、十神力量旺衰强弱的来源；格局论命派，把月令视作天命，以此立格局、论成败、定贵贱。盲派论命说，打开八字找财官，上哪找？上月令去找。总之月令乃提纲，司四时之令，布天地之气，命之富贵贫贱之根源。这是每个门派的共识。

《子平真诠》——定格取用总纲领：八字用神，专求月令，以日干配月令地支，而生克不同，格局分焉。财官印食，此用神之善而顺用之者也；煞伤劫刃，用神之不善而逆用之者也。当顺而顺，当逆而逆，配合得宜，皆为贵格。也就是说八字真神，要从月令上寻求。

《三命通会》——大凡看命，先看月支有无财官，方看其他，月令为命也。如月有正官及偏官，而时又入他格，只以月中取，他格无用。如月令全无可用，方看他格。先贤取用只重提纲，而用神等物又以财官为本。

用神只是月令定格之物，譬如甲木日元，生于金月，用神就是官杀；生于火月，用神就是食伤；生于水月，用神就是印绶等等，而这官杀、食伤、印绶，既是用神，也是定格之物。月令可用之物指六样东西，即：财、官、杀、印、食、伤。古人以此定为六大正格。用一句话说，就是月令除了比劫之外，其余的东西均可以取作用神。如果月令是比劫，就表示月令没有用神。这时，就须要在月令之外寻取财、官、杀、印、食、伤这六样东西，来取作用神。这是格局论命的基本规则。

黄大陆先生说：为什么要从月令取用神呢？因为月令是父母宫，人都是由父母所生，故而父母宫带有先天遗传信息。看命要先看你父母给你遗传了什么，或者说你想要立身处世，做一番事业，得先看你家里有什么东西可以为你所用。

盲人讲打开八字，专找财官。取用神又要以财官为主，为什么呢？因为官为名，财为利，世人整天起早贪黑忙忙碌碌的不都是为了这两个字吗？所谓“人生在世名利二字。”名利二字又可以延伸出很多社会现像。如官可以代表女性的丈夫，男性的儿女，财则代表男性的妻子，所以这财官二字囊括了人生的事业与婚姻，同时没有财所代表的物质、金钱我们就无法生存。没有官所代表的名誉、地位、法律、制度和秩序，我们就无法区别与其他动物。

盲派秘诀《先天本元诀》谈到：“先天一气是本元，要看清浊

分自然”。

何为 “先天一气”？从八字的宫位看，年为祖上，月为父母，也是日主出生的地方；从遗传的角度看，年月为先天，日时为后天。这里的先天主要指月令。

所谓本元就是根本，是第一位；很多古代命书，把月令比作“提纲”，月令是看命的关键。是体现命主一生富贵贫贱，人生品味层次高低的标尺。何谓清浊？月气有深浅，透藏分清浊。清纯与混浊主要讲阳阳搭配的问题。

《滴天髓》清气篇云：一清到底有精神，管取生平富贵真，
澄浊求清清得去，时来寒谷也回春。

浊气篇云：满盘浊气令人苦，一局清枯也苦人，
半浊半清犹是可，多成多败度晨昏。

盲派歌诀中有：“独财值千金，独官掌万民。”讲的是物以稀为贵。

清纯代表用神的能量级别。其中的道理可以从一个和尚挑水喝，两个和尚抬水喝，三个和尚没水喝的故事中去感悟。总之，清纯浊杂在这里是用来判断格局成败和能量大小的。下面我们看一个八字：

# 第2课、关系学是破译八字的重要思路

八字预测学实际是一种关系学，是用五行十神之间的关系来揭示人生之理。

八字所揭示的是人与自然的关系，人与社会的关系，人与家庭六亲的关系。

人与自然的关系，主要体现在在命主的出生环境，生存环境，身体状况等；人与社会的关系，主要体现在命主与财。官、印、食的关系，体现命主的富贵贫贱，人生价值，人生的层次等；人与家庭六亲的关系，主要体现命主的婚姻、父母、兄弟、子女等状况。

那么如何运用关系学具体破译八字呢？

第一步，先要分出宫位。年月为先天，为外环境，为国家，为社会，为他人，为祖辈、父辈；日时为后天，为内环境，为命主结婚后的家，夫妻的家，子女的家。一般来说，年月概括为他宫，（也可以叫宾位）日时概括为我宫（也可以叫主位）。之所以要分出我宫他宫，就是要看我与他都有什么关系。

第二步，具体理清八字的横竖关系。先横看天干的关系，横看地支关系，再竖看天干与地支的关系，最后看发表归根的关系。

# 第3课、体用论命的特点

盲派命理有三大支柱体系，一是体用论命突出理法，注重八字的组合结构优劣，二是像法论命，突出技法研究；三是神煞论命。

# 一、什么是体用？

四柱是由年来推月，由日来推时，这样本身就将四柱分成了两部分，即我宫，他宫。我宫就是日柱和时柱；他宫就是年柱和月柱。一个四柱是一个太极，太极分两仪，这两仪就是我宫，他宫。

关于体用，现代命理专家段先生，为了明确体用关系，他把体用换成，主宾，换了一个名词，似乎更接近生活，便于理解；是终南命理传人惭愧学人，用更通俗的两个词，我宫，他宫；把体用关系说的更明白。

更具体的说，日干代表我，（我个人）日柱代表我家，（包括，妻子、或丈夫）日时代表我宫，（包括老婆孩子，也包括我的财物，包括我的思想。）

# 二、通过体用要表达什么意思？

体用，表达的是，我与父母祖辈的联系，我与社会的联系，我与自然界的联系，内环境与外环境的联系，小环境与大环境的联

系，小家与大家的联系。

体用关系还表达了，命主的追求，命主一生是以求财为主，还是以求官为主。

体用关系还表达了命主有没有能力，有没有办法求到，财官。所以体用是盲派八字论命的入手点。

干查根源，支找去处，人生追求，财官印禄

有得有失，方法思路，天地君臣，配合适度，

内外感应，天地有情，我为他用，为人打工，

明看五行，暗推长生，神煞空亡，参考应用。

我宫为体，它宫为用，看财官在我宫还是他宫

四柱生成，先定体用，（排好八字，定出我宫，他宫：）

同根同旬，日时我宫。（原则上，是日时为我宫；还有一条就是同根同旬也是确定我宫的重要条件，有是日主与月柱同宫同旬，同地支，那么月日为我宫。）

干查根源，支找去处，（上一节讲过，每一个天干都要查他又没有根源，几个根源，根源是在哪？在我宫还是在他宫？都代表谁，或代表什么？每一个地支都要看他透没透出，透出几个，透到我宫还是他宫，要干什么？）

人生追求，财官印禄。财官印禄是人生追求的目标，命局中出现没有，出现了就看与我有没有关系，如果没有财官，有没有可以代表财财官的东西，然后看命主有没有能力、有没有办法获取到手？

有得有失，方法思路，得失是八字论命的核心，，人生都是在追求，只是目标不同，（有的求官贵有点求财，有的只求名气，有点追求享乐，有的只求一生平安等等）方法不同，有点用体力求财，有的用智慧，有的用技术，有的用艺术，有的用人脉，有的用非法手段，强抢、咋骗等等。

天地君臣，配合适度，天干为君，地支为臣，干支配合有情。

内外感应，天地有情，我为他用，为人打工，

内外感应，就是指我宫与他宫有什么感应，通过什么方式感应？是我为他用，还是他为我用，如果他被我用，我的地位高是我管别人，我被他用，是我给别人打工，给别人干活。

明看五行，暗推长生，神煞空亡，参考应用。

五行生克是根本不大法，哪一种论命思维都离不开五行生克。在运用生克的同时，要看每个天干的生存状态，即十二长生状态。盲派讲究三点一线，两项定一项，就是从不同的角度，用不同的方法都证明一件事，才可以铁口直断。所以我们断命要做到多角度求证。

坤造：丁 丙 壬 辛

# 巳午寅丑

年月他宫，日时为我宫。年月正偏财，是他宫之财，代表是别人的财，社会上的，国家的财，也代表是大财。那么这个财，命主能不能得到。如果我们用旺衰的思维看，这个八字，身弱不从，印星为用，年月全是忌神，哪有富贵？如果我们用体用的方法看就简单多了，月上丙火有两处根源，其中一处在日主坐下，有两个含义，一代表命主的才能，二代表命主的投资，（说明命主一生以求财为主）丙火定义了午火，说明午火被命主控制，午火透出丁火，又掌控了年上的巳火，年月的财都在命主掌控之中，只是掌控不行，还要有渠道，有办法拿到才行。

干查根源，支找去处，何处起根源，流向何处住。分辨体用之间存在什么关系，哪些关系？即命书上所说的发表归根，钩玄索引。体用双方的作用关系：一是天干与天干之间的作用关系，二是地支与地支之间的作用关系。（段先生把作用关系，称之为做工，更便于理解。这是一种创新）

我们拿到一个八字，先将我宫，他宫这个概念树立起来。然后看我宫的地支藏干是否透出，如果透出了，看他是透在在我宫还是他宫，透到他宫，看它下面所坐的是什么地支，这样才能知道我宫透出的天干在做什么事情。如果去做事了，看都在做什么事情，是对我有利的事情还是对我不利的事情。有利的事情就看有没有阻力，能否顺利的办成。如果做了对我不利的事情，就要看有没有化解之道，变不利为有利。

再看看他宫的地支藏干透没透在外，是透到了我宫，还是透到了他宫。透在我宫的话是否是在帮我做事，还是搞破坏。若透到他宫，也同样要看他们在做什么事情，是对我有利呢，还是对我不利呢？用《滴天髓》的一句话就是“配合干支仔细详，断人福祸与灾祥”。

有的时候透出的天干又形成了一个小的天地，这个小天地也同样蕴藏着某种玄机。比如说，癸水透在外，癸下坐卯，为癸水坐长生，长生的意思就是刚刚生长，如果长生的地支被坏掉了，那么癸水所代表的六亲就要遭殃。

# 四、如何确定体用？我宫和他宫的划分

一般情况下将四柱分成我宫和他宫，我宫指的是日柱和时柱，时柱并且为真我，代表一个人的归宿，尤为重要。他宫指的是年柱和月柱。

原则上，日时我宫。有时月日同根同旬，那么我宫在月日，年时为他宫。这个知识点需要慢慢感悟，不是一说就会的。等遇到时再讲。

看财官在我宫还是他宫？财官是我们每个人都想得到的东西。如果我宫能得到，那么福禄自有，反之，我宫的财官被他宫获取了，那么名利俱损。

# 第 4 课、得失是八字论命的核心

看任何一个八字，都是在看“我”得到了什么，“我”失去了什么。得是得，失是失，不能混为一谈。得：“我”用什么工具获取了“他”的什么。失：“他”用什么工具获取了“我”的什么。得和失都有一种途径或者路线。从路线来看，破坏路线有三种方法：

1、我要获取的对象在源头上被破坏。

2、我要获取对象的路线在中间环节有破坏。

3、我获取对象后在结果上被破坏。

反过也有是同理：

1、我要获取的对象在源头上得到优化。

2、我要获取对象的路线是优化的、简化的。

3、我获取对象后在结果上能被优化。

这一过程就存在了得到多少，失去多少，量的问题。得失都是组合，和位置关系。把先后、位置区分开来。虽关系错综复杂，亦能理清。工具、对象、途径在一个八字中往往不是唯一的。工具和途径往往又是统一的。对象和途径往往也是统一的。如八字中的透、归根、伏吟这就是统一。

# 第5课、阴阳是八字论命的灵魂

# 一、八字中阴阳的含义：

学习八字，首先得明白“阴阳”这个概念，八字命学之精华全在阴阳五行之变化之间。“能把阴阳颠倒颠，就是世上活神仙”，此乃易学宗师之感叹，

阴阳：最初涵义是很朴素的，表示阳光的向、背，向日为阳，背日为阴，后来引申为气候的寒暖，方位的上下、左右、内外，运动状态的躁动和宁静等。

以天地而言，天气轻清为阳，地气重浊为阴；体现在八字中天干主动，地支主静。天干为阳，地支为阴。

以水火而言，水性寒而润下属阴，火性热而炎上属阳。体现在五行上，木火为阳，金水为阴。

从进气退气而言，子水为阴极阳生，丑为木气生；午为阳极阴生，未为金气生。

从干支的阴阳来看，干属阳，主外，代表社会活动域，支属阴，主内，代表个人生存域；从柱位分，年月属外，主疏、远，日时属内，主亲、近；从柱位阴阳论，年日属阳，代表思维、无形、动象、转化，月时属阴，代表实体、有形、内部变化。

# 二、阴阳在八字论命中的应用

阴阳的四个基本特征：对立、互根、消长、转化。属于哲学范畴。

这里重点讨论阴阳在八字论命中的应用，其实在八字论命中，阴阳的应用非常广泛，有直观的，有隐藏的，有宏观的有微观的。这里重点讲几个直观的阴阳理念的体现。

1、阴阳感应：阴阳互藏交感，“交感”即交互感应，是指阴阳二气在运动中处于相互感应相互作用的过程之中。万物的化生源于阴阳之间的相互作用，如《荀子·礼记》中指出阴阳交感是万物化生的根本条件。

在宇宙自然界中，天之阳气下降，地之阴气上升，阴阳二气交感，化生出万物，并形成雨雾，雷电，雨露，阳光，空气等，水火相互交感，生命体方得以产生。在阳光雨露的沐浴滋润下，生物得以发育成长。

在人类，男女通过阴阳交构，新的生命个体得以诞生，代代相传，人类得以繁衍。所以，如果没有阴阳二气的交感运动，就没有自然界，就没有生命。

可见，阴阳交感又是生命活动产生的基本条件。

2、阴阳相互对立，即世间一切事物或现象都存在着相互对立的阴阳两个方面，就八字论命而言，天干为上属阳主动，地支为下属阴主静。动与静是相对的，动则能生能克，因此天干与天干之间遇到金则动而生水、木则动而能火、火则动而能克金、金则动而能克木，地支与地支之间，金静而不能生水、木静而不能生火、火静而不能克金、金静而不能克木，这些都是自然之理，显然不可违背。

3、阴阳转化在八字论命中最直观的是体现在寒、暖的转化上，命书言：寒虽甚，要暖有气，暖虽至，要寒有根，则能生成万物。若寒甚而暖无气，暖至而寒无根，必无生成之妙也。所谓阴极则阳生，阳极则阴生，此天地自然之理也。由此可见，阴阳对立与阴阳之间的转化是八字预测的灵魂。阴阳转化在八字的演算过程中是经常遇到的，最明显的，是大运的转化使原局五行发生阴阳消长。在八字预测术中，阴阳消长和阴阳转化一样都是指命局中的阴阳二气在行运中的变化状态的。

4、阴阳消长有两层含义，一是命局中阴阳二气在大运的变化，二是命局中的阴阳二气存在着吉凶，大运的到来必须分清吉凶的消长。

5、阴阳互根。阴阳互根就是矛盾的对立和统一，地支作用关系中的六合就是阴阳互根的表现。大家可以对比一下，在地支六合中，子水午火为阳极阴极之点与本气相合，其余的寅亥、卯戌、巳申、辰酉都是一阴一阳相互配合。发表归根也是一种阴阳互根。

6、阴阳相互依存，对立的阴阳双方相互关联，互相依存。从八字的宏观角度上看，天干地支相互依存，如干透财官，必察其阴位之根，若无则其用必不可久，或曰虚浮天干，空有其形，难用其能，必待大运或流年补全其质才能真正用其能；若支有财官，而干不透，则徒有其质，不显其能，孤阴不生，必虚位以待用，待大运流年透干方显其能，任何一方缺失都不符合阴阳之道。

同属性阴阳互相依存，比如木五行，甲乙同属木，但甲为阳乙为阴，看似两者属于阴阳对立。实则相互依存。最典型的是藤萝系甲，是乙依存于甲，甲用乙合庚，是甲依靠乙。更深层次的还有甲乃乙之气，乙乃甲之质，甲是乙散发出来的阳刚之气，乙是甲体现出来的阴柔之质，所以两者是互相依存的。

被金水之阴所包围的丙午却是阳中之阳，丙火虽然坐着帝旺，但是它没有根源；己未跟丙午并没有君臣隶属关系，丙午这日柱所代表的是命主自己，这样一来阳性阵营只能包夹阴性阵营的臣，而命主自己却被包夹在阴性阵营当中，这是第二个特征。

三、婚姻情况：上面说到这个年干己土跟时干的壬水都派出坐支中的乙木去坐大运地支的亥水跟日支午火暗合，这里面有说法。己土跟壬水的目的是不一样的。

己土本来他就是想通过自己的坐支未去合午的，他是想把丙午娶过门来的，只是无奈于这个丙火、午火被包围了，只好走曲线救国的路线，通过透出乙木坐在亥水上面去暗合她的下半身。己土想真心搞对象。

丙子大运，丙子的子水可是个大桃花，这个桃花不仅仅是年命纳音“火人猴鼠令人贱”的桃花，而且还是年支的咸池桃花，而且又是原命局癸水的禄地，那么命主在这个子运，她的桃花故事必然是连续剧。

果不其然，刚进入子运的乙酉年，她就认识了一东北老板，两个人好了一。到了2006的丙戌年，丙火坐在戌库这个太岁上面去刑年支未，这年她跟她的丈夫闹离婚，闹得还挺凶，但这一年并没有引动大运跟命局的子午冲，所以没离，至少在法律上面他们还是夫妻。不过呢，她跟她丈夫的婚姻已经是名存实亡了。

这里有个小技法，值得注意，月干的癸水是插入他们夫妻俩的第三者。癸水是从时支辰库中透出来的，这个壬辰是长流水啊，源源不断的，所以月干这个癸水官星所代表的肯定不止一个男人，何况大运地支这个子水为癸水的禄，又多了一个根源。

丁亥年，时干的这个壬水通禄了，而且午中的丁火透到天干去合壬水，所以这一年是命主男女关系最为混乱的一年了，她同时跟三个男人周旋。也就是说，除了她的丈夫之外还有两个男人，一个男友是壬辰，辰是水库，对于壬水来说是比劫库，是老板；另一个呢是癸酉，坐着印绶的公司经理。

到了 2010 的庚寅年，命主要进入丁丑运，日支午中的丁火给跑出来了，去冲癸水，去合壬水，跟情人闹矛盾了，对情人不满意了，而且又跟别的男人有了一腿。这个男人是壬水了，凑巧的是这次这个壬水呀也是个老板。
