# 一、什么是像法？

现代命书上都有论述，有人叫取像、类像、形象等等，如果给他下个定义，

所谓像法，就是围绕天干、地支、宫位、十神及神煞的基本含义，运用比喻、借代、类比的思维，加以最大限度的引申、移位，把抽象的干支、十神符号形象地转化为具体的人、事、物，就叫类象。其实很多高层断命都在基础当中，而且像法的技巧也都在基础之中。

# 二、八字取像论命的理念思路

# 1、像法论命的理念思路

学习任何技法，都要首先搞清理法。四柱八个字，是一个信息载体，它能够全息反映出命主各方面情况。四柱之所以能够全息反映命主各方面信息，是因为四柱的八个字与人的各方面具有一种一一对应关系，我们正是运用这种对应关系来提取命局的信息之象。对应关系是四柱预测学最基本最重要的关系，它是提取四柱信息最主要的依据，不知对应关系就谈不到提取信息之象。所以我们首先要掌握四柱中都有哪些对应关系，对应关系或者说对应点找得愈多愈准确，才能越全面越正确地提取命局信息之象。

四柱八个字，从身体角度来看，四个柱(年、月、日、时柱)

及八个干与支，就对应着命主的身体各部位和器官；从家庭角度讲，四个柱命局中的十神对应着具体的宫位和六亲；从社会角度讲，四柱八个字有对应着社会上与命主有关的形形色色的人与事物。

总之每一柱，每一字，都有很多的对应关系，只是角度不同，对应关系就不同。

五行的基本涵义，是提取命局信息之象的关键，作也是最基础的部分。如五行木：

木的方位应着：东方；季节对应着：春天(寅卯辰月)；人体对应着：肝胆、四肢、毛发、神经系统等：事物性质对应着：曲直向上，有条达、恻隐、善良、仁慈、倔强之性等等；还有五味对应着：主酸；五常对应着：主仁；五色对应：绿色。其他五行的基本含义在基础课程中已经经过，不再赘述。

# 2、提倡像法与其他论命方法的融合。

有人问道，像法论命与其他论命方法的关系？我的观点是这样的：像法论命是一种独立的论命方法，但在运用上，最好与其他论命方法相互配合。如子平论命，以格局为筋骨脉络，以易象为皮肉毛发。论格不论象，则只能论断富贵贫贱吉凶祸福，而不能论断吉凶祸福的各种类型与诸多细节。但若论象不论格，则象无所依附，吉凶无所凭据，从而无法论断富贵贫贱吉凶祸福。所以我提倡像法与其他论命方法的融合。

# 3、八字取象断事法的应用

四柱八字像法的运用，可分为整体大象、局部象易像、细微

小象的不同。

取象断事法，是八字命理学中一种技法，多用在局部、细节性断事上。侧重论断行业、职业、性格、人品、人事、婚恋、情感、灾病类型等等，以及六亲方面一些细节问题，及其流年中发生的一些具体事件。

取象占断，关键在于象，但有象必有象理，重点就在于象理上。若不懂得真正象理，见象取象、无原则取象、会导致牛头不对马面。若能精湛者运用取象之法，必能抽丝剥茧看到真象，达到铁口直断，料事必能如神。

# 2、八字类象的多重性原则（怎样理解多重性）

A、一个事物有多个符号表示：比如测身体方面：日干为头脑、甲为头脑，亥为头脑，戌为头脑，午为头脑，年柱为头脑，天干为头脑。

日干为头部，日干受克是头部被撞、破相、精神出问题等。一般情况下，日干有几个本气根就有几个家，无根表示游离态，一天到晚在外面忙碌。日支为日主的内心世界，日支在岁运伏吟为喜，则天天开心，日支伏吟为忌，则代表心里痛苦、抑郁、难过等象。

B、不同的符号能表示相近的事物：如测生活小事：酉代表说话或自言自语、申代表传话或通知、午代表谈话或聊天、巳代表吵架或辩论、丙代表演讲或推销、丁代表咨询或打听、食神代表称赞或表扬、伤官代表发牢骚或叨唠。

C、干为动，支为静，天干代表发表，地支为归根，天干代表意向及趋势；地支代表地盘、实力。天干主公开，地支主隐秘，天干为先，地支为后。看日主心性是看日干周围最近的三个位置的十神星，因它们直接左右日主的头部、思想等。

克是一种对立情绪，排斥心态。生的含义不光是爱护、喜欢、付出这么简单，更引申为追求、渴望、吸引、招引。

C、八字类象细断的基本思路是：大事看岁运，常事看年月，小事看月日，太细的看日时。看流年结合运，看流月结合年，看流日结合月，看流时结合日。

# 3、一点多像，反复定像原则

人生是复杂的，是多层面的，原局仅有把这个字，要想把人生百态，以及复杂的人际关系清晰的描绘出来，必须一字多用，反复定像，才能达到预期目标。像法重在“活”“变”。所谓活就是灵活，变，就是指取象思维要随着太极点的转变而转变。太极点就是测项，也就是说测官运，用测官的思维取象，测婚用测婚的思维取象。比如测本人事业，那就以日干为太极点，其它七个字就是日主所面临的社会环境，如测日主健康，虽然太极点没变，但取象思维变了，那么其他字则代表日主身体的各个部位；如果说测父亲情况，太极点就变了，取象思维也要随之而变。太极点的转变决定取象思维的转换

# 1、相合换像技法，

相合、有天干五合，和地支六合，相合所表示的像分为两种情况：一是表示阶级关系，夫妻关系。相合都是一阴一阳，成夫妻之像，相合的双方可以互用对方的象。相合如同夫妻结合成一家，虽然原来他们各不相同，但一旦走在一起，就是你代表我，我代表你。比如女命戊土日主，见局中癸水可以看成夫星，有人会认为这不是阴阳颠倒了吗，既然是夫妻，完全可以换像。这是取像的一种技法。二是干支相合代表事物或人际关系的像。

1、一方能定义另一方面的性质与特征：如“劫财合财”，能表示此财的性质，是“劫财而得到的财”；“食神合印”印是学问，食神是思想，可表示是一种思想性的学问，即为“学术”；如果“伤官合印”，伤官是技艺或艺术，那可能就是表示“技能”或“技艺”了。

2、一方与另一方表示同一种信息: 可以是同一个人, 也可以是同一个事物。比如: 伤官为子女, 官也为子女, 两者相合同表示子女。

3、一方与另一方象可以互换。如甲己相合，己是杀，甲是伤官，可以换过来用，既是将甲看作是杀，或将己看作是伤官。我们分别择例而论。

# 关于扎根像法点窍：

发表要归根，透出叫发表，透出的天干要找到他的根源。看他扎根在何处？这里主要指日主在地支的根，中级班讲过根的问题。如：甲日主在任何一个地支见到寅、卯或者见到辰、未、亥都可以为根。

顾名思义，扎根就好比一个人在某处安家，有了立足之地。八字原局中见到扎根，能反映什么象呢？这要结合宫位来判断。扎根理论其中最主要的一点用法，就是看某人能否吃到公家饭。有几处家？

公家饭，指某人的职业是国家性质的，可以分为公务员、事业人员或者国企人员。

在八字宫位类象中，年月柱就是代表国家的，所以当一个八字日主的根在年月支出现时，就可以代表日主在年月（这个国家领地）有一席之地，扎根或者立足了。假如年月支这个立足的地方不仅见根而且见到财星了，那就表示日主立足之处还能享受到应有的工资报酬及福利待遇（财星可表工资收入）。

# 第7课、甲乙木取像点窍

八字取象离不开大胆联想，丰富的想象会使八字推断妙趣横生、拍案叫绝，但是想象又不能天马行空、胡思乱想、胡编乱造，必须合情合理，有依有据。下面就聊聊十天干的奇妙取象。十干取像口诀：甲木为雷为大树，栋梁电杆与高楼，神位公门及首领，高贵宽仁与磊落。

十天干是甲乙丙丁等十个字，仓颉造字的时候绝非胡乱编造的，造字的过程也是一个类比的过程，所以中国的的文字叫象形文字。每个字不仅有发音，还有字形和字义，非常奥妙。我们就从音、形、义三个方面一个一个发掘十天干的一些微妙类象。先从甲木开始吧。

甲：表示开始、开端。为什么呢？看字形，甲字上面是个田字，为土地，中间一竖，是一个嫩芽，正孕育着准备破土而出，预示着事情的初始阶段，代表着萌发和开始的意思。既然甲代表开端，那么开端是什么？是第一、是领头的意思嘛，所以甲可以类象元首、领导、家长等表示尊贵性质的人和物。在人们心目中什么尊贵呢？有句话叫学而优则仕，仕是尊贵的，仕表公门，所以甲可以代表国家、公家。寅为宫门。

再从字形上理解，甲下一竖好比钻头，引申为摁钉、钉子、螺丝，表示一种钻劲和锲而不舍的精神，有了这个钻的意思，那么甲木的人做事情就会目标比较专一，孜孜以求，但是甲多了呢？因为过分专一，反而成一根筋了。比如我们看见甲辰一柱了，甲是钻，辰是水库，那就有钻井、开采的意思了；辰是水库，水是甲的印，印库可以看做书库、知识库、智慧库，这就有了教师、图书管理员的象。

甲木为神位，过去有钱人家都修建祠堂，供奉县人大地方，先人牌位，形状很像甲木，所以甲木代表神位。

甲形还像个话筒，所以甲日主的人喜欢讲话，喜欢指手画脚给人安排工作。

甲为铠甲、马甲，甲字形的顶端是平的，引申为平顶帽、武士穿的战袍，现代社会就是公检法部队等表示执法部门和武职的象。

甲在天为雷，六爻中为震卦，雷是空气震动产生的巨大声音，所以甲木人，个性比较要强、刚毅、性急，如果见到丙火，那么更是风风火火了、咋咋呼呼了。

甲木配四兽为青龙，青龙主贵、主高大上，甲木又是参天大树，所以甲木可以类比电线杆、高楼、桥梁、笔直的大路。

甲无论从字形还是从参天大树的角度，都是高大挺拔的，所以甲木的人，身板往往很直，挺胸抬头，气宇轩昂。

刚才讲到甲表开始、表尊贵，那么人体部位那部分与开始和尊贵对应呢？有句话叫“从头开始”，还有句话叫“低下那尊贵的头颅”，那么甲对应的是头部。

甲是树木，树木是生发的，形状是枝枝叉叉，四处扩散，人体中神经的分布与树木的形状是一致的，所以甲可以表示神经。

相对乙来讲，甲是粗壮的树干，人体可以对应四肢，而乙木一般对应指头了。

# 第8课、丙丁火取像点窍

丙火太阳为影视，权力名气与电气，

书画演说和装饰，正面多言与激情。

丙火为太阳之火，主光明、热情，显著也。古人把丙喻为太阳。因为太阳的光是有辐射并能不断向外扩张、扩散的，所以具有扩散、扩张、传播性质的象都是丙象，比如：发光体、发热体、核能辐射、媒体、影视、信息、娱乐、文化、语言、电器等。

有句话“万物生长靠太阳”，那么丙日主的人天生积极、乐观、自信、爱管事、有号召力和感染力。丙天生就是个领导，一呼百应，追随者众，因为他总认为自己是太阳，别人离不了他。

太阳带给万物光明和希望，所以丙可以类象眼睛、心脑、漂亮、美丽。太阳普照大地，其性无私无畏、光明磊落、温暖喜庆。

太阳代表神明、神灵，太阳出山鬼怪皆隐，太阳是一种文明的象征，文明才会懂礼，太阳属火，所以火主礼。

太阳的运行规律是东升西落，寅为艮为山，太阳从寅出，酉时落山，生生不息，所以丙火长生在寅，死在酉。

概括丙火喜忌最经典的论著要数《滴天髓》了，书言：丙火猛烈，欺霜侮雪。能煅庚金，逢辛反怯。土众成慈，水猖显节。虎马犬乡，甲来焚灭。这是什么意思呢？我们慢慢分析。丙火猛烈，欺霜侮雪----太阳的光热是非常厉害的，它不惧怕任何风雪，相反即使在冰天雪地里，只要太阳出来就会冰消雪融，这句话是概括丙火天性的。能煅庚金，逢辛反怯----古人写文章善于用运各种比拟手法和指代用语，十天干有各自的天象，庚金的天象是月亮，也是寒气，太阳出来月亮不显，寒气逼退，所以能煅庚金；辛金的天象是霜，太阳出来霜就变成雾了，看似丙克了辛，但是雾反过来又模糊了太阳的光线，所以丙遇辛是喜中有忧了；丙辛合不也是这个道理吗？表面是相互和合，实际丙辛合的结果是生出个火的天敌水，水去制约丙火，丙怎能不怯辛呢！土众成慈----土可以用戊己来表示，戊己的天象都是霞是云，戊是霞、己表乌云，土众说明云多了，云多的时候，太阳就钻到云朵里不见身影了，（丙火最忌己癸侵）光线被遮的时候是太阳显得最低调、最无奈、最悲催的时候。

水猖显节----水是火的天敌，水多了火是要灭掉的的，但是丙火却不是一般的火，它是天火太阳，这里的水猖我认为有两种理解，一种理解是水猖的水是专指壬水的，壬水的天象是天河，太阳照天河，是日照江河，雄伟壮观，相互仰映，这时候显出了太阳的气度；另一种理解，就是丙火在面对直接克制自己的敌人（水）时，一步都不会退缩，很难屈服于水的压力和气势，所以当我们看到丙日元的人时，不要轻意地说，从官或从杀，这样的人就象文天祥，死都不会投降敌人的，显示了一种士可杀不可辱的气节。

6、庚辰运、工作有较大变动，有些理想，计划得到实现，工作成就显著，财运大好，心情大好，虽然有些辛苦，但身心愉悦，很有成就感。庚辰大运是命主一生事业，运气的最高峰。

7、辛巳运、与月柱天克地冲。月柱代表命主的事业；事业遭到冲击；禄代表俸禄，钱财，也代表身体。因此这步大运会对命主带来沉重打击，是非口舌不断，事业一蹶不振，同时会有疾病、小人、破财等坏事连连。

8、人生遭到沉重打击，心灰意冷，看破红尘。命书有曰：身旺无依僧道之命；又曰：水多财绝，僧道为生；还有丑为道观，寅为山，丑见寅也是僧道的标志。也就是说命主八字先天就有皈依的信息。辛巳运，己丑年寺院钟鼓一响，皈依道教。

9、壬午运从霉运的低谷中走出来，命主换凳子，直接坐到午火上，取香火之财，寅午半合，财运会很好。但要注意心脏、泌尿系统容易发病，注意身体，健康是第一位的。

10、命主已近古稀之年，但仍然壮志暮年，雄心不减。命主很关心后步运气。我可以负责人告诉你，虽然你成不了一代宗师，但成为有名望，众所拥戴的导师是必然的。奉劝法师，人生都要遵循自然规律，孔子曰：三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲。可以有理想，但不要追求成果；有过程就是享受。
