## 四、与现有「盲派」体系的关系

```
高派（本文件夹）            现有 Skill 模块（盲派）
─────────────────          ─────────────────────
《理法篇》     →    模块 B（体用 / 主宾 / 格局清浊）
《象法篇》     →    附录-取象 + B（意向识别）

《车祸婚姻篇》 →    D（合婚铁律）+ cases/marriage
《择日篇》     →    模块 E（剖腹产择日）
《财运事业断法》→    B + C 综合 + cases/career
《神煞应用宝典》→    A + C + cases/health
```

> 高派 = 高德臣盲派体系（体用论命 + 象法 + 神煞三大支柱）
