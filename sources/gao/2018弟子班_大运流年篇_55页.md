# （四）、大运有修改格局成败之权

大运有成格变格之权。八字格局既定，大运还会使其变化吗？回答是肯定的，比如现实之中有些人先为官而后经商，或先为军人而后为艺术家，若说命格不变的话，这又怎么推断得出来呢？试想啊，假若岁运没有成格变格之权，那么，凡是破格的命主岂不是要倒一辈子的霉啊？现实中有些人早岁艰辛，中年腾达；而另一些人少年得志，中年败落；还有一些人一辈子只发过一年两年，如昙花一现，流星一闪，此后便贫穷度日，这就是命格的变化所致。

由此可见，职业的变化常常是命格变化所产生的结果，反过来说，命格变化一般都会导致职业、财运、婚姻等许多方面发生变化。命格绝不是一成不变的，而是常常会因大运的介入而产生变化的。成格之命在大运破坏格局时，轻者会失学、失业、失恋、失职，重者还会招来横祸飞灾；而破格之命在得到岁运救应时，轻者会平平安安，衣食无忧，重者会飞黄腾达，出人头地。所以《五言独步》说“格中若去病，财禄喜相随”。

# 第 3 课：大运的特殊规律研究

# 一、为什么行运，有些吉凶不验？

在行大运过程中，有行吉运而不见其吉，行凶运而不见其凶的情况，其中原因固然很复杂，但其中一个常见的原因就是忽略了大运的天干和地支之间的互相影响和照应，如果喜行木运，遇甲申乙酉，甲乙自坐绝地受克无力之类，故《滴天髓》说：“一运看十年，切勿上下截看……如上下截看，不论盖头截脚，则吉凶不验矣”。

这里也回答了，大运是分看还是统看的问题。盲派主张分看，其实是前五年，以看天干为主，不是不看地支，后五年以看地支为主不是不看天干。

大运的干支在什么情况下为比较有力或比较理想，《滴天髓》的作者对此也进行了大致的划分：如喜行木运，最好是甲寅乙卯，其次是壬寅癸卯、甲辰、乙亥。喜行火运，最好是丙午丁未，其次是丙寅丁卯、丙戌、丁巳。----

所以逢吉不见其吉，逢凶不见其凶，在很多情况下常常是大运的干支碰上盖头截脚的缘故。

# 四、交运与脱运规律

走完一个大运之后，交接下一大运，是为换运。由于大运的旧新交接，考虑到命格自身的适应情况，因此，反映出来的变化是最明显的，其影响也是非常深刻。古语曰：“交运脱运两头奔命”。即是说两个大运的交际之间，好像人脱胎换骨一样，特别是两个五行属性不相同的大运交脱之际的前后一两年最不太好过，就像人脱了层皮，苦闷忧郁，遇事往往心神不定，反反复复，或性情大变，冲动冒险，使事业、感情等方面出现动荡不安、奔波辛苦之象，严重者耗财生灾。

这里有几个关键点要注意：

1、交接之年，前后两部大运兼看。

# 二、几种容易出现凶灾的流年特征

# (一)、天克地冲

1. 命局有天罗、地网、灾煞、劫煞，流年大运遇之应防刑法、牢狱之灾。尤其注意流年与四柱中年月日时分别有天冲地克。

通常冲年柱为父母长辈有灾，本人也有不顺。冲月柱为同胞有灾，也会影响到本人。冲日柱为自己或妻子有灾，冲时柱是儿女有灾，或本人不顺。

# (二)、岁运并临

凡遇这些年份，青少年多死家中长辈，中老年则自己有死亡之灾，也有不死而生大病破大财的。如遇家中有喜事，可冲灾。

# (三)、丑未戌三刑

命局中全者，一生不顺之事多，命局有两个大运凑齐者，该运中不顺之事多，再遇流年又刑，有人合伙整命主，有走动或变环境稍好一些。三刑又遇天克地冲，灾重，有本人病，家中死人，长辈有病，本人破财或其它不顺之事。刑入年柱，长辈有灾，刑入月柱或日柱，同胞或本人、妻子有灾，刑入时柱子女有灾。丑未戌三刑是较轻的一种刑。

# (四)、寅巳申三刑

原局全灾重，易有牢狱之灾，或因财之事坐牢，大运有一支凑齐，十年中不顺事多，流年构成三刑，易有病灾，也有破财或有受骗上当的，重者为死灾。与天克地冲同时出现，灾重。
