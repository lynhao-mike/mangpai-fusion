# 机械铁断规则册 · mechanical-rules.md · v1.0

> **定位**：本文件为 mangpai-fusion 系统中**必须保留的机械铁断**规律清单。
>
> **背景**：用户明确要求——尽量减少分析报告中的"单条机械铁断"，对一条规律就直接下结论的写法应当软化为"多派联合判断"或"共识表达"。但有少数规律理论强度高、命中率稳定，必须作为铁断保留以指导分析。
>
> **使用方式**：
> - 分析师写报告时，**默认禁止**用单条规律直接下结论
> - 仅当某条判断属于本清单中的铁断时，才允许以"机械铁断"形式呈现
> - 分析报告中引用铁断时，必须标注本册编号（如 `MR-001`）以便追溯

最后更新：2026-05-23 · v1.0
来源案例：C-2026-013-壬申甲辰丙辰己丑（首批 6 条铁断由本案验证抽取）

---

## 一、机械铁断的认定标准

一条规律可入选本册，必须同时满足：

| 标准 | 要求 |
|---|---|
| **理论强度** | static_score ≥ 80 |
| **派别支持** | 至少 1 派主导 + 1 派旁证（不强求 4 派共识） |
| **命中样本** | 已在 ≥ 1 个真实案例中应验，无失验记录 |
| **应用边界** | 适用条件明确，不引发歧义 |
| **可证伪** | 命主反馈即可直接验证（外貌、出生事件、家庭事实） |

未满足以上标准的规律，**一律软化为多派联合表达**，不得当作铁断使用。

## 二、必须保留的铁断清单（v1.0 · 6 条）

### MR-001 · 月柱印星伏吟 → 母亲婚姻反复

| 字段 | 内容 |
|---|---|
| **派别** | 杨派主导 + 段派旁证 |
| **命题** | 月柱为母亲宫位 + 月柱印星与日柱同支伏吟 → 母亲感情/婚姻反复（多次婚配概率高） |
| **理论依据** | 月柱主母 · 伏吟主反复震荡 · 印星受冲扰 |
| **适用条件** | 月柱含印星（正印或偏印）且与年/日柱同支构成伏吟 |
| **不适用** | 月柱无印星；伏吟出现在其他柱位；月柱印星已被合化转化 |
| **应用案例** | C-2026-013-壬申甲辰丙辰己丑：月柱甲辰偏印 + 日柱丙辰 = 双辰伏吟 → 母三婚 ✅ |
| **置信度** | ★★★★★ (92%) |

### MR-002 · 命局土极旺（≥55%）→ 肤色偏暗

| 字段 | 内容 |
|---|---|
| **派别** | 高派主导 |
| **命题** | 八字五行结构中土所占比重达 55% 以上 → 肤色偏黄、偏暗或偏黑 |
| **理论依据** | 五行外应理论：土主肌肤·土厚则肌肤色重 |
| **适用条件** | 土的本气+中气合计 ≥ 55%；火金不极旺以平衡 |
| **不适用** | 命局火金极旺时（火主红润、金主白皙），可中和土色 |
| **应用案例** | C-2026-013-壬申甲辰丙辰己丑：己土+双辰土+丑土 ≈ 55%+ → 实际肤色偏黑 ✅ |
| **置信度** | ★★★★★ (93%) |

### MR-003 · 丙日主 + 食伤透干 → 大眼·嘴大·开朗外向

| 字段 | 内容 |
|---|---|
| **派别** | 高派主导 + 杨派旁证 |
| **命题** | 丙火日主主明目 → 大眼有神；食伤旺且透干 → 嘴阔善言；丙火主阳光性格开朗 |
| **理论依据** | 丙为太阳之火主明目 · 食伤主表达善辩 · 木火通明主阳气直 |
| **适用条件** | 日主丙火 + 食神或伤官透干（非藏支） |
| **不适用** | 日主非丙火；食伤全藏支不透干；命局火气被严重克泄 |
| **应用案例** | C-2026-013-壬申甲辰丙辰己丑：丙辰日主 + 己土伤官透时干 → 大眼·嘴大·开朗直爽 ✅ |
| **置信度** | ★★★★★ (90%) |

### MR-004 · 空亡支被流年冲动 → 应期前移可发生

| 字段 | 内容 |
|---|---|
| **派别** | 任派主导（M3 18 道法门） |
| **命题** | 命局某支位空亡（如时支空亡=子女宫空），并不必然意味"晚发生"或"不发生"——只要流年冲动该空亡支，对应宫位事件即可应期 |
| **理论依据** | 任派 18 道法门空亡机制：空亡需"填实/冲动"才被激活 |
| **适用条件** | 任何空亡宫位 + 流年地支与空亡支构成冲（六冲）或填实 |
| **不适用** | 空亡支同时被原局合住或绝地无气 |
| **应用案例** | C-2026-013-壬申甲辰丙辰己丑：时支丑空亡（子女宫）→ 2023 癸卯年卯冲辰、癸官透干→ 31 岁育女 ✅ |
| **置信度** | ★★★★★ (91%) |
| **配套校准规则** | **不允许**直接断"空亡=晚/无"，必须配合流年冲实分析才能给出应期 |

### MR-005 · 财动婚动 → 坤造婚期触发条件之一

| 字段 | 内容 |
|---|---|
| **派别** | 任派主导（M3 18 道法门） |
| **命题** | 坤造命主，流年财星（正/偏财）透干并旺时，构成"财动婚动"的婚期触发信号之一 |
| **理论依据** | 坤造以官星为夫、以财星生官，财动则官生有源 → 婚动 |
| **适用条件** | 坤造 + 流年天干为命主财星 + 流年地支不严重克身 |
| **不适用** | 流年财星被严重克破；身极弱无法承财；命主已婚则无效（婚动可能转化为夫财动） |
| **应用案例** | C-2026-013-壬申甲辰丙辰己丑：2021 辛丑年（辛=偏财）+ 庚子大运财官旺 → 29 岁首婚 ✅ |
| **置信度** | ★★★★★ (90%) |
| **配套校准规则** | **判坤造婚期必须三项联检**：① 官杀格局动；② 财动婚动；③ 空亡冲实。三者中至少 2 项满足才认定婚期年。 |

| 字段 | 内容 |
|---|---|
| **派别** | 杨派主导 |
| **命题** | 年柱含驿马（申子辰生人见寅；寅午戌见申；亥卯未见巳；巳酉丑见亥）→ 影响早期/首任配偶的"流动"或"远方"属性 |
| **理论依据** | 年柱主祖业及早年环境 + 驿马主流动远行 + 申金亦主军警金属类 |

## 三、已废弃/失验的铁断（黑名单）

以下规律曾被作为机械铁断使用，但因失验或过度机械化已废弃。**禁止**在分析中以铁断形式引用。

### XF-001 · "正官藏不透 → 晚婚铁律"❌ 已废弃

| 字段 | 内容 |
|---|---|
| **原命题** | 坤造正官三藏不透 + 食伤极旺克官 → 必为晚婚命，黄金婚期 35-41 岁 |
| **派别来源** | 杨派初判 + 共识层误判 |
| **失验案例** | C-2026-013-壬申甲辰丙辰己丑：命主 29 岁首婚（提前 6-12 年），原"晚婚铁律"完全失验 |
| **失验原因** | ① 官藏不透 ≠ 晚婚，仅代表缘分迂回 / 路径曲折；② 忽略了"官杀混杂"实际意味的"婚路曲折/易再婚"而非"晚婚"；③ 未联合财动婚动检验 |
| **替代规则** | 改用 MR-005 财动婚动 + MR-004 空亡冲实 + 多派联合判断"婚路曲折再婚命"，**严禁**单凭"官藏"直接断晚婚 |
| **降级处理** | 从铁断库移出 → 移入软化判断：仅作为"婚路迂回"的辅助信号之一 |

## 四、软化判断指南（替代铁断的写法）

当某项判断**不属于**第二节的 6 条铁断时，必须采用以下软化形式之一：

### 4.1 多派联合表达

```
[杨主导 + 段辅 + 任补充] ……(共 3 派联合) → 结论 ★★★★ (XX%)
```

### 4.2 共识层表达

```
[共识 · 4 派一致] ……(同向叠加加权) → 结论 ★★★★★ (XX%)
```

### 4.3 互补层联立加权

```
[互补 · 杨派事件维度 + 任派应期维度] → 结论 ★★★★ (XX%)
计算：joint_score = max(各派 score) + Σ (其余派 score × 0.15)
```

### 4.4 独门 + 旁证

```
[高派独门] …… ★★★★ (XX%)
[杨派持旁证支持]
```

### 4.5 禁止格式

❌ **禁止单条规律直接下铁断**（除非属于第二节 MR-XXX 已认证铁断）
❌ **禁止机械陈述**（如"必离""必晚""必无"等绝对化表达，铁断之外一律改为概率化语言）
❌ **禁止跳过依据**（每个判断必须挂派别 + 证据链）

## 五、铁断册的演化

### 增补流程
1. 案例分析中发现新规律高度命中（≥ 1 案例验证 + 派别理论支持）
2. 提交 candidate → 在 `cases/lessons.md` 标记 `candidate-MR`
3. 累计 ≥ 3 个独立案例应验且无失验 → 升级为正式 `MR-NNN`
4. 写入本册 + 更新 `META/rule-changelog.md`

### 降级流程
1. 已认证铁断在新案例中失验 ≥ 2 例
2. 进入 `黑名单`（第三节）
3. 替代规则在第四节登记软化形式
4. 同步更新 `META/rule-changelog.md`
