# Yang YAML topic/conclusion corrections and Gao source path fix

This change corrects metadata integrity across the theory knowledge base. Seventy-five yang rules (M2-Y-019..048 and M2-Y-144..163) had their `topic_label` and `raw_anchor` fields incorrectly set to "2019年班补全" — a label that properly belongs only to M2-Y-119..143. Those rules now carry the actual section headings from their authoritative extraction files (y01-y05-theory.md and y26-y29-theory.md). Separately, 72 placeholder conclusions that contained only section references like "§2.3" have been filled in with actual conclusion text from the registry. Nine gao extraction files also had their `来源文档` paths corrected from a legacy prefix to the current repo structure.

**Watch for:** One topic_label ("高维探索与活死五行") is an abbreviated form that doesn't exactly match its source heading "高维探索：活死五行与品质定位" [likely]. The conclusion texts are condensed summaries — verify a sample against the full rule definitions to ensure no meaning was lost [possible].

## High-level view

The yang YAML corrections split into two concerns: structural metadata (topic_label/raw_anchor) and content (conclusion text). The structural fix re-associates 50 rules with their true section headings from the raw extraction files, restoring the traceability chain that lets downstream tools and readers know which conceptual group a rule belongs to. The conclusion fix replaces 72 bare section references with actual rule summaries, making the YAML self-contained for consumers that don't cross-reference the registry.

The gao extraction path fix is a straightforward find-and-replace: the old prefix `高派命理理论知识库/` corresponds to the actual repo path `sources/gao/`. All 10 referenced target files have been confirmed to exist at the new paths.

The 50 topic_label corrections draw from two source files: `theory/raw/yang/y01-y05-theory.md` for M2-Y-019..048 (7 section groups) and `theory/raw/yang/y26-y29-theory.md` for M2-Y-144..163 (5 section groups). Cross-checking against source headings, 11 of 12 labels are verbatim copies of their section headings. The one exception:

**"高维探索与活死五行" (M2-Y-149..153) — abbreviated from "高维探索：活死五行与品质定位" (y26-y29 §2.2)** [confirmed]
