# reports/ · 正式报告归档

每个案例分析完成后，生成 `C-YYYY-NNN-report.md` 报告文件。

报告格式见 [`../templates/report.md`](../templates/report.md)（M6 阶段产出）。

**强制约束**：
- 每个 cases/C-YYYY-NNN/ 必须对应 reports/C-YYYY-NNN-report.md
- 二者必须在**同一次 commit** 中提交（不允许只提交案例不提交报告）
- 报告标题必须标注性别命式（乾/坤），格式如：`# 八字分析报告 · C-YYYY-NNN · 乾`
- 报告中所有断语必须双轨制置信度（★+%）+ 派别归属
