- 序 / P1: 子平之术，由来尚矣。自唐李虚中以年月日立命，至宋徐子平兼用时辰，而后八字之法定。其理本于阴阳五行，推究气运之盛衰，以定人之祸福寿夭。然世传诸书，或繁而无要，或简而疏漏，讹谬杂出，初学茫然。
- 序 / P2: 余涉猎子平有年，参究古法，删繁举要，辑为此编，名曰《子平真诠》。诠者，诠解命理之真义也。务使格局用神之理，条分缕析，一目了然，为后学入道之阶。非敢自诩精微，聊以正俗书之讹而已。
- 一、论十干十二支 / P1: 天地之间，一气而已。惟有动静，遂分阴阳；有老少，遂分四象。老者极动极静，为太阳太阴；少者初动初静，为少阳少阴。四象既定，由是生五行：水太阴也，火太阳也，木少阳也，金少阴也，土寄于四季，统摄四气。
- 一、论十干十二支 / P2: 五行布于天干地支，天干十：甲丙戊庚壬为阳干，乙丁己辛癸为阴干；地支十二：子寅辰午申戌阳支，丑卯巳未酉亥阴支。
- 一、论十干十二支 / P3: 甲乙属木，甲阳乙阴；丙丁属火，丙阳丁阴；戊己属土，戊阳己阴；庚辛属金，庚阳辛阴；壬癸属水，壬阳癸阴。
- 一、论十干十二支 / P4: 寅卯木，巳午火，申酉金，亥子水，辰戌丑未土。天干行天之气，地支凝地之形。甲禄在寅，乙禄在卯，丙禄在巳，丁禄在午，戊禄在巳，己禄在午，庚禄在申，辛禄在酉，壬禄在亥，癸禄在子。
- 一、论十干十二支 / P5: 以气论，阳干气盛；以质论，阴干质坚。俗以甲为大树宜斫、乙为小苗难伤，乃附会谬说，不足取。学命必先明干支阴阳本源。
- 二、论阴阳生克 / P1: 四时气运，相生轮转：木生火，火生土，土生金，金生水，水复生木，循环无端。
- 二、论阴阳生克 / P2: 相克之理：木克土，土克水，水克火，火克金，金克木。相生为和顺，相克为制伏。
- 二、论阴阳生克 / P3: 生之中有克，克之中有生。如木生火，木多火窒；火克金，金旺火熄。生克贵在中和，太过不及，皆为病。命理取用，全凭生克权衡旺衰。
- 三、论阴阳生死 / P1: 五行寄生十二宫：长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养，循环十二地支。
- 三、论阴阳生死 / P2: 长生者，气始生；帝旺者，气极盛；墓者气收藏；绝者气尽灭。阳干顺行，阴干逆行，乃阴阳消长自然之理。
- 三、论阴阳生死 / P3: 如甲木长生在亥，顺行十二宫；乙木长生在午，逆行十二宫。古法以生死定旺衰，不可执泥死绝便作衰弱，须看全局干支帮扶制化。
- 四、论十干配合性情 / P1: 十干五合：甲己合土，乙庚合金，丙辛合水，丁壬合木，戊癸合火。合有正合、化合之分。
- 四、论十干配合性情 / P2: 得月令之气则化，不得月令只作牵绊。天干相合，改变原干性情：甲用辛官，逢丙合辛，则官星被合无用；甲用己财，己被他甲合去，则财星落空。
- 四、论十干配合性情 / P3: 吉神被合则失用，凶神被合则化凶。男命财为妻，财被合夺；女命官为夫，官被合离，皆六亲不合之象。
- 五、论十干合而不合 / P1: 干合而不能化，谓之合而不合。一者化合无根，不得月令化神之气；二者中间隔干，被他干冲克；三者地支冲破合局。
- 五、论十干合而不合 / P2: 如甲己相逢，月令非土，地支寅木冲破，虽合不化，只作羁绊。凡合而不化，原干五行依旧论生克，不可妄作化格。
- 六、论十干得时不旺失时不弱 / P1: 旺衰以月令为提纲，然得地得助，虽失令亦不弱；失地无助，虽得令亦不强。
- 六、论十干得时不旺失时不弱 / P2: 甲木生申月失令，地支寅卯成林，则身旺；甲木生寅月得令，地支申酉重重克伐，则身弱。
- 六、论十干得时不旺失时不弱 / P3: 故断旺衰，月令为本，干支帮扶为辅，不可单执月令定强弱。
- 七、论刑冲会合解法 / P1: 地支有六合、六冲、三刑、三会、三合。
- 七、论刑冲会合解法 / P2: 六合：子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合；六冲：子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。
- 七、论刑冲会合解法 / P3: 三会聚一方之气，三合成局。冲则散局，合则解冲。如子午相冲，逢丑合子、逢未合午，即解冲。刑者，无恩之刑、恃势之刑、无礼之刑，刑破格局则损用神。
- 七、论刑冲会合解法 / P4: 解法全在合神、制神，以合解冲，以克解刑。
- 九、论用神成败救应 / P2: 如甲木日主，月令酉官为用，透出丁火伤官克官，逢壬水合丁护辛，则救应成格；无壬水救护，伤官破官，格局破败。
- 十、论用神变化 / P1: 用神随干支组合变换格局，正用可变偏用。月令本为官星，财多生官变财格；月令本为财星，官多护财变官格。
- 十、论用神变化 / P2: 因地支会合、天干透干，用神随时改换，不可死守月令原气。
- 十二、论用神格局高低 / P2: 有情者，顺生相辅；无情者，逆克相伐。同是官格，财生官、印护官为上格，伤官破官、七杀混官为下格。
- 十三、论用神因成得败因败得成 / P1: 格局本成，运岁克破用神则由成转败；格局本破，运岁补救用神则由败转成。
- 十三、论用神因成得败因败得成 / P2: 原局官格被伤，大运逢印制伤护官，败局复成；原局用神完备，大运忌神冲破，成局变败。
- 十四、论用神配气候得失 / P1: 用神兼顾四时气候，冬寒喜火调候，夏燥喜水润燥，调候为急，大于扶抑。
- 十四、论用神配气候得失 / P2: 水冷金寒，虽日主身旺，首取火暖局，次论克泄；炎燥木枯，先取水润土，再定用神。
- 十六、论杂气如何取用 / P1: 辰戌丑未为杂气，藏天地余气，分本气、余气、墓气。取杂气用神，以透出天干、地支会局为准，不透不聚，难作正用。
- 十六、论杂气如何取用 / P2: 辰藏戊乙癸，透戊取土格，透乙取木格，透癸取水格。
- 十七、论墓库刑冲之说 / P1: 墓库藏余气，静则收藏，冲则库开，余气透出。辰戌相冲开水土库，丑未相冲开木金库。
- 十七、论墓库刑冲之说 / P2: 宜冲则冲开取用，忌冲则冲破用神。身弱用神在库，逢冲出库帮身则吉；身旺用神藏库，逢冲破库损用则凶。
- 十八、论四吉神能破格 / P2: 官多身弱，官星克身破身格；财多身弱，财星耗身破格；印多身旺，印旺滞食破格；食神过旺泄身，食神破格。吉神泛滥，反为祸患。
- 二十、论生克先后分吉凶 / P1: 天干透干先后、地支生克次序，定吉凶轻重。先生后克，先生扶用神而后破用，凶轻；先克后生，先破用神而后补救，凶重。
- 二十一、论星辰无关格局 / P1: 纳音、神煞、二十八星宿，不关乎格局用神。命理以五行生克、格局用神为正宗，拘泥神煞星辰，误入旁门。
- 二十二、论外格用舍 / P1: 月令无可用之神，别位干支成局取用，名为外格。专旺、从强、从财、从杀，皆属外格。
- 二十二、论外格用舍 / P2: 日主真从则顺其旺势，不能真从仍作正格，从格最忌破格。
- 二十三、论宫分用神配六亲 / P1: 年柱祖辈，月柱兄弟父母，日柱夫妻自身，时柱子女。用神在年利祖辈，在月旺手足，在日利配偶，在时旺晚嗣。
- 二十四、论妻子 / P1: 男命以财为妻，财星有根无冲破，妻贤寿安；财星被劫合，刑妻克偶。女命以官为夫，官星纯粹无伤，夫君显贵；官星混杂受克，夫缘浅薄。时柱为子女宫，食伤旺而无破，儿女昌盛。
- 二十五、论行运 / P1: 运以月令起，阳年男顺排、阴年男逆排；阴年女顺排、阳年女逆排。大运管十年吉凶，流年司一岁祸福。
- 二十六、论行运成格变格 / P1: 原局格局已定，大运干支改变生克，可成格、可变格。原局破败，运补用神则成；原局完美，运破用神则败。
- 二十七、论喜忌支干有别 / P1: 天干主表面，地支主内里。天干喜神外露，外有贵人；地支喜神藏根，内享福泽。天干忌神在外，多招是非；地支忌神藏根，暗受灾祸。
- 二十八、论支中喜忌逢运透清 / P1: 地支藏干喜忌，大运引出透干，谓之透清。藏中用神，运岁透出，格局显用；藏中忌神，运岁透出，祸患显现。
- 二十九、论时说拘泥格局 / P1: 俗书死守定式格局，不问干支变化，胶柱鼓瑟。格局无定法，随生克会合改换，贵在变通。
- 三十、论时说以讹传讹 / P1: 坊间命书多承袭谬论，如甲木必喜庚、乙木必喜辛，五行定格死板，不知旺衰制化，不可盲从俗说。
- 三十一、论正官 / P1: 正官乃克我阴阳相配之神，扶身律己，最喜财生、印护。官星纯粹，无杀混杂、无伤官破损，正官成格。
- 三十二、论正官取运 / P1: 官格行运，喜财印生扶，忌伤官七杀旺运破官。身弱官旺，喜印绶化官生身；身旺官轻，喜财运生官。
- 三十四、论财取运 / P1: 财格运喜食伤生财、官星护财，忌比劫夺财、枭神克食。身弱财旺，宜印运扶身担财。
- 三十五、论印绶 / P1: 生我者为印，正印阴生阳、偏印阳生阴。印护身，喜官星生印，忌财星破印、食伤泄印。印旺身强，喜食伤吐秀。

## 6. 对未来案例校准可能造成的影响

1. 子平真诠覆盖率偏低会导致子平规则库偏向已抽取的用神、财官、调候和少数格局条目，未覆盖章节无法在反馈中获得校准机会。
