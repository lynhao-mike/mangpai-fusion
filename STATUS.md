## v1.0 能做什么

✅ **核心能力**
1. 接收问真八字 APP 排盘信息（按 templates/input-from-wenzhen.md 模板）
2. 4 派融合分析：共识层 → 互补层 → 独门层 → 冲突层
3. 双轨制置信度报告（★ + 百分比）
4. 派别归属显式标注，绝不隐瞒
5. 派别冲突显式呈现 + 仲裁裁决
6. 应期推断（4 模型交叉，年-月-日 精度）
7. 反馈回流校准（tools/calibrate.py 自动重算 hit_rate）
8. 案例归档（cases/C-YYYY-NNN + reports/C-YYYY-NNN-report.md）

✅ **理论库覆盖**
- 高派 261 条 / 段派 290 条 / 杨派 163 条 / 任派 200 条 = **914 条规律全部索引**
- 共识层 16 组 / 互补层 22 组 / 独门层 200 条 / 冲突 4 个登记

✅ **置信度算法**
- 静态分（理论强度，40% 权重）
- 动态分（实战命中率，60% 权重）
- 跨派联立加权 max + 0.15 × Σ
- 冲突场景 winner/loser 公式
- 5 大防作弊保护

✅ **领域权重矩阵**
- 段派 lead：格局/富贵/财运（3 领域）
- 杨派 lead：婚姻/调候/学业/六亲（4 领域）
- 高派 lead：神煞/健康（2 领域）
- 任派 lead：应期（1 领域）

## 立即可用的工作流

```
1. 用问真八字 APP 排盘 → 复制 → 按 templates/input-from-wenzhen.md 整理
2. 启动 Kiro 会话，激活 .kiro/skills/analyst.md
3. 贴入八字 + 分析重点 → 系统按策略 A/B/C 路由
4. 输出双轨置信度报告（共识/互补/独门/冲突 4 段）
5. 与命主对话，记录反馈
6. 把反馈填入 cases/C-YYYY-NNN/feedback.md
7. 运行 python3 tools/calibrate.py --case C-YYYY-NNN
8. 自动校准结果写回 theory/{school}/index.yaml
9. 升级/降级日志写入 META/rule-changelog.md
10. 新规律晋级到共识层 → 自动更新 mapping/consensus.md
```
